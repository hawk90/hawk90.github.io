\node[anchor=west, font=\tiny, align=left] at (0, -3) {
  1. IOVCC 1.8V Assert (digital IO)\\
  2. VCI 2.8V Assert (analog) — wait 1-10 ms\\
  3. Reset Low → High pulse — typically 5 ms reset, 10 ms wait\\
  4. DSI Host LP mode enter\\
  5. Send \texttt{Sleep Out} (0x11) → wait 120 ms (panel ID·gamma 로드)\\
  6. Send \texttt{Pixel Format Set} (0x3A) → RGB888 등\\
  7. Send vendor-specific gamma/power init (10-50 명령)\\
  8. Send \texttt{Display On} (0x29) → wait 20 ms → video stream 시작\\[2pt]
  ⚠ 각 명령 후 *데이터시트 wait time* 정확히. 빼먹으면 *첫 프레임 깨짐*.
