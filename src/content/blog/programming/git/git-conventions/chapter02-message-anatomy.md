## 작성 중

### 예정 내용
- 50/72 규칙 — subject 50자, body 72자 wrap
- 명령형 (imperative) — "Add" not "Added"
- subject — 한 줄 요약
- 빈 줄 → body
- body — why (무엇 / 왜), what is changed
- footer — refs, Co-Authored-By, BREAKING CHANGE
- 안티 패턴 — "fix bug", "update", "WIP"
