## 예정 내용
- Entity — 식별자로 구분 (lifecycle 추적)
- Value Object — 속성으로 정의 (불변 / 비교 가능)
- Service — 객체에 속하지 않는 도메인 동작
- Module — 응집 / 결합 단위
- 객체 vs 함수형 — 둘 다 가능
- 흔한 함정 — 모든 것이 Entity가 됨
