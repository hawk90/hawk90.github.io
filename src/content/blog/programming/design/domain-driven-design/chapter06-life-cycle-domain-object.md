## 예정 내용
- Aggregate — 일관성 경계 / 트랜잭션 단위
- Aggregate Root — 외부 접근 진입점
- 불변식 (invariant) 보장
- Factory — 복잡한 객체 생성
- Repository — 영속성 추상 (DB / 외부 인터페이스)
- 라이프사이클 — 생성 / 사용 / 보관 / 삭제
