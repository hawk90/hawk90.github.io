## 예정 내용

- Supporting Subdomain — 도와주는 부분
- Domain Vision Statement — 핵심 비전
- Highlighted Core — 코드 / 문서에서 강조
- Cohesive Mechanisms — 메커니즘 분리
- Abstract Core — 추상으로 도약
- Segregated Core — 분리 / 정제
