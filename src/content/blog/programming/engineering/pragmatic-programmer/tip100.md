## 이 팁의 메시지

> **Tip 100: It's Your Life.** You own it. You run it. You create it.

당신의 인생이다. 소유하고, 운영하고, 만들어 간다.

## 여정의 끝

100개의 팁을 함께 살펴봤다. 첫 번째 팁도, 마지막 팁도 같은 메시지다.

```text
Tip 1: Care About Your Craft
Tip 100: It's Your Life

당신의 기술, 당신의 인생, 당신의 선택.
```

## 실용주의 프로그래머의 본질

```text
실용주의 프로그래머는:
- 도구에 집착하지 않고 문제를 푼다
- 독단에 빠지지 않고 상황을 본다
- 완벽을 추구하되 적정선을 안다
- 혼자 똑똑하기보다 함께 나아간다
- 기술만이 아니라 사람을 이해한다
- 코드를 넘어 가치를 전달한다
```

## 핵심 원칙 정리

| 영역 | 핵심 원칙 |
|------|----------|
| 철학 | 장인 정신, 책임, 선택 |
| 설계 | DRY, 직교성, 가역성 |
| 도구 | 텍스트, 셸, 버전 관리 |
| 코드 | 계약, 단언, 예외 |
| 테스트 | 일찍, 자주, 자동으로 |
| 프로젝트 | 추정, 요구사항, 피드백 |
| 팀 | 소통, 협력, 자율 |
| 경력 | 학습, 성장, 주도 |

## 변하지 않는 것

```text
기술은 바뀌지만 원칙은 남는다:
- 20년 전의 DRY가 지금도 DRY다
- 20년 전의 테스트 원칙이 지금도 적용된다
- 20년 전의 소통 원칙이 여전히 중요하다

도구보다 원칙을 배운다.
```

## 계속 나아가기

```text
이 책을 읽은 후:
- 하나씩 적용한다
- 동료와 논의한다
- 실패하고 배운다
- 더 좋은 방법을 찾는다
- 다른 사람을 돕는다
```

## 커뮤니티

```text
혼자가 아니다:
- 같은 고민을 하는 개발자들
- 비슷한 문제를 푸는 팀들
- 지식을 나누는 커뮤니티
- 함께 성장하는 문화
```

## 마지막 생각

```text
소프트웨어 개발은 기술이면서 기예다.
코드를 작성하는 것을 넘어
문제를 이해하고
가치를 전달하고
사람과 협력하고
더 나은 세상을 만드는 일이다.

그리고 이 모든 것은
당신의 선택이다.
```

## 이 시리즈에서 다룬 것

**1-10:** 철학과 접근법
**11-20:** 설계 원칙
**21-30:** 도구와 환경
**31-40:** 코드 품질
**41-50:** 실용적 편집증
**51-60:** 유연성과 동시성
**61-70:** 코딩과 테스트
**71-80:** 요구사항과 문제 해결
**81-90:** 팀과 프로젝트
**91-100:** 테스트, 자동화, 윤리, 삶

## 정리

- 실용주의 프로그래머는 상황에 맞게 최선을 선택한다.
- 도구보다 원칙이, 유행보다 효과가 중요하다.
- 코드를 넘어 사용자 가치를 전달한다.
- 기술뿐 아니라 사람과 소통한다.
- 끊임없이 배우고 성장한다.
- 이 모든 것은 당신의 선택이다.

**당신의 인생이다. 만들어 가라.**
