## 이 팁의 메시지

> **Estimate to Avoid Surprises** — 추정은 — **사고의 도구**. 정확한 예측 X.

## 핵심 내용

- 추정 = 작업을 — 작게 분해.
- 분해 과정에서 — 위험·미지가 보인다.
- "놀라움"을 — 미리 발견.
- 결과 = 추정치 + **확신도**.

## 추정의 목적

- 정확한 시간 예측 X — 어차피 빗나간다.
- **위험 발견** — 추정 중 떠오른 미지.
- **계획** — 우선순위.
- **소통** — 이해관계자에게.

## 추정의 단위

- **시간** — 작은 작업.
- **이상 일수** — "이상적 환경에서 N일".
- **점수**(story points) — 상대적 크기.

## 추정 정확도

> 결과 = "약 3주" vs "3주 1일 4시간".

후자는 — 정확도 흉내. **신뢰감을 떨어뜨린다**.

```
적절한 단위:
- 며칠 → 시간.
- 몇 주 → 일.
- 몇 달 → 주.
```

## 추정 후

- 실제 시간을 — 기록.
- 추정 vs 실제 — 비교.
- 패턴을 — 학습.
- 다음 추정에 — 반영.

## 정리

- 추정 = 사고 도구.
- 위험·미지 발견이 핵심.
- 정확도가 아니라 — 적절한 단위.
- 기록·비교·학습.
