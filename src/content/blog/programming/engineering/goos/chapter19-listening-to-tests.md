## 예정 내용
- "I need to mock too many things" → 책임 너무 많음
- "Test setup is huge" → 의존 너무 많음
- "Logic in test" → 객체 잘못된 곳에 책임
- "Bursting at the seams" → 클래스 분리 필요
- 각 신호 → 리팩토링 액션
