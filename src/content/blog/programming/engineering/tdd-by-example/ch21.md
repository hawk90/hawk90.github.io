> **Test-Driven Development by Example** Chapter 21 요약

## 이 챕터의 메시지

지금까지 — print로 결과 보고. 더 정교한 추적 필요.

```python
class TestResult:
    def __init__(self):
        self.runCount = 0
        self.errorCount = 0
    
    def testStarted(self):
        self.runCount += 1
    
    def testFailed(self):
        self.errorCount += 1
    
    def summary(self):
        return f"{self.runCount} run, {self.errorCount} failed"
```

TestCase.run이 — TestResult 받음.

```python
def run(self, result):
    result.testStarted()
    self.setUp()
    try:
        method = getattr(self, self.name)
        method()
    except:
        result.testFailed()
    self.tearDown()
```

## 의미

매 테스트가 — 결과를 result에 등록. 끝에 — summary 출력.

```
test = WasRun("testMethod")
result = TestResult()
test.run(result)
print(result.summary())   # "1 run, 0 failed"
```
