## 이 챕터의 메시지

Dollar.times와 Franc.times — 거의 동일. Money로 끌어 올린다.

```java
class Money {
    Money times(int multiplier) {
        return new Money(amount * multiplier, currency);
    }
}
```

Money가 — 구체 객체가 됨. Dollar / Franc은 점차 의미를 잃는다.

## 테스트가 보호

리팩터링 중 — 매번 테스트 실행. 통과 → 안전. 실패 → 즉시 되돌림.
