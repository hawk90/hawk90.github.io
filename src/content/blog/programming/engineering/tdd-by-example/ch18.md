> **Bootstrapping의 문제** — 테스트 프레임워크 없이 테스트 프레임워크를 어떻게 테스트?

해답 — 매우 단순한 첫 버전부터.

## 가장 단순한 시작

```python
class TestCase:
    def __init__(self, name):
        self.name = name
    
    def run(self):
        method = getattr(self, self.name)
        method()
```

이게 — 최소한의 TestCase.

```python
class WasRun(TestCase):
    def __init__(self, name):
        TestCase.__init__(self, name)
        self.wasRun = None
    
    def testMethod(self):
        self.wasRun = 1

test = WasRun("testMethod")
print(test.wasRun)   # None
test.run()
print(test.wasRun)   # 1 (실행됨)
```

`run()`이 — `testMethod`를 호출했음을 확인.

## Print로 첫 검증

진짜 assertion 없음. `print`로 확인. 기능이 자라면 — 점차 정교하게.
