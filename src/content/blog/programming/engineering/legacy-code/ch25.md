## 주요 기법 — 발췌

### Adapt Parameter

테스트할 수 없는 타입을 — 인자로 받는 경우. 인터페이스로 추상화.

```java
// 이전:
void process(HttpRequest req) {   // 만들기 어려움
    // ...
}

// 이후:
interface Request { String getParam(String k); }
void process(Request req) { /* ... */ }
```

### Break Out Method Object

긴 메서드 — 별도 클래스로.

```java
// 이전: 200줄 메서드

// 이후:
class OrderProcessor {
    OrderProcessor(Order o) { /* ... */ }
    void execute() { /* 200줄 → 작은 메서드들 */ }
}

// 호출:
new OrderProcessor(order).execute();
```

### Definition Completion

C++의 분리 정의 활용. 헤더에서 — 다른 .cpp의 정의로 치환.

### Encapsulate Global References

```java
// 이전:
class Service {
    void run() {
        Globals.config.someValue;   // 전역
    }
}

// 이후:
class Service {
    ConfigProvider config;   // 주입
    void run() { config.someValue; }
}
```

### Expose Static Method

private static을 — public static으로 (테스트용).

### Extract and Override Call

문제의 메서드 호출을 — 별도 메서드로. 그 메서드를 — 오버라이드.

```java
// 이전:
void process() {
    int x = external_call();   // 테스트 환경에서 호출 어려움
    // ... x 사용 ...
}

// 이후:
void process() {
    int x = getExternalValue();
    // ... x 사용 ...
}
protected int getExternalValue() {
    return external_call();
}
// 테스트:
class TestService extends Service {
    @Override
    protected int getExternalValue() { return 42; }
}
```

### Extract and Override Factory Method

new 호출을 — 메서드로. 서브클래스에서 다른 객체 생성.

### Extract Interface

기존 클래스에서 — 인터페이스를 추출. 의존을 인터페이스로 전환.

### Introduce Instance Delegator

`static`을 — `instance` 메서드로 위임.

```java
// 이전: Util.compute() 정적 호출
// 이후:
class UtilDelegator {
    public int compute(int x) { return Util.compute(x); }
}
```

테스트에서 — `UtilDelegator`를 mock.

### Parameterize Constructor

생성자에 — 의존성 받게.

```java
// 이전: this.db = new Database();
// 이후: this.db = injectedDb;
```

### Pull Up Feature

수퍼클래스로 — 멤버를 이동.

### Push Down Dependency

서브클래스로 — 의존성을 이동.

### Replace Function with Function Pointer (C)

함수 호출을 — 포인터로. 런타임 교체.

### Subclass and Override Method

테스트용 서브클래스 — 메서드 오버라이드로 동작 변경.

### Supersede Instance Variable

테스트에서 — 인스턴스 변수를 직접 교체.
