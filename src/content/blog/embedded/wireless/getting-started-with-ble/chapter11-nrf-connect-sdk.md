> Outline — *Nordic 전환* — SoftDevice(독자 RTOS) → Zephyr RTOS 기반 nRF Connect SDK (NCS). *Zephyr Bluetooth stack* — Mbed TLS, controller, host. *Kconfig + devicetree* — BLE 설정. *Sample apps* — `peripheral_uart`, `peripheral_hr`, `central_bas`. *Provisioning* — DK 보드, J-Link 디버거. *Power profiling* — Power Profiler Kit II. *대안* — ESP32 (Bluedroid/NimBLE), STM32WB (Cortex-M0+ controller). 책 예제 (Bluez·Arduino)와 모던 SDK 비교.
