## 작성 중

Carnegie Mellon SEI의 *SEI CERT C Coding Standard* — 보안 결함 예방.

### 10장 구성
1. CERT란 / MISRA와 차이
2. Pre-processor (PRE)
3. Declarations / Initialization (DCL)
4. Expressions (EXP)
5. Integers (INT)
6. Floating point (FLP)
7. Arrays (ARR) / Strings (STR)
8. Memory Management (MEM)
9. Input/Output (FIO) / Environment (ENV) / Signals (SIG) / Errors (ERR)
10. POSIX (POS) / 동시성 (CON) / 시리즈 마무리

### 위상
- MISRA C와 결합 (보안 + 안전)
- 임베디드보다 일반 / 서버에도 적용
