## 작성 중

*Guidelines for the use of the C++14 language in critical and safety-related systems* — AUTOSAR consortium (2017+).

### 10장 구성
1. AUTOSAR란 / 도입 동기 / MISRA C++와 비교
2. 어휘 / 빌드 (M0-M2)
3. 표현식 / 변환 (A4-A5)
4. 함수 / 람다 (A7-A8)
5. 클래스 / 상속 (A10-A12)
6. 템플릿 (A14)
7. 예외 (A15-A16)
8. STL / 표준 라이브러리 (A18)
9. 동시성 / 메모리 (A21-A25)
10. 도구 / 인증 / 시리즈 마무리

### 위상
- MISRA C++ 후계 (MISRA C++:2008 → AUTOSAR C++14 → MISRA C++:2023)
- 모던 C++ idiom 포용 (RAII, smart pointer 등)
