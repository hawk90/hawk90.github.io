## 작성 중

MISRA C:2012 — Motor Industry Software Reliability Association (현재 MISRA C 2012 + Amendment 1-4).

### 10장 구성
1. MISRA란 / 도입 배경
2. 분류 — Required / Advisory / Mandatory. Directive vs Rule
3. 어휘 / 빌드 / 문서 관련 directives (Dir 1-4)
4. 구문 / 형식 (Rule 1-7) — 환경 / 미사용 / 주석 / 키워드 / 식별자 / 타입
5. 표현식 / 타입 변환 (Rule 8-11)
6. 포인터 / 배열 (Rule 18)
7. 제어 흐름 (Rule 14-15-16)
8. 함수 (Rule 17)
9. 메모리 / 라이브러리 (Rule 21-22)
10. 도구 / 인증 / 시리즈 마무리
