## 예정 내용
- Directive — 정적 분석 어려운 정책 (Dir 1.1 ~ 4.13)
- Rule — 분석기로 검증 가능 (Rule 1.1 ~ 22.10)
- Mandatory — 반드시 (deviation 불가)
- Required — 원칙적으로 (정당화 + 문서화 시 deviation)
- Advisory — 권고
- Decidable vs Undecidable
