> Outline — **F´** (F Prime) = JPL의 *모던 flight SW framework*. C++14·component·port·topology를 *FPP* (F Prime Prompt) 모델 언어로 명시. *type-safe message passing*. autocoder가 port·serial code 생성. 적용 — Ingenuity Mars Helicopter, ASTERIA, ISS SPHERES. cFS와의 비교 — F´는 *더 모던·정적 타이핑*, cFS는 *더 전통적·동적 메시지*.
