> Outline — TM Space Data Link Protocol — *Transfer Frame* (header + data + trailer). *Virtual Channel* (VC) — frame을 다중 채널로 다중화. *Master Channel* — 한 spacecraft 단위. AOS·CCSDS 표준 frame format. TC (Telecommand) — uplink, COP-1 (FARM/FOP) sliding window. *frame loss·retransmit* 처리. Reed-Solomon·LDPC·turbo coding과의 연계.
