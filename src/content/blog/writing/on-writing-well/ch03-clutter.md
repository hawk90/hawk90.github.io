> **On Writing Well** Chapter 3 요약

## 이 챕터의 메시지

2장이 단순함의 원칙이라면, 3장은 **구체적 잡초 식별**.

> **Clutter is the disease of American writing.**

미국 산문의 병 — 잡초. Zinsser가 챕터를 통해 그 잡초들을 분류.

## 잡초의 종류

### 1. Pompous Word (가식적 단어)

라틴어 / 학술적 단어가 평범한 단어를 대체.

```
❌ utilize         ✅ use
❌ implement       ✅ do
❌ facilitate      ✅ help
❌ ascertain       ✅ find out
❌ terminate       ✅ end / fire
❌ commence        ✅ start
❌ purchase        ✅ buy
❌ residence       ✅ home
❌ verbalize       ✅ say
❌ obligate        ✅ require
```

### 2. Filler (군더더기 단어)

```
- "I think"
- "I believe"
- "in my opinion"
- "it seems to me"
- "kind of", "sort of"
- "basically", "actually", "really"
- "in fact"
- "as a matter of fact"
```

종종 자신감 없을 때 끼움. 대부분 잘라낼 수 있다.

### 3. Verb의 Inflation

```
❌ "He took the decision to leave."
✅ "He decided to leave."

❌ "She made the recommendation that..."
✅ "She recommended..."

❌ "We have come to the conclusion that..."
✅ "We concluded..."
```

동사를 다른 단어로 부풀림. 그냥 동사로.

### 4. Redundant Phrases

```
❌ free gift             ✅ gift
❌ true facts            ✅ facts
❌ unexpected surprise   ✅ surprise
❌ end result            ✅ result
❌ each individual       ✅ each
❌ personal feelings     ✅ feelings
❌ honest truth          ✅ truth
```

### 5. Hedging

```
❌ "It is possible that we might consider..."
✅ "Let's consider..."

❌ "There seems to be a possibility that..."
✅ "Possibly..."

❌ "Studies have shown that there may be a tendency..."
✅ "Studies suggest..."
```

확신 없는 척하지만 — 종종 그냥 입버릇.

### 6. Long Prepositional Phrases

```
❌ "in spite of the fact that"     ✅ although
❌ "owing to the fact that"        ✅ because
❌ "with regard to"                ✅ about
❌ "in connection with"            ✅ about
❌ "on the basis of"               ✅ based on
❌ "for the reason that"           ✅ because
```

"of"가 많이 등장하면 — 의심.

### 7. Throat Clearing

문장 시작 전에 — 목 가다듬기.

```
❌ "I would like to point out that..."
❌ "It is interesting to note that..."
❌ "It should be mentioned that..."

→ 그냥 본론으로
```

이런 시작을 빼면 — 글이 즉시 시작.

## 잡초 자르기 절차

Zinsser의 권장 절차.

1. **첫 초안을 쓴다** — 자유롭게
2. **두 번째 초안에서 25% 자른다**
3. **세 번째 초안에서 또 다시 본다**
4. **소리 내 읽는다** — 어색한 곳 잡힌다

매 단계마다 잡초가 빠진다.

## 잡초의 부담

작가가 잡초를 두는 이유:

- 글이 더 길어 보임 (분량 채움)
- 더 격식 있어 보임
- 더 학술적
- 더 신중해 보임

독자의 입장:

- 의미 추출하기 어려움
- 시간 낭비
- 흥미 잃음
- 떠남

**잡초의 비용은 독자가 지불**.

## 자기 검토 — Zinsser의 방법

자기 글에서:

1. **각 단어에 묻는다** — "이게 없으면 의미가 바뀌나?"
2. 안 바뀌면 — 잘라낸다
3. **각 구절** — 짧게 가능?
4. **각 문장** — 50% 자를 수 있나?

이걸 매 글에 적용 — 빠르게 단순화 기술이 는다.

## "Brackets" 기법

Zinsser가 자기 학생들에게 가르치는 기법.

> 글을 인쇄해서 — 불필요한 단어 / 구를 **대괄호**로 묶는다.

```
"[It is currently our] understanding [that the situation] will [in 
fact] improve [in the near future]."
```

대괄호 친 부분이 잡초. 빼고 다시 읽으면:

```
"Our understanding: it will improve soon."
```

대괄호 기법으로 잡초가 가시화된다.

## "쓰기" vs "재쓰기"

좋은 글의 비밀.

```
나쁜 작가: 한 번에 완벽하게 쓰려 노력
좋은 작가: 첫 초안 후 여러 번 재작성
```

**Rewriting is the essence of writing**. 좋은 글은 다시 쓴 글.

## 정리

- Clutter = 미국 산문의 병
- 7가지 잡초 — Pompous word, Filler, Verb inflation, Redundancy, Hedging, Long prepositional, Throat clearing
- **25% 자르기**가 일반 규칙
- 잡초의 비용은 **독자**가 지불
- **Brackets 기법**으로 잡초 가시화
- **Rewriting**이 글의 비밀
