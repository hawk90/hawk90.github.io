2장이 단순함의 원칙이었다면, 3장은 *구체적 잡초의 식별*입니다.

> Clutter is the disease of American writing.

Zinsser는 미국 산문의 병을 *어수선함(clutter)*이라고 진단합니다. 이 챕터는 글에 끼어드는 잡초들을 일곱 종류로 분류합니다.

## 잡초 1 — Pompous Words (가식적 단어)

라틴 어원의 어려운 단어가 평범한 단어를 대체합니다.

```
회피             바름
utilize       →  use
implement     →  do
facilitate    →  help
ascertain     →  find out
terminate     →  end, fire
commence      →  start
purchase      →  buy
residence     →  home
verbalize     →  say
obligate      →  require
```

대부분의 경우 짧은 단어가 더 명확합니다.

## 잡초 2 — Filler (군더더기 단어)

자신감이 없을 때 끼워 넣는 단어들. 대부분 잘라낼 수 있습니다.

```
I think
I believe
in my opinion
it seems to me
kind of, sort of
basically, actually, really
in fact
as a matter of fact
```

## 잡초 3 — Verb Inflation (동사 부풀림)

동사를 *make/take/come to/give* + 명사로 부풀리는 패턴.

```
회피: He took the decision to leave.
바름: He decided to leave.

회피: She made the recommendation that ...
바름: She recommended ...

회피: We have come to the conclusion that ...
바름: We concluded ...

회피: I would like to give my support to ...
바름: I support ...
```

## 잡초 4 — Redundant Phrases (중복 표현)

```
free gift           →  gift
true facts          →  facts
unexpected surprise →  surprise
end result          →  result
each individual     →  each
personal feelings   →  feelings
honest truth        →  truth
basic fundamentals  →  fundamentals
```

수식어가 의미를 더하지 못하면 잡초입니다.

## 잡초 5 — Hedging (확신 없는 척)

```
회피: It is possible that we might consider ...
바름: Let's consider ...

회피: There seems to be a possibility that ...
바름: Possibly ...

회피: Studies have shown that there may be a tendency ...
바름: Studies suggest ...
```

진짜 불확실한 자리는 헤지가 필요하지만, *입버릇*처럼 끼는 헤지는 잡초입니다.

## 잡초 6 — Long Prepositional Phrases (긴 전치사구)

```
회피                          바름
in spite of the fact that  →  although
owing to the fact that     →  because
with regard to             →  about
in connection with         →  about
on the basis of            →  based on
for the reason that        →  because
```

*of*가 한 문장에 여러 번 등장하면 잡초의 신호입니다.

## 잡초 7 — Throat Clearing (목 가다듬기)

문장 시작 전에 끼우는 *의식적 도입*.

```
회피: I would like to point out that ...
회피: It is interesting to note that ...
회피: It should be mentioned that ...

→ 그냥 본론으로
```

이런 시작을 빼면 글이 즉시 시작됩니다.

## 잡초 자르기 절차

Zinsser가 권장하는 절차입니다.

```
1. 첫 초안 — 자유롭게 쓰기
2. 두 번째 초안 — 25퍼센트 잘라내기
3. 세 번째 초안 — 다시 보고 또 잘라내기
4. 소리 내 읽기 — 어색한 곳이 잡힙니다
```

매 단계마다 잡초가 빠집니다.

## 작가가 잡초를 두는 이유

```
작가 입장:
- 글이 더 길어 보임
- 더 격식 있어 보임
- 더 학술적
- 더 신중해 보임

독자 입장:
- 의미 추출이 어려움
- 시간 낭비
- 흥미 잃음
- 떠남
```

잡초의 비용은 *독자가 지불*합니다.

## Brackets 기법

Zinsser가 학생들에게 가르치는 시각화 기법입니다.

> 글을 인쇄해서 불필요한 단어와 구를 *대괄호*로 묶습니다.

```
"[It is currently our] understanding [that the situation] will [in fact]
improve [in the near future]."
```

대괄호 친 부분이 잡초입니다. 빼고 다시 읽으면:

```
"Our understanding: it will improve soon."
```

대괄호 기법으로 잡초가 시각적으로 드러납니다.

## Rewriting — 글의 비밀

Zinsser의 통찰 — 좋은 글은 *다시 쓴* 글입니다.

```
나쁜 작가:  한 번에 완벽하게 쓰려 함
좋은 작가:  첫 초안 후 여러 번 재작성
```

> Rewriting is the essence of writing.

*재작성이 글쓰기의 본질*. 한 번에 좋은 글이 나오리라는 기대를 버려야 합니다.

## 정리

- **Clutter**는 미국 산문의 병
- 일곱 종류의 잡초 — Pompous word, Filler, Verb inflation, Redundancy, Hedging, Long prepositional, Throat clearing
- 일반 규칙 — *25퍼센트 자르기*
- 잡초의 비용은 *독자가 지불*
- **Brackets 기법**으로 잡초 시각화
- **Rewriting**이 글의 비밀
