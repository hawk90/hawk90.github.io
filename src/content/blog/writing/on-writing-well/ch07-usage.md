영어 용법은 변합니다. 100년 전 옳지 않던 것이 지금 표준이 되고, 그 반대도 일어납니다.

> 누가 결정하는가? 작가들이, 시간을 두고.

규칙은 위에서 내려오지 않습니다. 사용에서 자라납니다.

## Usage Panel

Zinsser가 인용하는 *American Heritage Dictionary*의 Usage Panel은 작가와 학자로 구성된 자문 패널입니다. 각 단어와 용법에 대한 의견을 수렴합니다.

```
"hopefully"를 부사로 (희망적으로, 바라건대)
   1969: Panel의 86퍼센트가 반대
   1999: 의견 갈림
   2020: 더 받아들여짐
```

언어는 진화합니다. 어제의 잘못이 오늘의 표준이 될 수 있습니다.

## 두 진영

**Prescriptivist (규범주의)**:
- 옳은 용법이 있다
- 사전과 문법책의 권위
- 변화에 저항

**Descriptivist (기술주의)**:
- 사용된 대로가 영어
- 진화를 받아들임
- 규칙은 관찰의 결과

Zinsser는 *중간*에 섭니다. 좋은 영어는 권위에서 오지만, 권위는 작가들의 좋은 사용에서 나옵니다.

## 새 단어의 등장

매년 새 단어와 용법이 영어에 추가됩니다.

```
"Google"이 동사로 (2006년 OED)
"Selfie" (2013년 OED Word of the Year)
"Doomscroll" (코로나 시대)
"Vibe check"
```

처음엔 슬랭, 점차 정착, 결국 사전 등재입니다.

## "Different" — 한 단어의 진화

```
1700s: "different from" 만 표준
1800s: "different than" 등장
1900s: 둘 다 사용
오늘: "different from" (formal), "different than" (casual, AmE)

영국: "different to"도 흔함
```

같은 의미가 시대와 지역에 따라 다른 형태를 가집니다.

## 좋은 작가의 결정 방법

용법에 헷갈리면:

1. 사전 확인
2. 다른 좋은 작가의 사용 확인
3. 자기 청중의 기대 고려
4. *명확성 우선*

규칙에 따르지 말고 명확함을 우선합니다.

## Correctness의 함정

규칙에 너무 매이면 글이 어색해집니다.

```
"To whom did you give the book?" (격식 — whom)
"Who did you give the book to?"  (자연 — who)
```

격식이 필요한 자리(학술, 법률)는 첫 형태가 적절합니다. 일반 글은 두 번째가 더 자연스럽습니다. *상황에 맞는 영어*가 좋은 영어입니다.

## 흔한 혼동들

```
affect / effect           — 동사 / 명사
imply / infer             — 작가가 함축 / 독자가 추론
flaunt / flout            — 자랑 / 비웃음
disinterested / uninterested — 공정 / 무관심
literally                 — 글자 그대로 (강조로 X)
```

자세한 내용은 *Elements of Style* 4장에서 다룹니다.

## Split Infinitive

전통은 *split infinitive 금지*. 현대 작가는 거의 무시합니다.

```
"To boldly go where no man has gone before."   (Star Trek)
"To really understand ..."
```

분리가 문장을 더 자연스럽게 만들면 분리합니다. 규칙 때문에 어색해질 필요는 없습니다.

## 시대의 변화

영어가 빠르게 변하는 영역들입니다.

- *성중립 대명사* — 단수 *they*의 표준화
- *이메일 영어* — 더 캐주얼, 짧음
- *국제 영어* — 비-영어권의 영향
- *AI 영향* — 단어와 용법이 빠르게 변화

작가는 시대를 알면서 씁니다.

## 자기 글의 용법 점검

```
□ 이 단어를 사전이 어떻게 정의하는가?
□ 이 시대에 어떻게 쓰이는가?
□ 내 청중이 익숙한가?
□ 이 용법이 의미를 명확히 전달하는가?
```

네 질문에 자신 있으면 그 용법을 사용합니다.

## "Trust the language"

Zinsser의 결론입니다.

> 영어를 신뢰하세요. 규칙이 흐릿한 곳에서는 자기 귀를 믿으세요.

좋은 작가의 귀는 책을 많이 읽고 글을 많이 써서 만들어집니다. 그 귀가 사전보다 더 정확할 때가 있습니다.

## 정리

- 영어 용법은 *변합니다* — 시간을 두고
- Prescriptivist와 Descriptivist 사이에서 *명확함 우선*
- *사전 + 좋은 작가의 사용*이 권위
- 규칙 때문에 어색해지지 마세요 (split infinitive 등)
- 새 단어와 용법이 매년 등장
- *자기 귀*를 신뢰
