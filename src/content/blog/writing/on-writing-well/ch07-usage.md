> **On Writing Well** Chapter 7 요약

## 이 챕터의 메시지

영어 용법은 변한다. 100년 전 옳지 않던 것이 — 지금 표준. 그 반대도.

> **누가 결정하는가? 작가들이 — 시간을 두고.**

규칙은 위에서 내려오지 않는다. 사용에서 자라난다.

## "Usage Panel"

Zinsser가 인용하는 American Heritage Dictionary의 **Usage Panel** — 작가와 학자로 구성. 각 단어 / 용법에 대한 의견 수렴.

```
"hopefully"를 부사로 (희망적으로 / 바라건대)
   1969: Panel의 86%가 반대
   1999: 의견 갈림
   2020: 더 받아들여짐
```

언어는 진화. 어제의 잘못이 오늘의 표준이 될 수 있다.

## 두 진영

**Prescriptivist (규범주의)**:
- "옳은 용법이 있다"
- 사전 / 문법책의 권위
- 변화에 저항

**Descriptivist (기술주의)**:
- "사용된 대로가 영어"
- 진화 받아들임
- 규칙은 관찰의 결과

Zinsser는 **중간**. 좋은 영어는 — 권위에서 오지만, 권위는 작가들의 좋은 사용에서 나옴.

## 새 단어의 등장

매년 새 단어 / 용법이 영어에 추가.

```
"Google"이 동사로 (2006년 OED)
"Selfie" (2013년 OED Word of the Year)
"Doomscroll" (코로나 시대)
"Vibe check"
```

처음엔 slang, 점차 정착, 결국 사전 등재.

## 한 단어의 진화 — "Different"

```
1700s: "different from" (only)
1800s: "different than" 등장
1900s: 둘 다 사용
오늘: "different from" (formal), "different than" (casual, AmE)

영국: "different to"도 흔함
```

같은 의미 — 시대 / 지역마다 다른 형태.

## 좋은 작가는 어떻게 결정하나

용법에 헷갈리면:

1. **사전 확인**
2. **다른 좋은 작가의 사용 확인**
3. **자기 청중의 기대 고려**
4. **명확성 우선**

규칙에 따르지 말고 — **명확함**을 우선.

## "Correctness"의 함정

규칙에 너무 매이면 — 글이 어색해진다.

```
"To whom did you give the book?" (격식 — whom)
"Who did you give the book to?" (자연 — who)
```

격식이 필요한 자리(학술, 법률)는 첫 형태. 일반 글은 두 번째가 더 자연.

**상황에 맞는 영어** — 그게 좋은 영어.

## 흔한 혼동들

Zinsser가 다루는 몇 가지.

```
affect / effect      — 동사 / 명사
imply / infer        — 작가가 함축 / 독자가 추론
flaunt / flout       — 자랑 / 비웃음
disinterested / uninterested — 공정 / 무관심
literally            — 글자 그대로 (강조 X)
```

이런 단어들은 — 정확한 사용이 격을 결정.

자세한 내용은 Elements of Style 4장에서 다룬다.

## "Should I split this infinitive?"

전통 — split infinitive 금지.
현대 — 거의 모든 좋은 작가가 무시.

```
"To boldly go where no man has gone before."   ← Star Trek
"To really understand..." 
```

split이 문장을 더 자연스럽게 만들면 — 한다. 규칙 때문에 어색해질 필요 없음.

## 시대의 변화

영어가 빠르게 변하는 영역.

- **성중립 대명사** — "they" 단수 사용 표준화
- **이메일 영어** — 더 캐주얼, 짧음
- **국제 영어** — 비-영어권 영향
- **AI 영향** — 단어 / 용법이 빠르게 변화

작가는 — **시대를 알면서** 쓴다.

## 자기 글의 검토

용법 점검:

1. **이 단어를 사전이 어떻게 정의?**
2. **이 시대의 사용?**
3. **내 청중이 익숙한가?**
4. **이 용법이 의미를 명확히 전달?**

이 네 질문에 자신 있으면 — 그 용법 사용.

## "Trust the language"

Zinsser의 결론.

> **영어를 신뢰하라. 규칙이 흐릿한 곳에서는 — 자기 귀를 믿어라.**

좋은 작가의 귀는 — 책을 많이 읽고 글을 많이 써서 만들어진다. 그 귀가 — 사전보다 더 정확할 때가 있다.

## 정리

- 영어 용법은 **변한다** — 시간을 두고
- Prescriptivist vs Descriptivist 사이에서 — **명확함 우선**
- **사전 + 좋은 작가의 사용** = 권위
- 규칙 때문에 어색해지지 마라 (split infinitive 등)
- 새 단어 / 용법이 매년 등장
- **자기 귀**를 신뢰
