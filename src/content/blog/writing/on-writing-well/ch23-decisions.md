> **On Writing Well** Chapter 23 요약

## 이 챕터의 메시지

글쓰기 = 끝없는 결정.

- 어떤 단어
- 어떤 순서
- 어떤 길이
- 어떤 어조
- 무엇을 잘라
- 무엇을 더할

각 결정이 — 글을 형태짓는다.

## 결정의 무게

```
"It was a sunny day."   vs   "Sunlight cut through the clouds."
```

같은 사실. 다른 분위기. 단어 선택의 결정.

```
시작: 한 문장
끝: 한 단어
```

수백 / 수천 결정이 — 글을 만든다.

## "Choosing" — 작가의 일

좋은 작가의 차이 = **선택의 정확성**.

- 거의 비슷한 단어 두 개 중 — 진짜 옳은 하나
- 시작과 끝 사이의 — 무수한 분기점

이 선택이 — 시간이 든다.

## 결정의 자동화

연습으로 — 많은 결정이 자동화.

```
초보: 매 단어 고민
숙련: 자연스럽게 흐름
```

다만 **중요한 결정**은 — 항상 의식적.

## 정리

- 글쓰기 = **결정의 연속**
- 각 결정이 글을 형태짓는다
- 선택의 정확성 = 작가의 차이
- 연습으로 자동화, 그러나 핵심은 의식적
