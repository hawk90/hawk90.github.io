> **On Writing Well** Chapter 22 요약

## 이 챕터의 메시지

학교 / 직장이 가르치는 것 — **최종 결과물(final product)이 중요**.

Zinsser의 반박:

> **Process가 중요. Final product는 결과일 뿐.**

좋은 글은 — 좋은 process에서 나온다.

## "The tyranny of the final product"

완벽한 결과에 대한 강박 → 작가를 마비.

```
"이 글이 완벽해야 해."
→ 손이 안 움직임
→ 첫 줄도 못 씀
```

이게 폭정(tyranny). 완벽함이 — 시작 자체를 막는다.

## 해법 — Process 우선

```
1. 자유롭게 첫 초안 (perfectionism 끄기)
2. 비판적으로 수정 (perfectionism 켜기)
3. 다듬기
4. 발표 (또는 안 함)
```

각 단계가 — **다른 마인드**. 동시에 하지 마라.

## "Bad first draft"의 가치

좋은 작가는 — 나쁜 첫 초안을 환영.

```
Anne Lamott: "Shitty first drafts."
Hemingway: "The first draft of anything is shit."
```

첫 초안의 임무 = **존재하는 것**. 좋은 것이 아님.

수정에서 좋아진다. 그러나 — 첫 초안이 없으면 수정할 것도 없다.

## 학교가 만든 그릇된 가르침

학교 — 제출용 글만 평가. 그러나:

- Process도 가르쳐야
- 수정의 가치
- 실패한 시도의 의미

**좋은 작가는 완성품이 아니라 — 일하는 과정**.

## 작가는 평생 학생

```
어떤 위대한 작가도 — 다음 글에 대해 두려워한다.
어떤 위대한 작가도 — 매번 새로 배운다.
```

이 사실이 — 위안. 자기만 어렵다고 생각하지 마라.

## 정리

- Final product의 폭정 = 작가를 마비
- **Process가 중요**
- 나쁜 첫 초안은 — 좋은 출발점
- 학교의 결과 중심 평가는 — 작가를 망친다
- 모든 작가가 매번 — 새로 배움
