> **On Writing Well** Chapter 9 요약

## 이 챕터의 메시지

글의 가장 중요한 두 부분 — **시작 (lead)**과 **끝 (ending)**.

> **The first sentence must lure the reader into the second.**

첫 문장은 독자를 둘째 문장으로 유인. 둘째 문장은 셋째로. 그 사슬이 끊기면 — 독자가 떠난다.

## Lead — 첫 문장의 임무

첫 문장이 해야 할 것.

1. **호기심을 자극**
2. **분명한 약속**을 함
3. **신선한 정보 / 시각** 제공
4. **다음 문장으로 끌어 감**

```
❌ "This article will discuss the importance of..."
   (너무 일반, 약속 흐릿)

✅ "I never thought I'd find myself standing on a glacier in 
   Iceland at 3 a.m., wishing I had brought thicker socks."
   (구체, 호기심)
```

## "Lure"의 방법

Zinsser가 권장하는 시작 패턴.

**1. 일화 (anecdote)**

```
"When I was eight, my grandfather took me fishing for the 
first time. I caught a tin can."
```

작은 이야기로 시작. 독자가 따라옴.

**2. 놀라운 주장**

```
"Pickup trucks are now the most popular vehicle in America."
```

상식과 다른 사실 → 호기심.

**3. 강한 질문**

```
"Why do we hate Mondays?"
```

독자가 답을 알고 싶어짐.

**4. 인용**

```
"'It is impossible to write a memoir,' said E.B. White."
```

권위의 진술로 시작.

**5. 장면 묘사**

```
"The room was small, the light was harsh, and the chair 
was uncomfortable."
```

영화처럼 — 독자를 그 자리로.

## Lead의 길이

```
짧은 lead — 호흡, 강함
긴 lead — 점진적, 깊은 시작
```

대부분의 좋은 글은 — **짧은 lead로 잡고 펼친다**.

```
"Mountains are big.
But this one was bigger."
```

이런 짧음이 — 독자를 잡는다.

## Lead의 함정

피해야 할 시작.

**1. 흐릿한 일반화**

```
❌ "In today's modern world..."
❌ "Throughout history..."
❌ "It is widely believed that..."
```

**2. Throat clearing**

```
❌ "Before I begin, let me explain..."
❌ "I would like to talk about..."
```

**3. 약한 단어**

```
❌ "There is a topic that has been discussed..."
   (be 동사로 시작 — 약함)
```

이런 시작은 독자를 잡지 못함.

## Ending — 마지막 문장의 임무

끝은 시작만큼 중요.

> **The perfect ending should take the reader slightly by surprise and yet seem exactly right.**

완벽한 끝은 — 독자를 약간 놀라게 하면서, 동시에 정확히 옳다고 느끼게.

## Ending의 패턴

**1. 회기 (callback)**

시작의 어떤 요소를 끝에서 되울림.

```
시작: "I caught a tin can."
끝: "Forty years later, I still keep that tin can on my desk."
```

원이 닫힘. 독자 만족.

**2. 강한 한 문장**

```
끝: "He never came back."
```

짧고 결정적.

**3. 인용**

```
끝: "As my grandfather used to say: 'Patience is its own reward.'"
```

다른 사람의 말로 마무리. 권위.

**4. 의외의 진실**

```
끝: "After all, perhaps the trip was about the people, not the place."
```

기대와 다른 통찰. 독자에게 생각거리.

## Ending의 함정

**1. 요약**

```
❌ "In conclusion, this article discussed three topics..."
   (이미 본 것 반복 — 지루)
```

대학 작문이 가르치는 — 그러나 nonfiction에서는 피하라.

**2. 길게 끌기**

```
❌ "And so, after all is said and done, we can finally see 
   that, considering everything..."
   (마무리를 못 함)
```

끝나야 할 때 — 끝나라. 안 끌기.

**3. 미적지근**

```
❌ "It was an interesting experience overall."
   (할 말 없으면 안 마무리)
```

## "End when you have nothing more to say"

Zinsser의 단호한 원칙.

> **할 말이 끝났을 때 — 끝내라.**

대부분의 작가는 — 너무 많이 쓴다. 첫 초안의 마지막 단락 / 마지막 두 문장 — 종종 잘라낼 수 있다.

자기 글의 — 진짜 마지막을 찾아라.

## Lead와 Ending의 비율

좋은 글에 시간 배분:

```
Lead 작업    — 글 전체의 25%
본문 작업    — 50%
Ending 작업  — 25%
```

50%가 시작과 끝에. 그만큼 중요.

## "Surprise"의 가치

좋은 끝은 — **약간의 놀라움**.

너무 예측 가능 → 지루.
너무 비뚤 → 어색.

황금 균형 — 독자가 "오, 그렇구나"를 느낄 정도.

## 정리

- 글의 가장 중요한 두 부분 — **시작 (lead)**과 **끝 (ending)**
- 첫 문장은 둘째로 **유인 (lure)**
- 좋은 lead — 일화 / 놀라운 주장 / 질문 / 인용 / 장면
- 피할 lead — 흐릿한 일반화, throat clearing
- 좋은 ending — callback / 강한 한 문장 / 인용 / 의외 통찰
- 피할 ending — 요약 / 길게 끌기
- **할 말 끝났으면 끝내라**
