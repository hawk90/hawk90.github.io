> **On Writing Well** Chapter 8 요약

## 이 챕터의 메시지

좋은 글은 통일성이 있다. 한 글이 — 하나의 어조, 하나의 관점, 하나의 시제로 유지.

Zinsser가 강조하는 네 가지 통일성.

1. **어조 (tone)** — 진지 / 가벼움 / 비판적 / 회의적
2. **시제 (tense)** — 현재 / 과거 / 미래
3. **대명사 (pronoun)** — I / we / one / you
4. **관점 (point of view)** — 누구의 시각으로

## 1. 어조의 통일

한 글이 — 일관된 어조.

```
❌ 진지하게 시작 → 갑자기 농담 → 다시 진지
❌ 비판적으로 시작 → 갑자기 우호적

✅ 한 어조를 처음부터 끝까지
```

어조가 변하면 — 독자가 흔들린다. 작가가 누구인지 헷갈림.

다만 — 의도된 어조 변화는 OK. 한 단락의 진지함 후 — 짧은 농담이 강조를 만든다. 그러나 **일관성이 기본**.

## 2. 시제의 통일

이야기 / 보고서의 시제는 일관되게.

```
❌ "He went to the store. He buys some milk. Then he came home."
✅ "He went to the store. He bought some milk. Then he came home."
```

기본 시제 결정 후 — 그 시제로 유지. 변화는 — 시점 차이를 표시할 때만.

```
✅ "I am writing about my childhood. As a boy, I lived in Seoul."
        (현재 — 글의 시점)        (과거 — 회상 시점)
```

이건 의도된 변화. 자연스러움.

## 3. 대명사의 통일

자기를 어떻게 부를 것인가.

- **I** — 1인칭. 가장 직접적.
- **We** — 1인칭 복수. 저자 + 독자 또는 저자 + 팀.
- **One** — 보편적 주어. 격식.
- **You** — 2인칭. 독자에게 직접 말함.

```
❌ "When I was young, one would often play outside, and you 
   felt free."
   (I, one, you가 다 등장 — 누구의 시각?)

✅ "When I was young, I would often play outside, and I felt free."
   (모두 I)
```

대명사를 정한 후 — 끝까지.

## 4. 관점의 통일

누구의 시각에서 말하는가.

- **객관적 보고자** — 사실만, 의견 X
- **참여 관찰자** — 봤다고 보고
- **참여자** — 자기 경험
- **분석가** — 해석 / 평가

```
❌ "The event was held in the auditorium. I felt nervous as I 
   walked in. The crowd seemed bored."
   ↑                ↑                       ↑
   객관           참여자                   해석

✅ 관점을 정하고 — 끝까지
```

## "Make decisions before you write"

이 네 가지 통일성을 — **글 시작 전에 결정**.

```
1. 어조 — 어떤 분위기?
2. 시제 — 언제의 시점?
3. 대명사 — 어떻게 자기를 부를?
4. 관점 — 누구의 시각으로?
```

결정 안 한 채 쓰면 — 글이 흔들린다.

## "Reduce your big subject"

큰 주제를 작게.

```
❌ "여행" — 너무 큼
✅ "오사카에서 만난 한 카페" — 작게
```

한 글 = 한 작은 주제. 큰 주제는 — 여러 글로 나눠.

이게 unity의 또 다른 차원 — **주제의 통일**.

## "Choose your length"

길이를 미리 결정.

- 500단어 글
- 2000단어 글
- 10000단어 글

각 길이에 — 다른 양의 정보. 미리 결정해야 비례가 맞는다.

## 통일성의 점검

자기 글을 다 쓴 후 점검.

1. **어조** — 처음부터 끝까지 일관?
2. **시제** — 의도된 변화 외에 일관?
3. **대명사** — 같은 대명사로 자기 지칭?
4. **관점** — 한 시각으로 유지?

어느 하나라도 안 일관 — 재작성.

## 어조 변화의 위험

어조 변화는 — 종종 자연스러움 손실.

```
"This is a serious problem. We must address it immediately. 
And, well, sometimes things just happen, you know?"
```

진지함 → 캐주얼. 독자가 흔들. 첫 두 문장의 무게가 셋째 문장에 무너진다.

## Zinsser의 권장

"어떤 글을 쓸지 모르겠으면 — 자기에게 짧은 글을 쓴다."

```
"이 글은 X에 대해, Y한 어조로, Z 관점에서, 길이는 W."
```

이 한 문장이 — 전체 글의 방향. 결정 후 그대로 쓰면 — 통일성이 자동.

## 정리

- 좋은 글 = **통일성**
- 네 가지 통일 — **어조 / 시제 / 대명사 / 관점**
- 글 **시작 전에 결정**
- 의도된 변화는 OK — 무의식적 변화는 X
- 주제도 작게 — 한 글 = 한 작은 주제
- 길이도 미리 결정
