Part 3의 시작은 *작가의 태도*입니다. 무엇을 어떻게 쓰는가의 문제가 아니라, *왜 쓰는가, 어떤 자세로 쓰는가*의 문제로 넘어갑니다.

> Voice는 작가의 trademark입니다.

자기 voice가 있어야 *독자가 자기 글을 알아봅니다*. Voice가 없는 글은 *익명의 글*입니다. 무엇을 다루든, 다른 작가의 글과 *구별되지 않는* 글입니다.

## Voice란 무엇인가

Voice는 단순한 *어조*가 아닙니다. 작가의 *세상을 보는 방식, 단어 선택의 습관, 문장 리듬, 가치관*이 모두 합쳐진 것입니다.

```
같은 사건을 두 작가가 쓰면:

작가 A: "그날 비가 왔다. 우산이 없었지만 기분은 괜찮았다.
        때로 비가 정신을 맑게 해 준다."

작가 B: "기상청이 예고한 대로 오후 3시부터 비가 내렸다.
        나는 우산 없이 7분간 노출되었다. 결과 — 셔츠가
        젖었지만 사고는 없었다."
```

같은 비, 같은 사람이라도 *voice가 다르면* 다른 글이 됩니다.

## Voice의 발견

자기 voice는 *어떻게* 찾을까요?

### 1. 모방의 시기

대부분의 작가는 *다른 작가를 흉내내며 시작*합니다. 이것은 *어쩔 수 없는 단계*입니다. 처음에는 좋아하는 작가의 어휘, 문장 길이, 리듬을 그대로 빌립니다.

```
"내 첫 글들은 다 Hemingway 같았다. 그 다음은 Murakami 같았다.
이제는 — 누구 같지도 않다. 그게 내 voice가 자란 신호였다."
```

모방의 시기를 *부끄러워하지 마세요*. 좋은 모방이 *재료*가 됩니다.

### 2. 자기 패턴의 등장

여러 글을 쓰다 보면 *자기만의 패턴*이 드러납니다.

```
- 자주 쓰는 단어
- 즐겨 쓰는 문장 형태
- 어떤 비유를 자주 쓰는가
- 어떤 주제에 끌리는가
- 어떤 농담을 하는가
```

이 패턴들이 *자기 voice*입니다. 의식하지 못한 채 만들어집니다.

### 3. 평생의 다듬기

Voice는 *한 번 발견하면 끝*이 아닙니다. 평생에 걸쳐 *깊어지고 변합니다*. 20대의 voice와 50대의 voice가 다른 것이 자연스럽습니다.

## "Sound Like Yourself"

가장 단순한 가이드는 *자기가 말하는 것처럼 쓰는 것*입니다.

```
친구에게 이 주제를 설명하면 어떻게?
→ 그게 자기 voice의 출발점
```

가식이나 격식이 끼면 voice가 죽습니다.

### 회피해야 할 가식

```
회피 (가식적):
"It is the contention of the author that ..."
"본 연구의 함의는 다음과 같다고 사료된다."

선호 (자기 목소리):
"I think ..."
"이 연구는 ..."
```

학교 / 직장 / 학술계가 만든 *가식적 어조*에서 벗어나는 것이 voice 찾기의 첫 걸음입니다.

## Voice의 일관성

한 책 / 한 글에는 *한 voice*가 필요합니다.

```
회피: 농담 어조 → 학술 어조 → 다시 농담
선호: 일관된 voice
```

다양한 분야를 다뤄도 *같은 voice*를 유지합니다. 그것이 작가의 통일성입니다.

### 예시 — Voice의 변화

```
한 작가가 같은 주제를 두 어조로 쓰는 것은 잘못된 패턴:

오프닝 (학술적 톤):
"본 글은 디지털 화폐의 사회 경제적 함의를 다룬다.
이는 화폐 시스템의 변혁을 의미하며 ..."

중간 (캐주얼 톤):
"근데 진짜 이건 미친 짓 같아요. 비트코인 가격 보셨어요?
그냥 도박이죠."
```

독자가 *작가가 누구인지* 헷갈립니다. 어느 voice가 *진짜 작가*인지 모릅니다.

### Voice 일관성의 도구

```
□ 글 시작 전에 voice 결정 — 한 문장으로 자기에게 설명
□ 어휘 일관성 — 회화 어휘와 학술 어휘를 섞지 말 것
□ 문장 길이 — 일관된 리듬
□ 격식 수준 — 한 글에 한 수준
```

이 도구들이 *voice를 유지*합니다.

## Voice와 분야

자기 분야에 따라 *voice의 자유도*가 다릅니다.

```
학술 논문:    voice가 좁음 (분야 표준 우세)
저널리즘:     voice가 중간 (매체별 차이)
에세이:       voice가 넓음 (작가 자유)
블로그:       voice가 가장 넓음 (개인 표현)
```

학술 글에서도 *voice는 사라지지 않습니다*. 단지 *분야의 관습 안에서* 표현됩니다. Stephen Hawking의 voice와 Brian Greene의 voice는 *분명히 다릅니다* — 둘 다 물리학자지만.

## 모범 — 강한 Voice의 작가들

### George Orwell

명료하고 직설적. *Politics and the English Language*가 그의 voice 자체를 설명합니다. *Cliché*와 *기성 표현*을 거부합니다.

### Joan Didion

차갑고 정밀한 voice. 짧은 문장. 사실의 무게가 느껴집니다.

### David Foster Wallace

긴 문장, 각주의 폭주, 자기 의식. 한 줄로 *DFW*임을 알 수 있습니다.

### James Baldwin

격정과 통찰. 정치적 무게를 *시*로 풀어냅니다.

### 김영하

차분한 voice + 부조리. 일상의 작은 디테일이 큰 의미와 연결됩니다.

### 정지돈

문체 실험의 voice. 짧은 문장 + 단절 + 회상.

이 작가들의 글은 *한 문단만 봐도* 누구의 글인지 알 수 있습니다. 그것이 *강한 voice*의 신호입니다.

## Voice를 잃는 함정

작가가 voice를 잃는 흔한 자리들:

### 1. 너무 많은 모방

자기를 *없애고* 다른 작가의 그림자가 됨.

### 2. 시장에 맞추기

대중에 잘 팔리는 voice로 변형 → 자기 색깔 사라짐.

### 3. 학술계 / 직장의 압박

분야의 *익명 voice*에 맞추다 자기 voice 잃음.

### 4. 두려움

자기 voice가 *어리석어 보일까봐* 가식적 voice로 도망감.

이 함정들을 *의식*하는 것이 voice를 지키는 첫 걸음입니다.

## Voice의 다듬기 — 실용

자기 voice를 *발견하고 강화*하는 실용적 방법:

```
1. 자기가 쓴 글을 *소리 내 읽기*
   - 어색한 자리가 jal립니다
   - 자기가 평소 말하지 않는 단어가 들립니다

2. 자기가 좋아하는 작가의 글과 비교
   - 그 작가의 voice를 분석
   - 자기 voice의 *고유점* 찾기

3. 같은 주제를 *세 가지 voice*로 써 보기
   - 어느 voice가 가장 자연스러운가
   - 그게 자기 voice의 후보

4. 매년 자기 글 회고
   - voice가 자라고 있는가
   - 더 강해지고 있는가, 약해지고 있는가
```

## 정리

- *Voice는 작가의 trademark*
- 단순한 어조가 아니라 *세상을 보는 방식 + 단어 + 리듬 + 가치관*
- 발견의 단계 — 모방 → 자기 패턴 → 평생의 다듬기
- *자기가 말하는 것처럼* 쓸 것
- 한 글에 *한 voice* — 일관성
- 분야에 따라 voice의 *자유도*가 다름
- George Orwell, Joan Didion, 김영하 등 모범
