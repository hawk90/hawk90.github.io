> **On Writing Well** Chapter 10 요약

## 이 챕터의 메시지

Part 1의 마지막 장. **작은 가르침들의 모음** — 각각 한 챕터를 만들 만큼은 아니지만 중요한.

## Verb를 강하게

**약한 동사 + 부사** 대신 **강한 동사**.

```
❌ He walked slowly.
✅ He ambled.

❌ She said angrily.
✅ She snapped.

❌ He looked closely.
✅ He scrutinized.
```

특히 **be 동사**는 약함. 가능하면 강한 동사.

## Adverb를 줄여라

특히 **-ly**.

```
❌ "He suddenly appeared."
✅ "He appeared."   (또는 더 강한 동사: "He materialized.")

❌ "She quietly walked."
✅ "She tiptoed."
```

대부분의 -ly 부사 — 잘라 낼 수 있다.

## Adjective도 줄여

```
❌ "She is a very intelligent and creative and resourceful person."
✅ "She is intelligent." (또는 가장 정확한 한 단어)
```

여러 형용사 → 한 단어로 더 정확.

## "Qualifiers"

```
"a bit", "a little", "sort of", "kind of", "rather", "very", 
"quite", "in a sense"
```

이런 qualifier가 들어가면 — 글이 약해진다.

```
❌ "He was a bit tired."
✅ "He was tired."

❌ "It was rather impressive."
✅ "It was impressive." (또는 더 강한 단어)
```

## Punctuation의 절제

쉼표 / 세미콜론 / 대시 — 모두 도구. 그러나 과용 X.

```
❌ "He, however, decided, against all advice, to go — to Seoul."
✅ "He decided to go to Seoul, against all advice."
```

문장이 punctuation에 의지하면 — 구조 자체에 문제.

## Mood / Tone

같은 글에 — 같은 mood.

```
❌ 진지 → 농담 → 진지 (어색)
✅ 한 mood 유지
```

8장 unity의 연장.

## "Don't dialogue"

명사를 동사로 만든 신조어 피하기.

```
❌ "Let's dialogue."
✅ "Let's talk."

❌ "We will incentivize..."
✅ "We will reward..."

❌ "It impacts the result."
✅ "It affects the result."   (또는 "It changes...")
```

신조어가 항상 나쁜 건 아니지만 — 평범한 단어가 보통 더 강함.

## Sexist Language

성별 중립적 표현.

```
❌ "Each writer must find his own voice."
✅ "Writers must find their own voices."
✅ "Each writer must find their own voice." (singular they)
```

오늘날 표준 — 가능한 한 중립.

## Trust the Reader

좋은 작가는 — 독자를 신뢰.

```
❌ "Tom said angrily, his face red with rage, 'Get out!'"
✅ "Tom said, 'Get out!'"   (말이 분노를 보여줌)
```

다 설명하지 마라. 독자가 추론하게.

## "Show, don't tell"

문학 작법의 격언. Nonfiction에도.

```
❌ "It was a beautiful sunset."
✅ "The sky turned crimson, then violet, then deep purple."
```

구체적 디테일이 — "beautiful"이라는 추상보다 훨씬 강.

## Paragraphs — 단락의 길이

```
짧은 단락 — 강조, 호흡, 신문 스타일
긴 단락 — 깊은 분석, 학술
```

다양화. 같은 길이 반복 — 단조.

```
긴 단락 후 — 짧은 단락 (강조).
```

## Sexist / Racist 언어 신중

좋은 글은 — 누구도 배제하지 않음.

- 모든 의사가 남자라는 가정 X
- 모든 간호사가 여자라는 가정 X
- 인종 / 종교에 대한 무의식적 편견 X

```
❌ "Every doctor has his patients."
✅ "Doctors have patients."
```

## 자기 비판

Zinsser의 비유 — **자기 글의 첫 번째 비평가**가 자기.

```
1. 글 쓴 후 — 24시간 대기
2. 다시 읽으며 — 차갑게 비판
3. 자른다 / 수정한다
4. 다시 24시간
```

거리를 두고 봐야 — 객관성. 자기 글에 너무 가까이 있으면 — 약점 안 보임.

## 도구

좋은 작가가 가지는 것.

- **사전** — 매일
- **Thesaurus** — 자주
- **문법책** — 가끔
- **자기가 좋아하는 작가의 책** — 영감
- **자기 노트** — 좋은 표현 / 단어 모음
