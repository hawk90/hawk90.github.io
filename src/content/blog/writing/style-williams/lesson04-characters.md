> **Style: Lessons in Clarity and Grace** Lesson 4 요약

## 이 챕터의 메시지

3장이 동사를 다뤘다면, 4장은 **주어**를 다룬다.

Williams의 두 번째 큰 원칙:

> **주어는 character여야 한다. 동사는 그 character의 행위여야 한다.**

이게 명확한 글의 두 기둥 — character(행위자)와 action(행위)이 일치.

## Character란

**Character** = 이야기의 행위자.

- 사람 / 사람의 집합 (Alice, the team, the company)
- 살아 있는 존재 (the dog)
- 의도를 가진 시스템 (the algorithm)

추상은 character가 아니다.

```
❌ Character: implementation, decision, analysis
✅ Character: Alice, team, dog, algorithm
```

## 좋은 문장 — Character이 주어

```
✅ Alice decided to go.
   주어 = Alice (character)
   동사 = decided (action)
```

읽기 쉽다. 누가 무엇을 하는지 즉시.

## 나쁜 문장 — 추상이 주어

```
❌ The decision was made to go.
❌ A determination to depart was reached.
❌ It was decided that going would be done.
```

주어가 추상 (decision, determination, it). character가 사라짐. **누가 결정했는지 모름**.

## 진단 — "누가 무엇을 하는가"

문장을 분석할 때 두 질문.

1. **누가** 행위자인가? (Who/what is the character?)
2. **그가 무엇을** 하는가? (What is the action?)

두 질문에 대한 답이 **주어와 동사**여야 한다.

```
"The investigation of the committee revealed widespread fraud."

질문 1: 누가 행위자? → committee
질문 2: 무엇을 하는가? → investigated (또는 revealed)

수정:
"The committee investigated and revealed widespread fraud."
```

## Character는 종종 숨어 있다

흔히 character가 전치사구나 소유격으로 숨음.

```
❌ "The settling of the dispute was by the mediator."
                                      ↑ 진짜 character는 by 뒤에 숨음

✅ "The mediator settled the dispute."
```

```
❌ "It was the team's recommendation that..."
                ↑
✅ "The team recommended that..."
```

```
❌ "There exists in many companies a strong commitment to..."
✅ "Many companies are strongly committed to..."
```

## 추상이 주어인 글의 함정

학술 / 비즈니스 글에 흔한 패턴.

```
❌ "Recent observations suggest that current strategies are 
   inadequate, and a reconsideration of approaches is required."

→ 누가 observe? 누가 strategies를 사용? 누가 reconsider?
```

3개의 추상 주어 (observations, strategies, reconsideration). character가 없다.

```
✅ "We recently observed that our strategies are inadequate, 
   and we need to reconsider our approach."
```

훨씬 명확. 3개의 character (we, we, we).

## "Old-New" 원칙의 첫 등장

Character와 관련된 부원칙.

> **주어는 독자가 이미 아는 것 (old information).**
> **새 정보는 동사 다음에 (new information).**

```
"Yesterday, the team made a decision. The decision will affect everyone."
            ↑                         ↑
       처음 등장 (new)            이전 문장의 연결 (old)
```

이게 자연스러운 흐름. 5장(Cohesion)에서 더 다룬다.

## "Real Subject" 찾기

문법적 주어와 의미적 주어가 다를 때.

```
"It is important that the team analyze the data."

문법적 주어: It (의미 없음)
의미적 주어: team
```

수정:

```
"The team must analyze the data."
   ↑
의미적 주어를 문법적 주어로
```

대부분의 **"It is ... that"** 구문은 — 의미적 주어를 숨긴다.

## "By" 구문 — Passive Voice의 character

수동태도 character를 보존할 수 있다 — by 구문으로.

```
"The decision was made by the committee."
```

그러나 보통은 능동태가 더 강하다.

```
"The committee made the decision."   ← 더 강함
```

## 글이 흐려지는 두 가지 동시 패턴

3장 + 4장을 합치면.

**나쁜 글의 공식**:
- Nominalization (동사 → 명사) + 추상 주어 (character가 아닌 것)

```
❌ "The implementation of the policy resulted in the 
   transformation of the organization."
   
   nominalization: implementation, transformation
   주어: implementation (추상)
   character가 사라짐
```

**좋은 글의 공식**:
- 동사 그대로 + character가 주어

```
✅ "When we implemented the policy, the organization transformed."
   
   동사: implemented, transformed
   주어: we, organization (character)
```

이 두 원칙을 동시에 적용 → 거의 모든 흐릿한 글이 명확해진다.

## 학술 글의 도전

학술 글에서는 종종 — character를 안 노출하고 싶다.

```
"실험 결과를 보고합니다" (격식 차림)
   ↓
"실험 결과는 ... 을 보여준다"
```

이걸 영어로:

```
❌ "We found that X."   (직설적, 비격식)
✅ "The results show that X."   (격식)
```

**Williams의 입장** — 직설적이 더 좋다. 학술이라도 명확함이 우선.

그러나 분야 관습이 있다면 — 적당히 절충.

## 한국 학생의 Characters 함정

한국어 학술 글에 *추상 주어*가 흔하다. 영어로 옮길 때 *그대로 추상 주어*로 가면 글이 약하다.

```
회피 (추상 주어):
"본 연구의 수행은 X의 해결을 시도하였다."
→ "The conduct of this study attempted the resolution of X."

선호 (character 주어):
"우리는 X를 해결하려 했다."
→ "We attempted to solve X."

회피:
"It is necessary that consideration be given to..."

선호:
"We must consider..."
```

## 자기 글 점검

```
□ 자기 글의 주어가 *추상 명사*인가, *character*인가?
□ "누가 무엇을 하는가?" 답할 수 있는가?
□ "It is ... that" 패턴 *과다*하지 않은가?
□ 한국어 추상 주어 직역 회피?
```

## 정리

- 주어는 **character** (행위자)여야 한다
- character는 추상이 아니라 **사람 / 사람의 집합 / 살아 있는 존재**
- 진단 — "누가 무엇을 하는가?" 두 질문
- character는 종종 전치사구 / 소유격으로 숨어 있다
- "It is ... that" 구문은 의미적 주어를 숨긴다
- **나쁜 글의 공식**: nominalization + 추상 주어
- **좋은 글의 공식**: 동사 + character 주어
