> **Style: Lessons in Clarity and Grace** Lesson 6 요약

## 이 챕터의 메시지

5장이 문장의 **시작**(old info)을 다뤘다면, 6장은 문장의 **끝**(new info, stress).

Williams의 원칙:

> **문장의 가장 강한 위치는 끝이다. 가장 중요한 정보를 끝에 둔다.**

## 끝의 힘

```
He saw the man with the gun yesterday.
                            ↑
                       yesterday가 강조됨 (끝)
                       → 시점이 중요

He saw the man with the gun.
                  ↑
              with the gun이 강조됨 (끝)
              → 무엇을 들고 있었나가 중요

He saw the man.
       ↑
   man이 강조됨
   → 누구를 봤나가 중요
```

같은 의미의 문장 — 어느 위치에 정보를 두느냐가 인상을 결정.

## "End Focus" 원칙

영어의 자연스러운 문장 구조는:

```
[Old info] [Verb] [New info / Important info]
   주어         동사          목적어 / 보어
```

새 정보 / 중요 정보가 보통 끝.

## 강조의 두 위치

문장에는 두 개의 강한 위치.

1. **첫 단어** — primary focus (시작)
2. **마지막 단어** — stress position (끝, 더 강함)

두 위치가 가장 기억에 남는다. 중간은 흐려진다.

## 약한 끝 — 흔한 실수

문장이 약한 단어로 끝나면 — 인상이 약해진다.

```
❌ "His abilities, in my opinion, are remarkable in many ways."
                                              ↑
                                  "many ways" — 약함

✅ "In my opinion, his abilities are remarkable."
                                          ↑
                                    "remarkable" — 강함
```

```
❌ "The committee's decision is final, however."
                                          ↑
                                "however" — 약함

✅ "However, the committee's decision is final."
                                            ↑
                                       "final" — 강함
```

문장 끝에 throwaway 단어 (however, though, anyway)를 두지 마라.

## Old-to-New 다시

5장의 Old-to-New 원칙이 emphasis와 직접 연관.

```
Sentence 1: ... [new info: X]
Sentence 2: [X (now old)] ... [new info: Y]
```

각 문장의 끝(stress)이 다음 문장의 시작(topic). 이게 흐름을 만든다.

## 능동태 vs 수동태와 강조

수동태가 자연스러울 때 — **강조 위치**가 결정.

```
"The committee made the decision."  ← decision이 강조

"The decision was made by the committee."  ← committee가 강조
```

둘 다 정당. 강조하려는 정보에 따라 선택.

> "Voice is not a moral issue. Use the voice that puts your emphasis where you want it."

수동태가 잘못된 게 아니다 — **잘못된 강조**가 잘못.

## 긴 / 무거운 부분은 끝

영어의 일반 패턴 — **무거운 정보가 뒤로**.

```
❌ "That John would lie about something so trivial surprised me."
   ↑
   긴 주어 (that-clause)

✅ "It surprised me that John would lie about something so trivial."
                                          ↑
                                  긴 부분이 뒤로
```

이게 영어의 자연스러운 흐름. 주어는 짧게, 무거운 부분은 끝에.

## 동사 다음의 짧은 / 긴 패턴

```
"He gave a book to her."
   ↑
   짧은 직접 목적어 (a book) + 긴 간접 목적어 (to her) — 길이 비슷, OK

"He gave her a book about ancient Egyptian mythology."
   ↑
   짧은 간접 목적어 (her) + 긴 직접 목적어 — 자연스러움

"He gave a book about ancient Egyptian mythology to her."
   ↑
   긴 + 짧은 → 어색
```

영어는 **짧은 것 먼저, 긴 것 뒤**가 자연스럽다.

## "It-cleft" — 강조의 도구

```
정상: "John broke the window."

It-cleft:
"It was John who broke the window."  ← John 강조
"It was the window that John broke."  ← window 강조
```

평이한 문장에서 특정 요소를 강조할 때.

## "What-cleft"

```
"What I want is freedom."
"What surprised me was his honesty."
```

비슷한 강조 효과. 학술 / 격식 글에 자주.

## 강조의 도구들

| 도구 | 예 |
|---|---|
| End position | 중요 정보를 끝에 |
| Cleft (It / What) | 특정 요소 분리 강조 |
| Voice 선택 | 능동 / 수동의 강조 차이 |
| Length pattern | 짧은 + 긴 |
| Inversion | "Not until X did Y" |
| Italics / 강조 | 단어 자체 강조 (제한적 사용) |

## 강조의 과용

```
"This is VERY important. The MOST critical issue is X. 
We MUST address it IMMEDIATELY."
```

모든 단어를 강조하면 — 어느 것도 강조 안 됨. 강조는 **선택적**.

좋은 글은 한 단락에 한 강조점 정도. 그 한 점이 강하게 와닿는다.

## 한국어의 강조

한국어는 *동사가 문장 끝*에 온다. *자연 강조 위치*가 영어와 다르다.

```
한국어 (동사 끝):
"나는 어제 그 책을 *읽었다*."
→ "읽었다"가 끝, 동사 강조

영어 (목적어 끝):
"I read the book yesterday."
→ "yesterday" 끝, 시간 강조

도치로 강조:
"Yesterday, I read the book." (시간 도입)
"The book, I read yesterday." (목적어 강조)
```

### 한국어의 *만 / 도 / 까지*

```
조사로 명시적 강조:
- "그*만* 왔다" (only)
- "그*도* 왔다" (also)
- "끝*까지* 갔다" (even)
- "*그가* 왔다" (subject 강조)

영어 등가:
- "Only he came."
- "He also came."
- "He even went to the end."
```

## 자기 글 점검

```
□ 문장 *끝*에 강조하고 싶은 단어?
□ 약한 단어로 끝나지 않았는가?
□ Old-to-New 패턴?
□ Cleft 구문 (It is X that)이 *드물게* 효과적인가?
□ 강조가 *선택적*인가, *남발*인가?
```

## 정리

- 문장의 **가장 강한 위치는 끝** (stress position)
- 끝에 **새 정보 / 중요 정보**
- Old-to-New 원칙 — 시작은 old, 끝은 new
- 약한 단어 (however, though)로 끝나지 마라
- 수동태는 강조 도구 — 잘못된 게 아니라 다른 강조
- **짧은 + 긴** 패턴이 자연스럽다
- Cleft 구문 (It is X who...)으로 명시적 강조
- 강조는 **선택적** — 다 강조면 아무것도 안 강조
