> **Style: Lessons in Clarity and Grace** Lesson 5 요약

## 이 챕터의 메시지

3-4장은 개별 문장의 명확함. 5장은 **문장과 문장 사이**의 매끄러운 연결.

Williams는 두 개념을 구분.

- **Cohesion** — 인접한 문장이 연결되는 방식 (어떻게)
- **Coherence** — 전체 단락이 한 화제로 묶이는 방식 (무엇을)

## Cohesion — Old-to-New 원칙

가장 중요한 원칙.

> **각 문장은 익숙한 (old) 정보로 시작하고, 새 (new) 정보로 끝난다.**

```
Sentence 1: A → B (new)
Sentence 2: B → C (B는 이제 익숙)
Sentence 3: C → D
```

문장 끝의 새 정보가 다음 문장의 시작이 된다. 사슬처럼.

## 좋은 예 — Cohesion

```
Climate change affects polar ice caps. The melting of ice caps 
raises sea levels. Rising sea levels threaten coastal cities. 
These cities house most of the world's population.
```

각 문장의 끝이 다음 문장의 시작.

```
폴라 빙하 → 녹음 → 해수면 → 연안 도시 → 인구
```

자연스럽게 흘러간다.

## 나쁜 예 — Cohesion 없음

```
Climate change affects polar ice caps. Sea levels are rising. 
Coastal cities are threatened. Most of the world's population 
lives in coastal areas.
```

같은 주제. 그러나 문장 사이가 끊긴다. 독자가 매번 새로 시작해야.

## Coherence — 한 화제로 묶이기

Cohesion이 문장 사이의 연결이라면, **Coherence**는 단락 전체의 통일성.

```
좋은 단락: 모든 문장이 한 화제를 다룸
나쁜 단락: 화제가 여러 번 바뀜
```

진단 — **단락의 주어를 모아 본다**. 비슷하다면 coherent.

```
Sentence 1 subject: Climate change
Sentence 2 subject: Polar ice
Sentence 3 subject: Sea levels  
Sentence 4 subject: Coastal cities

→ 모두 환경 / 기후 관련, coherent
```

```
Sentence 1 subject: Climate change
Sentence 2 subject: The economy
Sentence 3 subject: Cooking
Sentence 4 subject: Politics

→ 흩어짐, incoherent
```

## "Topic" — 단락의 주어

각 문장의 주어가 **단락의 화제 (topic)**를 형성.

```
좋은 단락:
- 주어 1: AI
- 주어 2: AI systems
- 주어 3: These systems
- 주어 4: They

→ 화제: AI
```

각 문장이 같은 화제를 다양한 방식으로 가리킨다. **대명사 / 반복 / 동의어**로 cohesion 유지.

## Old-to-New 위반의 예

```
"Several factors contribute to climate change. Industrial 
emissions are one factor. Deforestation is another factor. 
Methane from livestock is a third factor."
```

각 문장이 "factor"로 끝난다. 끝이 강조 위치이지만 — 같은 단어 반복은 redundant.

수정:

```
"Several factors contribute to climate change. The most 
significant is industrial emissions. Deforestation makes 
a substantial contribution as well. Even methane from 
livestock plays a role."
```

각 문장이 다른 정보로 끝난다. 화제 — climate change의 원인 — 은 유지.

## 단락 시작의 "Topic Sentence"

전통적인 글쓰기 가르침 — **각 단락의 첫 문장이 topic sentence**.

```
Topic sentence: 단락의 주제 선언
Supporting sentences: 그 주제를 뒷받침
Concluding sentence: 마무리
```

좋은 가이드라인. 그러나 항상 적용 X — 좋은 작가는 가끔 topic sentence를 중간 / 끝에 두기도.

## "Transition" — 문장 사이 다리

문장과 문장 사이를 잇는 단어 / 구.

```
Therefore, however, moreover, in addition, on the other hand, 
similarly, in contrast, for example, ...
```

이런 transition을 적절히 쓰면 — 흐름이 명확.

**다만 과용 금지**. 매 문장마다 transition이면 글이 무거워진다.

## 단락 구조의 패턴

흔한 패턴들.

**1. General → Specific**

```
일반 진술 → 구체 예
```

**2. Cause → Effect**

```
원인 → 결과
```

**3. Problem → Solution**

```
문제 제시 → 해결책
```

**4. Compare → Contrast**

```
유사 → 차이
```

각 패턴이 자연스러운 cohesion을 만든다.

## 화제 변경 — 새 단락으로

화제가 바뀐다면 — 새 단락 시작.

```
[단락 1: 문제 진단]
The system has X problem. The root cause is Y. ...

[단락 2: 해결책]   ← 새 단락
We propose Z as solution. This approach ...
```

이게 단락 분할의 기준. 단순히 "너무 길어서"가 아니라 **화제 전환**이 기준.

## 진단 — 자기 글의 Cohesion

자기 글을 점검할 때.

1. **각 문장의 주어를 표시한다.**
2. 주어들이 비슷한 그룹인가? (coherent) → OK
3. 각 문장의 끝과 다음 문장의 시작이 연결되는가? (cohesive) → OK
4. 둘 다 안 되면 — 재작성

이게 단순한 절차이지만 — 단락의 흐름을 극적으로 개선한다.

## 한국어의 Cohesion

한국어에는 *조사*로 cohesion이 명시된다.

```
한국어 cohesion 도구:
- *은/는* (옛 정보, 주제)
- *이/가* (새 정보, 주제 도입)
- *이는, 이러한, 그것은* (대용어)
- *한편, 또한, 그러나* (연결어)

영어 cohesion 도구:
- Old-to-New (어순으로 표현)
- this, these, it, they
- However, Moreover, In addition
- 명사 반복 / 동의어

→ 한국어가 *조사로 명시*하는 것을
   영어는 *어순으로 표현*한다
```

## 자기 글 점검

```
□ 단락의 주어들이 *비슷한 그룹*인가? (coherent)
□ 문장 끝과 다음 문장 시작이 *연결*되는가? (cohesive)
□ Topic sentence가 단락 *시작*에 있는가?
□ Transition (However, Moreover) *과용* 없는가?
□ 화제 전환에 *새 단락*?
```

## 정리

- **Cohesion** — 문장 사이의 연결, Old-to-New 원칙
- **Coherence** — 단락 전체의 한 화제
- 진단 — **단락의 주어들을 모아 본다**
- Topic sentence — 단락 시작에 화제 선언
- Transition — 적절히, 과용 금지
- 화제 전환 = **새 단락**
- 주어들이 비슷 + 끝과 시작이 연결 → 자연스러운 흐름
