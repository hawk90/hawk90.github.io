> **Style: Lessons in Clarity and Grace** Lesson 2 요약

## 이 챕터의 메시지

영어 문법 규칙을 두고 사람들이 끝없이 다툰다.

- "don't split infinitives"
- "don't end a sentence with a preposition"
- "don't start a sentence with 'and' or 'but'"
- "don't use 'I' in formal writing"

이런 규칙이 진짜 규칙인가? Williams의 답 — **대부분 아니다**.

## 두 종류의 규칙

Williams는 규칙을 두 가지로 나눈다.

**1. Real Rules — 진짜 규칙**

영어 모국어 화자 모두가 자연스럽게 따른다. 어기면 즉시 어색.

```
✅ The cat sat on the mat.
❌ The cat the on sat mat.  (어순 파괴)

✅ I am going.
❌ I are going.              (수 일치 파괴)
```

이런 규칙은 가르치지 않는다 — 자연스럽게 학습된다.

**2. Folklore — 미신적 규칙**

누군가가 만들고 가르쳤지만, 좋은 작가들이 무시한다.

```
"Don't split infinitives"
→ 그러나 "To boldly go where no man has gone before" (Star Trek)

"Don't end with preposition"
→ Winston Churchill: "Up with which I will not put."
```

이런 규칙은 종종 18-19세기 라틴어 문법학자들이 영어에 강제로 적용한 것. **영어와 무관**.

## Folklore의 출처

대부분의 미신적 규칙은 다음에서 왔다.

- **라틴어 모방** — "infinitive를 쪼개지 마라"는 라틴어 infinitive가 한 단어라서. 영어와 무관.
- **18세기 문법학자의 강박** — Bishop Robert Lowth 등이 영어를 "정리"하려 함.
- **학교 선생의 가르침** — 한 세대가 다음 세대로 전수.

이런 규칙을 어겨도 — 좋은 작가들의 글에 자주 등장. 따라서 어겨도 OK.

## 실제 규칙들

진짜 어겨선 안 되는 규칙.

- 주어-동사 일치 (subject-verb agreement)
- 대명사 격 (who vs whom 같은 예외 빼고)
- 시제 일치
- 관사 (a / the) 사용
- 어순

이건 어기면 모국어 화자에게 즉시 어색.

## "Optional" 규칙

엄격하지 않지만 격식 있는 글에서 지키는 것이 좋은 규칙.

- "between you and me" (not "between you and I")
- "lay" vs "lie"
- "fewer" vs "less"
- "who" vs "whom" (격식 있는 글)

이걸 어기면 — 모국어 화자가 알아채지만 의사소통은 됨. 격식 차이.

## 미신적 규칙

좋은 작가들이 일상적으로 어김.

- **Split infinitive** — "to boldly go" OK
- **Preposition at end** — "the man I'm looking for" OK
- **Start with conjunction** — "And then..." OK
- **Sentence fragments** — 효과적이라면 OK
- **"I" in formal writing** — 의견 표현 시 OK
- **"They" as singular** — 성중립 대명사로 OK

## 왜 미신적 규칙이 살아남나

Williams의 통찰 — **위계의 도구**.

```
교육받은 사람 ← "I know the rules"
나머지 사람들  ← "I don't know"

→ 규칙은 신분의 표지
```

미신적 규칙을 강요하는 사람들은 종종 — 자기가 "교육받은 사람"이라는 신호. 진짜 의사소통이 아니라 **자기 권위 확립**.

> "Rules are sometimes used as a club to beat down rivals."

## 글쓰기 결정

따라서 글을 쓸 때 — **어떤 규칙을 따를 것인가**가 본인의 선택.

**격식 있는 글**:
- 진짜 규칙 + 옵션 규칙 + 일부 미신
- 안전한 선택

**캐주얼한 글**:
- 진짜 규칙만
- 미신 무시 OK

**창작 글**:
- 의도된 위반은 효과
- 모든 규칙을 알면서 어김

## 독자 인식

규칙을 어기면 — 누군가는 알아챈다. 그게 문제가 되는가는 독자에 따라.

```
독자가 미신적 규칙을 안다 → 어기면 "교양 없다"고 판단할 수도
독자가 모른다              → 차이 없음
```

전문가 독자에게는 — 미신적 규칙도 적당히 지키는 게 안전.

## "Whom"의 운명

대표적인 예 — "whom".

- **격식 있는 글**: 여전히 사용
- **현대 구어**: 거의 사라짐

```
Whom did you invite? (격식)
Who did you invite? (일상)
```

미래 어느 시점에는 — whom이 완전히 사라질 가능성. 언어는 변한다.

## 정리 — 규칙에 대한 실용적 자세

Williams의 권장.

1. **진짜 규칙은 안다** — 모국어 화자라면 자연스러움
2. **옵션 규칙은 격식에 따라** — 학술 / 비즈니스에서는 지킴
3. **미신적 규칙은 무시 OK** — 다만 독자에 따라
4. **의도된 위반은 효과** — 모든 규칙을 알면서

규칙을 두려워하지 마라. 그러나 **알고 어겨라**.

## 한국 학생의 미신적 규칙

영어 학습자는 *미신적 규칙*을 *진짜 규칙*보다 더 엄격히 지키는 경향이 있다. 두려움 때문.

```
한국 학생이 자주 지키는 미신:
1. *Never split infinitive* — 사실 분리 가능
2. *Never end with preposition* — 사실 끝 OK
3. *Never start with And/But* — 사실 가능
4. *Never use I in academic* — 분야에 따라 다름
5. *Always use whom* — 회화는 who OK

한국 학생이 자주 어기는 진짜 규칙:
1. 3인칭 단수 -s 누락
2. 관사 (a/the) 누락
3. Subject-verb agreement
4. Comma splice
5. Sentence fragment

→ 진짜 규칙 / 미신 *반대로 알고 있는 경우*가 흔하다
```

## 자기 글 점검

```
□ 미신적 규칙에 *과도하게* 매여 있지 않은가?
□ 진짜 규칙을 *실제로 지키고 있는가*?
□ 자기 분야 / 매체의 *관습*에 맞는 결정인가?
□ 위반이 *의도*인가 *실수*인가?
```

## 정리

- 문법 규칙은 두 종류 — **Real** vs **Folklore**
- 진짜 규칙은 자연스러움 — 모국어 화자가 안 어김
- 미신적 규칙은 18-19세기 학자들이 만든 것 — 좋은 작가들이 어김
- **Split infinitive**, **preposition at end** 등 — 미신
- 규칙은 종종 **위계의 도구** — 의사소통이 아니라 권위 확립
- 알고 어겨라 — 의도된 위반은 효과
- 한국 학생: 미신은 과잉 준수, 진짜 규칙은 자주 어김
