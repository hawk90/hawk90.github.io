> Avoid foreign languages.

가끔은 외국어가 *편리하거나 필요*합니다. 그러나 *과시를 위해* 외국어를 흩뿌리는 작가도 있습니다.

## 외국어 사용의 두 자리

```
정당한 사용:
1. 정확히 대응하는 영어 단어가 *없을 때*
   - 일본어: tsunami, sushi, karaoke
   - 프랑스어: cliché, déjà vu, faux pas
   - 독일어: schadenfreude, zeitgeist

2. 그 외국어의 *원전*을 인용할 때
   - 시 / 문학 인용
   - 학술 출처 (원문 + 번역)

3. 특정 *분야의 표준 용어*
   - 의학 라틴어: in vivo, in vitro
   - 음악 이탈리아어: allegro, forte
   - 법률 라틴어: pro bono, prima facie
```

## 과시를 위한 외국어

부적절한 사용은 *작가의 식견을 과시*하려는 경우입니다.

```
회피 (과시):
"Their relationship was, shall we say, a folie à deux."
"We must, after all, suivre les règles."
"This was, in a sense, a tour de force, n'est-ce pas?"
```

이런 문장은 *독자를 배제*합니다. 외국어를 모르는 독자는 *무시당했다는 느낌*을 받습니다.

## "Per Se", "Et Al." 같은 라틴어

학술 / 격식 글에서 라틴어 약자가 자주 등장합니다.

```
e.g. — exempli gratia (for example)
i.e. — id est (that is)
etc. — et cetera (and so on)
viz. — videlicet (namely)
cf. — confer (compare)
et al. — et alii (and others)
ibid. — ibidem (in the same place)
op. cit. — opere citato (in the work cited)
```

이 약자들은 *학술 / 격식 글*에서는 표준입니다. 일반 글에서는 *영어로 풀어 쓰는 것*이 더 친근합니다.

```
회피 (일반 글):
"There are many examples, e.g. the case of ..."

선호 (일반 글):
"There are many examples, for example, the case of ..."
또는: "For instance, the case of ..."
```

## 한국어 글의 외국어

한국어 글에 영어가 흩뿌려지는 흔한 패턴.

```
회피 (영어 과시):
"이번 프로젝트는 game changer가 될 거예요. 우리가 paradigm
shift를 만들면 disruptive innovation을 leverage할 수 있죠."

선호 (한국어로):
"이번 프로젝트는 큰 변화를 만들 수 있어요. 새 시각으로 기존
질서를 흔들면 우리만의 기회를 잡을 수 있어요."
```

이런 *비즈니스 영어 과시*가 한국 글에서 흔합니다. *명확한 한국어*로 충분히 표현 가능한 경우가 대부분입니다.

### 정당한 한국 글의 외국어

```
1. 정확한 영어 용어
   - "OOP는 1960년대에 ..." (인용 대상이 영어)
   - "stress test (스트레스 테스트)는 ..." (한국어 번역과 함께)

2. 분야 표준 용어
   - "AI (인공지능)", "ML (머신러닝)"
   - 의학, IT, 학술 등

3. 고유명사 / 인용
   - 책 제목, 사람 이름
   - 원전 인용 (필요 시)
```

## "Faux Pas" 같은 표현 — 영어가 받아들임

영어가 *외국어를 받아들여* 일상화한 표현들.

```
받아들임 (영어로 자연):
cliché, déjà vu, faux pas, tour de force
karaoke, tsunami, sushi
schadenfreude, zeitgeist, kindergarten
```

이런 표현은 *영어처럼* 사용됩니다. *이탤릭이나 인용부호* 없이.

## 외국어를 *어떻게* 표시할지

외국어를 표시하는 *형식*:

```
1. 이탤릭체:
   "Their joie de vivre was infectious."

2. 인용부호:
   "The 'savoir faire' she displayed..."

3. 표시 없이 (영어처럼 받아들여진 경우):
   "He had a déjà vu."
   "He had a tsunami of emotions."
```

자기 *글의 스타일*에 일관성을 둡니다.

## 한국어 글의 외래어 표시

한국 글에서도 비슷한 결정이 있습니다.

```
외래어 표기:
- 영어로 그대로 (이탤릭 또는 인용부호)
- 한글 표기 ("스트레스 테스트")
- 한자 표기 (학술적 자리)

표준 한글 표기 (국립국어원 권장):
- 컴퓨터 (X 콤퓨터)
- 카페 (X 까페)
- 디지털 (X 디지탈)
```

## 적용

- [ ] 자기 글의 외국어가 *정당한 자리*인가?
- [ ] 영어 / 한국어로 *대체 가능*한가?
- [ ] 외국어 표시 *형식*이 일관되는가?
- [ ] 학술 라틴어가 *분야 표준*인가?
- [ ] 한국 글의 영어 과시가 없는가?
