> Work from a suitable design.

무엇을 쓰기 전에 자기 작업의 *성격과 범위*를 가늠하고, 적합한 설계를 갖춰 두세요. 이 reminder는 Chapter II의 Rule 12와 짝을 이룹니다.

## 설계의 가치

설계는 가장 단순한 구조도 *조명*합니다.

```
텐트 한 채 — 4 모서리 + 1 입구의 단순 설계
대성당 — 수백 페이지의 청사진

규모는 다르지만 — *설계 없이는 둘 다 무너집니다.*
```

글도 마찬가지입니다. 한 단락의 작은 글이든 책 한 권이든 *설계*가 필요합니다.

## 설계의 정도 — 글의 종류별

| 글 종류 | 설계 깊이 |
|---------|-----------|
| 세탁물 목록 | 설계 거의 없음 (옷 더미 옆에서 적어 가며) |
| 짧은 메모 | 가벼운 설계 (요점 한두 개) |
| 이메일 | 헤더, 본문, 마무리 |
| 블로그 글 | 도입, 본문, 결론 |
| 에세이 | 큰 흐름 + 단락별 주제 |
| 연구 논문 | IMRD 구조 + 각 섹션의 sub-구조 |
| 책 | 챕터 outline + 각 챕터의 sub-구조 + 시리즈 흐름 |

설계의 *복잡도*는 글의 *규모와 야망*에 비례합니다.

## 전기를 쓴다면

세탁물 목록은 즉흥으로 가능하지만, *전기를 쓴다면* 최소한 *대강의 도면*이 필요합니다.

좋은 전기의 설계:
- 시작점 (출생 / 첫 사건)
- 전체 구조 (시대별 / 주제별)
- 핵심 사건의 위치
- 다른 인물과의 관계
- 마무리 (사망 / 마지막 사건 / 유산)

도면 없이 사실을 *쪼아 들어가면* — 결국 숲을 잃고 빠져나오지 못합니다.

## 설계의 형식

설계는 *문서로 적힐 필요는 없습니다*. 머릿속의 *큰 그림*이면 충분한 경우도 있습니다. 다만 글이 길고 복잡할수록 *물리적 outline*이 도움됩니다.

| 규모 | 설계 형식 |
|------|-----------|
| 한 페이지 글 | 머릿속 outline |
| 3-5 페이지 글 | 한 종이에 bullet point |
| 한 챕터 | 더 상세한 outline |
| 책 한 권 | 챕터별 outline + 각 챕터 안의 sub-outline |

## 설계와 자유 — 균형

설계가 *너무 단단하면* 글이 *기계적*이 됩니다.

**너무 단단한 설계:**
- 각 단락의 단어 수까지 미리 결정
- 모든 transition 미리 작성
- 변경 가능성 없음
- 결과: 글이 기계적, 발견의 즐거움 없음

**너무 느슨한 설계:**
- 어디로 갈지 모름
- 같은 주제를 반복
- 결론에서 헤맴
- 결과: 글이 산만, 독자 잃음

**적절한 설계:**
- 큰 흐름은 정함
- 세부는 자유롭게
- 발견에 따라 조정 가능

## 감정의 글 — 예외

다만 *감정과 충동*이 설계보다 더 강력해야 할 때가 있습니다.

- 자비를 호소하는 편지
- 사랑을 고백하는 편지
- 사후 추모 글
- 일기, 자유로운 회고

이런 글에서는 감정이 *흐트러진 상태로 남는 것*이 오히려 좋습니다. 너무 잘 짜인 *사랑 편지*는 *진심이 의심됩니다*.

## 글의 종류와 설계의 깊이

| 유형 | 설계의 깊이 |
|------|-------------|
| 일기 | 없음 / 가벼움 |
| 이메일 | 가벼움 |
| 블로그 글 | 중간 |
| 에세이 | 중간 ~ 깊음 |
| 연구 논문 | 깊음 (IMRD) |
| 책 | 매우 깊음 |

자기 글의 *유형*을 정한 뒤 그에 맞는 설계를 합니다.

## 적용 — 자기 글의 설계

- [ ] 이 글의 *유형*은 무엇인가
- [ ] 적절한 설계의 *깊이*는?
- [ ] 큰 흐름 -- 도입 / 본문 / 마무리는 어떻게?
- [ ] 본문 안의 *3-5 단락 주제*는?
- [ ] 마무리에서 *무엇을 남기고 싶은가*?

이 다섯 질문에 답한 뒤 쓰기 시작하면 글이 흔들리지 않습니다.
