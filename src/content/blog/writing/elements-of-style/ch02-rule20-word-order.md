문장에서 어순은 의미를 결정합니다. 관련 있는 단어들은 가까이, 관련 없는 단어들은 멀리.

```
모호: He noticed a large stain in the rug that was right in the center.
        (얼룩이 가운데? 양탄자가 가운데?)
명확: He noticed a large stain right in the center of the rug.
```

## 우스운 어순

```
우스움: New York's first commercial human-sperm bank opened Friday with
        semen samples from eighteen men frozen in a stainless steel tank.
바름:   ... opened Friday. The samples were then frozen in a stainless steel tank.
```

## 주어와 동사

주어와 동사 사이를 긴 구나 절로 끊지 마세요. 흐름을 끊습니다.

```
끊어짐: Toni Morrison, in Beloved, writes about characters who have escaped from slavery but are haunted by its heritage.
흐름:   In Beloved, Toni Morrison writes about characters ...
```

## 관계대명사

관계대명사는 선행사 바로 뒤에 옵니다.

```
모호: A proposal to amend the Sherman Act, which has been variously judged
       (제안이 평가됨? 법이 평가됨?)
바름: A proposal, which has been variously judged, to amend the Sherman Act
```

## 수식어

수식어는 수식 대상에 붙어야 합니다.

```
잘못: All the members were not present.   (전원 없음? 일부 없음?)
바름: Not all the members were present.
```

```
잘못: She only found two mistakes.
바름: She found only two mistakes.
```

## 핵심

- 관련 있는 단어는 가까이
- 주어와 동사 사이를 길게 끊지 말 것
- 관계대명사는 선행사 바로 뒤에
- 수식어는 수식 대상에 붙이기
