접속사 없이 쉼표만 쓰는 것은 *comma splice*라 불리는 가장 흔한 실수입니다. 두 독립절은 세미콜론으로 잇거나 따로 두 문장으로 나눠 씁니다.

```
Mary Shelley's works are entertaining; they are full of engaging ideas.
It is nearly half past five; we cannot reach town before dark.
```

같은 의미를 마침표로 분리해도 됩니다. 의미상 친밀하면 세미콜론, 그렇지 않으면 마침표입니다.

```
Mary Shelley's works are entertaining. They are full of engaging ideas.
```

## 접속사가 있는 경우

접속사가 들어가면 쉼표로 충분합니다(Rule 4).

```
Mary Shelley's works are entertaining, for they are full of engaging ideas.
```

## 부사가 이끄는 경우

두 번째 절을 *accordingly*, *besides*, *then*, *therefore*, *thus* 같은 부사가 이끌면, 접속사가 아니므로 세미콜론이 필요합니다.

```
I had never been in the place before; besides, it was dark as a tomb.
```

## 예외

매우 짧고 같은 형태의 절이거나, 가벼운 회화 톤일 때는 쉼표가 어울립니다.

```
Man proposes, God disposes.
The gates swung apart, the bridge fell, the portcullis was drawn up.
Here today, gone tomorrow.
```

## 핵심

- 두 독립절을 쉼표만으로 잇지 말 것 (comma splice)
- 세미콜론 또는 마침표로 분리
- 부사(accordingly, besides, then 등)가 이끄는 절도 세미콜론 필요
