> **The Elements of Style** Chapter 4 요약

## 이 챕터의 메시지

가장 긴 챕터. 약 50쌍 / 묶음의 자주 잘못 쓰이는 단어. 사전처럼 알파벳 순.

전부 다 외울 필요 없다. **자주 등장하는 함정 패턴**을 알면 충분. 가장 중요한 것들을 추린다.

## affect vs effect

가장 흔히 헷갈리는 쌍.

- **affect** (동사) — 영향을 주다
- **effect** (명사) — 영향, 결과
- **effect** (동사) — (드물게) 가져오다, 이루다

```
The weather affects my mood.       (영향을 준다)
The effect of the weather is...    (영향, 명사)
He effected a change in policy.    (이루었다, 드문 동사 용법)
```

## fewer vs less

- **fewer** — 셀 수 있는 명사
- **less** — 셀 수 없는 명사

```
Fewer apples (셀 수 있음)
Less water (셀 수 없음)
Fewer people, less crowd
```

슈퍼마켓의 "12 items or less" — 문법적으로 잘못. "12 items or fewer"가 옳다.

## that vs which

- **that** — restrictive (필수) 절. 쉼표 없음.
- **which** — non-restrictive (부가) 절. 쉼표 있음.

```
The book that I bought yesterday is on the table.
(여러 책 중 어제 산 책을 지칭 — 필수)

The book, which I bought yesterday, is on the table.
(책이 하나, 추가 정보 — 부가)
```

영국식 영어는 둘을 덜 구분. 미국식은 엄격.

## who vs whom

- **who** — 주격
- **whom** — 목적격

```
Who is going? (주격)
Whom did you invite? (목적격)
The man whom I saw. (목적격)
```

현대 영어에서 whom의 사용이 줄어듦. 격식 있는 글에서는 여전히 구별.

## lay vs lie

가장 헷갈리는 쌍.

- **lay** — "놓다" (타동사, 목적어 필요). 과거 = laid.
- **lie** — "눕다" (자동사). 과거 = lay (!)

```
I lay the book on the table.    (현재 — 놓다)
I laid the book yesterday.       (과거 — 놓았다)

I lie on the bed.                (현재 — 눕다)
I lay on the bed yesterday.      (과거 — 누웠다, 헷갈림!)
I have lain there for hours.     (완료)
```

"lie의 과거 = lay" 때문에 모두가 헷갈린다. 외워라.

## comprise vs compose

- **comprise** — "구성하다" (전체가 부분을). 수동태 X.
- **compose** — "구성하다" (부분이 전체를).

```
✅ The team comprises ten players.
✅ Ten players compose the team.
✅ The team is composed of ten players.

❌ The team is comprised of ten players.   (흔한 실수)
```

## farther vs further

- **farther** — 물리적 거리
- **further** — 비유적, 추상적 거리

```
He walked farther than I did.   (물리)
We need to discuss this further. (추상)
```

현대 영어에서 점차 혼용되지만, 격식에서는 구별.

## continual vs continuous

- **continual** — 자주 반복되는, 끊겼다 이어지는
- **continuous** — 끊김 없는, 진정으로 연속

```
The continual complaints annoyed him.   (반복되는)
A continuous stream of water.            (끊김 없는)
```

## different from / than / to

- **different from** — 표준 (미국)
- **different than** — 미국에서 점차 사용
- **different to** — 영국식

Strunk는 **different from**을 권장.

```
✅ A is different from B.
미국: ⚠️ A is different than B. (덜 격식)
영국: ✅ A is different to B.
```

## i.e. vs e.g.

라틴어 약어.

- **i.e.** — id est = "that is" (즉)
- **e.g.** — exempli gratia = "for example" (예)

```
Korean food, e.g., kimchi, bulgogi    (예시)
Korean food, i.e., food from Korea    (즉, 정의)
```

## "literally"의 오용

- **literally** — 글자 그대로 (강조 X)

```
✅ He was literally on fire (정말 불에 타고 있었다)
❌ He was literally dying of laughter (그냥 비유)
```

비유에는 figuratively 또는 그냥 강한 동사. 현대 영어에서 강조 용법이 점차 받아들여지지만 — 격식에서는 피함.

## "Hopefully"의 오용

- **Hopefully** = "in a hopeful manner" (희망에 차서)

```
✅ "I will return," he said hopefully.   (희망에 차서)

❌ Hopefully, it will rain tomorrow.      (X. Strunk 시점)
✅ I hope it will rain tomorrow.
```

현대 영어 일반어법에서는 "I hope" 의미로 받아들여짐. 그러나 학술/격식에서는 피함.

## "Like" vs "As"

- **like** — 전치사 ("처럼", 명사 앞)
- **as** — 접속사 ("것처럼", 절 앞)

```
✅ He looks like his father.        (전치사 + 명사)
✅ He acts as his father did.        (접속사 + 절)

❌ He acts like his father did.      (구어에선 흔하지만 격식 X)
```

## "Unique"는 강도 부사와 함께 안 씀

unique = 유일 = 그 자체로 절대값. "more unique" 같은 표현 안 함.

```
❌ Very unique
❌ More unique
❌ Most unique
✅ Unique
```

## fewer / less, that / which 같은 핵심 쌍

위의 50쌍 중 가장 자주 만나는 함정.

| 단어 쌍 | 핵심 차이 |
|---|---|
| affect / effect | 동사 / 명사 |
| fewer / less | 셀 수 있음 / 없음 |
| that / which | restrictive / non-restrictive |
| who / whom | 주격 / 목적격 |
| lay / lie | 타동 / 자동 |
| compose / comprise | 부분 / 전체 |
| farther / further | 물리 / 추상 |
| i.e. / e.g. | 즉 / 예 |
| like / as | 전치사 / 접속사 |

이 9쌍만 정확히 써도 영문 글의 격이 올라간다.

## 정리

- 50쌍 / 묶음의 자주 오용되는 단어
- 가장 자주 함정 — affect/effect, fewer/less, that/which, lay/lie
- 라틴어 약어 — i.e.(즉) / e.g.(예) 정확히
- "unique"는 절대값 — 강도 부사와 같이 안 씀
- **사전적 정확성**이 글의 신뢰를 결정
