자음으로 끝나든 모음으로 끝나든 's를 붙입니다.

```
Charles's friend
Burns's poems
the witch's malice
```

예외는 고대 고유명사뿐입니다.

```
Jesus' disciples
Achilles' heel
for conscience' sake
```

대명사 소유격(hers, its, theirs, yours, ours)에는 어포스트로피를 붙이지 않습니다. 부정대명사는 붙입니다.

```
one's rights
somebody else's umbrella
```

가장 흔한 실수는 *its*와 *it's*의 혼동입니다. `it's = it is`, `its = 소유격`.

```
The dog wagged its tail.
It's a wise dog that scratches its own fleas.
```

## 핵심

- 단수 소유격에는 's
- 고대 고유명사만 예외 (Jesus', Achilles')
- it's = it is, its = 소유격
