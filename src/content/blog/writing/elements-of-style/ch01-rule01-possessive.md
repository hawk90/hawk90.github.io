> Form the possessive singular of nouns by adding 's.

Strunk가 책의 가장 첫 자리에 두는 규칙. 단순해 보이지만 *영어 글쓰기에서 가장 자주 틀리는* 자리입니다.

## 기본 규칙

자음으로 끝나든 모음으로 끝나든 *'s*를 붙입니다.

```
Charles's friend
Burns's poems
the witch's malice
the bus's wheels
James's car
```

마지막 글자가 *s*로 끝나도 *'s*를 붙입니다. 이게 핵심입니다. 학교에서 *"s로 끝나면 어포스트로피만"*이라고 배운 경우가 많지만, Strunk의 입장은 다릅니다.

## 예외 — 고대 고유명사

```
Jesus' disciples
Achilles' heel
Moses' laws
for conscience' sake
for righteousness' sake
```

고대 그리스 / 히브리 고유명사가 *-es*나 *-is*로 끝날 때만 *어포스트로피만* 붙입니다. 그 외 모든 경우는 *'s*입니다.

### 대안 형식

```
Moses' laws  →  the laws of Moses
Isis' temple →  the temple of Isis
```

*of* 구문이 더 자연스러우면 그것을 사용. 예외 규칙을 외울 필요가 줄어듭니다.

## 대명사 소유격 — 어포스트로피 *없음*

대명사의 소유격에는 *어포스트로피를 붙이지 않습니다*.

```
hers   (her's 아님)
its    (it's 아님)
theirs (their's 아님)
yours  (your's 아님)
ours   (our's 아님)
```

이것이 영어 학습자가 자주 틀리는 자리입니다. *대명사*와 *명사*의 규칙이 다름을 기억합니다.

## 부정 대명사 — 어포스트로피 *있음*

부정 대명사는 *명사처럼* 어포스트로피를 붙입니다.

```
one's rights
somebody else's umbrella
anyone's guess
nobody's fool
```

*One, somebody, anyone, nobody* 등은 대명사지만 *부정 대명사*라 일반 명사처럼 처리합니다.

## *Its* vs *It's* — 가장 흔한 실수

영어 글쓰기에서 *가장 자주 틀리는 자리*가 이것입니다.

```
its   = 소유격
it's  = it is 또는 it has의 축약
```

```
The dog wagged its tail.        ← 소유격 (어포스트로피 없음)
It's a wise dog that scratches  ← it is (어포스트로피 있음)
   its own fleas.
It's been raining all day.       ← it has
```

### 점검 방법

의심되면 *it is*나 *it has*로 풀어 봅니다.

```
"The cat licked it's paw."
→ 풀어 보면: "The cat licked it is paw" (말이 안 됨)
→ 따라서 its (소유격)이 맞음

"It's a beautiful day."
→ 풀어 보면: "It is a beautiful day" (자연스러움)
→ 따라서 it's가 맞음
```

이 점검을 *모든 it's / its*에 적용하면 실수를 피할 수 있습니다.

## 복수 소유격

복수 명사의 소유격은 *s 뒤에 어포스트로피만*.

```
the boys' uniforms     (복수 boys + 어포스트로피)
the dogs' tails        (복수 dogs + 어포스트로피)
the students' books    (복수 students + 어포스트로피)
```

### 불규칙 복수의 소유격

```
the children's toys    (children은 이미 복수, 's 붙임)
the women's group      (women은 이미 복수)
the people's choice    (people은 이미 복수)
```

영어의 *불규칙 복수형*은 일반 명사처럼 *'s*를 붙입니다.

## 함정 — 1990s, the 90's

연대 표시에는 어포스트로피를 *붙이지 않습니다*.

```
회피: the 1990's, the 90's
선호: the 1990s, the 90s
```

연대는 *복수가 아니라 그냥 표기*이므로 어포스트로피가 불필요합니다.

## 함정 — Family Surnames

```
The Smith family → the Smiths (복수)
The Smiths' house (그 집의 소유)
```

*Smith's*는 *한 Smith*의 소유. *Smiths'*는 *Smith 가족*의 소유. 의미가 다릅니다.

## 한국어 글에서

한국어 글에서는 소유격에 *조사 ~의*를 사용합니다. 영어처럼 헷갈리는 경우가 적습니다.

```
영어: Charles's friend
한국: 찰스의 친구

영어: the dog's tail
한국: 그 개의 꼬리
```

다만 *영어 책 / 영어 학술 자료*를 한국 글에 인용할 때 영어 소유격을 자주 마주칩니다. 그때 위의 규칙을 적용합니다.

## 자기 점검

```
□ 단수 소유격에 *'s*가 있는가? (Charles's, not Charles')
□ 고대 고유명사만 예외 적용 (Jesus', Moses')
□ *its / it's* 혼동 없는가? (자기 글에서 모두 점검)
□ 대명사 소유격에 *어포스트로피 없음* (hers, theirs, ours)
□ 복수 소유격에 *어포스트로피만* (the boys')
□ 연대 표시에 어포스트로피 없음 (1990s)
```
