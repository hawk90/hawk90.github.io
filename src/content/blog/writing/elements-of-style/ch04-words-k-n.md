## K–L

### Kind of / Sort of

회화에서 *rather*, *something like*의 의미로 *쓰지 마세요*. 격식 글에서는 *본래 의미*로만.

```
회피 (회화 / 격식 글에 부적합):
"He was kind of tired."
"It was sort of strange."

선호 (격식):
"He was rather tired." / "He was somewhat tired."
"It was rather strange."

선호 (본래 의미):
"Amber is a kind of fossil resin."
   ← 종류
"What kind of dog is that?"
   ← 종류
```

### Lay / Lie

*가장 자주 혼동되는* 동사. 한국 학습자뿐 아니라 *영어 native*도 자주 틀림.

```
LAY (놓다, 타동사):
- Present: lay
- Past: laid
- Past Participle: laid
- "The hen lays an egg." (목적어 "an egg")

LIE (눕다, 자동사):
- Present: lie
- Past: lay  ← 헷갈리는 자리!
- Past Participle: lain
- "The playwright went home and lay down." (목적어 없음)

LIE (거짓말, 자동사) — 또 다른 lie:
- Present: lie
- Past: lied
- Past Participle: lied
- "He lied to me."
```

```
선호:
"I lay the book on the table." (지금 놓는다)
"I laid the book on the table." (어제 놓았다)

"I lie down to rest." (지금 눕는다)
"I lay down to rest." (어제 누웠다)  ← 발음 같은 lay지만 다른 동사
"I have lain there for hours." (완료)
```

이 *시제 패턴*이 *영어 학습자가 가장 자주 틀리는* 자리.

### Leave / Let

*Leave it stand* → *Let it stand*. `Let go`는 표준.

```
회피: "Leave him do it."
선호: "Let him do it."

선호 (let go):
"Let go of the rope."
"He just couldn't let go."
```

```
Leave = depart / abandon:
"I leave at noon."
"Leave the room."

Let = allow / permit:
"Let me try."
"Let it be."
```

### Less / Fewer

```
Fewer + 셀 수 있는 것:
"They had fewer workers than in the previous campaign."
"Fewer than ten people attended."

Less + 양 / 셀 수 없는 것:
"His troubles are less than mine."  ← 양적 비교
"Less water remained in the bucket."

대조:
"His troubles are less than mine." ← 무게 비교
"His troubles are fewer than mine." ← 개수 비교
```

```
회피: "They had less workers ..."
선호: "They had fewer workers ..."

회피 (슈퍼마켓 흔함): "Ten items or less"
선호 (격식): "Ten items or fewer"
```

### Like

*접속사가 아니라 전치사*. *명사·대명사 앞*에서만 씁니다. *절이나 구*를 이끌 때는 *as*.

```
회피:
"We spent the evening like in the old days."
"Chloë smells good, like a baby should."

선호:
"We spent the evening as in the old days."
"Chloë smells good, as a baby should."
```

```
선호 (like + 명사):
"He looks like his father."
"It tastes like chicken."

선호 (as + 절):
"He acts as if he knows everything."
"Do as I say."
```

이 *Like vs. As* 구분은 *Strunk-White 정통*. 현대 영어 회화에서는 *like*가 양쪽에 쓰이지만 격식 글에서는 *지킵니다*.

### Literal / Literally

*과장이나 비유의 강조*로 쓰지 마세요. 단어의 *원래 의미*는 *문자 그대로*입니다.

```
회피 (잘못된 강조):
"a literal flood of abuse"
"literally dead with fatigue"
"I was literally on fire with embarrassment"

선호:
"a flood of abuse" (또는 "a torrent of abuse")
"almost dead with fatigue"
"I was very embarrassed"

선호 (적절한 literal):
"The translation is literal."
   ← 직역
"I literally fell out of bed."
   ← 실제로 떨어졌다 (비유 아님)
```

*Literally*를 *figuratively*의 강조로 쓰는 것은 *역설적*. 영어 native도 자주 혼동.

### Lots / A lot of

*많음*의 회화 표현. 격식 글에서는 *many*, *much*, *several*.

```
회피 (격식): "Lots of people attended."
선호: "Many people attended."

회피 (격식): "A lot of research has been done."
선호: "Considerable research has been done."
```

## M–N

### Many a / Many

```
Many a + 단수 명사 + 단수 동사:
"Many a man has tried."
   ← 격식, 시적

Many + 복수 명사 + 복수 동사:
"Many men have tried."
   ← 일반
```

### Meaningful

*빈 형용사*입니다. *다른 단어*로 대체하거나 문장을 *다시 씁니다*.

```
회피:
"His was a meaningful contribution."
"We had a meaningful conversation."

선호:
"His contribution counted heavily."
"His contribution mattered."
"We had a deep conversation."
"We discussed important topics."
```

*Meaningful*은 *비즈니스 영어의 진부어*. 의미가 *흐릿*합니다.

### Memento

*Momento*가 아닙니다.

```
선호: "She kept the photo as a memento."
회피: "momento" — 흔한 철자 실수
```

```
어원:
memento ← Latin "remember"
momento ← 없음. 잘못된 철자.
```

### Most

*Almost*의 의미로 *형식적 글에 쓰지 마세요*.

```
회피 (회화):
"Most everyone agrees."
"Most all the work is done."

선호:
"Almost everyone agrees."
"Almost all the work is done."

선호 (적절한 most):
"Most people agree." (대부분)
"This is the most important point." (최상급)
```

### Myself

*I*나 *me* 대신 쓰지 마세요. 강조나 재귀 의미가 있을 때만.

```
회피:
"He invited my wife and myself to the party."
"Myself and John attended."

선호:
"He invited my wife and me to the party."
"John and I attended."

선호 (강조 / 재귀):
"I myself did it." (강조)
"I cut myself." (재귀)
```

*Myself*를 *I*나 *me* 자리에 쓰면 *과잉 격식*으로 들립니다.

### Nature

`Character`처럼 *군더더기*로 자주 쓰입니다.

```
회피                            선호
acts of a hostile nature   →   hostile acts
things of a similar nature →   similar things
decisions of a routine nature → routine decisions
```

`Lover of nature`, `poems about nature`처럼 *모호한 표현*도 피합니다. *어떤 자연*인지 *구체적*으로.

```
회피 (모호):
"He is a lover of nature."

선호 (구체):
"He loves the mountains and forests."
"He loves wildlife and wilderness areas."
```

### Nauseous / Nauseated

```
Nauseous → 메스껍게 *만드는*
"That smell is nauseous."

Nauseated → 메스꺼움을 *느끼는*
"I feel nauseated."

회피 (현대 흔함):
"I feel nauseous." (잘못, 형식적으로는)

선호 (격식):
"I feel nauseated."
```

현대 회화에서는 *nauseous = 메스꺼움 느끼는*이 흔하지만, *격식 글*에서는 *구분*합니다.

### Nice

*모든 의미를 흐릿하게* 만드는 *닳은 단어*입니다. 형식적 글에서는 *precise, delicate*의 의미일 때만 명확.

```
회피 (모호):
"A nice day."
"A nice person."
"A nice book."

선호 (구체):
"A sunny day." / "A warm day."
"A kind person." / "A generous person."
"A thoughtful book." / "An engaging book."

선호 (precise / delicate, 본래 의미):
"A nice distinction."
   ← 미묘한 구분
"A nice point of law."
   ← 미세한 법적 쟁점
```

### Nor

*부정문 뒤의 or 자리*에 *잘못 쓰지 마세요*. `Nor`은 *neither ... nor*의 쌍에서만, 또는 *독립된 부정절*을 이끌 때만.

```
회피:
"He cannot eat nor sleep."

선호:
"He cannot eat or sleep."
또는: "He can neither eat nor sleep."
또는 (두 부정절): "He cannot eat, nor can he sleep."
```

```
규칙:
1. Neither A nor B (쌍)
2. 첫 부정문 뒤 두 번째 부정절: "..., nor ..."
3. 단순 부정문 뒤 등위 항목: or
```

## 한국 학습자의 K~N 흔한 함정

```
영어                  한국 학습자 실수
─────────────────────────────────────
kind of / sort of     → 격식 글에 회화 사용
lay / lie             → 시제 변화 혼동
less / fewer          → 셀 수 / 양 구분 안 함
like / as             → 절을 이끌 때 like 사용
literally             → 비유 강조로 오용
meaningful            → 모호한 형용사 과용
most everyone         → "almost"가 표준
myself                → I / me 자리에 과잉 사용
nature                → 명사화 군더더기
nauseous              → "메스껍게 느낀다"는 nauseated
nice                  → 모든 자리에 사용
nor                   → 단순 부정문 뒤 or 자리에 오용
```

## 자기 글 점검

```
□ "Lay / lie" 시제 변화가 정확한가?
□ "Less / fewer" 셀 수 / 없는 구분이 맞는가?
□ "Like" → 절을 이끌면 "as"로
□ "Literally"가 *진짜 문자 그대로*인가?
□ "Meaningful" 같은 모호한 형용사 *구체화*?
□ "Nature, character" 군더더기 *제거*?
□ "Nice"를 구체적 형용사로 *대체*?
□ "Most everyone" → "Almost everyone"
□ "Myself" → I / me로 대체 가능?
□ "Nor"의 사용이 *문법 규칙*에 맞는가?
```
