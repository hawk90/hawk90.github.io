> Chapter 1 — *Elementary Rules of Usage*

Strunk가 책의 *첫 챕터*에 둔 11개의 기본 문법 규칙. *문장 단위*의 결정들 — 소유격, 쉼표, 세미콜론, 대명사 격, 현수 분사.

## 챕터의 위치

```
Chapter 1: Elementary Rules of Usage      ← 문장 단위 문법
Chapter 2: Elementary Principles of       ← 문장 / 단락 / 글 단위 구성
            Composition
Chapter 3: A Few Matters of Form          ← 형식 관습
Chapter 4: Words and Expressions          ← 단어 / 표현 사용
            Commonly Misused
Chapter 5: An Approach to Style           ← 글쓰기의 총체 원칙
```

Chapter 1은 *가장 작은 단위*. 한 문장 안에서 *어디에 쉼표를 둘 것인가*, *대명사의 격은 무엇인가* 같은 결정들.

## 11개 규칙 목록

### 구두점 관련 (Rule 1-8)

```
Rule 1  — 단수 명사의 소유격은 's
Rule 2  — 직렬 쉼표 (Serial Comma / Oxford Comma)
Rule 3  — 삽입 표현은 쉼표로 감싸기
Rule 4  — 등위 접속사 앞에 쉼표
Rule 5  — Comma Splice 금지
Rule 6  — 문장을 둘로 끊지 말 것
Rule 7  — 콜론의 용법
Rule 8  — 대시의 용법
```

### 문법 일치 (Rule 9-11)

```
Rule 9  — 주어의 수가 동사의 수를 결정
Rule 10 — 대명사의 적절한 격
Rule 11 — 현수 분사 (Dangling Participle)
```

## 핵심 원칙 — Rules 3, 4, 5, 6

Strunk의 강조:
> *Rule 3, 4, 5, 6은 구두점의 가장 중요한 원칙이다. 다른 모든 규칙보다 먼저 몸에 익혀야 한다.*

```
Rule 3: 삽입 표현 → 양옆 쉼표
Rule 4: 등위 접속사 → 앞에 쉼표
Rule 5: Comma splice 금지 → 세미콜론 / 마침표
Rule 6: Sentence fragment 금지 → 쉼표로 연결
```

이 네 규칙이 *영어 구두점의 핵심*. 영어 학습자가 *가장 자주 틀리는* 자리.

## 한국 학습자가 가장 자주 틀리는 자리

```
1. Its vs It's (Rule 1) — 가장 흔한 영어 실수
2. Comma splice (Rule 5) — 두 독립절을 쉼표로
3. Subject-verb agreement (Rule 9) — 3인칭 단수 -s
4. Pronoun case (Rule 10) — Between you and me
5. Dangling participle (Rule 11) — 영어 주어 일치
```

자기 글에서 *이 다섯 자리*를 점검하면 *대부분의 문법 오류*를 잡습니다.

## 규칙 적용 순서

```
글을 쓸 때 점검 순서:

1. 한 문장 — 문법 (Rules 9, 10, 11)
2. 한 문장 — 구두점 (Rules 1, 2, 3, 4)
3. 두 문장 사이 — 연결 (Rules 5, 6, 7, 8)

수정 우선 순위:
1. Comma splice / fragment 제거 (Rules 5, 6)
2. Pronoun case 점검 (Rule 10)
3. Subject-verb agreement (Rule 9)
4. Apostrophe 점검 (Rule 1)
5. 직렬 쉼표 (Rule 2)
6. 대시 / 콜론 사용 (Rules 7, 8)
```

## 자기 글 점검 — Chapter 1 통합

```
□ 단수 소유격에 's가 있는가? (Rule 1)
□ Its / It's 혼동 없는가? (Rule 1)
□ 세 항목 이상 나열에 직렬 쉼표? (Rule 2)
□ 삽입 표현이 양옆에 쉼표? (Rule 3)
□ 등위 접속사 앞에 쉼표? (Rule 4)
□ Comma splice 없는가? (Rule 5)
□ Sentence fragment 없는가? (Rule 6)
□ 콜론이 독립절 뒤에 있는가? (Rule 7)
□ 대시 사용이 과하지 않은가? (Rule 8)
□ 주어 / 동사 일치 정확? (Rule 9)
□ 대명사 격이 맞는가? (Rule 10)
□ Dangling participle 없는가? (Rule 11)
```
