> Do not use dialect unless your ear is good.

사투리를 흉내 내려면 그 사투리의 *진정한 학생*이어야 합니다. 어설픈 사투리 묘사는 *조롱*에 가까워지고 *진정성*을 잃습니다.

## 사투리 묘사의 어려움

```
회피 (어설픈 사투리):
"I dunno, I reckon we oughta go now."
"Ya'll comin' or what?"

회피 (캐릭터 특성을 단순히 사투리로 환원):
시골 사람 = 사투리 잔뜩
교양 없는 사람 = 사투리 잔뜩
```

이런 묘사는 *고정관념의 강화*에 가깝습니다.

## 사투리의 일관성

쓰기로 했다면 *일관성*을 지키세요.

```
같은 단어가 두세 가지로 표기되면 독자가 짜증.

예시:
"once"의 사투리 표기:
   - oncet
   - wunst
   - wonse

한 글에서 *한 표기*만 선택. 끝까지 일관.
```

### Strunk의 예시

```
"once"는 사투리로 *oncet*으로 쓰입니다.
그러나 발음은 *wunst*에 가깝습니다.
*oncet*은 *onset*처럼 잘못 읽힐 위험이 있습니다.

*wunst*라고 쓰는 게 더 정확하지만, *oncet*으로 정했다면 끝까지
*oncet*으로.
```

## 최소화의 원칙

최고의 사투리 작가는 표준에서 *벗어나는 양*을 *최소*로 유지합니다.

```
과도 (회피):
"Wal, I reckon them thar critters be comin' down the holler.
Ah'm gonna git me a rifle 'n' shoot 'em afore they git my chickens."

최소 (선호):
"Well, I reckon those critters are coming down the holler. I'm
going to get my rifle before they get my chickens."
```

*reckon*, *critters*, *holler*만 사투리로. 나머지는 표준 영어. 독자가 *읽기 쉬우면서도* 화자의 출신이 보입니다.

## 사투리 묘사의 윤리

```
회피해야 할 사투리 묘사:
- 인종 / 민족적 정형화
- 계급 / 학력 의식의 표시로
- 조롱 / 비웃음의 대상으로
- 자기 출신이 아닌 사투리

정당한 사투리 묘사:
- 인물의 *지역적 출신* 표현
- *문화적 진정성* 확보 (작가가 그 문화 일원일 때)
- 작품의 *시대 / 장소* 분위기
- 화자별 *어휘 차이*의 미세한 표현
```

## 한국어 사투리

한국어 사투리 묘사도 같은 원칙이 적용됩니다.

```
한국어 사투리 (지역별):
- 경상도 사투리 (대구, 부산 등)
- 전라도 사투리 (광주, 전주 등)
- 충청도 사투리
- 강원도 사투리
- 제주 사투리 (가장 다름)
```

### 한국 글의 사투리 묘사 함정

```
회피 (어설픈 묘사):
"하이고, 거시기 좀 가져 오니라" (전라도 정형화)
"이기 무신 짓이고" (경상도 정형화)

선호 (자연스럽게):
대화 일부에만 사투리 어휘 사용
표준어와 자연스럽게 섞임
화자의 특성으로서의 사투리, 표면적 효과 아님
```

박경리의 *토지*, 조정래의 *태백산맥*이 *전라도 사투리*를 *진정성 있게* 다룬 사례입니다. 작가가 *그 지역에 살아온* 사람이기 때문에 가능했습니다.

## 사투리를 *안 쓰는* 결정

자기가 그 사투리의 *진정한 학생*이 아니라면, *사투리를 안 쓰는* 것이 안전합니다.

```
대안 (사투리 없이도 인물 묘사):
- "그는 강한 경상도 억양으로 말했다." (서술로)
- 화자 특정 어휘 한두 개만 (대화 안에서)
- 인물의 배경을 *간단히* 언급
```

## 적용

```
□ 자기가 그 사투리의 *진정한 학생*인가?
□ 사투리 묘사가 *일관*되는가?
□ *최소화 원칙*을 따르는가?
□ 사투리가 *조롱*이 되지 않는가?
□ 표준 영어 / 표준 한국어로 *대체 가능*한가?
```
