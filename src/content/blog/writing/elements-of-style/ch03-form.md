이 챕터는 짧습니다. 규칙도 11개 항목이지만 하나하나가 작은 결정들입니다. 글의 *형식 관습*을 정리합니다.

## Colloquialisms — 구어체

구어체 표현이나 슬랭을 쓸 때는 그냥 씁니다. 따옴표로 감싸 "나는 이걸 안다, 하지만 알면서도 쓴다" 같은 신호를 보내지 마세요. 그건 *아는 사람들끼리*의 동아리에 독자를 초대하는 듯한 거만함을 풍깁니다.

```
잘못: He was a real "wise guy" about the whole thing.
바름: He was a real wise guy about the whole thing.
```

## Exclamations — 느낌표

평범한 진술에 느낌표를 붙여 강조를 *만들어 내려고* 하지 마세요. 느낌표는 진짜 감탄이나 명령에만 씁니다.

```
잘못: It was a wonderful show!
바름: It was a wonderful show.

OK:   What a wonderful show!
OK:   Halt!
```

## Headings — 머리글 / 제목

원고를 출판용으로 제출할 때는 첫 페이지 위에 공간을 충분히 둡니다. 제목은 페이지의 1/4 지점쯤에 배치하고 한 줄(또는 그만한 공간)을 비운 뒤 본문을 시작합니다. 다음 페이지부터는 위쪽 가까이 적되 너무 빽빽하지 않게 합니다. 제목 끝의 마침표는 생략합니다(물음표나 느낌표가 필요한 제목이면 둡니다).

## Hyphen — 하이픈

두 단어가 합쳐져 복합 형용사를 만들면 하이픈이 보통 필요합니다.

```
He belonged to the leisure class and enjoyed leisure-class pursuits.
She entered her boat in the round-the-island race.
```

하이픈 없이 한 단어로 쓰는 게 더 자연스러운 단어들에는 하이픈을 넣지 마세요.

```
water-fowl  →  waterfowl
wild-life   →  wildlife
bell-boy    →  bellboy
```

영어 단어의 진화는 *두 단어 → 하이픈 → 한 단어*의 방향을 따릅니다.

```
bed chamber → bed-chamber → bedchamber
```

하이픈을 잘못 넣어 우습게 만든 유명한 예가 있습니다. Chattanooga의 두 신문 *News*와 *Free Press*가 합병되어 *Chattanooga News-Free Press*가 되자, 마치 "뉴스 없는 신문"처럼 읽혔습니다.

## Margins — 여백

좌우 여백을 같은 폭으로 둡니다. 편집과 주석이 많이 들어갈 원고는 왼쪽 여백을 더 넓게 잡습니다.

## Numerals — 숫자

날짜와 일련번호는 풀어 쓰지 말고 숫자나 로마 숫자로 표기합니다.

```
August 9, 1988
Part XII
Rule 3
352d Infantry
```

대화에서는 보통 풀어 씁니다.

```
"I arrived home on August ninth."
"In the year 1990, I turned twenty-one."
"Read Chapter Twelve."
```

## Parentheses — 괄호

괄호를 포함한 문장의 구두점은 *괄호가 없는 것처럼* 처리합니다. 괄호 안의 내용은 자기 안에서 따로 구두점을 갖되, 끝의 마침표는 생략합니다(괄호 안이 물음표나 느낌표라면 둡니다).

```
I went to her house yesterday (my third attempt to see her), but she had left town.
He declares (and why should we doubt his good faith?) that he is now certain of success.
```

완전히 독립된 표현이나 문장이 통째로 괄호 안에 들어가면 마침표를 괄호 안에 둡니다.

```
(He had already decided to leave when the letter arrived.)
```

## Quotations — 인용

증거로 인용하는 *공식 인용*은 콜론 뒤에 따옴표로 묶습니다.

```
The United States Coast Pilot has this to say of the place: "Bracy Cove,
0.5 mile eastward of Bear Island, is exposed to southeast winds, has a
rocky and uneven bottom, and is unfit for anchorage."
```

동격이나 동사의 직접 목적어로 들어가는 인용은 쉼표 + 따옴표.

```
I am reminded of the advice of my neighbor, "Never worry about your
heart till it stops beating."
Mark Twain says, "A classic is something that everybody wants to have
read and nobody wants to read."
```

인용 뒤에 *attributive*(누가 말했다)가 따라오면 쉼표를 따옴표 안에 둡니다. 미국식 활자 관습입니다.

```
"I can't attend," she said.
```

여러 줄짜리 인용은 새 줄에서 시작하고 들여쓰기로 본문과 구분합니다. 이때 따옴표는 보통 생략합니다(원문에 따옴표가 있는 경우는 살립니다).

```
Wordsworth's enthusiasm for the French Revolution was at first unbounded:
    Bliss was it in that dawn to be alive,
    But to be young was very heaven!
```

`that`으로 이끄는 간접 인용은 따옴표를 쓰지 않습니다.

```
Keats declares that beauty is truth, truth beauty.
Dickinson states that a coffin is a small domain.
```

널리 알려진 격언이나 문학적 표현도 따옴표 없이 씁니다.

```
These are the times that try men's souls.
He lives far from the madding crowd.
```

## References — 참조 / 인용 표기

학술적 글에서 자주 등장하는 작품의 제목은 약어로 표기하고, 약어 일람을 글 끝에 알파벳순으로 둡니다. 참조는 본문이 아닌 괄호 안이나 각주에 둡니다. *act, scene, line, book, volume, page* 같은 단어는 둘 이상을 가리킬 때만 생략합니다.

```
in the second scene of the third act     →  in III.ii
After the killing of Polonius, Hamlet is placed under guard (IV.ii.14).
2 Samuel i:17–27
Othello II.iii.264–267, III.iii.155–161
```

## Syllabication — 음절 분리

줄 끝에서 단어를 분리해야 할 때는 사전에서 음절 구분을 확인합니다. 잘 편집된 책의 분리 위치를 참고하는 것도 좋은 방법입니다.

## Titles — 작품 제목

문학 작품의 제목은 보통 *이탤릭체 + 단어 첫 글자 대문자*로 표기합니다. 출판사와 편집자마다 관습이 약간씩 다릅니다. 어떤 곳은 이탤릭과 대문자, 어떤 곳은 로마체 + 따옴표, 또 어떤 곳은 따옴표 없이 로마체를 씁니다.

```
A Tale of Two Cities
Dickens's Tale of Two Cities      ← 소유격 앞에서는 A/The 생략
The Age of Innocence
Wharton's Age of Innocence
```

## 정리

- 구어체는 따옴표 없이 그대로
- 느낌표는 진짜 감탄·명령에만
- 머리글 형식, 여백, 들여쓰기의 기본 관습
- 복합 형용사에는 하이픈, 굳어진 합성어에는 한 단어
- 날짜·일련번호는 숫자, 대화에서는 풀어 쓰기
- 괄호 안 구두점, 인용의 따옴표 관습
- 참조 약어, 작품 제목의 이탤릭
