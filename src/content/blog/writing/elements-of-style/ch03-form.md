> **The Elements of Style** Chapter 3 요약

## 이 챕터의 메시지

이 챕터는 가장 짧다. **형식적 디테일**의 작은 모음 — 표기, 인용, 숫자 같은 것들. 글쓰기의 본질은 아니지만 일관성과 가독성에 필요.

## Headings

머리글(제목 / 소제목)은 명사구나 짧은 문장. 마지막에 마침표 안 찍음.

```
✅ The Three Branches of Government
✅ Economic Conditions in 1929
✅ How to Tie a Tie

❌ The Three Branches of Government.   (마침표 X)
❌ Why we should care about ecology     (제목은 보통 대문자)
```

대문자 규칙은 스타일 가이드에 따라 — APA, MLA, Chicago가 모두 다름.

## Numerals

**십 미만은 영어로**, 그 이상은 숫자.

```
There were three apples and 12 oranges.
The class has 25 students.
She has two cats.
```

문장 시작은 숫자라도 영어로.

```
❌ 25 students were in the class.
✅ Twenty-five students were in the class.
```

큰 숫자는 자리 구분.

```
1,234
1,000,000
```

## Quotations

직접 인용은 큰따옴표. 인용 안에 인용은 작은따옴표.

```
He said, "I just heard her say, 'I'm coming.'"
```

마침표 / 쉼표는 닫는 따옴표 **안**에 (미국식).

```
"I am ready," she said.    (American)
"I am ready", she said.    (British)
```

콜론 / 세미콜론은 따옴표 **밖**에.

```
He called himself an "expert"; we doubted it.
```

## Block Quotations

긴 인용 (보통 4행 이상)은 들여쓰기로 분리. 따옴표 없음.

```
He wrote:

    The quick brown fox
    jumps over the lazy dog
    every day.

— from "The Fable of Animals"
```

## Hyphens

연결되는 형용사구는 하이픈으로.

```
well-known author      (a well-known author)
two-thirds majority    
old-fashioned values
nineteenth-century literature
```

명사 뒤로 가면 보통 하이픈 안 씀.

```
The author is well known.   (well known, no hyphen)
The literature of the nineteenth century.
```

## Em Dash와 En Dash

**Em dash (—)** — 가장 긴, 강한 단절. 1장에서 다룸.

**En dash (–)** — 범위 표시.

```
1990–2000
pages 5–12
Seoul–Tokyo flight
```

## Parentheses (괄호)

부가 정보. 본문에 흡수되어도 의미가 통해야.

```
He is the CEO (and founder) of the company.
The result (see Table 3) was unexpected.
```

괄호 안 문장이 완결되면 — 마침표가 괄호 안에.

```
(This is a separate sentence.)
```

본문의 일부면 — 마침표가 괄호 밖에.

```
The event (which I missed).
```

## Footnotes

각주 번호는 본문에 위첨자.

```
The result was significant.¹

¹ See Smith (2020) for detailed analysis.
```

각주 번호는 보통 마침표 / 쉼표 뒤에 (미국식). 영국식은 앞에 두는 경우도.

## Capitalization

고유 명사, 문장 시작, 제목.

```
The book is called "A Tale of Two Cities."
He visited Korea and Japan.
The President spoke yesterday.   (직책으로서)
The presidents of two nations met. (일반)
```

## Spelling

**일관성**이 가장 중요. American vs British, 한 글에 섞으면 안 됨.

```
American: color, organize, traveled, theater
British:  colour, organise, travelled, theatre
```

학술 / 저널마다 요구하는 스타일 — 처음에 정하고 끝까지 일관.

## Syllabication (음절 분리)

줄 끝에서 단어 분리 시 음절 사이에서. 한 글자만 남기는 분리는 X.

```
re-               ❌ a-
flec-                bout
tion.
```

워드프로세서가 자동 처리. 손글씨 / 옛 출판에서 중요.

## 결론 — 형식은 도구일 뿐

이 챕터의 모든 규칙이 절대적이지 않다. **스타일 가이드**(APA / MLA / Chicago / 회사 내부 가이드)가 우선. 그러나 가이드가 없으면 — Strunk-White가 합리적 default.

핵심은 **일관성**. 한 글 안에서 같은 규칙을 끝까지.

## 정리

- **Headings** — 명사구, 마침표 없음
- **숫자** — 10 미만은 영어, 그 이상은 숫자
- **Quotations** — 미국식은 마침표가 따옴표 안
- **Block quotation** — 4행 이상, 들여쓰기
- **Hyphens** — 형용사구의 연결
- **Em dash / En dash** — 단절 vs 범위
- **Parentheses** — 부가 정보
- **Footnotes** — 위첨자, 마침표 뒤
- **일관성**이 모든 형식의 핵심
