> Use orthodox spelling.

일반 글에서는 *표준 철자*를 따릅니다. *nite, thru, pleez*를 쓰지 마세요.

## 단편적 변형의 문제

단순화된 철자 체계를 *완전히* 도입할 각오가 없다면, 단편적인 변형은 *독자의 주의를 분산*시킬 뿐입니다.

```
표준:        변형:
night    →   nite
through  →   thru
please   →   pleez
you      →   u
your     →   ur
```

이런 변형이 *한 글에 한 번* 등장하면 독자는 *작가의 의도*를 묻습니다. *왜 여기서만 변형인가? 일관성 있게 다 변형할 수는 없나?*

## 변형이 정당한 자리

```
1. 대화의 *방언 표현* (Reminder 15)
2. SNS / 메시지의 *캐주얼한 형식*
3. 광고의 *시각적 효과* (광고 카피)
4. 시 / 소설의 *의도된 효과*
```

이런 자리에서는 변형이 *기능적*입니다. 일반 산문에서는 *주의 분산*입니다.

## 언어의 변화

언어는 변합니다. White가 한 예시:

> *Thruway*는 우리 세기에 자리 잡은 단어입니다. 표지판의 *throughway*보다 *빠르게 읽히기에* 살아남았습니다.

```
사라진 변형:
nite (← night)
pleez (← please)
thru (← through, 비공식)

자리잡은 변형:
thruway (← throughway, 표지판에서)
OK (← all correct)
e-mail / email (← electronic mail)
```

언젠가 *through*가 *thru*가 될지도 모릅니다. 하지만 그때가 오기까지는 표준을 씁니다.

## AI 시대의 변형

AI / 인터넷이 새 변형을 만들고 있습니다.

```
- u → you (텍스트)
- r → are
- ur → your
- thx → thanks
- gr8 → great
- prolly → probably
- IIRC → if I recall correctly
- TL;DR → too long; didn't read
- LGTM → looks good to me
```

이 중 일부는 *전문 분야 (개발자, 학술)*에서 표준이 되었습니다. 다른 일부는 *캐주얼*에 머무릅니다.

## 한국어 글의 표준 vs 변형

한국어 글에서도 비슷한 결정이 있습니다.

```
표준:                변형:
했어요             →   해써요
같아요             →   가타요
그렇게             →   글케

웃음 표현:
ㅋㅋㅋ (캐주얼)
(웃음) (소설 / 대화 안)
스마일 / 윙크 (이모티콘)
```

격식 있는 글에는 *표준 한국어*를 씁니다. 캐주얼 글에는 변형 OK.

### 외래어 표기

외래어는 *국립국어원의 표준 표기*가 있습니다.

```
- 컴퓨터 (O) / 콤퓨터 (X)
- 인터넷 (O) / 인터넷트 (X)
- 데이터 (O) / 데이타 (X)
- 카페 (O) / 까페 (X)
```

자기 글의 매체에 따라 표준을 따를지 결정합니다. 학술 글은 표준, 캐주얼 글은 자유로움.

## 격식 수준별 철자

```
초격식 (법률, 학술):
- 모든 표준 철자
- 약어조차 풀어 씀
- e.g. → for example
- etc. → and so on

격식 (저널리즘, 책):
- 표준 철자
- 일반적 약어 OK
- "OK"는 가끔

중립 (일반 글, 블로그):
- 표준 + 일반적 약어
- 비공식 변형은 회피

캐주얼 (SNS, 메시지):
- 자유로움
```

자기 매체에 맞춰.

## 적용

```
□ 자기 글에서 *비표준 변형*이 있는가?
□ 변형이 *의도적*인가, *습관적*인가?
□ 매체에 맞는 격식 수준인가?
□ 외래어가 *국립국어원 표준*을 따르는가?
□ 약어가 *처음 등장*할 때 풀어 쓰였는가?
```
