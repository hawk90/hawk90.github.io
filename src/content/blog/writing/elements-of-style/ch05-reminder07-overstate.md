> Do not overstate.

과장하면 독자는 *즉시 경계 태세*에 들어갑니다. 그 뒤로 쓰는 모든 것이 *의심받습니다*.

## 과장의 신호

```
"the best ever"
"absolutely the worst"
"the most amazing"
"completely revolutionary"
"totally life-changing"
```

이런 *무모한 최상급*이 한 번 등장하면 독자가 *작가의 판단력*을 의심합니다.

## 한 번의 과장의 비용

한 글에 *한 번의 과장*이 들어가도 *글 전체*의 신뢰가 무너집니다.

```
"이 책은 *역사상 가장 위대한* 책이다. 작가의 통찰은 깊고,
산문은 정확하며, 구성은 견고하다."
```

첫 문장이 *과장*이라면, 뒤따르는 *합리적 평가*도 의심받습니다. 깊은 통찰이 있다고 해도 *역사상 가장 위대한*에 발이 묶입니다.

## 과장 vs 강한 의견

강한 의견과 과장은 다릅니다.

```
강한 의견:
"이 책은 올해 출간된 nonfiction 중 가장 강하다."
→ 명확한 의견 + 범위 (올해, nonfiction)

과장:
"이 책은 역사상 가장 위대하다."
→ 검증 불가능한 절대 평가
```

강한 의견은 *범위와 근거*가 있습니다. 과장은 *둘 다 없습니다*.

## 학생 글의 흔한 과장

```
- "사회 전체가 변화하고 있다"
- "AI가 모든 것을 바꿀 것이다"
- "이 문제는 시급히 해결되어야 한다"
- "역사상 처음으로 ..."
- "전 세계가 ..."
```

이런 표현은 *검증 불가능*하고 *모호*합니다. 더 구체적이고 정확한 표현으로 바꿀 수 있습니다.

```
회피: AI가 모든 것을 바꿀 것이다.
선호: AI는 지난 5년간 [구체적 분야]에서 변화를 만들었다. 향후
       10년 동안 [구체적 영역]에서 더 큰 변화가 예상된다.
```

## 한 번의 과장이 무너뜨리는 것

```
- 작가의 *판단력* 신뢰
- 글의 *나머지 주장* 신뢰
- 작가의 *지식 범위* 신뢰
- 작가의 *정직성* 신뢰
```

신뢰는 *천천히 쌓이고 빠르게 무너집니다*.

## 강한 의견을 *과장 없이* 표현하기

```
대신 사용:
- "내가 본 것 중 가장 ..." (자기 경험 범위)
- "이 분야에서 가장 ..." (분야 한정)
- "최근 [기간]에서 가장 ..." (시간 한정)
- "내가 아는 한 ..." (지식 한계 인정)

피해야 할 표현:
- "역사상 가장 ..."
- "절대적으로 최고의 ..."
- "전 세계에서 ..."
- "모든 사람이 ..."
```

## 한국어 글의 과장

한국어 글에서 자주 마주치는 과장:

```
- "정말 대박이다"
- "역대급"
- "최고의"
- "유일한"
- "절대 불가능한"
- "완벽한"
```

이런 표현이 들어가면 *광고*에 가까워집니다. 학술 글이나 비평 글에서는 회피합니다.

## 적용

```
□ 자기 글에서 *최상급*을 점검
□ 각 최상급이 *옹호 가능*한가?
□ 검증 불가능한 *전 세계, 역사상, 모든* 표현을 회피
□ 강한 의견은 *범위 한정*으로
```
