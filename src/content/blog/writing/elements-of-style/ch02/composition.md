> Chapter 2 — *Elementary Principles of Composition*

Strunk의 두 번째 챕터. *Chapter 1*이 *문장 단위 문법*이라면, *Chapter 2*는 *문장 / 단락 / 글 단위 구성*. 글쓰기 기술의 *핵심 원칙*들.

## 챕터의 위치

| Chapter | 내용 | 비고 |
|---------|------|------|
| Chapter 1 | 문장 단위 문법 | 한 문장의 정확성 |
| Chapter 2 | 글의 구성 원칙 | 글의 효과 / 명료성 ← 현재 |
| Chapter 3 | 형식 관습 | |
| Chapter 4 | 단어 사용 | |
| Chapter 5 | 스타일의 총체 | |

Chapter 2의 11개 규칙이 *Strunk-White 글쓰기의 정수*. 가장 유명한 *"Omit needless words!"*도 이 챕터.

## 11개 규칙 목록

### 글의 큰 구조 (Rule 12-13)

- **Rule 12** — 적절한 디자인을 선택하고 끝까지 지킬 것
- **Rule 13** — 단락을 작문의 단위로 삼을 것

### 문체의 핵심 (Rule 14-17)

- **Rule 14** — 능동태를 쓸 것 (Use the active voice)
- **Rule 15** — 진술은 긍정 형식으로 (Put statements in positive form)
- **Rule 16** — 구체적이고 구상적인 언어를 쓸 것
- **Rule 17** — 불필요한 단어를 빼라 (Omit needless words!)

### 문장의 리듬 (Rule 18-22)

- **Rule 18** — Loose Sentence의 연쇄를 피하라
- **Rule 19** — 평행 구조 (Parallel Construction)
- **Rule 20** — 관련 있는 단어를 함께 두라
- **Rule 21** — 요약문에서는 시제를 하나로
- **Rule 22** — 강조 단어는 문장의 끝에

## 가장 유명한 규칙 — Rule 17

Strunk가 학생들에게 *세 번 외친* 문장:

> *Omit needless words! Omit needless words! Omit needless words!*

```
원칙:
- 문장에 *불필요한 단어*가 없을 것
- 단락에 *불필요한 문장*이 없을 것
- 그림에 *불필요한 선*이 없을 것
- 기계에 *불필요한 부품*이 없을 것

→ 모든 *단어가 일해야* 합니다.
```

이 한 문장이 *글쓰기 책의 핵심*.

## 다른 핵심 규칙

### Rule 14 — 능동태

> *Use the active voice.*

| 유형 | 예문 |
|------|------|
| 약함 (수동) | "My first visit to Boston will always be remembered by me." |
| 강함 (능동) | "I shall always remember my first visit to Boston." |

### Rule 16 — 구체성

> *Use definite, specific, concrete language.*

| 유형 | 예문 |
|------|------|
| 모호 | "A period of unfavorable weather set in." |
| 구체 | "It rained every day for a week." |

### Rule 22 — 강조 자리

> *Place the emphatic words of a sentence at the end.*

| 유형 | 예문 |
|------|------|
| 약함 | "Humanity has hardly advanced in fortitude since that time, though it has advanced in many other ways." |
| 강함 | "Since that time, humanity has advanced in many ways, but it has hardly advanced in fortitude." |

## 한국 학습자가 가장 자주 깨뜨리는 규칙

| Rule | 문제 |
|------|------|
| Rule 14 (능동태) | 영어식 수동태 직역 |
| Rule 16 (구체성) | 추상 명사 (-tion, -ity) 과다 |
| Rule 17 (간결) | 군더더기 표현 |
| Rule 19 (평행) | 어미 불일치 |
| Rule 21 (시제) | 시제 흔들림 |

자기 글에서 *이 다섯 자리*를 점검하면 *학술 영어의 큰 발전*.

## 규칙 적용 순서

**글 쓸 때 점검 순서:**

1. **1단계 — 큰 구조 (Rules 12, 13):** 디자인 / 골격, 단락 단위
2. **2단계 — 문체 (Rules 14, 15, 16, 17):** 능동태, 긍정 형식, 구체성, 간결
3. **3단계 — 리듬 (Rules 18, 19, 20, 21, 22):** 문장 다양성, 평행 구조, 어순, 시제 일관, 강조 자리

## 자기 글 점검 — Chapter 2 통합

- [ ] 글에 *명확한 디자인*이 있는가? (Rule 12)
- [ ] 한 단락 = 한 주제? (Rule 13)
- [ ] 수동태 비율이 *10% 이하*? (Rule 14)
- [ ] "Not" 부정 → 긍정 단어로 압축? (Rule 15)
- [ ] 모호한 단어 → 구체적 단어? (Rule 16)
- [ ] "The fact that", "in order to" 등 *군더더기 제거*? (Rule 17)
- [ ] 모든 문장이 "and"로 이어지지 않는가? (Rule 18)
- [ ] 나열 / 비교가 *평행*인가? (Rule 19)
- [ ] 수식어가 *수식 대상에 인접*? (Rule 20)
- [ ] 시제가 *일관*인가? (Rule 21)
- [ ] 강조 단어가 *문장 끝*에? (Rule 22)
