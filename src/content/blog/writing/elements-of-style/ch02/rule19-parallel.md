> Express coordinate ideas in similar form.

같은 기능과 내용을 가진 표현은 형태도 같게 만듭니다. 이를 parallel construction(평행 구조)이라 부릅니다. 강한 글의 핵심 기법입니다.

## Beatitudes의 예

성경의 팔복(八福)은 평행 구조의 고전적 예입니다.

```
Blessed are the poor in spirit: for theirs is the kingdom of heaven.
Blessed are they that mourn:    for they shall be comforted.
Blessed are the meek:           for they shall inherit the earth.
```

```
패턴:
"Blessed are [주어]: for [결과]"

→ 모든 줄이 같은 형태
→ 리듬 + 음악 + 기억하기 쉬움
→ 평행 구조의 힘
```

## 일관성 > 변화

같은 정보를 굳이 다른 형태로 표현해 변화를 주려는 시도는 약합니다. 일관성이 변화보다 강합니다.

```
회피 (변화 시도):
"Formerly, science was taught by the textbook method, while
now the laboratory method is employed."

분석:
- 첫 절: "science was taught by the textbook method"
- 둘째 절: "the laboratory method is employed"
- 주어·동사 패턴이 다름
- 어색

선호 (평행 유지):
"Formerly, science was taught by the textbook method; now it
is taught by the laboratory method."

분석:
- 두 절 모두 "[방법] was/is taught"
- 같은 형태 → 강함
```

## 평행해야 하는 자리

```
1. 동사 시제 / 형태 일관성:
   회피: "She likes hiking, swimming, and to run."
   선호: "She likes hiking, swimming, and running."
   또는: "She likes to hike, to swim, and to run."

2. 명사 / 동명사 / 부정사 일관성:
   회피: "His hobbies are reading, music, and to travel."
   선호: "His hobbies are reading, music, and travel."

3. 절 / 구 일관성:
   회피: "She is intelligent, kind, and has a great sense of humor."
   선호: "She is intelligent, kind, and humorous."

4. 비교 구조:
   회피: "Studying alone is more productive than to study with friends."
   선호: "Studying alone is more productive than studying with friends."
```

## 관사 / 전치사의 평행

복수 항목에 같은 관사·전치사를 적용할 때는 첫 항목 앞에 한 번만 쓰거나, 모든 항목 앞에 반복합니다.

```
회피 (일관성 없음):
"the French, the Italians, Spanish, and Portuguese"

선호 (모두 the):
"the French, the Italians, the Spanish, and the Portuguese"

선호 (첫 번째에만):
"the French, Italians, Spanish, and Portuguese"
```

```
전치사 평행:
회피:
"He spoke to John, Mary, and to Tom."

선호 (한 번만):
"He spoke to John, Mary, and Tom."

선호 (모두 to):
"He spoke to John, to Mary, and to Tom."
```

## 상관 표현 (Correlative Conjunctions)

`Both ... and`, `not only ... but also`, `either ... or`, `neither ... nor`, `first ... second` — 양쪽 형태가 같아야 합니다.

```
회피:
"My objections are, first, the injustice of the measure;
second, that it is unconstitutional."

분석:
- first 뒤: 명사구 "the injustice ..."
- second 뒤: 절 "that it is ..."
- 불평행

선호:
"My objections are, first, that the measure is unjust;
second, that it is unconstitutional."

→ 둘 다 "that ..."
```

### *Not Only ... But Also*

```
회피:
"She not only sings beautifully but also dancing well."

선호:
"She not only sings beautifully but also dances well."
또는:
"She not only sings but also dances beautifully."

원칙: not only 뒤와 but also 뒤가 문법적으로 동일
```

### *Either ... Or* / *Neither ... Nor*

```
회피:
"You can either come with us or you can stay home."

선호:
"You can either come with us or stay home."
또는:
"Either you can come with us, or you can stay home."

원칙: 양쪽이 같은 단위(구 / 절)
```

## Comparisons — 비교 구조

비교 구조에서도 평행이 필수입니다.

```
회피:
"The sales were higher this year than last year's sales."

선호:
"The sales were higher this year than last year."
또는:
"This year's sales were higher than last year's sales."
```

```
회피:
"To know him is loving him."

선호:
"To know him is to love him."
```

## Lists와 Bullet Points

목록·불릿 포인트는 특히 평행이 중요합니다.

```
회피 (불평행 불릿):
The new policy will:
- Reduce costs
- The improvement of efficiency
- We will implement training programs
- To increase morale

선호 (평행 불릿):
The new policy will:
- Reduce costs
- Improve efficiency
- Implement training programs
- Increase morale

→ 모두 동사 + 목적어 형태
```

```
선호 (명사 평행):
Today's agenda:
- Sales review
- Budget discussion
- Personnel updates
- Closing remarks
```

## Master Examples

### Kennedy의 *Inaugural Address*

```
"Let every nation know, whether it wishes us well or ill, that
we shall pay any price, bear any burden, meet any hardship,
support any friend, oppose any foe to assure the survival and
the success of liberty."

평행 패턴:
- pay any price
- bear any burden
- meet any hardship
- support any friend
- oppose any foe

→ 모두 동사 + any + 명사
→ 리듬 + 무게 + 기억
```

### Lincoln의 *Gettysburg Address*

```
"government of the people, by the people, for the people"

→ [전치사] the people × 3
→ 가장 유명한 평행 구조
```

### Churchill

```
"We shall fight on the beaches, we shall fight on the landing
grounds, we shall fight in the fields and in the streets, we
shall fight in the hills; we shall never surrender."

패턴:
- we shall fight [전치사구]
→ 다섯 번 반복
→ 결단의 무게
```

## 평행 구조가 깨지면 일어나는 일

```
독자의 반응:
- 리듬이 끊김
- 문법 오류로 느낌
- 작가의 부주의로 해석
- 의미 혼란 가능

수정 후:
- 읽기 쉬움
- 기억하기 쉬움
- 프로의 인상
```

## 한국어의 평행 구조

한국어에서도 평행 구조가 강합니다.

```
선호 (평행):
"공부하고, 일하고, 사랑하라."
"가난한 사람은 행복하니, 천국이 그들의 것이다.
슬퍼하는 사람은 행복하니, 위로받을 것이다.
온유한 사람은 행복하니, 땅을 물려받을 것이다."

→ 같은 어미·같은 구조
→ 한국어에서도 평행이 강한 효과를 낸다
```

### 한국어 불평행 함정

```
회피 (어미 섞임):
"공부를 좋아하고, 운동도 좋아하며, 음악을 좋아한다."

선호 (평행):
"공부를 좋아하고, 운동을 좋아하고, 음악을 좋아한다."
또는: "공부, 운동, 음악을 좋아한다."
```

## 자기 글 점검

- [ ] 목록·나열에서 모든 항목이 같은 형태인가?
- [ ] 동사 시제·형태가 일관되는가?
- [ ] 명사·동명사·부정사가 섞이지 않았는가?
- [ ] 상관 표현(not only ... but also) 양쪽이 평행인가?
- [ ] 비교 구조에서 같은 단위를 비교하는가?
- [ ] 불릿 포인트가 모두 같은 형태인가?
- [ ] 한국 글의 어미가 일관되는가?
