> Use the active voice.

능동태는 보통 수동태보다 직접적이고 활기찹니다. Strunk의 가장 자주 인용되는 규칙 중 하나입니다.

## 기본 대조

| 분류 | 예문 |
|---|---|
| 능동 | I shall always remember my first visit to Boston. |
| 수동 | My first visit to Boston will always be remembered by me. |

수동태가 더 길고 어색합니다. 주어 → 동사 → 목적어의 자연 순서가 능동태입니다.

## 왜 능동태가 강한가

```
능동:
주어 (행위자) + 동사 (행위) + 목적어 (대상)
"The committee approved the proposal."
   ↑
누가 무엇을 했는가 — 명확
```

```
수동:
대상 + be + 과거분사 + (by + 행위자)
"The proposal was approved by the committee."
또는: "The proposal was approved."  ← 행위자 사라짐
```

수동태는 누가 했는가를 모호하게 만듭니다. 그래서 책임 회피에 자주 쓰입니다.

```
회피 (관료적 수동태):
"Mistakes were made."  ← 누가 했는가?
"It has been decided that ..."  ← 누가 결정했는가?
"The data was lost."  ← 누가 잃었는가?

선호 (능동태):
"I made mistakes."
"The board decided that ..."
"We lost the data."
```

## 수동태가 필요한 자리

수동태도 정당한 자리가 있습니다. 무엇을 강조하느냐에 따라 결정합니다.

### 1. 행위자를 모르거나 중요하지 않을 때

```
선호:
"The window was broken."  ← 누가 깼는지 모름
"The patient was admitted yesterday."  ← 누가 admit했는지 무관
"Korean is spoken in Korea."  ← 행위자는 명백
```

### 2. 대상이 주제일 때

```
선호:
"The dramatists of the Restoration are little esteemed today."
→ 극작가가 주제 (수동태가 맞음)

"Modern readers have little esteem for the dramatists of
the Restoration."
→ 독자가 주제 (능동태)
```

문맥에서 무엇을 강조하느냐가 결정합니다.

### 3. 과학·학술 글

학술·과학 글에서는 수동태가 표준인 자리가 있습니다.

```
선호 (Methods 섹션):
"The samples were heated to 100°C."
"The data were analyzed using SPSS."
"The experiment was conducted in triplicate."
```

이유:
- 연구자가 누구인지보다 무엇을 했는가가 중요
- 객관성의 인상

그러나 현대 학술 글에서는 능동태 사용도 점차 권장됩니다.

```
선호 (현대 학술):
"We heated the samples to 100°C."
"We analyzed the data using SPSS."
```

자기 분야의 관습을 확인합니다.

### 4. 법률·격식 글

```
선호 (법률):
"The defendant is hereby ordered to ..."
"Notice is given that ..."
"The motion was denied."
```

법률 글의 전통적 형식입니다. 행위자가 법이므로 명시가 불필요합니다.

## 능동태로 바꾸면 — 더 짧고 강함

수동태를 능동태로 바꾸면 문장이 보통 짧아지고 강해집니다.

```
수동 (16 단어):
"There were a great number of dead leaves lying on the ground."

능동 (5 단어):
"Dead leaves covered the ground."

→ 문장 길이 절반 이상 줄어듦, 의미 강해짐
```

```
수동:
"The reason he left college was that his health became impaired."

능동:
"Failing health compelled him to leave college."
```

문장이 강해지면 보통 짧아집니다. 간결은 활기의 부산물입니다.

## *There Is / Are* 패턴

`There is / there are` 구조는 능동태이지만 수동태처럼 약합니다.

```
회피 (there is):
"There is a great need for reform."
"There are many people who disagree."

선호 (강한 능동):
"Reform is greatly needed."  (또는 능동: "We greatly need reform.")
"Many people disagree."
```

`There + be`를 진짜 주어가 있는 능동태로 바꾸면 문장이 살아납니다.

## 명사화와 수동태

학술·비즈니스 글의 또 다른 문제는 명사화입니다. 동사를 명사로 바꿔 약하게 만드는 패턴입니다.

```
회피 (명사화 + 수동태):
"An investigation of the problem was conducted by the team."

선호 (능동 + 동사):
"The team investigated the problem."
```

```
회피:
"A decision regarding the matter was reached by the committee."

선호:
"The committee decided on the matter."
```

명사화 + 수동태는 최악의 조합입니다. 동사 + 능동으로 바꿉니다.

## 능동태가 맞는 일반 자리

- 일반 에세이
- 블로그 글
- 보고서 / 메모
- 이메일
- 뉴스 기사 (대부분)
- 광고 / 카피
- 소설 / 단편

```
능동:
"The team won the championship."
"She wrote a beautiful poem."
"The committee approved the plan."
```

## 한국어의 능동 / 수동

한국어도 능동태가 자연스럽습니다. 수동태가 번역체로 흔히 등장합니다.

```
회피 (영어식 번역체):
"그 결정은 위원회에 의해 내려졌다."
"많은 책이 그에 의해 읽혔다."
"문제가 해결되었다." (모호함)

선호 (한국 자연):
"위원회가 그 결정을 내렸다."
"그는 많은 책을 읽었다."
"우리는 문제를 해결했다."
```

영어 *be + 과거분사*를 피동 표현(-되다, -지다)으로 직역하면 어색합니다. 능동으로 재배치하는 편이 좋습니다.

### 한국어의 피동 남용

```
회피 (피동 남용):
"이 책은 많이 읽혀집니다."  (이중 피동)
"문제가 해결되어집니다."   (이중 피동)
"보고서가 작성되어집니다." (이중 피동)

선호:
"이 책은 많이 읽힙니다."
"문제가 해결됩니다."  / "우리가 문제를 해결합니다."
"보고서를 작성합니다."
```

*-되어지다* 같은 이중 피동은 한국어 오류입니다. 능동이나 단순 피동으로 수정합니다.

## Master Writer의 능동태

```
Hemingway:
"The old man caught the fish."
- 짧은 능동태, 강한 동사

Orwell ("Politics and the English Language"):
"수동태는 종종 의도된 모호함의 도구다."
- 명확한 글의 첫 번째 규칙 — 능동태

Strunk:
"Use the active voice."
- 가장 자주 인용되는 규칙
```

## 자기 글 점검

- [ ] 자기 글의 수동태 비율은? → 일반 글에서 10% 이하가 목표
- [ ] 수동태가 정당한가? (행위자 불명, 대상이 주제, 학술)
- [ ] "There is / there are" 패턴이 과다하지 않은가?
- [ ] 명사화 + 수동태 조합이 없는가?
- [ ] 한국 글에 영어식 피동이 직역되지 않았는가?
- [ ] *-되어지다* 같은 이중 피동이 없는가?
