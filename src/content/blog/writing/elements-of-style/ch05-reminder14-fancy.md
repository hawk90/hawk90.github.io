> Avoid fancy words.

*뽐내거나 거만하거나 귀엽거나 너무 영리한* 단어를 피하세요. 10센트짜리 단어가 옆에 있는데 *20달러짜리 단어*를 꺼낼 이유가 없습니다.

## 화려한 단어 vs 평범한 단어

```
화려함                평범함
utilize         →    use
ascertain       →    find out
endeavor        →    try
expedite        →    speed up
commence        →    start, begin
terminate       →    end, stop
purchase        →    buy
inquire         →    ask
furnish         →    give, provide
conceptualize   →    think of
```

대부분의 자리에서 *평범한 단어*가 더 강합니다.

## Anglo-Saxon vs Latin

영어 단어는 *두 출신*을 가집니다.

```
Anglo-Saxon:                Latin (격식):
strong                       robust
big                          substantial
help                         assist
ask                          inquire
end                          terminate
buy                          purchase
do                           perform
use                          utilize
```

> Anglo-Saxon은 Latin보다 더 *생생*합니다.

Anglo-Saxon 단어는 *감각적*이고 *직접적*입니다. Latin 단어는 *추상적*이고 *멀게* 느껴집니다.

## 단어의 좋고 나쁨

White가 중요하게 짚는 점:

> 단어 하나하나에 *좋고 나쁨이 정해져 있지는 않습니다*.

같은 단어가 *어떤 자리*에서는 좋고 *다른 자리*에서는 어색합니다.

```
"utilize"가 자연스러운 자리 (드물게):
- 정확한 의미 전달이 필요한 자리
- 기술 / 학술의 표준 용법
- "use"와 *미묘하게 다른 의미*가 필요할 때

"utilize"가 어색한 자리 (대부분):
- 평범한 일상 행위
- 친근한 글
- "use"로 충분한 자리
```

자기 *귀*가 결정합니다.

## 화려한 단어를 좋아하는 사람

White의 통찰:

> *Beauteous*, *curvaceous*, *prodigious*가 매력적으로 들리는 사람은 이 reminder를 *어길 가능성*이 큽니다.

화려한 단어에 끌리는 작가가 있습니다. 자기가 *영리하게 보일 것 같은* 단어를 선호하는 경향. 이런 경향이 있다면 *의식적으로 의심*해야 합니다.

### 귀 다듬기

좋은 단어 감각은 *책을 많이 읽어* 만들어집니다.

```
- 좋은 작가의 글을 자주 읽음
- 어떤 단어가 어디서 잘 쓰이는지 *관찰*
- 자기 글을 *큰 소리로* 읽음
- 어색한 자리를 *식별*하는 능력 키움
```

## Lincoln의 게티스버그 연설

White가 든 흥미로운 예시:

```
"Four score and seven years ago ..."
```

링컨은 *eighty-seven*이라고 할 수 있었습니다. 그러나 그의 귀가 *fanciness의 경계*까지 가도 좋다고 말했고, 그 결과 *운율*이 살아남았습니다.

```
"Eighty-seven years ago" — 평범, 정보 전달
"Four score and seven years ago" — 시적, 기억에 남음
```

*화려함의 경계*까지 가는 것이 정당한 자리가 있습니다. 다만 *그 자리를 알아야* 합니다.

## 화려한 단어가 정당한 자리

```
1. 정확성이 필요한 자리:
   - 법률 문서 (precise terms)
   - 학술 논문 (technical terms)
   - 시 / 문학 (rhythmic / aesthetic choice)

2. 의도된 효과:
   - 풍자 (mock-formal)
   - 캐릭터의 특성 (학자, 변호사)
   - 분위기 설정 (격식 있는 자리)
```

이 자리들 외에는 *평범한 단어*가 더 좋습니다.

## 한국어의 화려한 어휘

한국어에도 같은 함정이 있습니다.

```
화려함                     평범함
사용한다 (X 아님)            쓴다
검토하다 (격식)              살펴보다
시행하다 (행정)              한다 / 실행한다
획득하다 (행정)              얻다 / 가지다
숙고하다 (학술)              깊이 생각하다
영위하다 (격식)              살다 / 하다
도출하다 (학술)              끌어내다 / 찾다
```

학술 / 행정 글에서는 격식 어휘가 필요합니다. 그러나 *일반 글에 격식 어휘*가 들어오면 어색합니다.

```
회피 (격식이 어울리지 않음):
"저는 어제 친구와 식사를 영위했습니다."

선호:
"저는 어제 친구와 저녁을 먹었습니다."
```

## 적용

```
□ 자기 글에 *Latin 어원 단어*가 너무 많지 않은가?
□ 같은 의미의 *Anglo-Saxon 단어*가 있는가?
□ 평범한 단어로 *충분*하는가?
□ 화려한 단어가 *정당한 자리*인가?
□ 한국어 글에서 *격식 어휘 과용* 없는가?
□ 자기 *귀*로 어색함을 식별할 수 있는가?
```
