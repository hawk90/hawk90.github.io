> Write with nouns and verbs.

형용사와 부사가 아니라 *명사와 동사*로 씁니다. 약하거나 부정확한 명사를 형용사가 구해내지 못합니다.

## 형용사의 함정

```
약함:
"a very tall, extraordinarily slender, somewhat eccentric man"

강함:
"a giraffe of a man"
```

여러 형용사보다 *하나의 정확한 명사 / 비유*가 강합니다.

```
약함:
"He walked slowly and carefully."

강함:
"He tiptoed."
```

*형용사 + 부사*보다 *정확한 동사*가 강합니다.

## Allingham의 시 — 명사의 힘

> Up the airy mountain,
> Down the rushy glen.

*Airy*와 *rushy*가 색을 더하지만, 명사 *mountain*과 *glen*이 시의 *골격*입니다. 명사를 *fountain*과 *valley*로 바꾸면 시가 *완전히 달라집니다*.

```
형용사를 바꿔 보면:
   Up the windy mountain  → 비슷한 분위기
   Up the snowy mountain  → 다른 계절
   Up the steep mountain  → 다른 정서

명사를 바꿔 보면:
   Up the airy hill        → 작은 규모
   Up the airy peak        → 더 극적
   Up the airy summit       → 정점
```

명사가 글의 *기본 구조*를 정합니다.

## 동사의 힘

좋은 동사가 *글의 활기*를 만듭니다.

```
약한 동사:
"He went to the store quickly."

강한 동사:
"He raced to the store."
"He dashed to the store."
"He hurried to the store."
"He sprinted to the store."
```

각 동사가 *다른 그림*을 그립니다. *Went*는 색깔이 없습니다.

### Be 동사의 함정

```
약함:
"There is a problem in the system."
"The system has a problem."

강함:
"The system has crashed."
"A problem plagues the system."
```

*Be* 동사와 *have* 동사는 색깔이 없습니다. 더 *동사다운 동사*로 바꿀 수 있는지 점검합니다.

## 형용사와 부사 — 가끔의 도구

형용사와 부사가 *나쁜 것은 아닙니다*. *과용*이 나쁩니다.

```
좋은 형용사 사용:
"a giraffe of a man"   ← 비유가 정확
"a steely look"        ← 한 단어로 그림
"a tired smile"        ← 단순한 두 단어가 분위기

나쁜 형용사 사용:
"a very tall, very thin, very pale man"   ← 나열만
"absolutely, totally, completely wrong"    ← 강조 중복
```

좋은 형용사는 *정확*하고 *희소*합니다.

## 명사와 동사로 쓰는 실습

다음 단락을 점검합니다.

### 약한 버전 (형용사 / 부사 위주)

```
The very large, very old building was empty and silent. The lonely
old man walked slowly and sadly through the dark and dusty halls.
```

### 강한 버전 (명사 / 동사 위주)

```
The building, a relic of the war, stood empty. An old man wandered
its halls, his footsteps echoing in the dust.
```

같은 그림이지만 두 번째가 *훨씬 더 본다*는 느낌을 줍니다. *형용사를 명사로*, *부사를 동사로* 변환했습니다.

## 자기 글 점검

자기 글에서 다음을 찾아봅니다.

```
1. *Be* 동사가 많은가? (is, are, was, were)
   → 더 강한 동사로 대체 가능?

2. 형용사가 *연속*으로 등장하는가?
   → 그 중 하나를 *명사 / 동사*로 바꿀 수 있는가?

3. 부사가 *동사 옆*에 자주 오는가?
   → 부사 + 약한 동사를 *강한 동사 하나*로?

4. *Very, really, quite, somewhat* 같은 강조 부사가 있는가?
   → 강조 없이 강한 단어로?
```

## 한국어 글에서

한국어 글에서도 같은 원리. 한국어는 영어보다 *조사*가 많지만 명사와 동사의 *핵심성*은 같습니다.

```
약함: 매우 크고 매우 오래된 건물이 비어 있고 조용했다.
강함: 전쟁의 잔해 같은 그 건물이 비어 서 있었다.
```

*매우, 매우*를 빼고 *비유 명사 (전쟁의 잔해)*로 대체했습니다.
