## O–P

### Noun used as verb

*명사를 동사화*하는 흐름은 늘고 있지만, *모두 좋은 동사는 아닙니다*. `Host`, `gift`, `chair`, `debut`을 *동사로 쓰는 것*은 *의심스러운 사례*입니다.

```
회피:
"Be prepared for kisses when you gift your girlfriend with
this scent."

선호:
"Be prepared for kisses when you give your girlfriend this
merry scent."
```

```
의심스러운 동사화:
gift → give
host → hold (a meeting)
chair → preside over
debut → make a debut, introduce
impact → affect
parent → raise
medal → win a medal
journal → write in a journal
```

### Offputting / Ongoing

*Offputting*은 부적절. *Ongoing*은 과용. 적절한 한국어 같은 영어 표현으로.

```
회피                선호
offputting          disturbing / repellent
ongoing             continuing / current
ongoing project     current project / project under way
```

### One

*A person*의 의미일 때 `his` / `her`로 받지 *않습니다*. `One ... one's`로 *일관*.

```
회피:
"One must watch his step."

선호:
"One must watch one's step."
```

```
미국 회화는 *his / her*도 받아들이지만, 학술 / 격식 글은 *one's*로 유지.

영국 영어는 더 엄격: 항상 *one ... one's*.
```

### One of the most

*진부한 공식*. 문법에 문제는 없지만 *닳고 닳았습니다*.

```
회피 (진부):
"He is one of the most talented writers of our time."
"This is one of the most important issues we face."

선호 (단언):
"He is among the most talented writers of our time."
또는: "He is a leading writer of our time."
또는: "His writing is exceptional."
```

### -oriented

*거만하고 모호한 도구*. *방향*이나 *지향*을 말하는 *더 나은 표현*을.

```
회피:
"It was a manufacturing-oriented company."
"A goal-oriented person."
"A customer-oriented approach."

선호:
"It was a company chiefly concerned with manufacturing."
"A purposeful person." / "A driven person."
"An approach focused on customers."
```

### People

*정치적 용어*로의 *people*과 *일반 다수*로의 *public*은 다릅니다. *단순한 개수*에는 `persons`를 씁니다.

```
선호 (정치):
"We the people of the United States ..."
"The people demanded reform."

선호 (일반 다수 / public):
"The public was outraged."

선호 (단순 개수, 격식):
"Three persons were injured."
   ← 격식 글, 법률 문서
"Three people were injured."
   ← 일반 글, 신문
```

### Personalize

*쓰지 마세요*. *Make it good* 한 마디면 충분.

```
회피:
"We need to personalize the experience."
"Let's personalize our messaging."

선호:
"We need to tailor the experience to each customer."
"Let's adjust our messaging for different audiences."
```

### Possess

*Have, own*보다 *인상적으로 들리려는* 무의식적 선택일 때가 많습니다.

```
회피:
"She possessed great courage."
"He possessed many books."

선호:
"She had great courage."
또는: "She was very brave."
"He had many books." / "He owned many books."
```

```
선호 (적절한 possess):
"She is possessed by the idea."  ← 사로잡힘
"Demons possessed him."           ← 영적 점령
```

### Prestigious

*Famous*나 *important*보다 *과장된 인상*. *과용*되면 *의심*을 부릅니다.

```
회피 (과장):
"He won a prestigious award."
"A prestigious university."

선호:
"He won an important award." / "He won a major award."
"A renowned university." / "A well-known university."
```

### Presently

*두 의미*가 있어 *모호*합니다.

```
의미 1 (in a short while): 곧 — *가장 안전*
"I will be there presently."

의미 2 (currently): 지금 — *피하세요*
회피: "We are presently working on it."
선호: "We are now working on it."
또는: "We are working on it."
```

## R

### Refer / Allude

`Allude`와의 차이를 *다시 확인*. `Refer`는 *직접 언급*, `Allude`는 *간접 언급*.

```
Refer (직접):
"He referred to the new law."
   ← 명시적으로 언급

Allude (간접):
"He alluded to recent events."
   ← 직접 말하지 않고 암시
```

### Regretful / Regrettable

```
Regretful → 후회하는 *사람*
"She felt regretful about her decision."

Regrettable → 유감스러운 *일*
"The mixup was due to a regrettable breakdown
in communications."
```

```
회피:
"The mixup was due to a regretful breakdown ..."
   ← 통신이 후회를 느낀다?

선호:
"The mixup was due to a regrettable breakdown ..."
```

### Relate

*Tell*이나 *narrate*의 단순 대체로 쓰지 마세요. 또한 *can relate to*는 회화체.

```
회피: "I can relate to your feelings."
선호: "I understand your feelings." / "I share your feelings."

선호 (적절한 relate):
"He related the story of his journey." (이야기를 했다)
"This problem relates to the larger issue." (연관됨)
```

### Respective / Respectively

*대개 빼고* 더 자연스럽게 *다시 쓰는 게 낫습니다*.

```
회피 (과잉 격식):
"John, Mary, and Tom went to their respective homes."

선호:
"John, Mary, and Tom went to their own homes."
또는: "John, Mary, and Tom each went home."
```

```
회피:
"They were paid $500 and $700, respectively."

선호 (재배치):
"John was paid $500, and Mary was paid $700."
```

*Respectively*는 *목록 순서* 매칭이 필요할 때만. 그 외에는 *쓰지 않습니다*.

## S

### Same

*The same*을 회피 / 격식 도구로 *쓰지 마세요*.

```
회피 (관료적):
"Your letter has been received. Same will be answered shortly."

선호:
"Your letter has been received. It will be answered shortly."
```

### Secondly, Thirdly, etc.

*First, second, third*가 더 직접적. *Firstly, secondly*는 *과잉 격식*.

```
회피: "Firstly, ... Secondly, ... Thirdly, ..."
선호: "First, ... Second, ... Third, ..."
```

### Shall / Will

*형식적 글*에서 *미래*는 *I shall*, *you / he will*. *결심·약속*은 *I will*. *회화*에서는 이 구분이 거의 *사라졌습니다*.

```
격식 (전통):
"I shall return."          (미래)
"You will see him."        (미래)
"I will do it!"            (결심, 강조)
"You shall obey!"          (강제, 강조)

회화 (현대):
"I will return." (격식 구분 없음)
"I'll be there."
```

미국 영어는 *shall*을 거의 안 씁니다. *영국 영어*는 *형식*에 따라 남아 있음.

### So

*강조 부사*로 *쓰지 마세요* — *so good*, *so warm*, *so delightful*.

```
회피 (회화):
"It was so good."
"She was so happy."
"The dinner was so delicious."

선호 (격식):
"It was very good." / "It was excellent."
"She was very happy." / "She was overjoyed."
"The dinner was excellent."

선호 (so + that):
"It was so good that everyone wanted more."
   ← so + that 구조는 OK
```

### Split infinitive

`to + adverb + verb` 형태는 *가능한 한* 피합니다. 다만 *분리하면 의미가 더 분명*해지는 경우는 *분리*합니다.

```
회피 (의미 모호):
"to diligently inquire"

선호:
"to inquire diligently"
"diligently to inquire"
```

```
선호 (분리가 의미를 살림):
"to more than double the output"
   ← "to double more than the output"은 어색
```

*Star Trek*의 유명 문장: *"to boldly go where no man has gone before"*. *Split infinitive*이지만 *수용*되었습니다.

현대 영어 문법학자 대부분이 *split infinitive 규칙*을 *과거의 미신*으로 봅니다. 그러나 *Strunk-White 학파*는 *피하는 편*.

### State

*Say, remark*의 단순 대체어로 *쓰지 마세요*. *Express fully or clearly*의 의미일 때만.

```
회피 (단순 say 대체):
"He stated that he was tired."

선호:
"He said that he was tired."

선호 (적절한 state):
"The contract states the terms clearly."
   ← 명확한 진술
"He stated his position." (공식 입장 표명)
```

### Student body

*거의 항상 students*가 더 낫습니다.

```
회피: "The student body voted on the proposal."
선호: "The students voted on the proposal."
```

비슷한 패턴:
```
회피                  선호
the student body  →   students
the female sex   →    women
mankind          →    people / humanity
the human race   →    humans / people
```

## 한국 학습자의 O~S 흔한 함정

```
영어                    한국 학습자 실수
─────────────────────────────────────
gift (동사)             → give
host (동사)             → hold
impact (동사)           → affect
one / his               → one / one's
-oriented               → focused on / concerned with
people / persons        → 격식에서는 persons
personalize             → tailor / customize
possess                 → have
presently               → 두 의미 혼동
regretful / regrettable → 사람 vs 일
respective              → 거의 항상 제거 가능
shall / will            → 미국 영어는 will
so (강조)               → 격식 글에서는 very
split infinitive        → 가능한 분리 피함
state                   → 단순 say 대체로 사용
student body            → students
```

## 자기 글 점검

```
□ 명사를 *함부로 동사화*하지 않았는가?
□ "One"의 받음이 *one's*인가? (his / her 아님)
□ "One of the most" 진부 패턴 자제?
□ "-oriented" 같은 모호 형용사 *구체화*?
□ "Personalize, possess" 자제?
□ "Prestigious, presently" 과장 자제?
□ "Regretful / regrettable" 구분?
□ "Respective" 거의 항상 *제거*?
□ "Firstly" → "First"
□ "So good, so warm" → very / excellent
□ Split infinitive 자제 (의미가 살아나면 OK)
□ "State"가 *진짜 격식 진술*인가?
□ "Student body" → students
```
