## D

### Data

`Data`는 *복수형* (strata, phenomena, media처럼). 단수로 쓰는 용법도 점차 받아들여지지만, *형식적인 글*에서는 *복수*로 다룹니다.

```
선호 (격식 / 학술):
"The data are misleading."
"These data show ..."

회화 (받아들임):
"The data is interesting."
```

```
관련 라틴어 / 그리스어 복수:
datum     → data (라틴어)
stratum   → strata (라틴어)
phenomenon → phenomena (그리스어)
medium    → media (라틴어)
criterion → criteria (그리스어)
```

학술 글에서는 *복수 형식*이 표준.

### Different than / Different from

`Different from`이 표준. 비교의 `other than`, `unlike`와 같은 계통.

```
선호: "This book is different from that one."
회피: "This book is different than that one." (미국 회화에 흔하지만 비표준)
```

### Disinterested / Uninterested

`Disinterested` — *공정한, 편향 없는*. `Uninterested` — *관심 없는*. *완전히 다른 의미*.

```
"Let a disinterested person judge our dispute."
   ← 공정한 판결자 (이해 관계 없는)

"This man is obviously uninterested in our dispute."
   ← 무관심
```

한국 학습자가 자주 *동의어*로 오해. *Dis-*가 *not*이 아니라 *fair*에 가까운 자리.

### Doubt that / Doubt whether

```
긍정문 → doubt whether
"I doubt whether he will come."

부정문 → doubt that
"I don't doubt that he will come."
```

### Due to

부사구로서 *because of*, *through*, *owing to*의 의미로 *함부로 쓰지 마세요*. *Attributable to* 의미일 때만 정확.

```
회피 (부사구):
"He lost the first game due to carelessness."

선호:
"He lost the first game because of carelessness."

선호 (attributable, 형용사구):
"The accident was due to bad weather."
   ← 사고가 = 날씨 *때문*
```

### During / While

```
During + 명사:
"During the meeting, he took notes."

While + 절:
"While he was at the meeting, he took notes."
```

## E

### Effect

```
명사 (결과):
"The new law had a positive effect on the economy."

동사 (이루다, 가져오다):
"The committee effected major changes."

cf. Affect (동사, 영향을 끼치다):
"The weather affects my mood."
```

*Affect / Effect*는 *학습자가 가장 자주 틀리는* 자리.

### Either

*Or*와 함께 쓰일 때만 *둘 중 하나*. 셋 이상에는 부적합.

```
선호: "Either A or B" (둘 중 하나)
회피: "Either A, B, or C" → "any of A, B, or C"
```

### Enormity

*Monstrous wickedness* (거대한 악)의 의미. *Hugeness*의 뜻으로 *쓰지 마세요*.

```
선호: "The enormity of the crime shocked the nation."
   ← 범죄의 거대한 악

회피: "The enormity of the building impressed me."
   → "size", "vastness" 등으로
```

### Etc.

*And so forth*의 라틴어 약자. *Such as*, *for example* 같은 표현으로 시작한 리스트의 *끝에는 쓰지 않습니다*. 형식적인 글에서는 *가능한 한 피합니다*.

```
회피 (중복):
"Such as apples, oranges, etc."  ← such as와 etc. 중복

선호:
"Such as apples, oranges, and bananas."
또는: "Apples, oranges, etc."
```

### Everyone / Every one

```
한 단어 (everyone) = everybody:
"Everyone is invited."

두 단어 (every one) = each individual:
"Every one of the books is signed."
```

## F

### Fact

*직접 검증 가능한 일*에만 씁니다. *판단이나 견해*에는 쓰지 마세요.

```
선호 (사실):
"It is a fact that water boils at 100°C."
"The fact is, the meeting ended at noon."

회피 (의견):
"It is a fact that Napoleon was the greatest of modern generals."
   ← *의견*. 사실 아님.

수정:
"In my view, Napoleon was the greatest of modern generals."
```

### Facility

*감옥, 학교, 병원*을 굳이 *facility*로 부르는 *관료적 습관*을 피합니다.

```
회피: "He has been appointed warden of the new facility."
선호: "He has been appointed warden of the new prison."

회피: "The medical facility opened last week."
선호: "The hospital opened last week."
```

*Facility*는 *모호*. *구체 명사*가 *강함*.

### Factor

*진부한* 단어. *더 직접적인 표현*으로.

```
회피:
"Her superior training was the great factor in her winning
the match."

선호:
"She won the match by being better trained."
```

```
회피: "Several factors contributed to the outcome."
선호: "Three things led to the outcome: ..."
```

### Farther / Further

```
Farther → 거리:
"You chase a ball farther than the other fellow."
"The store is farther down the road."

Further → 시간 / 정도 / 추상:
"You pursue a subject further."
"We will discuss this further."
"Further developments are expected."
```

미국식 영어는 *farther = 물리적, further = 비유*. 영국식은 *섞임*.

### Fewer / Less

```
Fewer + 셀 수 있는 명사:
"Fewer people attended the meeting."
"Fewer than ten students passed."

Less + 셀 수 없는 명사:
"Less water remained in the bucket."
"Less than ten percent agreed."
```

```
회피: "Less than ten students passed."
선호: "Fewer than ten students passed."

회피: "Ten items or less" ← 슈퍼마켓에 흔하지만 비표준
선호: "Ten items or fewer"
```

### Finalize

*거만하고 모호한* 동사. *Conclude*, *complete*, *terminate*로 *바꿔 쓰세요*.

```
회피: "We need to finalize the plans."
선호: "We need to complete the plans."
또는: "We need to settle the plans."
```

비즈니스 영어의 *전형적 군더더기*. 학술 / 격식 글에서는 *피하세요*.

### Fix

미국에서는 *arrange, prepare, mend*의 *구어*. 본래 *figere*(굳히다, 위치를 정하다)에서 왔으므로 *형식적 글*에서는 *본래 의미*로만 씁니다.

```
구어 (회화):
"I'll fix dinner." (준비하다)
"Fix your hair." (정돈하다)
"Fix the broken chair." (수리)

격식 (본래 의미):
"Fix the time of departure." (정하다)
"The image was fixed in his memory." (굳히다)
```

격식 글에서는 *prepare, repair, set* 등이 더 명확.

### Flammable / Inflammable

*같은 의미*. *Inflammable*은 *접두사 in-*이 *부정*으로 오해될 우려가 있어 *flammable*이 안전 표지에 사용됩니다.

```
선호 (안전 표지):
"FLAMMABLE — Keep away from flame"
```

### Folk

*People*의 의미로 *단수형 folk*만 씁니다. *Folks*(가족, 부모)는 *너무 구어적*.

```
회피 (구어):
"Her folks arrived by the afternoon train."

선호 (격식):
"Her father and mother arrived by the afternoon train."
또는: "Her parents arrived by the afternoon train."
```

### Fortuitous

*우연한*의 의미. *Fortunate, lucky*와 *다릅니다*.

```
"The meeting was fortuitous."
→ 우연히 일어났다 (의도하지 않음)

"The meeting was fortunate."
→ 운이 좋았다 (긍정적)

회피 (혼동):
"It was a fortuitous outcome." (운 좋다는 의미면 fortunate)
선호: "It was a fortunate outcome."
```

*Chance*의 의미일 때만 *fortuitous*. *Good luck*의 의미면 *fortunate*.

### Free gift

*Gift*는 이미 *공짜*. *Free*는 *군더더기*.

```
회피: "Free gift with purchase!"
선호: "Gift with purchase!"
또는: "Free with purchase!"
```

## 한국 학습자의 D~F 흔한 함정

| 영어 | 한국 학습자 실수 |
|------|------------------|
| data is | 형식 글에서는 "data are" |
| different than | "different from" |
| disinterested | "uninterested" 대체로 오해 |
| due to | 부사구로 오용 (because of로) |
| effect/affect | 동사 / 명사 혼동 |
| enormity | "거대함"으로 오해 (악의 의미) |
| facility | 구체 명사로 바꾸기 |
| fewer/less | 셀 수 / 없는 구분 안 함 |
| finalize | 비즈니스 군더더기 |
| fortuitous | "운 좋은"으로 오해 |

## 자기 글 점검

- [ ] "Data are"인가 "data is"인가 (격식 글에서는 are)
- [ ] "Different from" -- than 아님
- [ ] "Disinterested" -- 공정한 / "uninterested" -- 무관심
- [ ] "Due to" -- attributable일 때만 부사구 사용 자제
- [ ] "Affect"(동사)와 "Effect"(명사/동사) 구분
- [ ] "Enormity" -- 거대한 악, 단순한 크기 아님
- [ ] "Facility" -- *prison, hospital* 같은 구체로
- [ ] "Factor" -- 더 직접적 표현으로
- [ ] "Fewer"(셀 수) / "Less"(셀 수 없음)
- [ ] "Finalize" -- *complete, conclude*로
- [ ] "Fortuitous" -- 우연 / "Fortunate" -- 운
