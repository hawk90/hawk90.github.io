## T

### Than

`Than`이 들어간 비교문에는 *생략된 단어가 명확한지* 확인.

```
모호:
"I'm probably closer to my mother than my father."

→ 두 해석 가능:
1. ... than [I am to] my father.  (어머니와 가까운 것이 아버지와 가까운 것보다)
2. ... than my father [is].         (어머니와 가까운 정도가 아버지가 어머니와 가까운 정도보다)

명확:
"I'm probably closer to my mother than to my father."
"I'm probably closer to my mother than my father is."
```

Rule 10에서도 다룬 *than I / than me* 격 결정과 함께 점검.

### That / Which

`That` — *defining (제한적)*, *restrictive*. `Which` — *nondefining (비제한적)*, *nonrestrictive*. Rule 3과 같은 원리.

```
"The lawn mower that is broken is in the garage."
   ← 부서진 잔디 깎이 (여러 대 중 그것)
   ← 제한적, that, no comma

"The lawn mower, which is broken, is in the garage."
   ← 그 잔디 깎이 (한 대), 게다가 부서졌다
   ← 비제한적, which, commas
```

*조심스러운 작가*는 *defining which*를 *that*으로 바꾸는 *사냥*을 자주 합니다.

```
회피 (defining which):
"The book which you sent me is interesting."

선호:
"The book that you sent me is interesting."
또는: "The book you sent me is interesting." (that 생략 가능)
```

미국 영어는 *that / which* 구분이 더 엄격. 영국 영어는 *섞임*.

### The fact is... / The truth is...

*문장을 시작하는 약한 도입*입니다. *그냥 사실을 말하세요*. *청구서를 미리 보내는 격*입니다.

```
회피:
"The fact is, the meeting was canceled."
"The truth is, we ran out of time."
"To tell the truth, I don't know."

선호:
"The meeting was canceled."
"We ran out of time."
"I don't know."
```

### They / He or She

선행사가 `each`, `everyone`, `every one`, `many a man`일 때 `they`로 받지 *마세요*. *단수 대명사*를 씁니다.

```
회피:
"Everyone in the community, whether they are a member of the
Association or not, is invited."

선호 (전통):
"... whether he is a member of the Association or not, is
invited."
```

```
현대 문제: he가 *대표 단수*로 어색
- 성중립 / 포괄성 요구

해결책 (Strunk-White 제안):
1. 복수 주어로 바꾸기:
   "All members of the community, whether they are members of
   the Association or not, are invited."

2. 대명사 빼기:
   "Everyone is invited, member or not."

3. He or she (격식):
   "... whether he or she is a member ..."  ← 길고 어색

4. Singular they (현대 받아들임):
   "... whether they are a member ..."  ← 점차 표준
```

```
선호 (현대, 다양한 옵션):
"The writer must address his readers' concerns."  (전통)
→ "Writers must address their readers' concerns."  (복수)
→ "The writer must address readers' concerns."  (대명사 빼기)
→ "As a writer, you must address your readers' concerns."  (2인칭)
→ "The writer must address their readers' concerns."  (singular they, 현대)
```

### This

*선행 문장이나 절 전체*를 가리키는 `this`는 *모호*할 수 있습니다. *가리키는 것*을 *명사구*로 *풀어 쓰는 게 안전*.

```
회피 (모호):
"The team won the game and the coach was promoted. This was
unexpected."

→ "This"가 무엇? 승리? 승진? 둘 다?

선호 (명확):
"The team won the game and the coach was promoted. This
promotion was unexpected."

또는: "The team won the game and the coach was promoted.
The coach's promotion was unexpected."
```

### Thrust

권력 · 성 · 과시의 *뉘앙스*가 있는 명사로, *광고와 정치 화법*의 단골. *구체적 의미*에서만 사용.

```
회피 (모호):
"The thrust of the policy is ..."
"The main thrust of his argument ..."

선호:
"The main point of the policy is ..."
"The main point of his argument is ..."
```

### Together with

`As well as`, `with` 등과 같이 *주어의 수를 바꾸지 않습니다* (Rule 9).

```
선호:
"The teacher, together with the students, was at the rally."
   ← 주어는 "teacher" 단수
```

### Tortuous / Torturous

```
Tortuous → 구불구불한
"A tortuous mountain road."

Torturous → 고통스러운
"A torturous experience."

회피 (혼동):
"The interview was tortuous." (고통의 의미면 torturous)
선호: "The interview was torturous."
```

### Transpire

*Happen, come to pass*의 의미로 *쓰지 마세요*. 원뜻은 *become known* (알려지다).

```
선호 (본래 의미):
"It transpired that he had been lying."
   ← 알려졌다

회피 (잘못된 의미):
"What transpired at the meeting?"
   → "What happened at the meeting?"가 표준
```

현대 영어는 *happen* 의미를 받아들이지만, 격식 글에서는 *주의*.

### Try

`Try to`. *Try and*는 *회화*로 굳어졌지만 *형식적 글*에서는 `try to`.

```
선호 (격식):
"Try to be on time."
"He will try to finish by Friday."

회화:
"Try and be on time." ← 회화에서 흔하지만 격식에서는 try to
```

### Type

`Kind of`의 *동의어가 아닙니다*.

```
회피:
"that type employee"

선호:
"that kind of employee"
또는: "that type of employee"
```

*Type + 명사*는 *명사 합성어*. *Type of + 명사*가 일반.

```
선호 (type + 명사):
"a Type A personality"
"a blood type test"

선호 (kind of, sort of):
"What kind of dog?"
"That sort of question."
```

## U–W

### Unique

*Without like or equal*. *정도가 없습니다*. *Very unique*, *most unique*는 *잘못*.

```
회피:
"a very unique opportunity"
"the most unique design"
"a fairly unique experience"

선호:
"a unique opportunity"
"the most distinctive design"
"a rare experience" / "an unusual experience"

→ Unique는 *수식 안 함*. (유일한 것은 정도가 없음)
```

비슷한 *절대 형용사*:
```
absolute formless equal infinite perfect
preferable square supreme total universal
```

이 형용사들은 *very, more, most*로 *수식하지 않음*.

### Utilize

`Use`가 더 좋습니다.

```
회피:
"We need to utilize this resource."
"The team utilized new technology."

선호:
"We need to use this resource."
"The team used new technology."
```

*Utilize*는 *use*의 *과장된 형식*. *비즈니스 영어의 진부어*.

### Verbal / Oral

```
Verbal → *말로든 글로든* (의사소통 일반)
"Verbal communication includes both speech and writing."

Oral → *말로만*
"An oral agreement is not legally binding without writing."

선호:
"An oral agreement" (말로 한 합의)
   ← verbal보다 정확
```

회화에서는 *verbal*이 *oral*의 의미로 자주 사용되지만, *법률 / 격식 글*은 *구분*.

### Very

*남발하지 마세요*. 강조가 필요하면 *그 자체로 강한 단어*를 고르세요.

```
약함:                    강함:
very good        →       excellent / superb
very bad         →       terrible / awful
very big         →       huge / enormous / massive
very small       →       tiny / minute
very fast        →       rapid / swift
very tired       →       exhausted / spent
very happy       →       elated / overjoyed
very sad         →       devastated / heartbroken
very hot         →       scorching / sweltering
very cold        →       freezing / frigid
```

*Very + 평범한 형용사 = 약함*. *강한 형용사 하나 = 강함*.

### Well-known

*Famous, renowned, notorious*보다 *약함*.

```
회피 (약함):
"a well-known author"

선호 (강함):
"a famous author" / "a renowned author"
"a celebrated author"
```

### While

*And, but, although*의 *대체어로 함부로 쓰지 마세요*. 본뜻은 *during the time that*.

```
회피:
"The office is on the ground floor, while the rest of the
building is used for manufacturing."

선호:
"The office is on the ground floor; the rest is used for
manufacturing."
```

```
선호 (본래 의미, during):
"While I was reading, the phone rang."
   ← 시간

선호 (although):
"Although he was tired, he kept working."  ← Although가 더 명확
회피: "While he was tired, he kept working."
```

### Who / Whom

Rule 10에서 다룸. *주격 who*, *목적격 whom*.

```
선호:
"Who is calling?"  (주격)
"Whom did you call?"  (목적격)
"To whom did you give it?"  (전치사 + 목적격)
```

현대 회화에서는 *whom*이 사라지지만, *격식 글*에서는 *유지*.

### Whose

*Of which*의 대체로도 사용됩니다 (사물에도).

```
선호:
"The book whose cover is torn ..."  ← 사물에도 OK
"The dog whose tail wags ..."

대체:
"The book of which the cover is torn ..." ← 길고 어색
```

### -wise

*Taxwise, pricewise, marriagewise* 같은 *임의의 접미사*는 *피합니다*. *Clockwise*처럼 *in the manner of*의 의미로 *굳어진 단어*만 표준.

```
회피 (임의 접미사):
"Taxwise, this is a good year."
"Pricewise, the deal is fair."
"Marriagewise, things are going well."

선호:
"With respect to taxes, this is a good year."
"In terms of price, the deal is fair."
"In our marriage, things are going well."
```

```
선호 (굳어진):
"Turn it clockwise."
"Watch your step otherwise."
```

### Worth while / Worthwhile

*행위*에만 — *Is it worth while to telegraph?*. *형용사 worthwhile*은 *닳은 단어*이므로 *더 강한 형용사*로.

```
회피 (약함):
"a worthwhile project"
"a worthwhile experience"

선호 (강함):
"a promising project" / "a useful project" / "an exciting project"
"a valuable experience" / "a rewarding experience"
```

### Would

*습관적 · 반복적 행동*의 표시. 같은 의미가 *once a year*, *each Sunday* 같은 구절로 *이미 표현*된다면 *과거 시제*가 더 강하고 짧습니다.

```
약함:
"Once a year he would visit the old mansion."

강함:
"Once a year he visited the old mansion."
```

```
회피:
"Every morning he would walk to the park."

선호:
"Every morning he walked to the park."
```

```
선호 (would 정당):
"He would often visit the old mansion."  ← 부사 없으면 would로 반복 표현
"In those days, we would walk for hours."  ← 옛 회상
```

## 정리 — *Words Commonly Misused*

**가장 자주 헷갈리는 쌍:**
- affect / effect
- imply / infer
- lay / lie
- less / fewer
- that / which
- like / as
- fortuitous / fortunate
- regretful / regrettable
- tortuous / torturous
- continuous / continual
- nauseous / nauseated

**가장 자주 군더더기:**
- the fact that
- in terms of
- currently
- he is a man who
- one of the most
- as yet (yet으로)
- in regards to (in regard to로)

**닳은 형용사:**
- meaningful
- nice
- interesting
- worthwhile
- well-known
- prestigious

**거만한 동사:**
- finalize
- contact
- utilize
- personalize
- impact (동사)
- gift (동사)

**피해야 할 접미사:**
- -ize (utilize -- use)
- -ization (utilization)
- -oriented (manufacturing-oriented)
- -wise (taxwise, pricewise -- clockwise는 OK)

**강조 회피:**
- very (강한 단어로 대체)
- so (격식에서는 very)
- literally (잘못된 강조)
- definitely
- absolutely

## 한국 학습자의 T~Z 흔한 함정

| 영어 | 한국 학습자 실수 |
|------|------------------|
| than 비교 | 생략된 단어 모호 |
| that / which | defining에 which 사용 |
| the fact is... | 약한 도입 |
| they (단수 선행사) | 격식 글 비표준 |
| this (모호) | 가리키는 명사 추가 |
| transpire | "happen"로 오용 |
| try and | 격식에는 "try to" |
| unique (very, most) | "Unique"는 절대 형용사 |
| utilize | use |
| very + 약한 형용사 | 강한 형용사 하나 |
| while (= but/although) | "during"이 본 의미 |
| -wise (임의) | 비표준 |
| worthwhile (약함) | 구체 형용사 |
| would (반복) | 단순 과거가 더 강함 |

## Chapter 4 마무리

Chapter 4는 *영어의 흔한 함정 사전*. *60여 단어*가 *학습자가 가장 자주 틀리는* 자리.

권고:
1. 의심되는 단어는 *사전 확인*
2. 격식 글에서는 *전통 규칙* 적용
3. 회화 영어와 격식 영어 *구분*
4. *닳은 단어*는 *구체 단어*로
5. *거만한 단어*는 *직접 단어*로
6. *Show, don't tell*의 정신
