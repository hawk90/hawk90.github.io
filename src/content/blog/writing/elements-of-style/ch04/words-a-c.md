> Words and Expressions Commonly Misused.

Chapter 4는 *자주 잘못 쓰이는 단어 / 표현*의 사전입니다. Strunk가 *학생들의 글에서 본 패턴*을 정리했습니다. 알파벳 순.

## A

### Aggravate / Irritate

`Aggravate`는 *이미 있는 문제를 더 심하게 만들다*. `Irritate`는 *짜증 나게 하다*.

```
The medicine aggravated his symptoms.    ← 증상을 악화시켰다
The noise irritated him.                  ← 짜증 나게 했다

회피 (혼동):
"The dog aggravated me."  ← 짜증의 의미라면 irritate
선호:
"The dog irritated me."
```

한국 학습자 함정: *aggravate*를 *짜증*의 의미로 잘못 사용. *Worsen / make worse*의 뜻이 정확.

### All right

*한 단어 alright가 아니라 두 단어*로 씁니다. 의미는 *agreed*, *go ahead*, *OK*.

```
선호:
"All right, let's go."
"Are you all right?"

회피 (비표준):
"alright" — 신문 / 책에서 받아들이지 않음
```

### Allude / Elude

`Allude`는 *간접 언급*. `Elude`는 *피하다 / 빠져나가다*.

```
She alluded to his earlier remark.   ← 간접 언급
The thief eluded the police.          ← 피했다
```

`Allude`는 `refer`(직접 언급)와도 다릅니다.

```
Refer to → 직접 언급
"In her speech, she referred to the new law."

Allude to → 간접 언급
"In her speech, she alluded to recent events."
```

### Allusion / Illusion

```
Allusion → 간접 언급
"His novel contains many allusions to Greek mythology."

Illusion → 환영 / 헛것
"The magician created the illusion of a floating ball."
```

발음이 비슷해 혼동되지만 *의미가 완전히 다름*.

### Alternate / Alternative

`Alternate`는 *번갈아 가며*. `Alternative`는 *둘 중 선택*.

```
"As the flooded road left them no alternative, they took the
alternate route."

분석:
- "no alternative" → 선택의 여지 없음
- "alternate route" → 우회로 (대체 길)
```

```
선호:
"She works on alternate days." ← 격일
"We have two alternatives: stay or leave." ← 두 선택지
```

### Among / Between

*셋 이상은 보통 among*. *둘이거나, 셋 이상이라도 각자 개별적으로 다룰 때는 between*.

```
선호 (셋 이상, 집단적):
"The money was divided among the four players."

선호 (둘):
"The agreement was between the two countries."

선호 (셋 이상, 개별적):
"An agreement between the six heirs."
   ← 각자가 개별 당사자
```

### And/or

*문장을 망치는 약자*. *의미를 풀어 쓰는 게* 더 명확.

```
회피:
"Would an honor system cut down on the amount of stealing
and/or cheating?"

선호:
"Would an honor system reduce the incidence of stealing or
cheating or both?"
```

법률 / 행정 문서가 아니라면 *피하는 것*이 안전.

### Anticipate

*Expect*의 뜻으로 *함부로 쓰지 마세요*. `Anticipate`는 *예상하고 미리 행동*한다는 뉘앙스.

```
선호 (단순 예상):
"I expected he would look older."

선호 (예상 + 미리 행동):
"He anticipated the upturn in the market."
→ 예상하고 주식을 미리 매수했다는 함의
```

### Anybody / Any body

한 단어면 *any person*, 두 단어면 *any corpse* 또는 *any human form*.

```
"Anybody can do this." ← 누구든 (한 단어)
"They could not find any body in the wreckage." ← 시신 (두 단어)
```

### As good or better than

문장을 *다시 정리*합니다.

```
회피:
"My opinion is as good or better than his."

분석:
- "as good ... than" 불가 (as good as ...)
- "better than" OK
- 두 비교가 *섞임*

선호:
"My opinion is as good as his, or better."
"My opinion is as good as his, if not better."
```

### As to whether

`Whether` 하나로 충분.

```
회피: "the question as to whether"
선호: "the question whether" 또는 "whether"
```

### As yet

`Yet`이 *거의 항상 더 짧고* 같은 의미.

```
"No agreement has been reached as yet."
→ "No agreement has yet been reached."
```

## B

### Being

`Regard ... as` 뒤에 `being`은 *군더더기*.

```
회피: "He is regarded as being the best dancer in the club."
선호: "He is regarded as the best dancer in the club."
```

### But

`Doubt`와 `help` 뒤의 `but`은 *불필요*.

```
회피: "I have no doubt but that he will come."
선호: "I have no doubt that he will come."

회피: "I cannot help but admire him."
선호: "I cannot help admiring him."
```

`But`을 *자주 쓰면* Loose sentence(Rule 18) 함정에 빠지기 쉽습니다.

## C

### Can / May

```
Can → 능력 (할 수 있다)
"He can swim."

May → 허가 / 가능성
"You may leave now." (허가)
"It may rain." (가능성)
```

회화에서는 *can*이 *허가*에도 사용되지만, *격식 글*에서는 *may*.

### Case

대부분 *불필요*.

```
회피: "In many cases, the rooms lacked air conditioning."
선호: "Many of the rooms lacked air conditioning."

회피: "In some cases, the problem can be solved."
선호: "Sometimes the problem can be solved."
```

### Center around

*Center on*이 맞음. 중심은 *어디*에 *있다*.

```
회피: "The story centers around a young boy."
선호: "The story centers on a young boy."
```

### Certainly

*과용*되는 부사. 진짜 확실할 때만.

```
회피 (과용):
"This is certainly an important issue. We are certainly going
to address it. The committee will certainly review the matter."

선호 (간결):
"This is an important issue. We will address it. The committee
will review the matter."
```

### Character

`Acts of a hostile character` → `hostile acts`. 자주 *빼낼 수 있습니다*.

```
회피: "His remarks were of a critical character."
선호: "His remarks were critical."

회피: "The decisions were of a controversial character."
선호: "The decisions were controversial."
```

### Claim (verb)

*Lay claim to*의 의미입니다. `Declare`, `maintain`, `charge`의 대체어로 *쓰지 마세요*.

```
회피: "He claimed that he was innocent."
선호: "He maintained that he was innocent."
또는: "He insisted that he was innocent."

선호 (적절한 claim):
"The pioneers claimed land in the territory." ← 소유권 주장
```

### Clever

사람과 동물에 대해 *의미가 다릅니다*. *Clever horse*는 영리한 말이 아니라 *기질이 좋은 말*.

```
"A clever boy" ← 영리한
"A clever horse" ← 다루기 쉬운, 기질이 좋은
```

### Compare to / Compare with

```
Compare to → 같은 종류가 아닌 것들 사이의 *닮음* 강조 (비유)
"Life has been compared to a pilgrimage."

Compare with → 같은 종류 사이의 *차이* 따짐 (비교)
"Paris may be compared with modern London."
```

### Comprise

*문자 그대로 embrace, include*. *동물원이 동물들을 comprises*하지, *동물들이 동물원을 comprise*하지 않습니다.

```
선호: "A zoo comprises mammals, reptiles, and birds."
선호: "The mammals, reptiles, and birds constitute a zoo."

회피: "A zoo is comprised of mammals, reptiles, and birds."
   → 잘못된 수동태 사용
```

*Comprise = include*. *Constitute = make up*. 방향이 *반대*.

### Consider

*Believe to be*의 의미일 때 `as`가 *따라오지 않습니다*.

```
회피: "I consider him as competent."
선호: "I consider him competent."

선호 (examined, discussed의 의미, as 가능):
"He was considered as a candidate for the position."
   → 후보로 *검토*되었다
```

### Contact (verb)

타동사로서의 *contact*는 *모호하고 거만*하게 들립니다.

```
회피: "I will contact him."
선호: "I will phone him." / "I will email him." / "I will meet him."
```

*Get in touch with*, *look up*, *phone*, *find*, *meet* 중 *더 구체적인 것*을 씁니다.

### Continual / Continuous

```
Continual → 자주 반복되는 (중간에 끊김)
"He had continual interruptions all day."

Continuous → 끊임없이 이어지는
"The hum of the engine was continuous."
```

### Currently

*Now* + 현재 시제와 함께 쓰면 *군더더기*. *빼거나* 더 정확한 시간 표시를.

```
회피: "We are currently reviewing your application."
선호: "We are reviewing your application."

선호 (정확한 시간):
"As of January 1st, we are reviewing your application."
```

## 한국 학습자의 흔한 함정

| 영어 단어 | 한국 학습자 실수 |
|-----------|------------------|
| aggravate | "짜증" 의미로 잘못 사용 (irritate가 맞음) |
| allude / refer | 둘을 같은 의미로 사용 |
| alternative | "대안" 단수가 사실은 복수 의미 가능 |
| comprise | 수동태 "is comprised of" 잘못 사용 |
| contact | 타동사로 거만하게 사용 |
| currently | "지금" 과용 |
| character | 한국어 "성격"의 직역 |

## 자기 글 점검

- [ ] "Aggravate"가 *악화*인가 *짜증*인가? (짜증이면 irritate)
- [ ] "Allude"가 *간접*인가? (직접이면 refer)
- [ ] "Comprise / constitute" 방향이 맞는가?
- [ ] "Currently + 현재 시제" 군더더기 없는가?
- [ ] "Contact"를 *더 구체적 동사*로 바꿀 수 있는가?
- [ ] "Compare to / with" 구분이 맞는가?
- [ ] "Case", "character" 군더더기를 *제거*했는가?
