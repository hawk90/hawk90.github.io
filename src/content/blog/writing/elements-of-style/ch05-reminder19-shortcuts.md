> Do not take shortcuts at the cost of clarity.

조직이나 운동의 이름을 *알려진 약자*로만 쓰지 마세요. *MADD*가 *Mothers Against Drunk Driving*인 줄 모르는 독자가 매분 새로 태어납니다.

## 약자의 가정

작가가 자주 빠지는 함정 — *내가 아는 약자는 모두가 안다*고 가정.

```
회피:
"MADD reports that..." (Mothers Against Drunk Driving?)
"NATO expanded..." (North Atlantic Treaty Organization?)
"the EU directive..." (European Union? End User?)
```

자기 분야 / 자기 나라에서 *익숙한 약자*가 다른 독자에게는 *암호*입니다.

## 좋은 규칙

> 처음에는 풀어서 쓰고, 자리 잡으면 줄여서 쓰세요.

```
첫 등장:
Mothers Against Drunk Driving (MADD) reports that ...

이후 사용:
MADD's recent data show ...
```

이 패턴이 *명료성과 효율*을 모두 살립니다.

## 약자의 효율은 *환상*

약자를 모아 두는 것이 *효율*이라는 환상이 있습니다. 그러나 자주 *잘못된 효율*입니다.

```
효율의 환상:
"NLP를 활용한 LLM 기반 RAG 시스템의 BERT vs GPT 비교"

→ 한 줄에 5개 약자
→ 독자가 멈춰서 각각을 해독해야 함
→ 효율은 *작가*에게만, 독자에게는 *비용*
```

좋은 글은 *작가의 시간*보다 *독자의 시간*을 아낍니다.

## 약자가 적절한 자리

```
1. 같은 약자를 *글에서 반복적*으로 사용
   - 한 글에 10번 이상 등장한다면 약자 OK

2. 표준 약자
   - USA, NASA, NATO, AI
   - 거의 모든 독자가 안다고 가정 가능

3. 자기 분야의 표준
   - 분야 내 독자만 대상이면 OK
   - 분야 외 독자 가능성 있으면 *처음에 풀어 쓰기*

4. 표 / 그림의 라벨
   - 공간 제약
   - 본문에서 한 번 풀어 쓰기
```

## 약자가 부적절한 자리

```
1. 자기 분야 외 독자 대상 글
2. *한 번만* 등장하는 약자
3. *비슷한 약자*가 여러 개인 경우
   - 예: AI(Artificial Intelligence) vs AI(Avian Influenza)
4. 다국적 / 다문화 독자
```

## 인터넷 시대의 약자

오늘날 *인터넷 약자*가 매우 흔합니다.

```
IMO — in my opinion
IIRC — if I recall correctly
TL;DR — too long; didn't read
LGTM — looks good to me
WIP — work in progress
FYI — for your information
ASAP — as soon as possible
FAQ — frequently asked questions
```

이 중 일부는 *전문 분야*나 *비공식 매체*에서 표준이 되었지만, *격식 있는 글*에서는 풀어 쓰는 것이 안전합니다.

## 한국어 글의 약자

한국 글에서도 같은 함정이 있습니다.

```
- 국세청 (잘 알려짐) OK
- 식약처 (덜 알려짐) — 식품의약품안전처로 풀어 쓰기 권장
- 행안부 — 행정안전부
- 산자부 — 산업통상자원부

특수 약자:
- KISS — Korea Institute for ...
- KAIST — 잘 알려짐 OK
- KOSPI — 잘 알려짐 OK
```

자기 분야 한국 약자가 *일반 독자에게 통하는지* 점검합니다.

## 약자가 의미를 *바꾸는* 위험

약자가 *원래 의미*에서 멀어지는 경우.

```
"AIDS" — Acquired Immune Deficiency Syndrome
   → 지금은 그냥 "에이즈"라는 *고유명사*로 사용

"radar" — Radio Detection and Ranging
   → 지금은 일반명사로 사용

"laser" — Light Amplification by Stimulated Emission of Radiation
   → 일반명사
```

이런 *시간이 지난 약자*는 원래 의미를 풀어 쓰지 않아도 OK입니다.

## 적용

```
□ 자기 글의 약자가 *모든 잠재 독자*에게 익숙한가?
□ 처음 등장 시 *풀어 쓰고 괄호 안 약자*?
□ 한 글에서 *5번 이상* 등장하면 약자 OK
□ 비슷한 약자가 *충돌*하지 않는가?
□ 인터넷 약자는 *매체 격식*에 맞는가?
```
