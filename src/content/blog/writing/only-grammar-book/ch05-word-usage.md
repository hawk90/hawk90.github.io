문법이 맞아도 *단어 선택*이 잘못되면 의미가 어긋납니다. 영어에는 비슷하게 들리거나 비슷하게 생긴 단어가 많아 자주 혼동됩니다. 이 챕터는 흔히 혼동되는 단어 쌍을 *알파벳 순*으로 정리합니다.

## Accept / Except

```
Accept (동사) — 받아들이다:
   She accepted the gift.

Except (전치사) — 제외하고:
   Everyone came except John.
```

## Adverse / Averse

```
Adverse (형용사) — 부정적인, 해로운:
   The team faced adverse conditions.

Averse (형용사) — ~을 싫어하는:
   He is averse to taking risks.
```

## Advice / Advise

```
Advice (명사):
   I gave him good advice.

Advise (동사):
   I advise you to wait.
```

## Affect / Effect

(Chapter 4에서도 다룸)

```
Affect (보통 동사) — 영향을 끼치다:
   The decision affected everyone.

Effect (보통 명사) — 결과:
   The effect was immediate.

Effect (드물게 동사) — 이루다:
   They effected significant change.

Affect (드물게 명사, 심리학) — 정서:
   The patient showed flat affect.
```

## All Ready / Already

```
All ready — 모두 준비된:
   The children are all ready to go.

Already — 이미:
   I have already eaten.
```

## All Together / Altogether

```
All together — 다 함께:
   We sang all together.

Altogether — 완전히, 전적으로:
   The project was altogether successful.
```

## Allusion / Illusion

```
Allusion — 간접 언급:
   His speech contained an allusion to Shakespeare.

Illusion — 환영, 착각:
   It was just an optical illusion.
```

## Among / Between

```
Among — 셋 이상:
   The money was divided among the four players.

Between — 둘, 또는 셋 이상이 *개별적*일 때:
   An agreement between the six countries.
```

## Anxious / Eager

```
Anxious — 불안한, 걱정스러운:
   She is anxious about the test.

Eager — 열망하는:
   He is eager to start the new job.
```

회화 영어에서 *anxious*가 *eager*의 의미로 쓰이지만, 격식 글에서는 구분합니다.

## Bad / Badly

```
Bad (형용사) — 나쁜:
   I feel bad about the mistake.

Badly (부사) — 잘못, 심하게:
   He sang badly.

회피: I feel badly.   (느낌 동사는 형용사)
바름: I feel bad.
```

`Feel, look, smell, taste, sound`는 *연결 동사* — 형용사 보어를 받습니다.

## Bring / Take

```
Bring — 화자 쪽으로:
   Bring me the book.

Take — 화자에게서 멀어지는:
   Take this to your sister.
```

## Can / May

```
Can — 능력:
   I can swim.

May — 허락:
   May I leave now?
```

회화에서 *can*이 허락의 의미로 자주 쓰이지만, 격식에서는 *may*.

## Capital / Capitol

```
Capital — 수도, 대문자, 자본:
   Seoul is the capital of Korea.

Capitol — 의회 건물:
   The Capitol building in Washington.
```

## Cite / Site / Sight

```
Cite (동사) — 인용하다:
   She cited three sources.

Site (명사) — 장소, 웹사이트:
   The construction site.

Sight (명사) — 시야, 광경:
   The sunset was a beautiful sight.
```

## Complement / Compliment

```
Complement — 보완하다 / 보완물:
   The wine complements the meal.

Compliment — 칭찬 / 칭찬하다:
   She paid him a compliment.
```

## Comprise / Compose

```
Comprise — 구성하다 (전체가 부분을):
   The orchestra comprises 80 musicians.

Compose — 구성하다 (부분이 전체를):
   80 musicians compose the orchestra.

수동: The orchestra is composed of 80 musicians.
```

`Is comprised of`는 *비표준*. *Comprises* 또는 *is composed of*를 씁니다.

## Continual / Continuous

```
Continual — 빈번한, 반복적:
   Continual interruptions disrupted the meeting.

Continuous — 끊임없는, 중단 없는:
   The continuous flow of the river.
```

## Council / Counsel

```
Council (명사) — 위원회:
   The city council met yesterday.

Counsel (명사) — 조언, 변호사:
   I sought legal counsel.

Counsel (동사) — 조언하다:
   She counsels students.
```

## Disinterested / Uninterested

```
Disinterested — 공정한, 편향 없는:
   A disinterested judge.

Uninterested — 관심 없는:
   He was uninterested in the conversation.
```

회화에서 두 단어가 혼용되지만, 격식 글에서는 구분.

## Emigrate / Immigrate

```
Emigrate — (자기 나라에서) 떠나다:
   She emigrated from Korea.

Immigrate — (다른 나라로) 이주하다:
   She immigrated to Canada.
```

방향이 다릅니다.

## Eminent / Imminent

```
Eminent — 저명한:
   An eminent scholar.

Imminent — 임박한:
   A storm was imminent.
```

## Farther / Further

```
Farther — 물리적 거리:
   The store is farther than I thought.

Further — 추상적 진전, 정도:
   We need further discussion.
   Further information is needed.
```

회화에서 둘이 혼용됩니다.

## Fewer / Less

(Chapter 4에서 다룸)

```
Fewer — 셀 수 있는 것:
   Fewer people attended.

Less — 양:
   Less time than expected.
```

슈퍼마켓 표지판의 *10 items or less*는 *문법적으로는 fewer*가 맞지만, 회화에서 *less*가 굳어졌습니다.

## Good / Well

```
Good (형용사):
   She is a good writer.
   I feel good. (정서 — 형용사)

Well (부사):
   She writes well.

Well (형용사, 건강):
   I feel well. (건강 — 형용사)
```

`I feel good`(정서)과 `I feel well`(건강)이 *모두 맞습니다*. 의미가 다릅니다.

## Imply / Infer

```
Imply — 암시하다 (화자가):
   The CEO implied that layoffs were coming.

Infer — 추론하다 (청자가):
   We inferred from her tone that something was wrong.
```

*화자가 imply, 청자가 infer*. 자주 혼동됩니다.

## Insure / Ensure / Assure

```
Insure — 보험을 들다:
   I need to insure my car.

Ensure — 보장하다:
   We must ensure quality.

Assure — 안심시키다 (사람을 대상으로):
   I assure you it's safe.
```

## Lay / Lie

(Chapter 4에서 다룸)

```
Lay (타동사):   Lay the book down.   (lay, laid, laid)
Lie (자동사):   I want to lie down.   (lie, lay, lain)
Lie (거짓말):   Don't lie.            (lie, lied, lied)
```

## Lead / Led

```
Lead (현재) — 이끌다:
   I lead the team now.

Led (과거):
   I led the team last year.
```

`Lead`는 명사일 때 *납*이라는 의미도 있습니다(발음 다름).

## Loose / Lose

```
Loose (형용사) — 헐거운:
   The screw is loose.

Lose (동사) — 잃다, 패배하다:
   Don't lose your keys.
```

## Maybe / May Be

```
Maybe (부사) — 아마도:
   Maybe she will come.

May be (조동사 + 동사):
   She may be late.
```

## Passed / Past

```
Passed — pass의 과거:
   We passed the test.

Past — 지나간, 과거:
   In the past, things were different.
   Walk past the store.
```

## Personal / Personnel

```
Personal — 개인적:
   This is a personal matter.

Personnel — 인사, 직원:
   The personnel department.
```

## Precede / Proceed

```
Precede — 앞서다:
   A brief introduction precedes the main section.

Proceed — 계속하다, 진행하다:
   Please proceed to the next room.
```

## Principal / Principle

```
Principal (형용사) — 주된:
   The principal reason for delay.

Principal (명사) — 교장, 원금:
   The school principal.
   Principal plus interest.

Principle (명사) — 원칙:
   A matter of principle.
```

## Stationary / Stationery

```
Stationary — 정지된:
   A stationary object.

Stationery — 문구류:
   Office stationery.
```

## Than / Then

```
Than — 비교:
   She is taller than I am.

Then — 시간, 순서:
   First we ate, then we left.
```

## Their / There / They're

```
Their — 소유격:
   Their house is large.

There — 장소:
   The book is over there.

They're — they are:
   They're arriving soon.
```

## To / Too / Two

```
To — 전치사 / 부정사:
   Go to school.
   I want to learn.

Too — 또한, 너무:
   Me too.
   It's too hot.

Two — 숫자:
   I have two pens.
```

## Who / Whom / Whose

```
Who — 주격:
   Who called?

Whom — 목적격:
   Whom did you call?

Whose — 소유격:
   Whose book is this?
```

## Whose / Who's

```
Whose — 소유격:
   Whose book is this?

Who's — who is / who has:
   Who's coming? (who is)
   Who's been here? (who has)
```

## Your / You're

```
Your — 소유격:
   Your idea is great.

You're — you are:
   You're correct.
```

## 격식 / 비격식 어휘

학술 / 격식 글에서 회화 어휘를 피합니다.

| 회화 / 비격식 | 격식 |
|--------------|------|
| get | obtain, acquire, receive |
| got | obtained, received |
| big | large, substantial, significant |
| little | small, minor |
| a lot of | many, numerous, considerable |
| kids | children |
| guys | people, individuals |
| stuff | items, materials, objects |
| things | items, matters, issues |
| really | very, extremely, particularly |
| pretty (= rather) | rather, somewhat, quite |
| OK | acceptable, adequate, satisfactory |
| nice | pleasant, favorable, appealing |
| about (= around) | approximately, roughly |
| like (= such as) | such as, including |

## 자기 글 점검

```
□ 자주 혼동하는 쌍이 정확히 쓰였는가?
   - affect/effect, than/then, their/there/they're
   - lay/lie, fewer/less, who/whom
□ 회화 어휘가 격식 글에 섞이지 않았는가?
□ 동음이의어(homophone)의 의미가 맞는가?
□ Comprise / Compose의 방향이 맞는가?
□ Disinterested / Uninterested 같은 정확한 구분
```

## 한국 학생의 Word Usage 함정

```
1. *Utilize* 남용 — "use"가 거의 항상 정답
2. *Personalize, finalize, optimize* 등 *-ize*  과시
3. *Very* + 약한 단어 — 강한 단어 하나로
   "very tired" → "exhausted"
4. *Nice, interesting* 남발 — 구체적 형용사로
5. *Contact* 동사 — phone, email 같은 구체 동사로
6. *Significant* 통계적 의미와 일반 의미 혼동
   학술: significant = statistically significant
   일반: significant = important
7. 한국어 직역 함정
   "consider as" → "consider" (as 불필요)
   "discuss about" → "discuss" (about 불필요)
```

## 정리

- 동음이의어 / 비슷한 단어 쌍의 *정확한 의미* 구분
- 자주 혼동되는 쌍 — affect/effect, lay/lie, fewer/less, imply/infer
- 격식 / 비격식 어휘 — 회화 어휘를 격식 글에서 피함
- 단어 선택은 *문법만큼 중요* — 의미가 어긋남
- *사전*과 *코퍼스*로 의심스러운 사용 점검
