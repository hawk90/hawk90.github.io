문법이 *맞는 문장*과 *좋은 문장*은 다릅니다. 이 마지막 챕터는 문법을 넘어선 *문체와 명확성*의 원리를 정리합니다. Thurman의 책이 *실용 레퍼런스*로서 문법을 넘어 *작가의 결정*에 닿는 자리입니다.

## 명확성의 원리

좋은 글의 가장 중요한 미덕은 *명확성*입니다. 독자가 *글의 의미*에 집중할 수 있어야 합니다.

1. 능동태 우선
2. 짧고 구체적인 단어
3. 불필요한 단어 제거
4. 평행 구조
5. 좋은 어순
6. 적절한 어조

이 원리들은 모두 *서로 강화*합니다. 능동태 문장은 보통 짧고 구체적입니다.

## 1. 능동태 우선

능동태는 *주어가 행위자*. 수동태는 *주어가 대상*.

- **능동**: *The committee approved the proposal.*
- **수동**: *The proposal was approved by the committee.*

### 능동의 장점

- 더 짧음 (능동 5단어 / 수동 7단어)
- 행위자가 명확
- 더 직접적이고 활기참

Strunk의 *Rule 14*와 같은 메시지.

### 수동이 적절한 자리

1. **행위자가 중요하지 않거나 모름**: *The window was broken.*
2. **결과나 대상이 강조점**: *The Mona Lisa was painted in the early 1500s.*
3. **학술 / 과학 글 — 방법 / 결과에 초점**: *The samples were collected weekly. / The results are presented in Table 1.*
4. **행위자를 숨기는 외교적 표현**: *Mistakes were made.* (정치 화법)

학술 글에서는 *능동과 수동의 균형*이 중요합니다(Unit 3.3 참고).

## 2. 짧고 구체적인 단어

같은 의미면 *짧고 익숙한 단어*가 명확합니다.

| 긴 / 어려운 | 짧고 명확 |
|---|---|
| utilize | use |
| ascertain | find out, learn |
| demonstrate | show |
| endeavor | try |
| commence | begin, start |
| terminate | end |
| in the event that | if |
| due to the fact that | since, because |
| at this point in time | now |
| in order to | to |
| prior to | before |
| subsequent to | after |
| the majority of | most |

### 구체성

- 모호: *The animal moved.*
- 구체적: *The wolf padded silently across the snow.*
- 모호: *She is good at her job.*
- 구체적: *She closes 80% of her sales calls.*

Strunk의 *Rule 16*과 같습니다.

## 3. 불필요한 단어 제거

영어 글쓰기의 *가장 자주 인용*되는 원리. Strunk의 *Rule 17*.

### 흔한 군더더기

| 군더더기 | 간결 |
|---|---|
| the question as to whether | whether |
| there is no doubt but that | no doubt |
| used for fuel purposes | used for fuel |
| he is a man who | he |
| in a hasty manner | hastily |
| this is a subject that | this subject |
| Her story is a strange one. | Her story is strange. |
| the reason why is that | because |
| the fact that he had not succeeded | his failure |
| in spite of the fact that | although |
| due to the fact that | because |
| during the course of | during |
| at the present time | now |
| in close proximity to | near |
| along the lines of | like |

### 명사화 vs 동사화

명사화된 표현이 길어집니다. *동사화*하면 짧아집니다.

| 명사화 | 동사화 |
|---|---|
| She made a decision to go. | She decided to go. |
| The implementation of the plan was carried out by the committee. | The committee implemented the plan. |
| There was a discussion of the issue. | They discussed the issue. |

### 군더더기 형용사·부사

| 군더더기 | 간결 |
|---|---|
| 완전히 제거하다 | 제거하다 (제거는 완전한 행위) |
| totally destroy | destroy |
| absolutely essential | essential |
| end result | result |
| final outcome | outcome |
| past history | history |
| free gift | gift |
| unexpected surprise | surprise |
| basic fundamentals | fundamentals |

## 4. 평행 구조

짝지은 항목은 *같은 문법 형태*. Strunk의 *Rule 19*.

- 잘못: *She likes hiking, fishing, and to swim.*
- 바름: *She likes hiking, fishing, and swimming.*
- 잘못: *The job requires strong communication, leadership, and to think critically.*
- 바름: *The job requires strong communication, leadership, and critical thinking.*
- 잘못: *To enroll, students must complete the form, paying the fee, and they should bring ID.*
- 바름: *To enroll, students must complete the form, pay the fee, and bring ID.*

평행 구조가 *리듬*과 *명확성*을 동시에 만듭니다.

## 5. 어순과 강조

영어 문장의 강조는 *끝*에 옵니다. Strunk의 *Rule 22*.

- 약함: *Humanity has hardly advanced in fortitude since that time, though it has advanced in many other ways.*
- 강함: *Since that time, humanity has advanced in many ways, but it has hardly advanced in fortitude.*

### 관련 단어를 *가까이*

Strunk의 *Rule 20*.

- 모호: *He noticed a large stain in the rug that was right in the center.*
- 명확: *He noticed a large stain right in the center of the rug.*

### 주어와 동사 가까이

- 끊어짐: *Toni Morrison, in Beloved, writes about characters who have escaped from slavery but are haunted by its heritage.*
- 흐름: *In Beloved, Toni Morrison writes about characters …*

## 6. 적절한 어조

글의 *목적과 독자*에 맞는 어조.

### 격식의 범위

| 수준 | 자리 |
|---|---|
| 매우 격식 | 학술 논문, 법률 문서, 공식 보고서 |
| 격식 | 비즈니스 편지, 보고서, 공식 이메일 |
| 중립 | 대부분의 비즈니스 의사소통, 신문 기사 |
| 비격식 | 친한 동료 이메일, 블로그, 캐주얼 글 |
| 구어 | 대화, 소셜 미디어, 친구에게 쓰는 글 |

### 어조의 일관성

같은 글 안에서 *어조가 일관*되어야 합니다.

**부조화**

> *The implementation of the new policy was undertaken by management. Anyway, things have been kinda weird since.*

**조화 (격식)**

> *The implementation of the new policy was undertaken by management. The transition has been challenging.*

**조화 (비격식)**

> *Management rolled out the new policy. Things have been kinda weird since.*

### 회피해야 할 톤

- **거만함**: *The author obviously fails to grasp the fundamental concepts.*
- **비하**: *Anyone with half a brain can see that …*
- **감정적 / 과장**: *This is absolutely the worst idea ever conceived.*
- **자기 비하 / 과도한 헤지**: *I'm probably wrong, but maybe perhaps it could possibly be that …*

학술 글의 적절한 톤은 *균형 잡힌 자신감* — Skelton의 *confidently uncertain*.

## Wordiness — 장황함의 패턴

자주 마주치는 *장황함*의 패턴.

### Empty Phrases — 빈 어구

| 빈 어구 | 처리 |
|---|---|
| in my opinion, I think | 1인칭만 / 또는 그냥 진술 |
| as a matter of fact | 종종 빼도 됨 |
| needless to say | 정말 needless이면 빼기 |
| at the end of the day | 회피 |
| in the final analysis | 회피 |
| all things considered | 회피 |
| for all intents and purposes | 회피 |

### Redundant Pairs — 중복 쌍

| 중복 | 줄임 |
|---|---|
| each and every | *each* 또는 *every* |
| first and foremost | first |
| hopes and dreams | 하나만 |
| null and void | *null* 또는 *void* |
| basic and fundamental | *basic* 또는 *fundamental* |
| true and accurate | *true* 또는 *accurate* |

### Redundant Modifiers — 중복 수식

| 중복 수식 | 단순 |
|---|---|
| absolutely necessary | necessary |
| totally complete | complete |
| totally exhausted | exhausted |
| fully completed | completed |
| completely surrounded | surrounded |
| exact same | same |
| just exactly | exactly |

## 학술 글 vs 회화 — 의식적 선택

같은 의견을 두 어조로 표현할 수 있습니다.

### 회화

- *Stuff like that isn't really a big deal.*
- *You gotta be kidding me!*
- *We totally need to fix this asap.*

### 학술

- *Issues of this nature are not particularly significant.*
- *This conclusion appears questionable.*
- *Immediate revision is warranted.*

자기 글의 *목적과 독자*에 맞게 *의식적으로* 어조를 선택합니다.

## 6원리의 통합 점검

자기 글의 한 단락을 다음 6원리로 점검합니다.

- [ ] 능동태와 수동태가 적절히 섞였는가?
- [ ] 짧고 익숙한 단어가 우선되는가?
- [ ] 불필요한 단어가 없는가?
- [ ] 평행 구조가 짝지은 항목에 적용되었는가?
- [ ] 강조 단어가 문장 끝에 오는가?
- [ ] 어조가 일관되고 적절한가?

이 점검만으로 자기 글의 *명확성*이 즉시 개선됩니다.

## Style Guides — 추가 자료

영어 글쓰기의 *문체와 명확성*에 대한 더 깊은 자료:

- **The Elements of Style** (Strunk & White) — 짧고 단호한 규칙
- **On Writing Well** (Zinsser) — Nonfiction 글쓰기
- **Style: Lessons in Clarity and Grace** (Williams) — 문체의 깊이
- **The Sense of Style** (Pinker) — 인지과학 관점
- **The Chicago Manual of Style** — 학술 / 출판 표준
- **Garner's Modern English Usage** — 어휘 / 문법 레퍼런스

각 책이 *다른 관점*에서 영어 문체를 다룹니다.

핵심 메시지 — *문법은 도구*입니다. 좋은 글의 목적은 *문법의 정확성*이 아니라 *명확한 의사소통*. 문법은 그 의사소통을 *방해하지 않도록* 알고 있어야 할 도구입니다.

## 한국 학생의 Style / Clarity 함정

1. **Wordy 표현** — 한국어 격식의 직역
   - *due to the fact that* → *because*
   - *in order to* → *to*
2. **수동태 남용** — 격식의 환상
   - 해결: 능동이 자연스러우면 능동
3. **명사화 `-tion` 과다**
   - *the implementation of* → *implementing*
4. **Hedging 과다**
   - *may possibly perhaps* → 한 헤지만
5. **한국어 추상 명사 → 영어 추상 명사** → 무거움
   - *the necessity of consideration of …* 같은 직역 회피
6. **단조로움** — 모든 문장이 같은 길이 / 구조
   - 해결: 문장 다양화 (Rule 18)

## 자기 글 점검 — Style

- [ ] 모든 단어가 *일하고* 있는가? (Omit needless words)
- [ ] 능동 / 수동이 *적절한 자리*?
- [ ] 명사화 과다는 없는가?
- [ ] 헤지 *균형*?
- [ ] 문장 *다양*?
- [ ] 한국어 추상 명사 직역 회피?
