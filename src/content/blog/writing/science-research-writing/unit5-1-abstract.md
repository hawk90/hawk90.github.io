Abstract는 논문 전체를 200-300 단어로 압축한 것입니다. 가장 읽히는 부분이지만 가장 마지막에 작성됩니다. 논문의 모든 섹션이 완성된 뒤에야 진짜 abstract를 쓸 수 있습니다.

## Abstract의 역할

```
1. 논문 검색의 진입점 — 데이터베이스 검색 결과로 노출
2. 빠른 평가의 도구 — 독자가 전체를 읽을지 결정
3. 학회의 선정 기준 — Abstract만 보고 발표 채택
4. 외부 인용의 근거 — 다른 연구자가 자기 글에 인용
```

## 표준 구조 — 4 부분

```
1. Background — 연구 문제 / 맥락 (1-2 문장)
2. Methods    — 어떻게 했는가 (1-2 문장)
3. Results    — 무엇을 발견했는가 (2-3 문장)
4. Conclusion — 의미와 함의 (1-2 문장)
```

200-300 단어 안에 논문 전체를 압축합니다.

## 실제 Abstract 예시

```
[Background]
Caffeine consumption has been linked to reduced sleep quality, but
the role of consumption timing remains poorly characterized.

[Methods]
We surveyed 245 college students about their caffeine intake patterns
and measured sleep efficiency using actigraphy over a 7-night period.
Multilevel regression models examined the relationship between
within-person variation in caffeine timing and same-night sleep
outcomes.

[Results]
Caffeine consumption within 6 hours of bedtime was associated with
a 12% reduction in sleep efficiency (β = -0.34, p < .001), even
after controlling for total daily intake. Contrary to prediction,
this effect did not vary by genetic markers of caffeine metabolism
(p = .58).

[Conclusion]
These findings suggest that timing of caffeine consumption — not
just quantity — is a key determinant of sleep quality in young adults,
with implications for public health messaging.
```

분량 분석을 보면 4 부분이 대략 균등하게 차지합니다.

## Abstract의 시제

각 부분이 다른 시제를 가집니다.

```
Background:
   Present Perfect (분야의 현재 상태)
   has been linked / has been examined
   
   또는 Present (일반 진리)
   is a major public health concern

Methods:
   Past Simple (이미 한 일)
   We surveyed / We measured / We analyzed

Results:
   Past Simple (관찰된 사실)
   was associated / did not vary / increased

Conclusion:
   Present (현재의 의미)
   These findings suggest / have implications
```

위 사례에서 시제 변화를 확인할 수 있습니다.

## Move별 핵심 어휘

### Background

```
[Topic] has received attention / has been studied / remains poorly
understood.

The relationship between A and B has been examined in [populations].

Despite [progress], several questions remain.
```

### Methods

```
We examined / investigated / measured / analyzed ...
A total of N participants ...
Using [method/instrument] ...
Multilevel / mixed-effects / linear regression models were used to ...
```

### Results

```
[Variable] was significantly associated with [outcome] ...
The intervention reduced / increased [outcome] by [%].
Contrary to expectation, [unexpected finding].
The effect size was [value], indicating ...
```

### Conclusion

```
These findings suggest / indicate that ...
The results have implications for [field / practice].
Future research should examine ...
```

## Abstract 작성의 함정

### 1. 너무 일반적인 도입

| 분류 | 예문 |
|---|---|
| 회피 | This study examines an important topic. |
| 바름 | This study examines the relationship between caffeine timing and sleep quality in young adults. |

구체성이 검색 키워드를 만듭니다.

### 2. 결과 미명시

| 분류 | 예문 |
|---|---|
| 회피 | We found interesting effects. |
| 바름 | Caffeine consumption within 6 hours of bedtime was associated with reduced sleep efficiency. |

무엇이 발견되었는지 명시해야 합니다.

### 3. Statistical 디테일 누락

| 분류 | 예문 |
|---|---|
| 회피 | Differences were significant. |
| 바름 | Differences were significant (t = 3.45, p < .001, d = 0.44). |

가능하면 효과 크기와 p-value를 포함합니다(분야 표준에 따릅니다).

### 4. Methodology 디테일 부족

| 분류 | 예문 |
|---|---|
| 회피 | We analyzed the data. |
| 바름 | We analyzed the data using mixed-effects models with random intercepts for participants. |

심사자가 방법의 적절성을 abstract만 보고 평가할 수 있어야 합니다.

### 5. 약속 후 미이행

Abstract에서 한다고 했는데 본문에서 안 한 것은 자주 마주치는 함정입니다.

```
Abstract: We examine three hypotheses ...
본문:    [실제로는 두 개만 다룸]
```

본문 완성 후 Abstract를 다시 쓰면 이 문제가 해결됩니다.

## Abstract 길이

```
저널마다 다름:
   150 단어 — 짧은 저널
   200-250 단어 — 가장 흔함
   300 단어 — 긴 저널
   500 단어 — 일부 의학 저널
```

저널 요구사항을 반드시 확인합니다.

## Structured vs Unstructured Abstract

### Unstructured (전통적)

```
한 단락에 4 부분이 자연스럽게 흐름
대부분의 과학 저널
```

### Structured (구조화)

```
헤더로 명시:
Background: ...
Methods: ...
Results: ...
Conclusions: ...

의학·임상 저널에 흔함
```

자기 분야와 자기 저널의 표준을 따릅니다.

## 다른 abstract 형식

### Graphical Abstract

```
일부 저널 (특히 화학·재료) — 그림 abstract
시각적 요약
```

### Highlights

```
일부 저널 — 3-5개 짧은 bullet
"What's new in this paper"
```

### Lay Summary

```
일부 저널 — 일반 독자용 abstract
non-technical 언어
```

자기 저널의 요구를 확인합니다.

## Abstract 작성 절차

```
1. 논문 본문이 완성된 후 작성
2. 각 섹션에서 핵심 한두 문장만 추출
3. 4 부분 구조로 결합
4. 단어 수 점검 — 너무 길면 자르기
5. 키워드 점검 — 검색에 노출될 단어가 포함되었는가
6. 본문과의 일치 점검
7. 외부에 보여 주고 논문 의도가 명확한지 확인
```

## 정리

- Abstract는 논문 전체를 200-300 단어로 압축한 글입니다.
- 4 부분은 Background, Methods, Results, Conclusion입니다.
- 시제는 Background에서 Perfect나 Present, Methods와 Results에서 Past, Conclusion에서 Present를 씁니다.
- 흔한 함정은 일반적 도입, 결과 미명시, 통계 누락, 약속 후 미이행입니다.
- 본문 완성 후 Abstract를 작성합니다.
- 자기 저널의 형식 표준을 확인합니다(structured와 unstructured).

## 한국 연구자의 Abstract 함정

```
1. 너무 일반적 도입 ("In modern science...")
2. 결과 미명시 — "interesting results"
3. 통계 누락
4. 약속 후 미이행
5. 본문 완성 전 작성
```
