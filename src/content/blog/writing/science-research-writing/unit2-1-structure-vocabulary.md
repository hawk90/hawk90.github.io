Methodology는 모래시계의 좁은 중심부입니다. 자기 연구의 재현 가능성을 만드는 자리입니다. 이 글은 Methodology의 표준 구조와 자주 쓰이는 어휘를 다룹니다.

## Methodology의 목적

```
1. 재현 가능성 — 다른 연구자가 같은 절차를 반복할 수 있게
2. 신뢰성 검증 — 심사자가 방법의 적절성을 평가할 수 있게
3. 한계 명시 — 사용한 방법의 경계를 분명히
```

좋은 Methodology는 충분한 디테일과 과한 디테일 사이의 균형입니다.

## 표준 구조

Glasman-Deal이 정리한 Methodology의 모델은 보통 다음 흐름을 따릅니다.

```
1. 무엇을 / 누구를 — Materials / Participants
2. 어떻게 준비 — Preparation
3. 무엇을 했는가 — Procedure (단계별)
4. 어떻게 측정 — Measurement
5. 어떻게 분석 — Analysis
```

### 자연과학 / 공학 — Chemistry 예시

```
The polymer blends were prepared using a melt-blending technique
at 180°C. Approximately 50 g of PLA was dried under vacuum at
40°C for 24 h before processing. The PLA was then blended with
PI at ratios of 95:5, 90:10, and 85:15 in a twin-screw extruder
rotating at 60 rpm. The extruded blends were cooled to room
temperature and cut into pellets of approximately 3 mm in length.
The pellets were then compression-moulded at 175°C and 5 MPa
for 5 min to produce test specimens. Mechanical properties were
measured using a universal testing machine (Instron 5582)
according to ASTM D638. Tensile tests were conducted at a
crosshead speed of 50 mm/min at 23 °C and 50% relative humidity.
```

이 단락에 5 단계가 모두 들어 있습니다.

- Materials — PLA, PI
- Preparation — *dried under vacuum*
- Procedure — *blended, extruded, cooled, cut, compression-moulded*
- Measurement — *universal testing machine, ASTM D638, tensile tests*

### 생명과학 / 의학 — Sample 처리

```
Cells were cultured in Dulbecco's modified Eagle's medium (DMEM)
supplemented with 10% fetal bovine serum and 1% penicillin-
streptomycin at 37°C in 5% CO2. At 80% confluence, cells were
trypsinized and seeded into 96-well plates at a density of 1×10⁴
cells/well. After 24 h, cells were treated with various
concentrations of the test compound (0–100 μM) for 48 h. Cell
viability was assessed using the MTT assay. Absorbance was
measured at 570 nm using a microplate reader.
```

### 사회과학 / 심리학 — Participants

```
A total of 245 participants (M_age = 22.3 years, SD = 3.1; 54%
female) were recruited from three universities. All participants
provided informed consent and received course credit for
participation. The study was approved by the Institutional
Review Board.
```

## 분야별 섹션 제목

```
자연과학 / 공학:    Methods / Methodology / Experimental
의학 / 보건과학:   Materials and Methods
사회과학:        Research Design / Methodology / Methods
컴퓨터 과학:      Experimental Setup / System Design
```

자기 분야의 모범 논문 제목을 확인합니다.

## 흔한 sub-section 구조

긴 Methodology는 sub-section으로 나뉩니다.

```
2. Methodology
   2.1 Materials
   2.2 Sample Preparation
   2.3 Experimental Procedure
   2.4 Data Analysis

또는:

2. Materials and Methods
   2.1 Participants
   2.2 Apparatus
   2.3 Stimuli
   2.4 Procedure
   2.5 Statistical Analysis
```

분야와 저널에 따라 다릅니다.

## Vocabulary — Materials와 준비

### Materials 표현

```
materials were obtained from / purchased from / provided by ...
samples were prepared / synthesized / collected / cultured ...
solutions were made up / dissolved in / diluted to ...
```

### 농도와 단위

```
a concentration of 0.5 M
a dilution of 1:1000
ratios of 95:5 (w/w)   ← weight/weight
volumes of 10 mL each
```

## Vocabulary — 실험 절차

### 흔한 동사

```
add, mix, stir, heat, cool, filter, dry, weigh
transfer, dissolve, dilute, evaporate, distill
incubate, centrifuge, sonicate, lyophilize
inject, infuse, administer
record, observe, monitor, calibrate
```

### Sequence (순서) 표현

```
First, ...
Next, ...
Then, ...
Subsequently, ...
After [period], ...
Once [condition], ...
Following [event], ...
Finally, ...
```

이 시간 신호어가 단계 순서를 명확히 합니다.

## Vocabulary — 분석

### 통계 분석

```
data were analyzed using ...
statistical analysis was performed in [software]
significant differences were determined using [test]
p-values < 0.05 were considered significant
```

### 정량 분석

```
the concentration was calculated from ...
the rate was determined by ...
the spectra were recorded on ...
the values were averaged over [N] replicates
```

## Build a Model — Methodology Template

위 예시들을 일반화한 template입니다.

```
[Section 1: Materials]
[Reagent/material] was obtained from [supplier]. [Material] was
prepared by [method] using [equipment]. The [concentration/ratio/
purity] was [value].

[Section 2: Preparation]
[Sample] was [treatment 1] for [time] at [temperature]. The
[treated sample] was then [treatment 2].

[Section 3: Procedure]
The experiment was conducted as follows. First, [step 1]. Next,
[step 2]. Then, [step 3]. Finally, [step 4].

[Section 4: Measurement]
[Property] was measured using [instrument] according to [standard].
[Conditions] were maintained at [values].

[Section 5: Analysis]
Data were analyzed using [software]. [Statistical test] was used
to determine [outcome]. All experiments were performed in
triplicate. Values are reported as mean ± SD.
```

자기 분야 모범 논문에서 같은 분석을 하면 더 정확한 template이 됩니다.

## 자기 분야 동사 보강

각 분야는 고유한 기술 동사를 가집니다.

### 화학 / 재료

```
synthesize, polymerize, crystallize, anneal
extract, purify, characterize, functionalize
```

### 생물학 / 의학

```
isolate, culture, transfect, infect
amplify, sequence, clone, digest
quantify, immunoblot, immunoprecipitate
```

### 컴퓨터 / 정보과학

```
implement, evaluate, benchmark
train, validate, fine-tune, deploy
parse, tokenize, embed, classify
```

### 물리학 / 천문학

```
detect, observe, simulate, model
calibrate, normalize, fit, deconvolve
```

자기 분야의 표준 동사 20-30개를 정리해 두면 Methodology 작성이 훨씬 빠릅니다.

## 함정 — 흔한 실수

### 1. 시제 일관성 누락

```
회피: We collected samples. They are analyzed using GC-MS.
선호: We collected samples. They were analyzed using GC-MS.
```

같은 연구의 절차는 모두 과거 시제로 통일합니다.

### 2. Agent 누락의 모호함

```
회피: The compound was synthesized.
       (우리가? 다른 연구자가? 시판?)

선호: The compound was synthesized as described in Brown (2020).
또는: We synthesized the compound following the procedure of Brown (2020).
```

### 3. 디테일 부족

```
회피: The cells were treated with the drug.
선호: The cells were treated with 10 μM of the drug for 24 h at 37°C.
```

재현 가능성에 필요한 디테일을 빠뜨리지 않습니다.

### 4. 결과 섞임

```
회피: We measured the response. The response was 30% higher.
선호: We measured the response. [결과는 Section 3 Results에]
```

Methodology에는 어떻게 했는지만 적습니다. 결과는 Results 섹션의 몫입니다.

## Write Your Methodology — 통합 과제

```
1. 자기 실험의 단계를 시간 순서로 적기
2. 각 단계의 재료, 기기, 조건을 표로 정리
3. 위 template으로 첫 draft 작성
4. 시제 일관성 점검
5. Agent 명시 점검
6. 자기 분야 모범 논문과 디테일 수준 비교
7. 수정
```

## 정리

- Methodology는 다섯 단계로 구성됩니다. Materials, Preparation, Procedure, Measurement, Analysis 순서입니다.
- 분야마다 섹션 제목과 sub-section 구조가 다릅니다.
- 핵심 동사로 *prepare*, *measure*, *analyze*, *perform*이 자주 쓰입니다.
- Sequence 신호어로 단계 순서를 명확히 합니다.
- 자기 분야의 표준 동사 목록을 정리해 두면 효율적입니다.
- 흔한 함정은 시제 불일치, agent 모호, 디테일 부족, 결과 섞임입니다.

## 한국 연구자의 Methodology 함정

```
1. 디테일 부족 또는 과다 — 재현 가능성 기준
2. Sequence 신호어 누락
3. Results 정보 섞임
4. Brand name / 모델 번호 누락
```
