Methodology는 *수동태와 과거 시제*가 압도적으로 자주 쓰이는 자리입니다. 영어가 모국어가 아닌 작가에게 가장 어려운 자리는 *관사 a/the*와 *부사 위치*입니다.

## Passive Voice의 사용

### 왜 Methodology는 수동인가

과학 글의 *과정 묘사*에서 수동태가 흔한 이유는 *초점*에 있습니다. *행위자*가 아니라 *과정 자체*가 중요합니다.

```
수동 (과정 중심):
The samples were collected at three sites. Each sample was filtered
through a 0.45 μm membrane and stored at −20°C until analysis.

능동 (행위자 중심):
We collected samples at three sites. We filtered each sample
through a 0.45 μm membrane and stored them at −20°C until analysis.
```

수동 버전에서 독자의 주의는 *samples*에 갑니다. 능동 버전은 *we*에 갑니다. Methodology는 보통 *과정에 초점*이 가는 수동을 선택합니다.

### 능동이 적절한 자리

```
1. 새로운 / 비표준 방법을 우리가 개발했을 때:
   We developed a new technique that combines X and Y.

2. 다른 연구자의 방법을 우리가 따랐을 때:
   We followed the procedure of Brown (2020) with minor modifications.

3. 자기 결정을 명시할 때:
   We chose this concentration based on preliminary experiments.
```

### Methodology의 시제

기본은 *Past Simple* (과거 시제)입니다. 이미 한 일이기 때문입니다.

```
선호: Samples were collected weekly.
선호: We measured response times with a millisecond-accurate timer.
선호: Data were analyzed using mixed-effects models.
```

### 예외 — Present Simple

```
1. 표 / 그림 참조:
   Figure 1 shows the experimental setup.

2. 알려진 사실 / 일반 절차:
   The Stroop task measures interference between conflicting stimuli.

3. 방법론적 정당화:
   This approach is widely used in similar studies.

4. 공식 / 방정식:
   Equation 1 represents the relationship between X and Y.
```

같은 단락에 *현재와 과거*가 섞일 수 있습니다.

```
The temperature was monitored every 5 minutes (Past — 우리가 했음).
The system shows good thermal stability (Present — 일반 사실).
```

## Articles — a / an / the / Ø

영어가 모국어가 아닌 작가의 가장 큰 함정. Glasman-Deal이 자세히 다룹니다.

### 기본 규칙

```
a / an  — 가산명사, 처음 언급, 일반화
the     — 가산/불가산, 이미 언급됨 또는 특정됨
Ø (관사 없음) — 불가산, 일반화 / 복수의 일반화
```

### 처음 언급 vs 다시 언급

```
첫 언급 (a):
We added a sample to the reactor.

다시 언급 (the):
The sample was then heated to 100°C.
```

처음 등장할 때 *a*, 두 번째부터 *the*. 이미 정해진 것을 가리키기 때문입니다.

### 일반화

```
가산명사 + 복수 (일반):
Polymers are widely used in industry.

불가산명사 (일반):
Water is essential to life.
Information is power.

대조 — 특정한 것 (the):
The polymer used in this study was PLA.
The water sample contained ...
```

### 과학 글의 흔한 패턴

```
1. 일반 분류 (no article 또는 a):
   Polymers offer many advantages.
   A polymer is a long-chain molecule.

2. 특정 분류 (the):
   The polymer used in this study ...

3. 측정값 (the):
   The temperature was 100°C.
   The concentration was 0.5 M.

4. 도구 / 장비 (the):
   The thermometer was calibrated.
   The spectrometer recorded ...
```

### 모범 논문에서 점검

```
회피: We used spectrometer to record spectra.
선호: We used a spectrometer to record the spectra.
       (a spectrometer — 처음 언급, the spectra — 특정)

회피: Spectrometer was calibrated using standard.
선호: The spectrometer was calibrated using a standard.
       또는: A spectrometer was used; it was calibrated using a standard.
```

자기 분야 모범 논문의 *관사 사용*을 자세히 보면 패턴이 보입니다. 한국어 화자가 가장 자주 빠뜨리는 것이 *the*입니다.

### Ø article (관사 없음)

복수 일반화와 불가산 일반화는 관사가 없습니다.

```
Polymers are useful materials.   ← polymers (복수 일반)
Water boils at 100°C.            ← water (불가산 일반)
Research suggests that ...        ← research (불가산)
```

```
회피: Researches show that ...    (research는 불가산)
선호: Research shows that ...
또는: Studies show that ...
```

*research*는 불가산. *studies*는 가산.

## Adverbs와 Adverb 위치

학술 영어에서 부사는 자주 *mid-position*에 옵니다 (Unit 1 참고). Methodology에서도 같은 패턴입니다.

### 시간·빈도 부사

```
일상 (문장 끝):
The samples were heated quickly.
The experiment was performed initially.

학술 (mid-position):
The samples were quickly heated.
The experiment was initially performed.
```

### 양식 부사

```
일상: The mixture was stirred carefully.
학술: The mixture was carefully stirred.
```

Mid-position 부사가 *학술 톤*을 더합니다.

### Split Infinitive

```
전통: to + verb + adverb를 분리하지 말 것
   We needed to adequately meet the requirements.   ← 분리

표준 권장: to + adverb + verb
   We needed to meet the requirements adequately.   ← 비분리
```

Glasman-Deal은 split infinitive를 *피하라*고 권합니다. 다만 분리가 *명확성에 더 좋으면* 허용합니다.

## Quantity 표현

Methodology에 자주 등장하는 수량 표현입니다.

### Mass / Volume

```
50 g of PLA
10 mL of solution
1.5 mg of sample
0.5 L (= 500 mL)
```

*of*가 들어가는 패턴 — `[수치] [단위] of [물질]`.

### Concentration

```
0.5 M (= mol/L)
50 mM (= mmol/L)
1% (w/v) — weight/volume
2% (v/v) — volume/volume
```

### Ratio

```
ratios of 95:5
a ratio of 3:1
in equal parts
```

### Approximate

```
approximately 50 g
about 100 mL
roughly 30 mg
some 200 samples
```

### Range

```
ranging from 0 to 100 μM
between 25 and 30°C
50 ± 5 mg   ← 표준편차
```

## 사례 — 한 단락 점검

다음 단락을 문법 관점에서 점검해 봅니다.

```
The polymer blends were prepared using a melt-blending technique
at 180°C. Approximately 50 g of PLA was dried under vacuum at 40°C
for 24 h before processing. The PLA was then blended with PI at
ratios of 95:5, 90:10, and 85:15 in a twin-screw extruder rotating
at 60 rpm. The extruded blends were cooled to room temperature
and cut into pellets of approximately 3 mm in length.
```

분석:

- **Passive** — *were prepared, was dried, was blended, were cooled, cut* (수동 일관)
- **Past Simple** — 모든 동사 (과거 행위)
- **Articles**:
  - *a melt-blending technique* — 처음 언급 (a)
  - *The PLA* — 이미 언급된 PLA (the)
  - *a twin-screw extruder* — 처음 언급 (a)
  - *The extruded blends* — 직전에 만든 것 (the)
- **Mid-position adverb** — *then blended*
- **Quantity** — *50 g of PLA*, *180°C*, *60 rpm*, *3 mm*

한 단락에 이 책의 주요 문법 요소가 모두 들어 있습니다.

## 자가 점검 체크리스트

```
□ 모든 절차가 *과거 시제 수동*인가?
□ 일반 사실은 *현재 시제*인가?
□ 처음 등장하는 사물 / 도구에 *a/an*이 있는가?
□ 다시 언급되는 것에 *the*가 있는가?
□ 불가산 명사 (research, information)에 *복수형*이 없는가?
□ 부사가 *mid-position*에 적절히 배치되었는가?
□ 측정값에 *단위*가 명시되었는가?
□ Agent가 모호한 자리가 없는가?
```

## 정리

- Methodology의 기본 — *Past Simple 수동*
- 예외 — 표/그림 참조, 일반 사실, 정당화에 *Present Simple*
- 관사 — 처음 *a/an*, 재언급 *the*, 복수/불가산 일반화 *Ø*
- 한국어 화자가 자주 빠뜨리는 것은 *the*
- 부사는 *mid-position*이 학술적
- Quantity 표현은 분야 표준 단위 사용
- *Agent 모호*에 주의
