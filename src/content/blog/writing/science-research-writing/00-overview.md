Hilary Glasman-Deal의 *Science Research Writing*은 Imperial College London에서 30년 가까이 가르친 경험을 바탕으로 한 *실용 매뉴얼*입니다. Swales와 Feak의 *Academic Writing for Graduate Students*가 *왜 그렇게 쓰는가*를 설명한다면, 이 책은 *어떻게 빠르게 쓰는가*에 집중합니다.

> Things should be made as simple as possible, but not any simpler. — Albert Einstein

대상 독자는 *영어가 모국어가 아닌 과학·공학 연구자*입니다. IELTS 6.0이나 TOEFL 91 이상 수준의 영어를 가진 사람을 가정합니다.

## 책의 핵심 전략

```
1. 좋은 예 (자기 분야의 모범 논문)를 자세히 본다
2. 구조 · 문법 · 어휘의 패턴을 발견한다
3. 그 패턴을 자기 글의 model로 만든다
4. model을 자기 연구에 적용한다
```

저자의 통찰 — *언어 능력이 완벽하지 않을 때, 정보를 관습적인 방식으로 조직하고 관습적인 언어로 표현하는 것이 매우 중요*합니다. 관습적 모델을 따르면 *독자가 무엇을 예상해야 할지 안다*는 것이 핵심입니다.

## Research Article의 모래시계

이 책의 출발점은 연구 논문의 *모래시계 구조*입니다.

![Research Article의 모래시계 구조](/images/blog/science-research-writing/diagrams/overview-hourglass.svg)

이 모양이 다섯 Unit의 구조를 결정합니다.

## 다섯 Unit 구조

```
Unit 1 — Introduction
Unit 2 — Methodology
Unit 3 — Results
Unit 4 — Discussion / Conclusion
Unit 5 — Abstract (그리고 Title)
```

각 Unit이 같은 패턴을 따릅니다.

```
1. Sample 분석 — 실제 논문 구절을 자세히 봅니다
2. Grammar & Writing Skills — 그 섹션에 자주 쓰이는 시제와 문법
3. Build a Model — sample을 일반화한 재사용 가능한 template
4. Vocabulary — 그 섹션에 자주 쓰이는 동사와 표현
5. Write — 자기 연구로 직접 작성
```

## AWG와의 비교

| 항목 | AWG (Swales & Feak) | SRW (Glasman-Deal) |
|------|---------------------|---------------------|
| 범위 | 학술 영어 전반 | 과학 연구 논문 |
| 깊이 | 이론적, 분석적 | 실용적, template 중심 |
| 분량 | 415 페이지 | 245 페이지 |
| 접근 | "왜 그렇게 쓰는가" | "어떻게 빠르게 쓰는가" |
| 대상 | 대학원생, 박사과정 | 박사과정, 박사후, 연구원 |
| 특수 | 정의·요약·비평 | Abstract, Title 강조 |

두 책은 *대체*가 아니라 *보완*입니다. AWG가 학술 영어의 *문법과 원리*를 가르치면, SRW는 *과학 논문의 즉시 적용 가능한 template*을 제공합니다.

## 시리즈 구성

```
Unit 1. How to Write an Introduction
   — 4-Move 모델, 시제 쌍, signalling language

Unit 2. Writing about Methodology
   — Passive 위주, 시제 일치, 'a'와 'the'의 사용

Unit 3. Writing about Results
   — Sequence, Frequency, Quantity, Causality

Unit 4. Writing the Discussion / Conclusion
   — 결과 해석, 한계 인정, 함의 도출

Unit 5. Writing the Abstract (and Title)
   — 200-300 단어로 논문 전체 압축, 좋은 제목의 조건
```

## 자기 분야 모범 논문 — 미리 준비

이 시리즈를 활용하려면 *자기 분야 모범 논문 3-4편*을 미리 모아 두는 것을 권합니다. Glasman-Deal의 권고입니다.

```
□ 영어권 기관의 연구자가 쓴 것
□ 15페이지 이하 (그림 / 표 포함)
□ 자기 연구 주제와 가까운 것
□ Introduction / Methodology / Results / Discussion 구분이 명확
```

이 모범 논문이 *target article*입니다. 각 Unit을 읽으며 자기 target article에서 같은 패턴을 *찾아내는 것*이 학습의 핵심입니다.

## 사용법

이 시리즈는 *읽기만 해서는 효과가 적습니다*. 각 Unit의 *Build a Model*과 *Write* 과제를 직접 해야 합니다.

```
1. Unit의 sample을 자세히 읽기
2. Grammar & Writing Skills의 패턴을 자기 target article에서 확인
3. Build a Model — 자기만의 template 작성
4. Vocabulary 목록을 자기 분야 어휘로 보완
5. Write — 자기 연구로 그 섹션 작성
```

다섯 Unit을 마치면 *완전한 연구 논문 draft*가 만들어집니다.

## 한국 연구자에게 *이 책*이 유용한 이유

```
1. *Template 중심* — 빠른 적용 가능
   한국 학생은 보통 *명시적 패턴 학습*에 강함
   → SRW의 template 접근이 잘 맞음

2. *비영어권 대상* — IELTS 6.0+ 가정
   AWG는 *원어민에 가까운 학습자* 가정
   SRW는 *영어가 두 번째 언어*인 연구자 가정

3. *모범 논문 분석* — 한국 학습 문화와 자연
   "좋은 예 모방하기" 전략이 *공식화*

4. *자연과학 / 공학 위주*
   한국의 STEM 박사과정에 *직접 적용*
```

## SRW vs AWG — 어디서 시작?

```
당신이 STEM 박사과정 학생이라면:
   SRW부터 → 빠른 논문 작성 가능
   그 뒤 AWG → 깊이 있는 학술 영어

당신이 인문·사회과학이라면:
   AWG부터 → 비평 / 정의 / 종합
   SRW는 *과학 논문 부분만* 참고

당신이 *처음* 영어 논문을 쓴다면:
   SRW Unit 1부터 → template 따라 작성
   AWG의 Unit 1-2를 보완으로
```

## 시작점

먼저 **Unit 1: How to Write an Introduction**부터. Introduction이 *모래시계의 가장 넓은 입구* — 가장 일반적인 곳에서 시작해 구체로 좁아집니다.
