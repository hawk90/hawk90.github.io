Results는 모래시계의 좁은 끝입니다. 발견 자체에 집중하고 해석은 Discussion으로 미룹니다. 이 글은 Results의 구조와 어휘를 다룹니다.

## Results의 목적

```
1. 발견을 명확히 보고
2. 표·그림으로 데이터를 시각화
3. 해석은 최소로 (Discussion의 영역)
```

Results는 객관적 보고의 자리입니다. 해석과 의견은 다음 섹션에서 다룹니다.

## 표준 구조

```
1. 데이터 설명 — 무엇을 측정했는가
2. 핵심 발견 — 가장 중요한 결과부터
3. 추가 분석 — 보조적 결과
4. 시각 자료 — 표 / 그림 / 그래프
```

### 자주 쓰이는 패턴

```
[그림/표 참조] + [관찰] + [정량적 디테일]

Figure 1 shows the effect of temperature on yield. The yield
increased from 65% at 80°C to 87% at 100°C. Above 100°C, the
yield remained constant.

As shown in Table 2, treatment A produced significantly higher
response rates than treatment B (45% vs 28%, p < 0.001).
```

## Vocabulary — 그림·표 참조

### 능동 참조

```
Figure 1 shows the experimental setup.
Table 2 presents the descriptive statistics.
Figure 3 illustrates the time course of the response.
Table 4 summarizes the key findings.
```

### 수동 참조

```
The experimental setup is shown in Figure 1.
The descriptive statistics are presented in Table 2.
The time course is illustrated in Figure 3.
```

### As-Clause 참조

```
As shown in Figure 1, the yield increased with temperature.
As can be seen in Table 2, the difference was significant.
As illustrated in Figure 3, the response peaked at 200 ms.
```

### 괄호 참조

```
The yield increased with temperature (Figure 1).
The difference was significant (Table 2, p < 0.001).
```

같은 단락에 형태를 섞어 단조로움을 피합니다.

## Vocabulary — Sequence (순서)

Results에서 사건의 순서를 표현하는 어휘입니다.

### 단계 순서

```
First, ...
Initially, ...
Then, ...
Subsequently, ...
After [time/event], ...
Following [event], ...
Finally, ...
```

### 진행

```
gradually
progressively
steadily
rapidly
quickly
slowly
```

### 발생

```
occurred
took place
arose
emerged
appeared
```

## Vocabulary — Frequency (빈도)

```
always, invariably, consistently
usually, generally, typically, normally
often, frequently, regularly
sometimes, occasionally
rarely, seldom, infrequently
never
```

### 학술적 강도

```
in most cases
in the majority of cases
in a significant minority of cases
in a few cases
```

## Vocabulary — Quantity (양)

### 크기

```
large / small
substantial / minor
significant / negligible
considerable / minimal
substantial / slight
```

### 변화의 정도

```
increase / decrease / fluctuate
rise / fall / stabilize
grow / decline / remain constant
peak / bottom out / plateau
```

### 정량적 표현

```
by 30% — 30 percent만큼
to 50% — 50 percent까지
from 20% to 80% — 20에서 80으로
between 10 and 20°C
approximately 200 samples
roughly twice as many
```

### 비교

```
A is larger than B
A is twice / three times as large as B
A is 30% larger than B
The difference between A and B was X
Compared to A, B is ...
```

## Vocabulary — Causality (인과)

Results에서 상관과 인과를 구분하는 것이 중요합니다.

### 약한 표현 (상관)

```
A is associated with B
A is correlated with B
A is linked to B
A coincides with B
```

### 강한 표현 (인과)

```
A causes B
A leads to B
A results in B
A produces B
A induces B
A triggers B
```

### 매개

```
A is mediated by B
B mediates the effect of A on C
A contributes to B
A is partially responsible for B
```

자기 데이터가 상관만 보였다면 *causes*가 아니라 *is associated with*를 씁니다.

## Build a Model — Results Template

```
[일반 도입]
The results of the [experiments/analyses] are presented in [Figures/
Tables].

[첫 발견 — 가장 중요한 것부터]
As shown in [Figure 1], [main finding]. Specifically, [quantitative
detail]. [Comparison or trend].

[두 번째 발견]
[Figure 2 / Table 2] presents [data type]. [Observation]. [Statistical
finding if applicable: t = ..., p = ...].

[보조 발견 / 추가 분석]
In addition, [secondary finding]. [Detail].

[Negative findings if applicable]
[Variable] did not significantly affect [outcome] (p = [value]).
```

자기 분야 모범 논문의 Results section을 같은 방식으로 분석하면 정확한 template이 만들어집니다.

## 분야별 어휘 — 자기 분야

### 자연과학 / 공학

```
spectra were recorded, peaks were observed
the yield was [value]
the rate constant was determined
phase transition occurred at ...
the morphology was analyzed
```

### 생물학 / 의학

```
expression levels were measured
the survival rate was ...
treatment group differed significantly from control
the IC50 value was ...
```

### 사회과학 / 심리학

```
participants reported / rated / responded
the mean score was ...
the correlation was significant
the regression model explained [variance]
```

### 컴퓨터 / 정보과학

```
the algorithm achieved [accuracy]
the model was evaluated on [dataset]
the runtime was [value]
the F1 score reached ...
```

자기 분야 모범 논문에서 Results에 자주 등장하는 동사 20-30개를 정리합니다.

## 통계 표기 형식

```
t-test:
   t(243) = 3.45, p = .001, d = 0.44

ANOVA:
   F(2, 240) = 8.92, p < .001, η²_p = .07

Correlation:
   r(243) = -.34, p < .001, 95% CI [-.45, -.22]

Regression:
   β = -.32, p < .001, R² = .13

Confidence Interval:
   The mean difference was 2.3 points (95% CI [1.4, 3.2])
```

분야 / 저널마다 형식이 다릅니다. 모범 논문의 표기를 따릅니다.

## 발견의 강도

```
강함:
The data demonstrate that ...
The results clearly show that ...
The findings establish that ...

중간:
The data indicate that ...
The results suggest that ...

부드러움:
The data may suggest that ...
The results appear to indicate that ...
```

자기 데이터의 결정성에 맞춰 동사를 선택합니다.

## 함정

### 1. 해석 섞임

| 분류 | 예문 |
|---|---|
| 회피 | Group A scored higher, suggesting the training was effective. |
| 바름 | Group A scored higher than Group B (Results). [Discussion에서: The higher score may reflect the effectiveness of the training.] |

### 2. 모든 숫자 반복

| 분류 | 예문 |
|---|---|
| 회피 | Group A had a mean of 45.32 (SD = 5.61, range = 32–58), while Group B had a mean of 41.87 (SD = 6.23, range = 28–55) ... |
| 바름 | Group A scored higher than Group B (t = 3.45, p = .001; see Table 1 for details). |

표가 모든 숫자를 보여 줍니다. 본문은 핵심만 적습니다.

### 3. 부정 결과 숨김

부정 결과(가설 미지지)도 반드시 보고합니다. 학술 정직성의 핵심입니다.

```
선호: Contrary to our hypothesis, treatment X did not significantly
       reduce Y (p = .42).
```

## 자기 점검

```
□ 가장 중요한 발견이 앞에 있는가?
□ 표/그림이 본문에서 언급되었는가?
□ 모든 통계값에 효과 크기가 함께 있는가?
□ 해석이 과도하게 들어가지 않았는가?
□ 부정 결과가 빠지지 않았는가?
□ 시제가 Past Simple인가? (관찰된 사실)
```

## 정리

- Results의 표준 흐름은 데이터 설명 → 핵심 발견 → 추가 → 시각 자료입니다.
- 그림·표 참조는 능동, 수동, as-clause, 괄호 형태를 섞어 단조로움을 피합니다.
- Sequence, Frequency, Quantity, Causality 어휘를 익혀 둡니다.
- 상관과 인과를 구분합니다. *causes*와 *is associated with*는 의미가 다릅니다.
- 분야 표준 통계 표기를 따릅니다.
- 흔한 함정은 해석 섞임, 숫자 반복, 부정 결과 숨김입니다.

## 한국 연구자의 Results 함정

```
1. 해석을 Results에 — Discussion으로 옮기기
2. 모든 데이터를 본문에 반복
3. 부정 결과 숨김 — 학술 정직성 위반
4. 상관과 인과 표현 혼동
5. 통계 표기 분야 표준 무시
```
