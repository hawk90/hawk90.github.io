Results의 문법은 *시제*와 *비교 표현*에 집중됩니다. 자기 발견을 어느 강도로 표현하는가가 voice를 결정합니다.

## Results의 시제

기본은 *Past Simple* — 관찰된 사실입니다.

```
The yield increased with temperature.
Group A scored higher than Group B.
The reaction proceeded smoothly.
```

### Present Simple — 일반화 / 그림 참조

```
일반화:
The data show a clear trend.
The compound exhibits high stability.

그림 참조:
Figure 1 shows the temperature dependence.
Table 2 presents the descriptive statistics.
```

### Past Simple — 특정 결과

```
The yield was 87% at 100°C.
Participants in Group A responded faster (M = 245 ms) than those
in Group B (M = 312 ms).
```

### 같은 단락의 시제 혼합

```
The temperature dependence of the yield is shown in Figure 1.
The yield increased from 65% at 80°C to 87% at 100°C. Above 100°C,
the yield remained constant.

[현재: 그림 참조]   [과거: 관찰된 사실]
```

이 혼합이 자연스럽습니다.

## Comparative와 Superlative

### 형용사 비교급·최상급

```
1-2 음절: -er, -est
   tall → taller → tallest
   busy → busier → busiest

3음절 이상: more, most
   important → more important → most important
   significant → more significant → most significant

불규칙:
   good → better → best
   bad → worse → worst
   far → farther/further → farthest/furthest
   much → more → most
   little → less → least
```

### 비교 패턴

```
A is larger than B.
A is much larger than B.        ← 정도 부사
A is significantly larger than B.
A is approximately twice as large as B.
```

### Times / Folds

```
A is twice as large as B.
A is three times as large as B.
A is ten-fold higher than B.
A is an order of magnitude greater than B.
```

### Percent / Degree

```
A is 30% larger than B.
A is 30 percent higher than B.
A increased by 30%.    ← 변화량
A reached 30%.         ← 도달값
```

### 비교의 함정

```
회피: A is larger.
       (무엇과 비교? 모호)

선호: A is larger than B.
또는: A is larger than the control group.
```

비교는 항상 *대상 명시*가 필요합니다.

## 정도 부사 — Modifying Comparatives

```
약함 → 강함:
slightly larger
somewhat larger
moderately larger
considerably larger
significantly larger
substantially larger
much larger
significantly larger (with p-value)
```

```
slightly larger     — 약간 큼
considerably larger — 상당히 큼
significantly larger — 통계적으로 유의
```

*significantly*는 학술 글에서 *통계적 유의성*을 가리키므로 *일반적 강조*에는 쓰지 않습니다.

## Statistical Significance — 표현

### 유의함

```
significantly differed
significantly higher than
significantly correlated with
showed a significant effect
```

### 유의하지 않음

```
did not significantly differ
no significant effect was observed
the difference was not statistically significant
```

### 경향 (marginal)

```
approached significance (p = .07)
   ← 비판받기 쉬운 표현. 더 정확한 표현이 안전:

not significant but in the predicted direction (p = .07)
a trend toward [direction] was observed
```

## 함정 — Negative Results

### 1. 부정을 약화

```
회피 (애매한 약화):
There was perhaps a slight tendency toward a small difference.

선호 (명확한 부정):
No significant difference was found between groups.
```

부정 결과도 *명확하게* 보고합니다.

### 2. 결과 누락

가설을 검증하기 위한 모든 분석을 보고합니다 — 부정 결과도.

```
선호: Of the five hypothesized effects, three were confirmed
       (H1, H2, H5), while two were not supported (H3, H4).
```

### 3. p-value 강조

```
회피 (강한 표현 + 약한 p):
The data strongly suggest a significant effect (p = .049).

선호 (p에 맞는 강도):
The data suggest a marginally significant effect (p = .049).
```

`p < .05`라도 *.049*는 *.001*보다 약합니다. 그에 맞는 어조를 씁니다.

## 발견의 강도 조절

### Strong (강함)

```
The data demonstrate that ...
The results clearly show that ...
The findings establish that ...
```

### Moderate (중간)

```
The data indicate that ...
The results show that ...
```

### Soft (부드러움)

```
The data suggest that ...
The results imply that ...
The findings appear to indicate that ...
```

### Tentative (잠정적)

```
The data may suggest that ...
The results possibly indicate ...
The findings could be interpreted as ...
```

자기 데이터의 *결정성*에 맞춰 동사를 선택합니다.

## 사례 — 한 단락

다음 단락을 문법 관점에서 점검합니다.

```
The temperature dependence of the yield is shown in Figure 1. As
the temperature increased from 80°C to 100°C, the yield rose
significantly from 65% to 87% (a 22-percentage-point increase).
Above 100°C, the yield remained constant, suggesting that the
reaction reached saturation. In contrast, the yield was unaffected
by changes in stirring speed (p = .54), indicating that mass
transfer was not rate-limiting under our conditions.
```

분석:

- **Present (그림 참조)** — *is shown*
- **Past (관찰)** — *increased, rose, remained, was unaffected*
- **비교** — *from 80°C to 100°C*, *from 65% to 87%*
- **정량 정도** — *significantly*, *a 22-percentage-point increase*
- **인과** — *suggesting that*, *indicating that* (약한 인과)
- **부정 결과** — *was unaffected ... p = .54* (명확히 보고)

이 단락이 Results의 좋은 모델입니다.

## 자기 점검

```
□ 기본 시제가 *Past Simple*인가?
□ 그림 참조가 *Present Simple*인가?
□ 비교에 *대상이 명시*되었는가?
□ *significantly*가 통계적 의미로 사용되었는가?
□ 부정 결과가 *명확히* 보고되었는가?
□ 발견의 강도가 데이터에 맞는가?
```

## 정리

- Results의 시제 — 발견은 *Past*, 그림 참조와 일반화는 *Present*
- 비교는 *대상 명시* + 정도 부사
- *Significantly*는 통계적 유의성 — 일반 강조 금지
- 부정 결과도 *명확히* 보고
- 발견의 강도(*demonstrate / indicate / suggest*)를 데이터에 맞추기

## 한국 연구자의 Results Grammar 함정

```
1. Significantly 일반 강조 → 통계적 의미만
2. 비교 대상 누락
3. Past / Present 자리 흔들림
4. 동사 강도 데이터와 불일치
```
