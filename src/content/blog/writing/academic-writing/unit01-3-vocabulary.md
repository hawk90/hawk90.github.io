학술 영어의 어휘는 *일상 영어와 패턴이 분명히 다릅니다*. Swales와 Feak는 *두 가지 어휘 이동* (vocabulary shift)을 강조합니다.

```
Shift 1: Phrasal verb (동사 + 전치사) → 단일 라틴 동사
Shift 2: 동사 표현 → 명사구 (nominalization)
```

두 이동 모두 글을 *더 압축적이고 격식 있게* 만듭니다.

## Shift 1 — Phrasal Verb → 단일 Latinate Verb

영어는 *한 동작*을 표현하는 데 보통 *두 가지 선택지*가 있습니다.

```
선택지 1: 동사 + 전치사 (phrasal / prepositional verb)
   - Anglo-Saxon 어원이 많음
   - 일상 회화 / 강의에 흔함
   - 짧고 친숙

선택지 2: 단일 라틴 동사 (Latinate verb)
   - 라틴 / 프랑스어 어원
   - 학술 영어 산문에 흔함
   - 압축적이고 격식적
```

```
일상:
"Given our fast-paced society, people must routinely put creative
solutions to unexpected problems into practice."

학술:
"Given our fast-paced society, people must routinely implement
creative solutions to unexpected problems."
```

*put ... into practice* (5단어, 분리) → *implement* (1단어, 압축).

### 자주 등장하는 짝

```
일상 (phrasal)          학술 (Latinate)
─────────────────────────────────────
put up with             tolerate, endure
look into               investigate, examine
figure out              determine, ascertain
come up with            develop, devise, propose
make up                 constitute, comprise
get rid of              eliminate, remove
go up to                reach, attain
keep up                 maintain, sustain
go down                 decrease, decline
think about             consider, contemplate
bring up                raise (an issue)
run into                encounter
show up                 appear, emerge
look at                 examine, investigate
break down              analyze, decompose
build up                develop, accumulate
find out                determine, discover
get back                recover, retrieve
go through              undergo, experience
look for                seek, search for
make out                discern, perceive
point out               indicate, note
put off                 postpone, defer
set up                  establish, install
take in                 absorb, comprehend
work out                resolve, calculate
```

*전부 외울 필요는 없습니다*. 다만 *글을 다시 읽을 때 phrasal verb가 눈에 띄면* *단일 동사로 바꿔 볼지 검토*. 글이 *자연스럽게 학술적*이 됩니다.

### 모든 phrasal verb를 바꿀 필요는 없음

일부 phrasal verb는 *영어에 깊이 뿌리내려* 단일 동사 대체가 *어색*합니다.

```
학술 영어에서도 흔히 쓰이는 phrasal verb:
- bring about (결과를 가져오다)
- carry out (수행하다) ← 학술 영어에서 매우 흔함
- set up (구성 / 설정하다)
- point out (지적하다)
- give rise to (~의 원인이 되다)
- result in (~로 귀결되다)
- consist of (~로 구성되다)
- depend on (~에 의존하다)
- focus on (~에 집중하다)
- rely on (~에 의존하다)
```

```
선호 (학술에서 자연):
"We carried out three experiments."
   → "We conducted three experiments."도 OK
   → 둘 다 자연스러움

"The committee pointed out several issues."
   → "The committee identified several issues."도 가능
   → 둘 다 자연스러움
```

핵심은 *나의 글이 회화체로 흐르고 있는지* 점검하는 것. *모든 phrasal verb를 추방*하는 것이 아닙니다.

## Shift 2 — Nominalization (명사화)

영어는 어휘가 매우 풍부해서 *같은 생각*을 *동사로* 표현할 수도, *명사구로 압축*할 수도 있습니다. 학술 영어는 *명사구로 압축*하는 쪽을 즐깁니다.

### 명사구의 확장 — *Language* 예시

단순한 명사 *language*를 *점점 확장*해 가는 예.

```
language
    ↓
the language of scientific communication
    ↓
the international language of scientific communication
    ↓
English as the international language of scientific
communication
    ↓
the emergence of English as the international language of
scientific communication
```

마지막 명사구는 *그 자체로 한 사건 / 한 현상*을 가리킵니다. 이를 *주어*로 받아 *한 문장*에 압축할 수 있습니다.

```
선호 (압축):
"The emergence of English as the international language of
scientific communication has been widely documented."
```

같은 내용을 *명사화 없이 풀어 쓰면* *두 문장*이 됩니다.

```
풀어 쓰기 (두 문장):
"English has emerged as the international language of
scientific communication. This phenomenon has been widely
documented."
```

두 표현 모두 *문법적으로 맞지만*, 학술 산문에서는 *압축된 명사구*가 더 잘 어울립니다. *밀도 높은 정보*를 *짧게* 전달.

### Nominalization의 이점

```
1. 정보 밀도 — 한 문장에 더 많이 담음
2. Flow — 주제 (theme)와 평언 (rheme)을 명사구로 미리 묶음
3. 객관성 — 행위자가 흐릿
4. 추상화 — 사건 / 현상을 한 단위로 다룸
```

### Nominalization의 위험

지나치면 글이 *무겁고 추상적*이 됩니다. Strunk가 *Elements of Style*에서 경고한 *Latinate 과잉*과 같은 위험.

```
회피 (지나친 명사화):
"The implementation of the optimization of the configuration
of the system requires the consideration of the minimization
of the cost."

→ 5개의 명사화 (implementation, optimization, configuration,
   consideration, minimization)
→ 무겁고 추상

선호 (풀어 쓰기):
"To configure the system optimally, we need to minimize cost."
→ 명사 → 동사로 환원
→ 가볍고 구체
```

```
회피:
"A discussion of the impact of climate change on agricultural
productivity in developing nations was conducted."

선호:
"We discussed how climate change affects agricultural
productivity in developing nations."
```

학술 영어에 *명사화가 흔하다*는 것이 *모든 동사를 명사화하라*는 뜻은 *아닙니다*. 자기 분야의 *모범 논문*이 *어느 정도* 명사화를 쓰는지 *관찰*하고 그 정도에 맞춥니다.

### 명사화 패턴

```
동사 → 명사 변환:
implement   → implementation
analyze     → analysis
optimize    → optimization
investigate → investigation
develop     → development
configure   → configuration
minimize    → minimization
decide      → decision
discuss     → discussion
emerge      → emergence
react       → reaction
construct   → construction

형용사 → 명사:
important   → importance
complex     → complexity
significant → significance
relevant    → relevance
```

이 접미사 (-tion, -ment, -sion, -ance, -ence, -ity)가 *명사화의 신호*. *과다하면* 글이 *추상*해짐.

## Latinate vs. Anglo-Saxon

영어는 *두 큰 어휘 출처*를 갖습니다 — *Anglo-Saxon*과 *Latin / Greek*. 보통 *같은 의미*의 두 단어가 *양쪽에서* 옵니다.

```
Anglo-Saxon (앵글로색슨)        Latinate (학술적)
─────────────────────────────────────
begin                          commence, initiate
end                            conclude, terminate
help                           assist, facilitate
show                           demonstrate, indicate
think about                    consider, contemplate
find out                       determine, ascertain
use                            utilize, employ
get                            obtain, acquire
big                            substantial, significant
small                          minor, insignificant
about                          concerning, regarding
build                          construct, establish
make                           produce, manufacture
need                           require, necessitate
ask                            inquire, request
keep                           maintain, retain
try                            attempt, endeavor
go up                          ascend, increase
go down                        descend, decrease
break                          fracture, rupture
```

학술 영어는 *라틴 어원*을 선호. 다만 *Strunk의 경고*:

> *Anglo-Saxon is a livelier tongue than Latin.*

라틴 어원을 *과시*하는 데 쓰지 말고, *정확성과 압축*이 필요한 자리에만 씁니다.

```
회피 (과시):
"We utilized this resource to obtain the requisite data."

선호:
"We used this resource to get the necessary data."
```

```
선호 (정확성):
"We consider three factors."  (Anglo-Saxon "think about"보다 압축적)
"We obtain the result."        (적절한 격식)
```

판단의 기준은 *과시인가, 정확성인가*. *Utilize*는 거의 항상 *use*로 충분. *Consider*는 *think about*보다 *압축적*이라 자연.

## 분야별 어휘 농도

```
어휘 농도 (라틴 어원 / 명사화의 비율):

높음 (75%+):
- 법학, 철학
- 의학 (특히 임상)
- 화학, 물리학 (이론)

중간 (50-70%):
- 사회과학 (사회학, 정치학)
- 공학 (응용)
- 경영, 경제

낮음 (30-50%):
- 인문학 (문학, 역사)
- 교육학
- 인류학 (질적)
- 저널리즘 학술

매우 낮음 (20% 이하):
- 일반 대중 글
- 블로그
- 회화
```

자기 분야의 *모범 논문 5편*에서 *임의의 단락*을 *어휘 분석*해 보면 *분야 농도*가 보입니다.

## 실용적 점검 — Task

학술 어휘로 바꾸는 연습. 다음 문장에서 italic 부분을 *단일 동사*로.

```
1. Software manufacturers in developed countries *put up with*
   widespread copyright violations.
   → tolerate

2. Scientists are *looking into* innovative drug delivery systems.
   → investigating

3. The purpose of this paper is to *figure out* what is lacking
   in our understanding of corrosion.
   → determine

4. Researchers have *come up with* plug-in hybrid vehicles.
   → developed / devised

5. Rice and aquatic products *make up* a major part of the diet.
   → constitute

6. Touch screen voting could *get rid of* many problems.
   → eliminate

7. Worldwide consumption has *gone up to* 2.6 million metric tons.
   → reached

8. Labor unions have been able to *keep up* their membership
   numbers.
   → maintain

9. The number of mature green turtles has *gone down* from 1,280
   to 145.
   → decreased / declined

10. Many funding agencies are *thinking about* ways to give
    researchers more opportunities.
    → considering

11. We need to *find out* what causes this.
    → determine / ascertain

12. The committee *brought up* several concerns.
    → raised

13. The data *shows* a clear trend.
    → indicates / demonstrates / shows (격식 글에서도 OK)

14. We *got* the results from the database.
    → obtained / retrieved

15. The team *put off* the meeting.
    → postponed / deferred
```

## 한국어와의 비교

### 한국어 학술 어휘의 두 출처

```
한국어:
1. 순한국어 (Native Korean)
2. 한자어 (Sino-Korean, 학술적)

영어:
1. Anglo-Saxon (Native English)
2. Latinate (학술적)

→ 두 언어 모두 *두 어원이 공존*
→ 학술 글은 *후자*를 선호
```

```
순한국어                  한자어 (학술)
─────────────────────────────────────
시작하다                  개시하다, 착수하다
끝내다                    종료하다, 완료하다
도와주다                  지원하다, 협력하다
보여주다                  제시하다, 시현하다
생각하다                  고려하다, 사고하다
알아내다                  파악하다, 확인하다
사용하다                  활용하다, 이용하다
얻다                      획득하다, 취득하다
크다                      거대하다, 막대하다
작다                      미미하다, 소규모이다
```

영어 *Latinate* 선호와 *비슷한 구조*. 한국 학술 글도 *한자어 중심*.

### 한국 학생의 직역 함정

```
회피 (영어 phrasal 직역):
한국어 직역: "이 문제를 들여다보겠다"  (look into)
선호 (한자어): "이 문제를 조사하겠다"

회피 (영어 nominalization 직역):
한국어 직역: "시스템의 구성의 최적화의 구현"
선호 (한국어 자연): "시스템을 최적으로 구성하기"
```

영어 *학술 어휘 패턴*을 *그대로 한국어로* 옮기면 *번역체*. *각 언어의 학술 관습*을 따릅니다.

## 자기 글 점검 — 어휘 농도

```
□ Phrasal verb가 *과다*하지 않은가?
   - 5개 이상 / 단락 → 검토
□ Phrasal을 Latinate로 바꿀 때 *자연스러운가*?
   - 무리하면 회피 (carry out → conduct는 OK, 그러나 강제 아님)
□ 명사화 (-tion, -ment 등)가 *과다*하지 않은가?
   - 한 문장에 3개 이상 → 검토
□ 자기 분야의 *어휘 농도*에 맞는가?
   - 모범 논문과 비교
□ 라틴 어원을 *과시*하지 않는가?
   - utilize → use (대부분)
□ 한국 글에 *영어 학술 어휘 직역*이 없는가?
```

## 정리

- 학술 영어 어휘의 *두 이동* — **phrasal verb → 단일 라틴 동사**, **동사 표현 → 명사구**
- 일상 회화: *put ... into practice*. 학술: *implement*.
- *긴 명사구*가 학술 영어의 특징. *한 문장에 정보를 압축*
- Anglo-Saxon vs. Latinate — 학술은 후자를 선호, 다만 *과시*는 금물
- 자기 분야 *모범 논문의 어휘 농도*를 관찰하고 맞춤
- 한국어도 *순한국어 / 한자어*의 이분 — 영어와 같은 구조
- 명사화 과잉 — Strunk가 경고한 *Latinate 과잉*과 같은 위험
