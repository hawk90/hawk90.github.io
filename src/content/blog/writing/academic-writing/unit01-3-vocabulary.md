학술 영어의 어휘는 일상 영어와 패턴이 분명히 다릅니다. Swales와 Feak는 *두 가지 어휘 이동(vocabulary shift)*을 강조합니다. 첫째, phrasal verb(동사+전치사)를 *단일 라틴 동사*로 바꾸기. 둘째, 동사 + 명사 표현을 *명사구(nominalization)*로 압축하기. 두 이동 모두 글을 더 *압축적이고 격식 있게* 만듭니다.

## Shift 1 — Phrasal Verb → 단일 Latinate Verb

영어는 한 동작을 표현하는 데 보통 두 가지 선택지가 있습니다. 하나는 *동사 + 전치사*(phrasal verb 또는 prepositional verb)이고, 다른 하나는 같은 의미의 *라틴 어원 단일 동사*입니다. 일상 회화와 강의에서는 phrasal verb가 흔하지만, 학술 영어 산문에서는 *단일 라틴 동사*가 선호됩니다.

```
일상:   Given our fast-paced society, people must routinely put
        creative solutions to unexpected problems into practice.
학술:   Given our fast-paced society, people must routinely implement
        creative solutions to unexpected problems.
```

`put ... into practice` (5단어, 분리되어 있음) → `implement` (1단어, 압축).

### 자주 등장하는 짝

다음은 학술 영어에서 자주 바뀌는 phrasal verb 짝입니다.

```
put up with    →  tolerate
look into      →  investigate
figure out     →  determine
come up with   →  develop, devise
make up        →  constitute
get rid of     →  eliminate
go up to       →  reach
keep up        →  maintain
go down        →  decrease, decline
think about    →  consider, contemplate
bring up       →  raise (an issue)
run into       →  encounter
show up        →  appear, emerge
look at        →  examine, investigate
```

이런 짝을 다 외울 필요는 없습니다. 다만 *글을 다시 읽을 때 phrasal verb가 눈에 띄면*, 단일 동사로 바꿔 볼지 검토하면 됩니다. 글이 자연스럽게 학술적이 됩니다.

### 주의 — 모든 phrasal verb를 바꿀 필요는 없음

일부 phrasal verb는 영어에 깊이 뿌리내려 단일 동사 대체가 어색합니다.

```
bring about (어떤 결과를 가져오다)
carry out (수행하다, 학술 영어에서 매우 흔함)
set up (구성하다, 설정하다)
point out (지적하다)
```

`carry out` 같은 표현은 학술 영어에서도 *자연스럽고 흔히 쓰입니다*. 무리해서 *conduct*로 바꾸지 않아도 됩니다. 핵심은 *나의 글이 회화체로 흐르고 있는지* 점검하는 것입니다.

## Shift 2 — Nominalization (명사화)

영어는 어휘가 매우 풍부해서 같은 생각을 *동사로* 표현할 수도, *명사구로 압축*할 수도 있습니다. 학술 영어는 명사구로 압축하는 쪽을 즐깁니다. 그래서 학술 영어 텍스트에는 *긴 명사구*가 많이 등장합니다.

다음은 단순한 명사 `language`를 점점 확장해 가는 예입니다.

```
language
    ↓
the language of scientific communication
    ↓
the international language of scientific communication
    ↓
English as the international language of scientific communication
    ↓
the emergence of English as the international language
of scientific communication
```

마지막 명사구는 그 자체로 *한 사건 / 한 현상*을 가리킵니다. 이를 주어로 받아 한 문장에 압축할 수 있습니다.

```
The emergence of English as the international language of scientific
communication has been widely documented.
```

같은 내용을 명사화 없이 풀어 쓰면 두 문장이 됩니다.

```
English has emerged as the international language of scientific
communication. This phenomenon has been widely documented.
```

두 표현 모두 문법적으로 맞지만, 학술 산문에서는 *압축된 명사구*가 더 잘 어울립니다. *밀도 높은 정보를 짧게* 전달하기 때문입니다.

### Nominalization의 이점

- 한 문장에 *더 많은 정보*를 담을 수 있음
- 주제(theme)와 평언(rheme)을 *명사구로 미리 묶음* — flow가 매끄러워짐
- 객관적이고 *행위자(agent)가 흐릿한* 느낌 — 학술 영어 톤에 맞음

### Nominalization의 위험

지나치면 글이 *무겁고 추상적*이 됩니다. Strunk가 *Elements of Style*에서 경고한 *Latinate 과잉*과 같은 위험입니다.

```
지나친 명사화:
The implementation of the optimization of the configuration of the
system requires the consideration of the minimization of the cost.

풀어 쓰기:
To configure the system optimally, we need to minimize cost.
```

학술 영어에 *명사화가 흔하다*는 것이 *모든 동사를 명사화하라*는 뜻은 아닙니다. 자기 분야의 모범 논문이 *어느 정도* 명사화를 쓰는지 관찰하고 그 정도에 맞춥니다.

## Latinate vs. Anglo-Saxon

영어는 두 큰 어휘 출처를 갖습니다 — *Anglo-Saxon*과 *Latin/Greek*. 보통 같은 의미의 두 단어가 양쪽에서 옵니다. 학술 영어는 라틴 어원 단어를 선호합니다.

| Anglo-Saxon | Latinate (학술적) |
|-------------|------------------|
| begin | commence, initiate |
| end | conclude, terminate |
| help | assist, facilitate |
| show | demonstrate, indicate |
| think about | consider, contemplate |
| find out | determine, ascertain |
| use | utilize, employ |
| get | obtain, acquire |
| big | substantial, significant |
| small | minor, insignificant |
| about | concerning, regarding |

다만 Strunk의 경고 — *Anglo-Saxon is a livelier tongue than Latin*. 라틴 어원을 *과시*하는 데 쓰지 말고, *정확성과 압축*이 필요한 자리에만 씁니다. `Utilize`는 거의 항상 `use`로 충분합니다. 그러나 `consider`는 `think about`보다 *압축적*이라 자연스럽습니다. 판단의 기준은 *과시인가, 정확성인가*입니다.

## 실용적 점검 — Task

학술 어휘로 바꾸는 연습입니다. 다음 문장에서 italic 부분을 단일 동사로 바꿉니다.

```
1. Software manufacturers in developed countries put up with widespread
   copyright violations.   → tolerate

2. Scientists are looking into innovative drug delivery systems.   → investigating

3. The purpose of this paper is to figure out what is lacking in our
   understanding of corrosion.   → determine

4. Researchers have come up with plug-in hybrid vehicles.   → developed

5. Rice and aquatic products make up a major part of the diet.   → constitute

6. Touch screen voting could get rid of many problems.   → eliminate

7. Worldwide consumption has gone up to 2.6 million metric tons.   → reached

8. Labor unions have been able to keep up their membership numbers.   → maintain

9. The number of mature green turtles has gone down from 1,280 to 145.   → decreased

10. Many funding agencies are thinking about ways to give researchers
    more opportunities.   → considering
```

## 정리

- 학술 영어 어휘의 두 이동 — **phrasal verb → 단일 라틴 동사**, **동사 표현 → 명사구**
- 일상 회화: *put ... into practice*. 학술: *implement*.
- *긴 명사구*가 학술 영어의 특징. 한 문장에 정보를 압축
- Anglo-Saxon vs. Latinate — 학술은 후자를 선호, 다만 *과시*는 금물
- 자기 분야 모범 논문의 *어휘 농도*를 관찰하고 맞춤
