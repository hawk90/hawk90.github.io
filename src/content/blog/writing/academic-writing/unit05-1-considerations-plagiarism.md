학술 글쓰기에서 *요약(summary)*은 거의 모든 글에 들어갑니다. 문헌 리뷰의 각 문단이 요약이고, 토론 섹션의 *기존 연구 정리*도 요약입니다. Unit 5의 첫 글은 요약을 쓰기 *전에* 결정해야 할 것들과, 학술 정직성의 문제인 *표절*을 다룹니다.

## 요약을 쓰기 전에 — Considerations

좋은 요약은 *결정의 결과*입니다. 텍스트를 그대로 줄이는 작업이 아닙니다.

### 다섯 가지 결정

```
1. 목적 — 왜 이 요약을 쓰는가?
2. 길이 — 얼마나 길게?
3. 독자 — 누가 읽을 것인가?
4. 강조 — 무엇을 부각할 것인가?
5. 자기 입장 — 요약 안에 자기 입장이 들어가는가?
```

같은 논문을 두 사람이 요약하면 보통 *다른 요약*이 나옵니다. *목적과 강조*가 다르기 때문입니다.

### 목적의 종류

```
- 정보 전달 — 무엇이 발견되었는지 알리기
- 비판 준비 — 자기 비판의 *대상*으로 자리매김
- 자기 연구의 *맥락* — 기존 연구와의 위치 잡기
- 학습 — 자기 이해를 정리
```

목적이 다르면 강조점이 다릅니다.

```
같은 논문 (Smith, 2020 — 카페인과 수면):

정보 전달용 요약:
Smith (2020) found that caffeine intake of more than 400mg
significantly reduced sleep efficiency in young adults.

비판 준비용 요약:
Smith (2020) reported a negative relationship between caffeine intake
and sleep efficiency, although the study relied solely on self-reported
sleep data, which may have introduced significant measurement error.

맥락용 요약:
Building on earlier work that focused on adults aged 30–50, Smith
(2020) extended the analysis to young adults and confirmed similar
patterns.
```

같은 사실에 *다른 정보가 강조*됩니다.

## 요약의 길이

```
한 문장 요약 — 인용으로 들어가는 짧은 요약
한 단락 요약 — 한 연구를 다루는 단락
한 절 요약 — 여러 연구의 통합
한 chapter 요약 — 책 한 권의 핵심
```

각각 다른 어휘 밀도와 인용 패턴이 필요합니다. *한 문장 요약*은 가장 어렵습니다 — 한 문장에 *연구의 핵심*이 들어가야 합니다.

## 자기 입장의 표시

요약 안에 자기 입장이 *암묵적으로* 또는 *명시적으로* 들어갈 수 있습니다.

```
중립:
Smith (2020) reported a negative relationship.

암묵적 비판:
Smith (2020) reported a negative relationship, based solely on self-
reported data.   ← 한계 명시로 약화

명시적 비판:
Although Smith (2020) reported a negative relationship, the study has
significant methodological limitations.

암묵적 동의:
Confirming earlier findings, Smith (2020) reported ...

명시적 통합:
Smith (2020) provides further evidence for the negative impact of
caffeine on sleep.
```

학술 글의 어조는 *과도한 비판도, 과도한 동조도* 피합니다. 균형 잡힌 *위치 잡기*가 학술적 신뢰를 만듭니다.

## Plagiarism — 표절

표절은 학술 정직성의 가장 심각한 위반입니다. 학위가 취소되거나 학술적 경력이 끝날 수 있는 사안입니다.

### 표절의 정의

```
다른 사람의 *말, 생각, 데이터*를 *출처 표시 없이* 자기 것처럼 쓰는 것
```

세 가지 모두 표절입니다.

```
1. 말 — 문장이나 구를 그대로 또는 거의 그대로 옮김
2. 생각 — 다른 사람의 *개념, 가설, 해석*을 출처 없이 사용
3. 데이터 — 다른 사람의 *측정값, 표, 그래프*를 자기 것처럼 제시
```

### 표절의 유형

```
직접 표절 — 한 문장을 그대로 옮김
패치워크 표절 — 여러 출처의 문장을 짜깁기
패러프레이즈 표절 — 단어만 바꾸고 구조는 그대로
자기 표절 — 자기 이전 글을 출처 없이 재사용
모자이크 표절 — 단어 일부만 바꿔 새 문장처럼 보이게
```

자기 표절도 *문제*라는 점이 학생에게 종종 놀랍습니다. 같은 결과를 두 논문에 발표하면 *학술 데이터베이스의 신뢰*를 깎습니다. 자기 이전 글을 활용할 때도 *적절한 인용*이 필요합니다.

### 표절을 피하는 방법

```
1. 모든 인용에 *출처 표시*
   - 직접 인용: 따옴표 + 페이지 번호
   - 패러프레이즈: 저자 + 연도

2. 자기 생각과 인용을 *명확히 구분*
   - "Smith (2020) argues that ... I would add that ..."

3. 패러프레이즈 시 *충분히 다시 쓰기*
   - 단어만 바꾸지 않고 *문장 구조*도 변경

4. 메모할 때 *출처 함께* — 나중에 헷갈리지 않게

5. 의심스러우면 *인용*
   - "이건 표절일까?" 고민되면 출처 표시
```

### 직접 인용의 형식

```
짧은 인용 (3-4줄 이하) — 따옴표 안에:
According to Hyland (2008), "Control of this voice is central to building
a convincing discourse" (p. 17).

긴 인용 (4줄 이상) — 별도 단락, 들여쓰기, 따옴표 없이:
   Skelton (1988) describes the challenge facing students:
       It is important for students to learn to be confidently
       uncertain. (p. 178)
```

직접 인용을 *남발*하지 마세요. *3% 이하*가 일반적 기준입니다. 자기 글이 *인용 모음*이 되면 *자기 분석*이 약해 보입니다.

### Common Knowledge의 예외

*상식*이나 *일반적으로 알려진 사실*은 인용이 필요 없습니다.

```
인용 불필요:
- The Earth orbits the Sun.
- Water boils at 100°C at sea level.
- World War II ended in 1945.

인용 필요:
- 통계 (특정 연도, 특정 조사)
- 해석 (다른 사람의 관점)
- 특정 연구의 발견
- 새로운 이론이나 개념
```

*상식이 무엇인가*는 분야와 독자에 따라 다릅니다. *학부 1학년 교과서*에 나오는 것은 보통 상식입니다. *전공 논문*에 나오는 것은 보통 인용이 필요합니다.

## 표절이 일어나는 흔한 자리

### 1. Literature Review

```
잘못된 패턴:
원문: "Caffeine has been shown to increase alertness through adenosine
       receptor antagonism."
학생: "Caffeine has been shown to increase alertness through adenosine
       receptor antagonism (Brown, 2020)."  ← 인용은 했지만, 문장이 거의 같음
```

인용을 했어도 *직접 인용 표시*가 없으면 표절입니다. 따옴표를 두거나, 패러프레이즈를 해야 합니다.

### 2. Methods Section

다른 연구의 *방법*을 그대로 옮기는 경우. *Methods 섹션은 자기 말로 다시 써야* 합니다. 같은 방법이면 *Brown et al. (2019)의 방법을 따랐다*고 명시합니다.

### 3. Translated Content

다른 언어로 된 원문을 번역해 *자기 글*로 쓰는 것. 이것도 표절입니다. 번역도 *원저자에게 출처 표시*가 필요합니다.

### 4. AI-Generated Content

LLM이 생성한 내용을 *자기 글*로 제출하는 것. 많은 학교가 이를 *표절*로 분류합니다. 학교의 정책을 확인하세요.

## 학술 정직성의 더 큰 그림

표절 회피는 *규칙의 문제*만이 아닙니다. 학술 공동체의 *대화*에 참여하는 *방식*입니다.

```
내 글에 다른 사람의 말이 들어옴 → 인용
다른 사람의 생각을 빌림 → 인용
나의 새 기여가 무엇인지 명확히 표시 → 명시적
```

좋은 학술 글은 *기존 연구와의 대화*입니다. 모든 인용이 *자기 위치 잡기*의 도구입니다. 표절은 단순한 *규칙 위반*이 아니라 *대화의 거부*입니다.

## 정리

- 요약은 *결정의 결과* — 목적, 길이, 독자, 강조, 자기 입장
- 같은 텍스트도 *목적에 따라* 다른 요약이 나옴
- **표절** — 말, 생각, 데이터를 출처 없이 사용
- 표절의 유형 — 직접, 패치워크, 패러프레이즈, 자기, 모자이크
- 회피 방법 — 출처 표시, 자기 생각 구분, 충분한 패러프레이즈, 메모와 출처 함께
- 직접 인용은 *짧고 드물게*
- *Common knowledge*는 인용 불필요, 다만 기준은 *분야*에 따라 다름

## 한국 학생의 표절 함정

```
1. 문화적 인식 차이 — 명시가 필수
2. 한국어 책 인용 시 *원본 출처도 표기*
3. AI 도구 사용 시 *AI 출처 표기*
4. 자기 표절 — 자기 이전 글도 출처
```

## 자기 글 점검

```
□ 모든 인용에 *출처 표기*?
□ 직접 인용은 *짧고 드물게*?
□ 패러프레이즈가 *충분히 변형*?
□ 한국 / AI / 자기 출처도?
```
