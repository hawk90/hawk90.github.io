학술 영어 문체는 어휘만의 문제가 아닙니다. *문장 수준의 작은 결정들*이 모여 학술 톤을 만듭니다. Swales와 Feak는 열 가지 항목을 정리합니다 — 이 글이 그것을 한 자리에 모았습니다.

## 1. *I* / *we* — 1인칭을 쓸 것인가

분야마다 다릅니다. 일부 분야(공학, 자연과학)에서는 단일 저자도 *we*를 쓰는 관습이 있습니다. 협업 연구가 많기 때문입니다. 인문학에서는 *I*가 더 흔합니다. *MICUSP* 코퍼스 데이터에 따르면 1만 단어당 *I* 출현 빈도는 컴퓨터 과학에서 5.6, 철학에서 53.9로 큰 차이가 납니다.

```
1인칭 사용:
In this paper I argue that small incentives can lead to greater
participation in surveys.

3인칭 / 수동:
This paper argues that small incentives can lead to greater
participation in surveys.
```

연구가 보여 주는 것 — *I*와 *we*가 학술 영어에 등장하는 것은 분명한 사실. 다만 *새 저자는 사용을 주저*하는 경향이 있습니다. 자기 분야의 모범 논문이 어느 쪽인지 확인하고 그에 맞춥니다.

## 2. Contractions — 줄임말

대부분의 학술 분야에서 contractions(`don't`, `won't`, `it's`)는 *드물게* 사용됩니다. 격식 있는 글에서는 보통 풀어 씁니다.

```
회화체: Export figures won't improve until the economy is stronger.
학술:   Export figures will not improve until the economy is stronger.
```

다만 *반론*이나 *강한 부정 강조*가 필요한 자리에서는 contraction이 의도적으로 쓰일 수도 있습니다.

## 3. 부정형 — *not* 의 두 위치

같은 부정도 *형태*에 따라 격식이 달라집니다. 학술 영어에서는 *not + 일반 단어* 대신 *부정 의미가 내장된 단일 단어*를 선호합니다.

```
회화체:                          학술적:
The analysis didn't yield        The analysis yielded
any new results.                  no new results.

The government didn't            The government allocated
allocate much funding.           little funding.

This problem doesn't have        This problem has
many sustainable solutions.      few sustainable solutions.
```

`Not ... any` → `no`, `not ... much` → `little`, `not ... many` → `few`로 압축합니다.

같은 패턴 — `not ... different` → `similar`, `not ... important` → `trivial`. Strunk의 Rule 15 "Put statements in positive form"과 같은 메시지입니다.

## 4. *etc.*, *and so forth*, *and so on*

저널 에디터 다수가 이 표현들에 *반대*합니다. 정확하지 않고, 독자에게 *빠진 정보를 채워 넣으라고* 요구하기 때문입니다.

```
약함:
Micropumps can be used in drug delivery, lab-on-a-chip analysis, etc.

강함:
Micropumps can be used in drug delivery, lab-on-a-chip analysis,
ink dispensing, and other specialized applications that require
self-contained, low-power, miniature pumps.
```

목록의 *마지막에 *etc.**를 두는 것은 그 항목이 *중요하지 않다*는 신호와도 같습니다. 정말 중요한 항목이면 *이름을 부르면* 됩니다. Strunk도 같은 입장입니다(Ch4 항목 `Etc.` 참조).

## 5. 독자 직접 호명 — *you*

대부분 분야에서 학술 영어는 *you*로 독자를 직접 부르지 않습니다(교과서나 매뉴얼 같은 *지시문*은 예외).

```
회화체: You can see the results in Table 1.
학술:   The results can be seen in Table 1.
```

수동태가 자연스럽게 쓰이는 자리이기도 합니다. *you* 회피는 *수동태 사용*과 연결됩니다.

## 6. 직접 / 간접 질문

직접 의문문은 *주의를 끄는* 강한 수단입니다. 그러나 학술 영어에서 더 흔한 것은 *간접 의문*입니다.

```
직접:
Why has antibiotic resistance increased?

간접:
Many studies have investigated why antibiotic resistance has increased.
It is important to understand why antibiotic resistance has increased.
It remains unclear why antibiotic resistance has increased.
```

논점을 *질문으로 던지는 것*이 효과적인 자리(서론 끝, 토론 전환점)에서는 직접 의문이 좋습니다. 그 외에는 간접이 차분합니다.

## 7. 부사 위치 — Mid-position 선호

학술 영어에서 부사는 *문장 중간*(주어와 본동사 사이, 또는 조동사와 본동사 사이)에 자주 옵니다. 일상 회화에서는 부사가 문장 처음이나 끝에 놓이는 일이 흔합니다.

```
회화체:
This model was developed by the IMF originally and was adapted by
Lalonde and Muir (2007) later.

학술적:
This model was originally developed by the IMF and was later adapted
by Lalonde and Muir (2007).
```

`originally`, `later`, `frequently`, `previously` 같은 시간/빈도 부사가 자주 *mid-position*에 들어갑니다.

## 8. Split Infinitive — *to* + 부사 + 동사

전통적 규범은 *to*와 동사 사이에 부사를 넣지 말라고 합니다(*to sharply rise* 대신 *to rise sharply*). 그러나 *어색함이나 모호함*을 피하기 위해 분리가 필요할 때가 있습니다.

```
원문 (분리):
We need to adequately meet the needs of those enrolled in the program.

수정 (비분리):
We need to meet the needs of those enrolled in the program adequately.
```

원문의 *to adequately meet*은 *adequately가 meet을 수식*함을 강력히 보입니다. 수정문에서는 *adequately*가 문장 끝에 가서 *프로그램*을 수식하는 것처럼 읽힐 위험이 있습니다.

저자가 의도적으로 *split infinitive*를 쓴다면 보통 *어색함을 피하려는 결정*입니다. Swales와 Feak도 *적당히 분리를 허용*합니다. Chris와 John 본인들이 자기 글에서 split infinitive를 쓴다고 명시합니다. Strunk(Reminder 14)도 *귀로 결정하라*고 합니다.

## 9. 단어 수 — 너무 많지 않게

필요한 만큼은 쓰되, *불필요하게* 늘리지 않습니다.

```
지나침:
It may be difficult to make a decision about the method that should
be used.

압축:
Choosing the proper method may be difficult.

지나침:
There are some inorganic materials that can be used in tissue
engineering by bioengineers in the process of tissue engineering
that have been shown to be very promising.

압축:
Some inorganic materials have shown great promise in tissue engineering.
```

Strunk의 Rule 17(Omit needless words)와 같습니다.

## 10. 능동 / 수동 — 둘 다 사용

문법 검사기는 종종 수동태를 피하라고 권하지만, 학술 영어에서 수동은 *흔히* 쓰입니다. *행위자가 중요하지 않거나 자명할 때* 수동이 자연스럽습니다. Unit 3에서 더 자세히 다룹니다.

```
능동: We measured the temperature using thermocouples.
수동: The temperature was measured using thermocouples.
```

`방법 섹션`에서는 수동이 흔하고, `토론 섹션`에서는 능동이 흔합니다. 어느 voice가 *적절한지* 자리마다 판단합니다.

## 정리

| 항목 | 회화체 / 비격식 | 학술적 |
|------|---------------|--------|
| 1인칭 | *I, we* 자유롭게 | 분야에 따라, 단 *we*는 흔함 |
| Contraction | *won't* | *will not* |
| 부정 | *not ... any* | *no* |
| Etc. | 끝에 *etc.* | 풀어 쓰기 |
| 호명 | *you* | 수동 또는 일반화 |
| 의문 | 직접 의문 | 간접 의문이 흔함 |
| 부사 | 문장 처음/끝 | 자주 mid-position |
| Split infinitive | 흔함 | 모호함 피하려 때때로 |
| 단어 수 | 풀어 쓰기 | 압축 |
| Voice | 능동 위주 | 능동/수동 둘 다 |
