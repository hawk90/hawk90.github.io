문제-해결 텍스트의 핵심은 *원인과 결과의 연결*입니다. *왜 문제가 생겼는가*, *해결이 어떻게 효과를 내는가*를 명확히 표현해야 합니다. Swales와 Feak는 인과 표현의 세 그룹을 정리합니다 — 동사 패턴, 전통적 연결어, *-ing* 절. 이 글은 그중 *-ing* 절에 집중합니다.

## 인과 표현의 세 가지 도구

### 1. 동사 패턴

```
A causes B.
B is triggered by A.
A leads to B.
A results in B.
A produces B.
A contributes to B.
A is associated with B.
```

```
An increase in demand causes a rise in prices.
The tsunami was triggered by a very powerful earthquake.
```

### 2. 전통적 연결어

```
A. As a result / Therefore / Thus / Consequently / Hence, B.
Because A, B.
A. For this reason, B.
```

```
The magma flows into the pores of the rocks; as a result, the rocks rupture.
A current is sent through the material. As a result, the electrons are polarized.
```

### 3. *-ing* 절 (이 글의 초점)

학술 영어에서 자주 등장하지만 익숙하지 않은 형태입니다.

```
Researchers worldwide are increasingly pressured to publish in
English language journals, thus leading to a decline in publications
written in languages other than English.
```

*thus leading to ...*가 *-ing 절*입니다. 앞 절의 *결과*를 압축적으로 표현합니다.

## *thus + -ing* 패턴

가장 흔한 형태는 *thus + -ing*입니다. 두 문장을 *한 문장으로 압축*하는 효과가 있습니다.

### 두 문장 vs 압축

```
두 문장:
The magma flows into the pores of the rocks. As a result, the rocks rupture.

압축:
The magma flows into the pores of the rocks, thus causing them to rupture.
```

```
두 문장:
A current is sent through the material. As a result, the electrons are polarized.

압축:
A current is sent through the material, thus polarizing the electrons.
```

### *thus*는 선택적

*Thus*를 생략해도 *-ing* 절이 결과를 표현합니다.

```
With thus:
The magma flows into the pores of the rocks, thus causing them to rupture.

Without thus:
The magma flows into the pores of the rocks, causing them to rupture.
```

*Thus*가 있으면 *인과 관계가 더 강조*됩니다. 없으면 좀 더 부드럽습니다.

### *thereby + -ing*

*Thereby*도 비슷하게 쓰입니다 — *그것에 의해*의 뉘앙스.

```
The laser light forms an EM field, thereby slowing the vibration of the atoms.
```

*Thereby*는 *thus*보다 약간 더 격식적입니다.

## 세 부분 구조

긴 *-ing* 절 패턴은 다음 셋으로 이루어집니다.

```
(선택) 부속절 :  When the ABS controller senses that a wheel is about to lock up,
주절          :  it automatically changes the pressure in the brake lines of the car
(선택) -ing 절 :  thus preventing the lockup, resulting in maximum brake performance.
```

세 부분이 *원인 → 사건 → 결과* 흐름을 한 문장에 압축합니다.

```
When the ABS controller senses that a wheel is about to lock up, it
automatically changes the pressure in the car's brake lines to prevent the
lockup, (thus) resulting in maximum brake performance.
```

이 구조가 *문제-해결 텍스트*에 특히 유용합니다. 한 문장 안에 *과정의 다음 단계*, *발생하는 문제*, 또는 *해결의 결과*를 표현할 수 있습니다.

## 문제-해결의 세 변형

같은 구조에 *과정*, *문제*, *해결* 셋을 넣을 수 있습니다.

```
Process : Prices rise, thus leading to a drop in demand.
Problem : Prices rise, thus increasing the chance of hyperinflation.
Solution: Prices rise, thus increasing earnings that can then be reinvested
          in the enterprise.
```

같은 첫 절(*Prices rise*)이 세 가지 다른 *-ing* 절로 이어집니다. 같은 사건이 *맥락에 따라* 과정/문제/해결로 자리 잡습니다.

## *-ing* 절의 다른 형태

*-ing* 절은 *causing*과 *leading to*뿐 아니라 다양한 동사로 만들 수 있습니다.

```
... thus creating ...
... thus enabling ...
... thus increasing ...
... thus decreasing ...
... thus supporting ...
... thus undermining ...
... thus allowing ...
... thus preventing ...
... thus contributing to ...
... thus suggesting ...
```

### 사례

```
Contact among humans, livestock, and wildlife may increase, thus
creating opportunities for the emergence of new livestock diseases.

Users have access to information, thus supporting smarter purchasing
decisions that affect a company's bottom line.

The propellant evaporates, leaving behind only the desired product.
```

*Leaving behind*는 *thus*가 없고 *thus*-부재 *-ing* 절입니다. 자연스럽습니다.

## 인과 연결의 강도

학술 영어에서 인과 표현의 *강도*가 다릅니다. 강한 표현일수록 *증거가 더 필요*합니다.

| 강도 | 표현 |
|------|------|
| 매우 강함 | *causes*, *is caused by*, *results from* |
| 강함 | *leads to*, *results in*, *produces* |
| 중간 | *contributes to*, *is associated with* |
| 약함 (인과 아님) | *is linked to*, *is correlated with*, *coincides with* |
| 가능성 | *may cause*, *might lead to*, *could result in* |

상관관계와 인과관계를 혼동하지 않는 것이 학술적 정직함입니다. 자기 데이터가 *상관관계*만 보였다면 *causes*가 아니라 *is associated with*를 씁니다.

## 짧은 연습

다음 두 문장을 *-ing* 절로 결합해 봅니다.

```
1. Technical improvements in resource efficiency can lower demand for
   resources. This results in lower prices.

   → Technical improvements in resource efficiency can lower demand
     for resources, (thus) resulting in lower prices.

2. The plants extract nickel and zinc; hence, the soil is left uncontaminated.

   → The plants extract nickel and zinc, (thus) leaving the soil
     uncontaminated.

3. Rainfall levels plummeted. A slow but steady loss of grasses occurred.
   As a result, the region was transformed into a desert.

   → Rainfall levels plummeted, resulting in a slow but steady loss of
     grasses, (thus) transforming the region into a desert.
```

마지막 예에서 *-ing* 절이 두 개 겹쳐졌습니다 — *resulting in ... transforming*. 여러 결과를 *연쇄*로 표현합니다.

## 학술 문맥에서의 사용 — Wildlife Strikes

다음은 *항공기와 야생동물의 충돌*에 관한 학술 단락의 일부입니다. 빈칸을 *-ing* 절로 채워 봅니다.

```
Since the onset of air travel in the early 1900s, aircraft collisions with
birds and other wildlife have been an ongoing threat to human safety.
These collisions, known as wildlife strikes, occur on average nearly 20
times per day and have damaged or destroyed more than 400 aircraft,
___________________________________________________________.
```

선택지:

```
a. ; as a result, the average cost is $117,787 per incident
b. , resulting in an average cost of $117,787 per incident
c. . These collisions have resulted in an average cost of $117,787 per incident
```

세 선택지 모두 문법적입니다. (b)가 *-ing* 절로 가장 압축적이고, 인과 관계가 *직전 사건의 결과*임을 자연스럽게 보입니다. (a)는 *as a result*가 약간 과한 느낌, (c)는 *반복*이 됩니다.

학술 영어는 *간결과 흐름*을 중시하므로 (b)가 더 자주 선택됩니다.

## 다른 인과 패턴

### *due to* / *owing to* / *because of*

전치사구 형태로 *원인*을 표현합니다.

```
Due to budget cuts, several programs were eliminated.
Owing to the heavy rainfall, the conference was rescheduled.
Because of the labor shortage, wages have risen.
```

### *as a consequence* / *consequently*

*결과*를 명시하는 연결어.

```
The drought lasted three years. Consequently, the crop yields decreased
by 40%.

The economy contracted. As a consequence, unemployment rose sharply.
```

### *given that* / *in light of*

*상황을 전제로* 결과를 도출.

```
Given that resources are limited, prioritization is necessary.
In light of recent findings, the hypothesis must be revised.
```

## 정리

- 학술 영어 인과 표현의 세 도구 — **동사 패턴**, **전통적 연결어**, ***-ing* 절**
- ***thus + -ing*** 절이 두 문장을 한 문장으로 압축
- 구조 — *(부속절) + 주절 + (thus/thereby) + -ing 절*
- 문제-해결 텍스트에서 *과정 / 문제 / 해결*을 같은 구조에 담을 수 있음
- 인과 표현의 *강도* — *causes* (강) → *is associated with* (약)
- 상관과 인과를 혼동하지 않는 것이 학술적 정직함
