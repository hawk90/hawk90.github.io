공학, 경영학, 정책, 공중보건 같은 분야의 학술 글은 *문제 해결*을 중심으로 흐릅니다. Swales와 Feak는 이 패턴을 *SPSE — Situation, Problem, Solution, Evaluation*으로 정리합니다. Unit 3의 첫 글은 이 네 단계 구조와, 문제-해결 텍스트에 자주 등장하는 *mid-position 부사*를 다룹니다.

## SPSE — 네 단계

```
Situation  — 현재 상황, 배경 (어떻게 되고 있는가)
    ↓
Problem    — 그 안에서 무엇이 어긋났는가
    ↓
Solution   — 어떻게 풀 수 있는가
    ↓
Evaluation — 해결이 얼마나 잘 되었는가, 한계는 무엇인가
```

이 네 단계가 한 단락에 압축될 수도, 한 논문 전체로 확장될 수도 있습니다. *학위 논문*은 보통 한 chapter 단위로, *짧은 보고서*는 한 단락 안에서 펼쳐집니다.

## 예 — Fog Harvesting (안개 수확)

칠레의 안개 수확 프로젝트를 다룬 한 단락입니다.

```
[Situation]
❶ Many small communities in arid coastal regions face severe water
   shortages.
❷ In Chungungo, a village in northern Chile, residents historically
   relied on weekly truck deliveries that supplied only 14 liters of water
   per person per day—far below the World Health Organization's
   minimum recommendation.

[Problem]
❸ Trucked water was costly, unreliable, and insufficient to support the
   village's economic development.
❹ As the population grew and demands for water-intensive activities
   such as agriculture and tourism increased, the inadequacy of this
   supply became increasingly apparent.

[Solution]
❺ In 1992, researchers installed an array of fog-collecting nets on the
   ridges above the village.
❻ The nets, made of fine polypropylene mesh, intercepted droplets
   from passing fog, which then drained into reservoirs and flowed by
   gravity to homes below.
❼ At peak production, the system delivered up to 11,000 liters of
   water per day, providing each resident with more than 30 liters.

[Evaluation]
❽ Despite the success of fog harvesting in Chungungo, the system is,
   unfortunately, no longer in use (de la Lastra, 2002).
❾ The availability of water led to a tripling of the population from
   300 to 900, putting pressure on the water supply (IDRC, 2003).
❿ Because the community lacked a clear commitment to the project
   (see Diehl, 2010, for a full explanation), they did not add new nets
   to increase the water supply, and instead petitioned for water to be
   piped in from 20 km away.
⓫ Although the village abandoned this viable alternative technology,
   the Chungungo experience has led to successful implementation of
   fog harvesting initiatives in other mountainous coastal areas of
   Chile, Ecuador, Mexico, and Peru, providing much needed fresh
   water to small communities.
```

각 단계가 또렷이 보입니다.

- **Situation** (❶–❷) — 일반적 배경 + 구체적 마을
- **Problem** (❸–❹) — 무엇이 문제이며 왜 점점 심각해지는가
- **Solution** (❺–❼) — 어떤 기술이 어떻게 작동하는가, 결과는?
- **Evaluation** (❽–⓫) — 결과의 평가 + 부작용 + 더 넓은 함의

### Evaluation의 두 얼굴

❽–❿는 *부정적 평가*(시스템 폐기)이고, ⓫는 *긍정적 평가*(다른 곳에서의 성공)입니다. *Evaluation*은 단순한 *성공/실패* 판정이 아닙니다. 한 문장 안에 *복잡한 평가*가 들어갈 수 있습니다.

## SPSE의 변형들

기본 구조에서 강조점에 따라 여러 변형이 가능합니다.

### Problem 중심

```
Situation → Problem → Problem의 더 깊은 분석 → 미해결 → 추가 연구 필요
```

학위 논문 *literature review*에서 자주 — 자기 연구의 *gap*을 만들기 위해.

### Solution 중심

```
Situation → Problem → Solution 1 → Solution 2 → Solution 3 → 비교 → 권장
```

공학·정책 보고서에서 자주.

### Evaluation 중심

```
Situation → Problem → Solution → 단기 평가 → 장기 평가 → 함의
```

사례 연구(case study)에서 자주.

## 자기 SPSE 작성 — 시작 표현

각 단계의 *전형적인 시작 표현*입니다.

### Situation 도입

```
In recent decades, ...
Many [industries / countries / fields] face ...
The use of X has become increasingly widespread ...
Currently, ...
```

### Problem 신호

```
However, ...
Despite this, ...
A major challenge is that ...
The difficulty arises because ...
One concern with ... is that ...
This raises the question of ...
```

*However*가 가장 흔한 신호입니다. *Problem* 부분은 보통 *contrast 연결어*로 시작합니다.

### Solution 도입

```
To address this issue, ...
A possible solution is ...
Researchers have proposed ...
One approach is to ...
Recent advances suggest that ...
```

### Evaluation 표현

```
This approach has proven effective in ...
The results have been mixed.
While X has succeeded in [area], it has failed to ...
Despite [success], ...
The long-term sustainability remains uncertain.
```

평가는 *균형*이 중요합니다. 일방적 칭찬이나 일방적 비판은 학술적 신뢰를 깎습니다.

## Mid-Position Adverbs

문제-해결 텍스트(및 학술 영어 전반)에서 *부사가 mid-position*에 자주 옵니다. 이 위치가 *학술적 톤*을 강화합니다.

### Mid-Position이란

```
주어 + [부사] + 본동사
주어 + 조동사 + [부사] + 본동사
```

부사가 *주어 직후*나 *조동사와 본동사 사이*에 들어갑니다.

### 일상 vs. 학술

```
일상:
This model was developed by the IMF originally.
Then the morphology of the samples was analyzed.

학술 (mid-position):
This model was originally developed by the IMF.
The morphology of the samples was then analyzed.
```

*Originally*, *later*, *then*, *frequently*, *previously*, *currently*, *now*가 자주 mid-position에 들어갑니다.

### 의미 차이

Mid-position이 부사를 *주절 전체*가 아니라 *동사 자체*에 묶습니다.

```
The system has unfortunately been abandoned.   ← unfortunately가 abandon에 묶임
Unfortunately, the system has been abandoned.   ← 화자의 태도, 문장 전체에
```

학술 글에서 *작가의 태도가 동사에 묶이는* mid-position이 더 차분합니다. 문장 처음의 부사는 *작가의 주관*을 더 강하게 드러냅니다.

### 자주 mid-position에 오는 부사

```
시간:    originally, previously, currently, recently, then, later, eventually
빈도:    typically, frequently, occasionally, rarely, always, often, generally
정도:    largely, mainly, partly, fully, primarily, considerably
태도:    unfortunately, surprisingly, interestingly, notably
확신:    likely, probably, certainly, perhaps, possibly
양식:    automatically, gradually, slowly, rapidly, carefully
```

자기 글에서 부사를 처음이나 끝에만 두고 있다면, mid-position으로 옮기는 연습을 해 봅니다. 문장이 즉시 학술적이 됩니다.

## SPSE 작성 시 흔한 함정

### 1. Problem이 약함

```
약함:  The cost of solar panels is high.
강함:  The cost of solar panels remains 30% higher than coal-generated
       electricity in most developed markets, despite a decade of
       subsidies and technological improvement (IEA, 2023).
```

*문제의 규모와 지속성*을 수치와 인용으로 받쳐 줍니다.

### 2. Solution이 일방적

```
약함:  Wind power will solve the energy crisis.
강함:  Wind power can contribute meaningfully to a diversified energy
       portfolio, although its intermittency requires complementary
       baseload sources or storage technologies.
```

*절대적 해결*을 약속하지 말고 *역할과 한계*를 함께 말합니다. 학술 글의 *positioning*(Unit 1.6)과 직접 연결됩니다.

### 3. Evaluation이 빠짐

학생 글에서 가장 흔한 누락은 *Evaluation*입니다. Solution만 제시하고 끝내면 글이 *처방전*처럼 보입니다. *짧은 한 단락이라도* 평가가 들어가야 학술적입니다.

### 4. 단계가 섞임

```
[Situation] Renewable energy is increasingly used.
[Solution] Solar panels can be installed on rooftops.   ← Problem 없이 Solution
[Problem] Cost is high.                                  ← Problem이 뒤늦게
```

순서가 어긋나면 독자가 혼란스럽습니다. *Situation → Problem → Solution → Evaluation*을 의식적으로 유지합니다.

## 정리

- 문제-해결 텍스트의 표준 구조 — **SPSE** (Situation–Problem–Solution–Evaluation)
- 각 단계에 *전형적 시작 표현*이 있음 — *However*, *To address this*, *Despite ...*
- **Evaluation**은 *균형* — 일방적 칭찬·비판 피함
- **Mid-position adverbs**가 학술 톤을 강화
- 흔한 함정 — 약한 Problem, 일방적 Solution, 누락된 Evaluation, 단계 섞임
