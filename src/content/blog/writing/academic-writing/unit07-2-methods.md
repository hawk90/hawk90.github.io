Methods 섹션은 IMRD의 가장 기술적인 부분입니다. 어떻게 했는지를 재현 가능하게 기술하는 것이 목적입니다. 이 글은 Methods의 표준 구조, 시제·voice 관습, 자주 쓰는 표현을 정리합니다.

## Methods 섹션의 목적

```text
1. 재현 가능성 — 다른 연구자가 같은 절차를 반복할 수 있게
2. 신뢰성 검증 — 심사자가 방법의 적절성을 평가할 수 있게
3. 한계 명시 — 사용한 방법의 경계를 분명히
```

좋은 Methods 섹션은 충분한 디테일과 과한 디테일 사이의 균형입니다.

## 표준 구조

분야마다 다르지만, 다음이 일반적 흐름입니다.

```text
1. Participants / Subjects / Samples / Materials
   — 누구를 / 무엇을 / 얼마나
2. Design / Setting
   — 실험 / 관찰 / 조사 설계
3. Procedure
   — 단계별 과정
4. Measures / Instruments
   — 어떻게 측정 / 기록했는가
5. Data Analysis
   — 어떻게 분석했는가
```

### 사회과학 / 심리학

```text
1. Participants
2. Materials / Measures
3. Procedure
4. Data Analysis
```

### 생물학 / 의학

```text
1. Subjects (or Animals, Cell Cultures)
2. Materials and Reagents
3. Experimental Procedures
4. Statistical Analysis
```

### 공학 / 컴퓨터 과학

```text
1. Experimental Setup / System Design
2. Data / Dataset
3. Algorithms / Models
4. Evaluation Metrics
```

자기 분야의 모범 논문에서 섹션 제목과 순서를 확인합니다.

## 시제 — Past Simple

Methods 섹션의 기본 시제는 과거 단순형입니다. 이미 한 일이기 때문입니다.

```text
Participants were recruited from three universities.
The survey was conducted in March 2023.
Data were analyzed using SPSS version 28.
We measured response times with a millisecond-accurate timer.
```

### 예외 — Present Simple

다음 경우에 현재 시제가 쓰입니다.

```text
1. 알려진 사실 / 일반적 절차:
   The Stroop task measures interference between conflicting stimuli.
   
2. 공식·방정식 설명:
   Equation 1 represents the relationship between X and Y.

3. 그림 / 표 참조:
   Figure 1 shows the experimental setup.

4. 방법론적 정당화:
   This approach is widely used in similar studies.
```

### Past Continuous / Past Perfect

```text
Past continuous — 진행 중인 절차:
   Participants were being monitored throughout the session.

Past perfect — 사전 단계:
   Before the experiment, all participants had completed a screening.
```

## Voice — 수동태 위주

Methods 섹션은 수동태가 표준입니다. 행위자(we, researchers)보다 절차 자체에 초점이 가기 때문입니다(Unit 3.3 참고).

```text
수동:
Samples were collected from three sites.
Temperature was measured every 5 minutes.
Data were analyzed using mixed-effects models.

능동 (덜 흔함):
We collected samples from three sites.
We measured temperature every 5 minutes.
We analyzed the data using mixed-effects models.
```

### 분야에 따른 능동의 사용

```text
자연과학·공학: 수동이 압도적
의학·생명과학: 수동이 우세, 일부 능동
사회과학·심리학: 능동도 흔함 (특히 "We")
인문학: 능동이 우세
```

최근 능동의 사용이 늘고 있습니다. 그러나 Methods는 여전히 수동 우세 영역입니다.

## Participants / Subjects

연구 대상을 어떻게 모집하고 묘사할지.

### 인간 참여자

```text
Participants were N students recruited from [source]. The sample
consisted of N1 women and N2 men, ranging in age from X to Y years
(M = ..., SD = ...). All participants provided informed consent and
received [compensation].
```

핵심 정보:
- 수
- 모집 방법 (어디서, 어떻게)
- 인구학적 특징 (성별, 나이, 분포)
- 동의
- 보상 (있다면)
- Ethics 승인 (필요한 분야)

### 동물 / 세포 / 화학물질

```text
Adult male Sprague-Dawley rats (N = ..., 200–250 g, [source]) were
housed in standard cages with food and water ad libitum. All
procedures were approved by the [Institutional Animal Care and Use
Committee].

HEK293 cells (ATCC, Manassas, VA) were maintained in DMEM
supplemented with 10% FBS at 37°C in 5% CO2.

Lithium chloride (≥99% purity) was purchased from Sigma-Aldrich
and used without further purification.
```

각 분야의 표준 디테일이 있습니다. 자기 분야의 모범 논문을 참고합니다.

## Procedure — 절차

단계별로 무엇을 했는지. 시간 순서가 보통.

```text
The experiment was conducted in three phases. In Phase 1,
participants completed a baseline questionnaire and a 5-minute training
session. In Phase 2, they performed the main task, which consisted of
60 trials over approximately 30 minutes. During the task, response
times and accuracy were recorded. In Phase 3, participants completed
a debriefing questionnaire.
```

### 시간 표현

```text
First, ...
Then, ...
Next, ...
Following [phase], ...
After [event], ...
Before [event], ...
During [phase], ...
Finally, ...
```

### 조건 / 그룹

```text
Participants were randomly assigned to one of three conditions:
[condition 1], [condition 2], or [control].

The samples were divided into two groups: experimental (N = ...) and
control (N = ...).
```

## Measures / Instruments — 측정 도구

각 변수를 어떻게 측정했는지.

```text
Reading speed was measured as the number of words read per minute,
using a digital timer.

Anxiety was assessed using the State-Trait Anxiety Inventory (STAI;
Spielberger, 1983), which consists of 40 items rated on a 4-point Likert
scale.

Surface temperature was measured with a calibrated thermocouple
(Type K, ±0.5°C, model ABC-123, Manufacturer Inc.).
```

핵심:
- 변수 + 측정 단위
- 도구 / 척도 + 출처 (인용)
- 신뢰도 / 정확도 (있다면)
- 모델 / 제조사 (장비)

### 표준 도구의 인용

```text
The Beck Depression Inventory (BDI-II; Beck et al., 1996) was used.
Statistical analyses were performed in R version 4.2.0 (R Core Team, 2022).
Imaging data were processed using FreeSurfer 7.0 (Fischl, 2012).
```

자주 쓰는 도구도 첫 등장 시 인용. 분야의 표준 도구임을 보입니다.

## Data Analysis — 분석

어떤 통계 / 분석 방법을 썼는지.

```text
Descriptive statistics were calculated for all variables. Group differences
were examined using independent-samples t-tests. The relationship
between X and Y was assessed with Pearson correlations. To control
for [confound], multiple regression was used with [predictors] as
independent variables and [outcome] as the dependent variable. All
analyses were conducted in SPSS version 28, with α set at .05.
```

핵심:
- 어떤 통계 방법
- 왜 그 방법 (선택의 정당화, 필요 시)
- 가정 / 보정 (covariates, multiple comparisons)
- 소프트웨어 / 버전
- 유의 수준 (α)

### 분석 방법의 정당화

```text
Linear regression was chosen because the relationship between X and Y
was expected to be linear, as suggested by previous research (Smith, 2020).

Mixed-effects models were used to account for the nested structure of
the data (participants within groups).

Bonferroni correction was applied to control for multiple comparisons.
```

방법 선택의 근거가 있으면 한 줄이라도 적습니다.

## Methods의 흔한 함정

### 1. 과한 디테일

```text
회피:
At 10:34 AM on March 17, 2023, the experimenter pressed the red
button on the Trinity TXR-9000 timer, which had been calibrated the
previous day according to ...
```

재현에 필요하지 않은 디테일. 과한 정확은 부정확과 같이 나쁨.

### 2. 부족한 디테일

```text
회피:
We measured the response.
```

어떻게 측정했는지 빠짐. 재현 불가.

### 3. 행위자 혼란

```text
혼란:
The temperature was measured. Then we recorded the data. The
analysis was conducted by us.
```

같은 행위자가 수동/능동을 오가면 어색합니다. 일관성을 유지합니다.

### 4. Methods에 Results 섞임

```text
회피:
Participants were recruited. The recruitment yielded 245 useful
responses.   ← "yielded ... responses"는 결과
```

Methods는 절차. 결과의 수치는 Results에 둡니다.

## Methods 섹션의 길이

```text
짧은 분야 (한 문단): 응용 수학, 이론 컴퓨터 과학
중간 (1–3 페이지): 대부분의 실험 분야
긴 분야 (5+ 페이지): 임상 시험, 대규모 데이터 연구
```

부적절한 길이가 심사에서 자주 지적됩니다. 자기 분야의 적절한 길이를 모범 논문에서 확인합니다.

## 정리

- **Methods**의 목적 — 재현 가능성, 신뢰성 검증, 한계 명시
- 표준 구조 — Participants / Materials → Design → Procedure → Measures → Analysis
- 시제 — **Past simple** 위주 (이미 한 일)
- Voice — **수동태 우세** (절차 중심)
- 핵심 — 어떻게 + 왜 + 무엇으로 (도구)
- 함정 — 과한 디테일, 부족한 디테일, voice 혼란, Results 섞임
- 자기 분야의 모범 논문에서 길이와 디테일 수준 확인

## 한국 학생의 Methods 함정

```text
1. 디테일 부족 / 과다
2. 시제 흔들림
3. 모든 게 수동 / 능동 (분야 관습)
4. Results / Methods 섞임
```

## 자기 글 점검

```text
□ 재현 가능한 디테일?
□ 시제 일관 (과거)?
□ Voice 분야 관습?
□ Results 정보 끼지 않음?
```
