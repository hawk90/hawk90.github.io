Unit 1의 마지막 두 주제는 *Presentation*과 *Positioning*입니다. 앞의 글들이 *문장과 단락 안*의 문제를 다뤘다면, 이 둘은 *글 전체와 저자 자신*의 문제입니다.

## Presentation — 시각적 표현

학술 글의 외양은 *서식*과 *시각 요소*로 구성됩니다.

### Formatting

- **여백** — 좌우 1인치, 위아래 1인치가 영어권 표준
- **글꼴** — Times New Roman 12pt 또는 비슷한 serif 글꼴
- **줄간격** — 초안과 제출본은 double-spacing이 흔함
- **들여쓰기** — 단락 첫 줄 0.5인치 또는 들여쓰기 대신 단락 사이 공백
- **헤더 / 페이지 번호** — 저널·학회 요구에 맞춤

자기 분야의 *모범 논문*을 한 편 옆에 두고 그 서식을 흉내 내는 것이 가장 빠른 학습입니다.

### Visual Elements — 표, 그림, 그래프

학술 글에서 표와 그림은 *데이터를 압축해서 보여 주는 강력한 도구*입니다. 본문이 30 문장으로 풀어 쓸 내용을 한 표가 한눈에 보여 줄 수 있습니다.

각 시각 요소에는 다음이 필요합니다.

```
Table 1.   Number 표시 (Table / Figure 등)
Caption    제목 + 짧은 설명
Source     출처 (인용일 때)
Note       추가 설명 / 약어 풀이
```

본문에서 시각 요소를 *반드시 언급*합니다. 독자가 본문만 읽고 표를 무시하지 않도록.

```
잘못된 패턴:
[표만 있고 본문에서 언급 안 됨]

올바른 패턴:
As shown in Table 1, the response rate increased significantly ...
```

본문이 *모든 데이터를 반복*할 필요는 없습니다. 표가 보여 주는 내용 중 *가장 중요한 것* 하나, 둘만 본문에서 짚고, 그것의 *해석*에 집중합니다(Unit 4의 *Data Commentary*에서 자세히).

### Headings — 절 제목

긴 학술 글은 *절 제목*으로 구조를 시각화합니다.

```
1. Introduction
2. Methods
   2.1 Participants
   2.2 Materials
   2.3 Procedure
3. Results
4. Discussion
5. Conclusion
```

제목 자체는 *명사구*가 일반적입니다 — *Recruitment Procedure*가 *How We Recruited Participants*보다 학술적입니다. 다만 분야에 따라 다릅니다. 일부 인문학 학술지는 *문장형 제목*도 허용합니다.

## Positioning — 자기 자리 만들기

학술 글쓰기는 *내용 전달*만이 아니라 *학문 공동체의 한 구성원으로서 자기 자리를 잡는 일*입니다. Swales와 Feak는 이것을 *positioning*이라 부릅니다.

### Positioning이 일어나는 자리들

#### 1. 인용 — 누구를 인용하는가

자기가 인용하는 학자들이 자기의 *학문적 가족*을 보여 줍니다. 인지심리학자가 인용하는 사람과 사회심리학자가 인용하는 사람은 거의 겹치지 않습니다. *내가 어느 전통의 후예인가*가 인용 명단에 드러납니다.

```
같은 데이터를 가지고도:
[인지 전통] Tversky, Kahneman, Gigerenzer를 주로 인용
[사회 전통] Asch, Milgram, Cialdini를 주로 인용
```

새 분야에 들어갈 때는 *그 분야의 핵심 인용 패턴*을 흉내 내야 합니다. 인용 명단이 어긋나면 심사위원이 *outsider*로 느낍니다.

#### 2. 표현 — 어떤 헤지를 쓰는가

자기 주장을 얼마나 *강하게* 또는 *부드럽게* 표현하는지가 positioning입니다. 너무 강하면 *오만하다*는 인상, 너무 부드러우면 *자신 없다*는 인상을 줍니다.

```
강함:  Our findings prove that ...
중간:  Our findings show that ...
부드러움: Our findings suggest that ...
헤지:  Our findings may suggest that ...
이중 헤지: Our findings appear to suggest that ...
```

학술 영어는 *적절한 헤지*를 선호합니다. *Prove*는 수학과 논리에서만, 실증 연구에서는 보통 *suggest*나 *indicate*가 알맞습니다.

자주 쓰이는 헤지 표현:

```
modal verbs:    may, might, could, would
verbs:          appear, seem, suggest, indicate, tend to
adverbs:        possibly, perhaps, probably, likely, apparently
adjectives:     possible, likely, probable, potential
phrases:        it is possible that, there is some evidence that,
                in some cases, to some extent, in general
```

#### 3. 어휘 — 학문 공동체의 어휘를 빌리기

자기 분야의 *전문 용어*를 정확히 쓰는 것이 *insider*임을 보입니다. 같은 현상을 가리키는 *분야의 표준 용어*가 있는데 다른 단어를 쓰면, 그 분야 사람이 아니라는 신호가 됩니다.

```
경제학: marginal utility, opportunity cost
사회학: social capital, structural inequality
공학:   stress, strain, yield point
```

전문 용어를 흉내 내는 것은 *과시*가 아닙니다. *해당 공동체의 언어로 말하는 것*입니다.

#### 4. *Gap Statement* — 무엇이 빠져 있는가

자기 연구의 의의를 보이는 가장 흔한 방식은 *기존 연구의 빈 자리(gap)*를 지목하는 것입니다.

```
While many studies have examined X, few have considered Y.
Previous research has focused on X, but Z remains poorly understood.
Although X has been widely studied, less is known about Y in the context of Z.
```

이 *gap statement*가 자기 글의 자리를 *학문 지도 위에* 표시합니다. Unit 8에서 *Creating a Research Space* (CARS 모델)로 더 자세히 다룹니다.

#### 5. 자기 분야의 *positioning 관습* 익히기

분야마다 positioning 관습이 다릅니다.

```
물리학  — 매우 자신 있는 톤, 강한 동사, 짧은 문장
인문학  — 부드러운 톤, 다양한 헤지, 긴 문장
경영학  — 데이터 강조, 통계 헤지
법학    — 강한 인용, 단정적 결론
```

자기 분야의 모범 논문 *세 편*을 골라 다음을 점검합니다.

```
□ 어떤 modal verb를 자주 쓰는가? (may / might / can / could)
□ 헤지가 한 문단에 평균 몇 번 나오는가?
□ 자기 결과를 처음 소개할 때 어떤 동사를 쓰는가? (show / find / demonstrate / suggest / argue)
□ 결론에서 얼마나 단정적으로 끝맺는가?
□ Gap statement가 도입부 어디에 위치하는가?
```

이런 관찰이 자기 분야의 *positioning grammar*를 익히는 가장 빠른 길입니다.

## Unit 1 마무리 — 여섯 가지 고려 사항의 통합

Unit 1을 다 지나면, 다음 여섯 가지가 자기 글에 적용되어야 합니다.

```
1. Audience      — 누가 읽는가?
2. Purpose       — 왜 쓰는가?
3. Organization  — 어떤 순서로 펼치는가?
4. Style         — 어떤 어휘와 문체로?
5. Flow          — 문장 사이를 어떻게 잇는가?
6. Presentation  — 어떻게 보여 주는가?
```

이 여섯을 의식하며 자기 글을 *처음부터* 점검하는 습관이 학술 글쓰기의 토대입니다.

### 자기 점검 — 한 단락을 다음 질문으로

자기가 쓴 단락 하나를 골라 다음 질문에 답해 봅니다.

```
1. 이 단락은 누구를 위한 것인가? (Audience)
2. 이 단락의 목적은 — 설명? 논증? 분석? (Purpose)
3. 첫 문장이 topic sentence인가? (Organization)
4. 회화체 어휘가 있는가? phrasal verb 중 단일 동사로 바꿀 만한 것은? (Style)
5. 옛 정보가 주어 자리에 오는가? linking이 필요한 자리는? (Flow)
6. 표·그림과 어떻게 연결되는가? (Presentation)
```

대부분의 학술 글쓰기 개선은 *이 여섯 질문에 정직하게 답하는 데서* 시작합니다.

## 한국 학생의 Positioning 함정

```
1. 인용 부족 — 자기 주장만 펴고 근거 약함
   해결: 분야 모범 논문 *5편*의 인용 패턴 따라

2. 헤지 부족 또는 과다
   부족 → 오만하다는 인상
   과다 → 자신 없다는 인상
   해결: 분야 평균 헤지 빈도 관찰

3. 한국어 학술 용어 직역
   "본 연구에서는..."  →  "In this study, ..."
   "선행 연구에 따르면" → "Previous research indicates..."
   해결: 영어 학술 관용 표현 학습

4. Gap statement 약함
   "이 주제는 중요하다" → 약함
   "X는 광범위하게 연구되었으나 Y는 미해결" → 강함
   해결: CARS 모델 (Move 1, 2, 3) 학습

5. 전문 용어 사용의 일관성 부족
   같은 개념을 여러 단어로 → 혼란
   해결: 분야 표준 용어 *한 번 정하고* 일관 사용
```

## 한국 학술 영어의 정중함 (Politeness)

```
한국어 학술 글:
- 정중한 어조 표준
- 단언 회피 ("~인 것 같다")
- 권위에 의존 ("선행 연구에 따르면")

영어 학술 글:
- 직접적이되 헤지로 균형
- 자신감 있는 단언 + 적절한 헤지
- 권위에 의존하되 자기 기여 명시
```

```
회피 (한국 직역, 과도 정중):
"본 연구에서 X에 관해 살펴보고자 합니다."

선호 (영어 학술):
"This study examines X."
또는: "We examine X."
```

## 자기 글 점검 — Positioning

```
□ 인용 명단이 *자기 분야 전통*에 맞는가?
□ 헤지가 *너무 강하거나 약하지 않은가*? (분야 평균 비교)
□ 전문 용어가 *분야 표준*인가?
□ Gap statement가 도입부에 있는가?
□ 자기 기여가 *명시*되어 있는가?
□ 동사 선택이 분야 관습에 맞는가? (show/argue/suggest)
□ 한국어 직역 표현이 없는가?
```

## 정리

- **Presentation** — 서식, 시각 요소, 헤더의 시각적 일관성
- 표/그림은 *반드시 본문에서 언급*. 본문은 *해석*에 집중
- **Positioning** — 학문 공동체의 한 구성원으로 자기 자리를 잡는 일
- 인용, 헤지, 전문 어휘, gap statement 모두 positioning의 도구
- 분야마다 positioning 관습이 다름 — 모범 논문에서 *자기 분야의 grammar*를 읽어 냄
- Unit 1의 여섯 가지 고려 사항을 자기 글의 점검 체크리스트로
- 한국 학생: 헤지 균형, 영어 학술 관용 표현 학습이 핵심
