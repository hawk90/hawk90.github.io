Unit 3 마지막 글은 문제-해결 텍스트에서 *해결을 도입하는 방법*과, 학술 영어에서 자주 쓰이는 *간접 의문문*을 다룹니다.

## Indirect Questions — 간접 의문문

학술 영어에서 직접 의문문은 *드물게* 등장합니다. 대부분 *간접 의문문*으로 같은 효과를 냅니다.

### 직접 vs 간접

```
직접:
Why has antibiotic resistance increased?
How does the immune system respond?
What causes inflation?

간접:
Many studies have investigated why antibiotic resistance has increased.
It is important to understand how the immune system responds.
It remains unclear what causes inflation.
```

### 간접 의문문의 구조

```
주절 + question word + 평서문 어순
```

직접 의문문의 *주어-동사 도치*가 *사라지고*, 평서문처럼 정상 어순이 됩니다.

```
직접: Why does the response vary?
간접: ... why the response varies.   ← does가 사라지고, vary가 varies로
```

### 간접 의문문을 도입하는 표현

```
연구·조사:
Many studies have investigated [why ...].
Recent research has explored [how ...].
Researchers have examined [whether ...].
We sought to determine [whether ...].

관심·중요:
It is important to understand [why ...].
The question of [how ...] is central to ...
A key question is [whether ...].

지식의 한계:
It remains unclear [why ...].
Little is known about [how ...].
It is not yet established [whether ...].

가능성:
This raises the question of [whether ...].
The issue of [how ...] deserves further study.
```

### 간접 의문문의 장점

```
1. 학술 톤 — 직접 의문은 *대화체*에 가까움
2. 흐름 — 의문문은 단락 흐름을 끊을 수 있음
3. 헤지 — *it remains unclear* 같은 표현이 자연스러운 헤지
4. 인용 — 다른 연구의 *질문*을 자기 문장 안에 통합 가능
```

### 직접 의문문이 효과적인 자리

학술 글에서도 직접 의문문이 *효과적*인 자리가 있습니다.

```
1. 도입의 *연구 질문* 제시:
   This study addresses the following question: How does sleep
   duration affect academic performance among graduate students?

2. 단락 전환점:
   But what about the long-term effects?

3. Rhetorical question — 독자의 주의 환기:
   How, then, should we interpret these conflicting findings?
```

이런 자리에서는 직접 의문이 *강조*를 만듭니다. 다만 *드물게* 써야 효과가 있습니다. 한 글에 의문문이 너무 많으면 *대화체*로 미끄러집니다.

## Introducing the Solution

문제-해결 텍스트에서 *Problem*에서 *Solution*으로 넘어가는 자리는 글의 *분기점*입니다. 자연스러운 전환이 글의 질을 결정합니다.

### Solution 도입의 전형 표현

```
직접적:
To address this issue, ...
To overcome this problem, ...
To meet this challenge, ...
To resolve this difficulty, ...

연구·제안:
Researchers have proposed ...
Several approaches have been suggested ...
A possible solution is to ...
One approach involves ...

가능성·잠재력:
A promising avenue is ...
A potential remedy lies in ...
This problem may be addressed by ...

명시적 전환:
The solution to this problem came in [year/era] with the development of ...
The breakthrough came when ...
A new approach emerged ...
```

### 약한 도입 vs 강한 도입

```
약함:
There is a solution. The solution is X.

강함:
To address the limitations of conventional methods, researchers have
proposed a novel approach: X.
```

약한 도입은 *솔루션이 존재한다는 사실을 새 문장에 분리*해서 *흐름*을 끊습니다. 강한 도입은 *문제-해결 연결을 한 문장에 압축*합니다.

## 학술 단락의 분기점 — Airbag 사례

다음은 학생 글의 *수정 전 / 수정 후* 비교입니다. 강사 코멘트가 함께 있습니다.

### 수정 전

```
❶ Automotive airbag is occupant restraint system.
❷ It provides protection for occupant of vehicle in crash.
❸ Although airbags may seem to be somewhat recent innovation, rapidly
   inflating air cushions designed to prevent crash injuries existed for
   quite some time.
❹ Before being used in the automobiles.
❺ In fact researchers filed very first patents for inflatable safety cushion to
   be used in airplanes during World War II.

❻ A recent study by the National Highway Traffic Safety Administration
   concluded that airbags save nearly 1,000 lives annually.
❼ In the future even more lives will be saved as new airbag technologies
   are developed.
❽ Currently, for example, research is being done on as many as six
   different types of airbags that will offer protection in a wider range of
   accidents beyond front-end and side-impact collisions.

❾ Automotive airbag technology developed between 1940 and 1960
   was quite similar to that of airbags currently in use.
❿ Those early airbag systems were very difficult to implement and costly.
⓫ The main concern for design engineers at the time centered on storage.
⓬ And the efficient release of compressed air.
⓭ The housing of the system had to be large enough for a gas canister.
⓮ The canister had to keep the gas at high pressure for a long period of
   time.
⓯ The bag itself had to have a special design.
⓰ It would deploy reliably and inflate within 40 milliseconds.
⓱ The solution to these problems came in the early 1970s with the
   development of small inflators.
⓲ Inflators used hot nitrogen instead of air to deploy the bag.
⓳ It allowed the widespread installation of airbags in vehicles beginning
   in the 1980s.
```

### 강사 코멘트

```
1. 정식 sentence definition으로 시작하기.
2. 본문 흐름을 situation-problem-solution-evaluation 순서로 재정렬.
3. 문장 2–4는 통합. 9–10, 15–16, 17–18도 통합.
4. 연결어 추가 — Sentence 10, 13, 14, 15에.
5. 관사 (a, an, the, Ø) 점검 — 첫 단락이 특히.
6. Sentence 19를 *this + summary word*로 (대명사 *It* 대신).
7. Sentence 18 — *nitrogen*에 짧은 정의 추가.
8. Sentence 6, 8 — 부사를 mid-position에.
```

### 학습 포인트

이 코멘트들이 Unit 1–3의 거의 모든 요소를 다룹니다.

- **Sentence definition** (Unit 2.2)
- **SPSE 구조** (Unit 3.1)
- **문장 통합** — 짧은 문장 → 인과 연결 (Unit 3.2)
- **연결어 / mid-position 부사** (Unit 1.4, 1.5)
- **관사** — *a/an/the/Ø* (Unit 1.4)
- **This + summary word** (Unit 1.5의 old-to-new)

수정 후 글은 *훨씬 짧고, 흐름이 매끄럽고, 학술 톤이 강합니다*. 학생 글쓰기 개선은 거의 항상 *이미 다룬 도구들의 적용*입니다.

## Solution 평가 — Evaluation

해결을 도입한 뒤에는 *평가*가 와야 합니다. 학생 글에서 가장 자주 빠지는 부분입니다.

### 평가의 양면

```
긍정:
This approach has proven effective in ...
The method offers several advantages over ...
The results have been promising.

부정:
However, the method has several limitations ...
The approach has faced criticism for ...
Implementation has been hindered by ...

균형:
While [method] has succeeded in [aspect], it has been less effective in [other aspect].
Despite the promise of [method], several challenges remain.
The results have been mixed.
```

### 강한 평가 — 한계의 인정

학술 글에서 *한계를 인정*하는 것이 글의 신뢰를 *깎지 않고 오히려 높입니다*. 일방적 칭찬은 *광고*처럼 들립니다.

```
약함 (광고처럼):
This approach is a complete solution to the problem.

강함 (학술적):
This approach addresses the primary issue effectively, though questions
remain about its scalability to larger systems.
```
