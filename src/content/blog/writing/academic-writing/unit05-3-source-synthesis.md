여러 출처에서 정보를 가져와 *자기 글의 흐름에 통합*하는 것이 학술 글쓰기의 핵심 기술입니다. 이 글은 *출처 도입의 두 방식*과, 여러 출처를 묶는 *Synthesis*를 다룹니다.

## Integral vs Non-Integral Citation

학술 영어의 인용은 두 큰 형태로 나뉩니다.

### Integral Citation — 본문에 통합

저자 이름이 *문장의 일부*가 됩니다.

```
Brown (2020) found that caffeine consumption is associated with
reduced sleep quality.

According to Smith (2019), the effect varies by genetic background.

As noted by Lee and Park (2021), individual differences must be
considered.
```

저자가 *문장의 행위자*가 됩니다. *Brown found*, *Smith noted*, *Lee and Park noted*.

### Non-Integral Citation — 괄호 안에

저자 이름이 *괄호 안*에만 등장합니다.

```
Caffeine consumption is associated with reduced sleep quality
(Brown, 2020).

The effect varies by genetic background (Smith, 2019).

Individual differences must be considered (Lee & Park, 2021).
```

본문은 *내용*에 집중. 저자는 *부차적 정보*입니다.

### 둘의 선택

```
Integral 선호:
- 저자가 *유명함* (그 분야의 권위)
- 저자의 *방법론*이 중요
- 저자들 간 *대조*를 만들고 싶음 (X said Y, but Z argued W)
- 같은 출처가 *여러 문장*에 걸쳐 다뤄짐

Non-integral 선호:
- *내용*이 저자보다 중요
- 여러 출처를 *한 문장*에 압축
- 일반적인 *합의 사실*
- 문장 흐름을 *깨지 않고* 출처 보호
```

학술 영어 코퍼스에서 *non-integral이 약간 더 흔합니다* (특히 자연과학·공학). 그러나 인문·사회과학은 *integral*도 자주 씁니다.

## 보고 동사 — Reporting Verbs

Integral citation에서 사용하는 *보고 동사*의 선택이 *작가의 태도*를 표시합니다.

### 중립 동사

```
state, report, observe, note, write, mention, describe, present
```

```
Smith (2019) observed a correlation between X and Y.
Brown (2020) reported similar findings.
```

### 약한 동조

```
suggest, indicate, propose, point out
```

```
Smith (2019) suggested that the relationship may be causal.
```

### 강한 동조

```
demonstrate, prove, show, establish, confirm
```

```
Smith (2019) demonstrated that ...
The findings establish a clear connection ...
```

### 비판적 / 거리 두기

```
claim, allege, contend, argue
```

```
Smith (2019) claims that ...   ← claim은 종종 *의심* 함축
Brown (2020) alleges that ...   ← 회의적
```

`Claim`은 *약간 의심*의 뉘앙스가 있어 학생이 *Smith claims that*을 쓰면 의도치 않게 비판적으로 들릴 수 있습니다. *States, reports*가 더 중립적입니다.

### 종합 / 정리

```
review, summarize, synthesize, integrate
```

```
Lee (2021) reviewed the relevant literature and concluded that ...
```

## 출처 도입의 다양한 형태

같은 인용을 여러 형태로 도입할 수 있습니다.

```
1. As Smith (2019) noted, ...
2. According to Smith (2019), ...
3. Smith (2019) argues / suggests / contends that ...
4. In her recent study, Smith (2019) found that ...
5. Smith's (2019) analysis reveals that ...
6. The data presented by Smith (2019) indicate that ...
7. Building on earlier work, Smith (2019) extends ...
8. Smith (2019) is among those who have argued that ...
```

같은 단락에 모두 같은 형태를 쓰면 단조롭습니다. 형태를 *섞어* 흐름을 만듭니다.

## 시제 선택

영어 학술 인용에서 *시제 선택*이 미묘합니다.

### Past simple

```
Smith (2019) found that caffeine reduces sleep quality.
Brown (2020) reported similar findings.
```

특정 연구의 발견을 *과거의 일*로 봅니다. 가장 흔한 시제입니다.

### Present simple

```
Smith (2019) argues that caffeine reduces sleep quality.
The relationship is well-established (Brown, 2020).
```

저자의 *주장*이 *현재도 유효*하다고 볼 때. 또는 *합의된 사실*.

### Present perfect

```
Several studies have shown that caffeine reduces sleep quality (Smith,
2019; Brown, 2020).
Recent research has documented this pattern.
```

*과거에서 현재까지 이어지는* 연구 흐름. *여러 출처*를 묶을 때 자주.

### 시제의 일관성

한 단락 안에서 *시제를 섞지 말 것*입니다 — 모든 출처를 *같은 시제*로 다루는 것이 일반적입니다.

```
일관:
Smith (2019) found ... Brown (2020) reported ... Lee (2021) observed ...

비일관:
Smith (2019) found ... Brown (2020) reports ... Lee (2021) has observed ...
```

## Synthesis — 여러 출처의 통합

여러 출처를 *나열*하는 것과 *통합*하는 것은 다릅니다.

### 약한 통합 — 나열

```
Smith (2019) found a positive correlation. Brown (2020) found a similar
correlation. Lee (2021) also found a positive correlation.
```

세 문장이 같은 사실을 *반복*합니다. 통합이 아니라 *나열*입니다.

### 강한 통합 — 묶기

```
Multiple studies have documented a positive correlation between X and
Y (Smith, 2019; Brown, 2020; Lee, 2021).

Across several studies, a consistent positive correlation has emerged
(Smith, 2019; Brown, 2020; Lee, 2021).

Smith (2019), Brown (2020), and Lee (2021) all report a positive
correlation, suggesting a robust relationship.
```

한 문장에 세 출처를 묶고, *공통점*을 명시합니다.

### 대조 통합

```
While Smith (2019) reports a strong correlation, Brown (2020) found
the relationship to be weaker, and Lee (2021) suggests it may depend
on context.

The literature presents conflicting findings: a strong correlation (Smith,
2019), a weaker one (Brown, 2020), and a context-dependent
relationship (Lee, 2021).
```

세 출처의 *차이*를 명시합니다.

### 시간 흐름 통합

```
Early studies (Smith, 2010; Brown, 2012) suggested a simple causal
relationship. More recent research has complicated this picture,
identifying mediating factors (Lee, 2018) and individual differences
(Park, 2020).
```

*과거의 합의* + *현재의 변화* 구도. 학술 글에서 *gap*을 만드는 흔한 패턴입니다.

## 유사점과 차이점의 표현

여러 출처 사이의 *유사점*과 *차이점*을 명시하는 어구입니다.

### 유사점

```
Similarly, ...
Likewise, ...
In line with this, ...
Consistent with [Smith]'s findings, ...
[Smith]'s findings echo those of [Brown]'s.
Both [Smith] and [Brown] reported ...
A similar pattern was observed by [Lee] ...
```

### 차이점

```
In contrast, ...
However, ...
While [Smith] argues X, [Brown] suggests Y.
[Smith]'s findings differ from those of [Brown].
[Lee] takes a different position, arguing that ...
Whereas [Smith] focuses on X, [Brown] examines Y.
```

### 부분적 일치

```
[Smith] and [Brown] agree on X, but differ on Y.
While the studies converge on the general pattern, they diverge in their
specific predictions.
[Lee] confirms part of [Smith]'s findings while questioning others.
```

## 문헌 리뷰 단락의 예

다음은 *Synthesis*가 잘 된 문헌 리뷰 단락입니다.

```
❶ The relationship between caffeine consumption and sleep quality has
   been examined extensively over the past two decades.
❷ Early studies focused on adults aged 30–50 and consistently reported
   a negative association (Smith, 2010; Brown, 2012).
❸ More recent research has extended this work to younger and older
   populations, with mixed results.
❹ While Lee (2018) confirmed the negative association in college students,
   Park (2020) found that the effect was substantially weakened in adults
   over 65.
❺ A growing body of research now points to genetic variation as a key
   moderator (Kim, 2021; Chen, 2022), suggesting that population-level
   findings may obscure substantial individual differences.
❻ This complexity underscores the need for more nuanced approaches
   to the caffeine-sleep relationship.
```

분석:

- ❶ — *맥락 설정* (어떤 주제가 다뤄졌는지)
- ❷ — *초기 연구 묶음* (두 출처 한 문장에)
- ❸–❹ — *최근 변화* (두 출처 대조)
- ❺ — *현재 방향* (또 다른 두 출처 묶음)
- ❻ — *함의*

6 문장 안에 6개 출처가 자연스럽게 통합됩니다. 각 출처가 *나열*이 아닌 *논증의 일부*입니다.


- *Citation* — Integral과 Non-integral, 보고 동사로 태도 표시
- *Synthesis* — 출처를 *나열*이 아니라 *논증의 일부*로 통합
