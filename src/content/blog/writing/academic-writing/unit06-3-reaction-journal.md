Unit 6 마지막은 두 가지 짧은 비평 형식 — *Reaction Paper*와 *Journal Review*입니다. 둘 다 학위 과정에서 자주 마주치는 글이지만, 형식이 명확히 정해져 있지 않아 학생이 혼란스러워합니다.

## Reaction Paper — 반응 논문

*Reaction paper*는 *한 텍스트*에 대한 짧은(보통 1–3 페이지) 반응 글입니다. 강의 과제로 자주 부여됩니다.

### Reaction Paper의 목적

```
- 학생이 텍스트를 *제대로 읽었는지* 확인
- 학생이 텍스트에 *적극적으로 반응*할 수 있는지 확인
- 강의 토론의 *준비 자료*
- 비평적 사고의 *연습*
```

### Reaction Paper의 표준 구조

엄격한 형식은 없지만, 다음이 흔합니다.

```
1. 짧은 요약 (텍스트의 핵심 1–2 문장)
2. 자기의 반응 (동의 / 의문 / 확장)
3. 텍스트와 *다른 자료*의 연결 (선택)
4. 결론적 입장
```

### 예 — Reaction Paper

```
[요약]
In *The Sense of Style*, Pinker argues that prose style should be
grounded in cognitive science: rules that survive empirical scrutiny
deserve attention, while traditional prescriptions without cognitive basis
should be reconsidered.

[자기 반응]
While I find Pinker's empirical approach compelling, two aspects of his
argument deserve scrutiny. First, the move from cognitive findings to
prescriptive rules is not straightforward — Pinker often takes a single
finding (e.g., readers prefer subject-verb proximity) and elevates it into
a general principle, despite the contextual nature of the original research.
Second, Pinker's preferred style — what he calls the "classic style" — is
itself a culturally specific tradition, not an empirical discovery; treating
it as the default seems to undermine his own commitment to empirical
grounding.

[다른 자료와 연결]
This tension between empirical method and cultural assumption recalls
Hyland's (2008) work on disciplinary voice, which shows that "good"
writing varies substantially by field. A purely cognitive approach risks
treating field-specific conventions as inefficiencies to be corrected.

[결론적 입장]
Pinker's book is most successful when it treats cognitive science as one
input to stylistic decision-making, not the only one. The contribution
lies in *expanding* the considerations writers can bring to bear, not in
replacing the older tradition.
```

이 reaction paper가 *균형* 잡혀 있습니다. *Pinker의 강점* 인정 + *두 가지 비판* + *다른 자료 연결* + *자기 입장 정리*.

### Reaction Paper의 톤

```
지나치게 비판적:
This book is misguided from start to finish.

지나치게 동조적:
Pinker is brilliant. Every word is gold.

학술적 균형:
While Pinker's approach has significant strengths, two aspects deserve
careful scrutiny.
```

균형이 *반응의 신뢰성*을 만듭니다.

### Reaction Paper의 길이별 전략

```
1 페이지 (300–500 단어):
- 요약 1 문장
- 자기 반응 1–2 단락
- 결론 1 문장

2 페이지 (500–1,000 단어):
- 요약 1 단락
- 자기 반응 2–3 단락
- 다른 자료 연결 또는 자기 적용 1 단락
- 결론 1 단락

3 페이지 (1,000–1,500 단어):
- 더 발전된 단락 구조
- 여러 측면의 평가
- 자기 분야 / 자기 연구와의 명시적 연결
```

## Journal Review — 저널 심사 보고서

석사·박사 과정 후반부터, 또는 학회 참여 시작 시점부터 *저널 심사*를 부탁받는 일이 생깁니다. 학술 공동체의 *peer-review* 활동의 일부입니다.

### 심사 보고서의 표준 구조

대부분의 저널이 다음 구조를 요구합니다.

```
1. Summary of the manuscript        — 논문 내용 요약
2. Overall assessment                — 전체적 평가
3. Major issues                      — 주요 문제 (필수 수정)
4. Minor issues                      — 작은 문제 (선택적 수정)
5. Recommendation                    — 추천 (Accept / Minor revision / Major revision / Reject)
```

저널마다 형식이 다르므로 *심사 요청서의 지침*을 따릅니다.

### 1. Summary of the Manuscript

심사 보고서의 *summary*는 두 가지 역할을 합니다.

```
1. 심사자가 *논문을 제대로 읽었음*을 보여 줌
2. 저자에게 *심사자가 어떻게 이해했는지* 알림
```

```
The manuscript investigates the relationship between caffeine
consumption and sleep quality in college students. Using a survey of
2,500 undergraduates at three universities, the authors examine
self-reported caffeine intake, sleep duration, and sleep efficiency. The
main finding is a negative association between caffeine intake and
sleep efficiency, particularly for consumption within 6 hours of bedtime.
The authors propose that early-evening intake represents a critical
threshold.
```

3–5 문장으로 충분합니다. *판단*은 들어가지 않습니다.

### 2. Overall Assessment

```
The manuscript addresses an important topic in sleep research and
presents original data from a relatively large sample. The methods are
generally sound, and the main conclusions are supported by the
analysis. However, several issues require attention before the paper can
be considered for publication. The most significant concerns relate to
the measurement of caffeine intake and the interpretation of the
threshold effect.
```

3–5 문장. *전체적 인상*과 *큰 방향성*을 제시합니다.

### 3. Major Issues — 주요 문제

심사의 *심장*입니다. 출판 *가부*를 결정하는 문제들.

```
1. Measurement of Caffeine Intake (Methods, p. 7)
The reliance on self-reported caffeine intake from the previous week
is concerning. Recall over a week's period is unreliable, particularly
for substances consumed multiple times daily. The authors should:
(a) acknowledge this limitation in the discussion, (b) preferably
present a sensitivity analysis using only the most recent 48 hours,
and (c) consider replicating with a different measurement approach.

2. Threshold Interpretation (Results, p. 14)
The "critical threshold" at 6 hours pre-bedtime appears to be derived
post hoc from the data rather than tested as an a priori hypothesis.
This raises concerns about multiple testing and over-fitting. The
authors should: (a) clarify whether this threshold was pre-specified,
and (b) if not, present the analysis as exploratory and call for
replication.
```

좋은 major issue 보고의 구조:

```
1. *어디*에 문제가 있는지 (페이지, 섹션)
2. *무엇*이 문제인지 (구체적 설명)
3. *어떻게* 수정할 수 있는지 (제안)
```

*"이 부분이 약하다"* 같은 모호한 평가는 *도움이 되지 않습니다*.

### 4. Minor Issues — 작은 문제

```
- p. 3, line 12: "subjects" → "participants" (현대 학술 영어 관행)
- Table 2: Confidence intervals should be reported alongside p-values
- The reference list is missing the citation to Lee (2020) noted on p. 9
- Figure 1: Y-axis label is unclear; please specify units
```

타이포, 표 형식, 인용 누락 등 *수정 가능한 작은 문제*.

### 5. Recommendation

저널마다 카테고리가 다르지만, 보통 네 가지입니다.

```
Accept             — 거의 그대로 출판 가능 (드물게)
Minor revision     — 작은 수정 후 출판
Major revision     — 상당한 수정 후 재심사
Reject             — 출판 부적합
```

이 결정은 *Major issues의 심각성*으로 결정됩니다. Minor issues만 있으면 *Minor revision*. Major issues가 있어도 *수정 가능*하면 *Major revision*. 수정이 어려운 *근본 문제*가 있으면 *Reject*.

## 심사 보고서의 톤

### 회피해야 할 톤

```
회피:
This paper is poorly written.
The methodology is wrong.
The authors don't know what they're doing.
The conclusions are absurd.
```

이런 표현은 *공격적*이고 *건설적이지 않습니다*. 학술 공동체의 표준에 어긋납니다.

### 학술적 톤

```
학술적:
The manuscript would benefit from additional attention to writing
clarity, particularly in Sections 3 and 4.

The methodological approach raises several concerns that the authors
should address.

The interpretation of these results appears to overstate the strength of
the evidence.

The conclusions extend beyond what the data can support.
```

같은 비판이지만 *건설적이고 옹호 가능*합니다. *Author*가 아닌 *manuscript*가 평가의 대상입니다.

### 비판과 칭찬의 균형

좋은 심사 보고서는 *비판*만이 아니라 *강점도 인정*합니다.

```
긍정 측면 인정 + 한계 지적:
The study addresses an important question and uses a relatively large
sample. However, the measurement of caffeine intake is a significant
limitation, ...

균형 잡힌 결론:
With the revisions outlined above, this manuscript has the potential
to make a useful contribution to the literature on sleep and stimulants.
```

저자의 *노력을 인정*하면서 *수정을 요청*합니다.

## 심사 보고서 작성 시 주의

### 1. 자기 분야의 표준 사용

```
□ 저널의 *심사 지침* 확인
□ *익명성* 유지 — 자기 이름·연구를 드러내지 않기
□ *기밀 유지* — 논문 내용 외부 공유 금지
□ *시간 약속* 지키기 — 심사 기한
```

### 2. 일관된 형식

```
□ Major issues는 *번호*로
□ 각 issue에 *위치 표시* (페이지, 섹션, 그림 번호)
□ *구체적 제안* — "어떻게 고치라"
□ Minor issues는 *목록*으로 압축
```

### 3. 자기 한계의 인정

```
"이 통계 방법은 내 전문 영역 밖이므로 통계 전문가의 추가 심사를 권합니다."
"이 분야의 최근 5년 문헌은 충분히 follow하지 못해 ..."
```

자기 전문 영역의 *경계*를 인정하는 것이 *심사의 신뢰*를 만듭니다.

## Unit 6 마무리

Unit 6에서 다룬 것:

```
6.1 Book Reviews와 의견 표명
6.2 Evaluative Language와 Critical Reading
6.3 Reaction Papers와 Journal Reviews (이 글)
```

핵심:

- 학술 비평의 톤 — *균형*과 *근거*
- 평가어의 *스펙트럼* 이해 — 강함 vs 부드러움
- *Critical Reading*이 비평의 토대
- *Reaction Paper* — 강의 토론의 준비, 짧고 균형
- *Journal Review* — peer-review의 의무, 건설적·구체적

## 한국 학생의 Reaction / Peer Review 함정

```
1. Reaction paper — 단순 요약만
   해결: 자기 반응 / 입장 명시
2. Peer review — 일방적 비판
   해결: 건설적 제안 동반
3. 영어 학술 톤 부족 — 직접적이되 정중
4. 익명성 — 저자에게 직접 말하듯
```

## 자기 글 점검

```
□ Reaction에 자기 *입장*?
□ Peer review가 *건설적*?
□ 톤이 *전문적*?
□ 구체적 *제안 / 예*?
```
