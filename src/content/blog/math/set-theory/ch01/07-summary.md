## 한 장의 이야기

처음에는 모든 게 좋아 보였다.

> *조건만 적으면 그것을 만족하는 모든 것의 집합이 존재한다.*

그러다 Russell 의 한 줄이 그 우주를 무너뜨렸다.

> $R = \{x \mid x \notin x\}$. *$R \in R$ 인가?*

Zermelo 는 방향을 바꿨다. *집합이 무엇인가* 를 묻는 대신 — *어떤 집합이 존재한다고 약속할지* 를 정했다.

각 약속은 *바로 앞에서 부족했던 무엇* 을 메운다.

| 부족했던 것 | 새 약속 |
|---|---|
| "같다" 의 기준 | **외연** |
| 출발점 | **공집합** |
| 두 개를 묶는 법 | **짝** |
| 평평하게 풀어 헤치기 | **합집합** |
| 한 차원 위로 | **멱집합** |
| 안전하게 자르기 | **분리** |
| 무한의 존재 | **무한** |
| 함수의 상까지 | **치환** |
| 우주의 모양 | **정칙성** |
| 무한히 많은 동시 선택 | **선택** |

각 약속이 등장할 때마다 *그 직전엔 못 했던 일* 이 가능해졌다.
이 *부족 → 약속* 의 사슬이 ZFC 의 정체.

## 한눈에 보는 ZFC

| # | 공리 | 핵심 한 줄 |
|---|---|---|
| 1 | Extensionality | 같은 원소를 갖는 두 집합은 같다 |
| 2 | Empty Set | $\emptyset$ 가 존재한다 |
| 3 | Pairing | $\{a, b\}$ 를 만들 수 있다 |
| 4 | Union | 집합족을 한 단계 평평하게 푼다 |
| 5 | Power Set | 모든 부분집합을 한 데 모은다 |
| 6 | Separation (도식) | *기존 집합 안에서만* 조건으로 자른다 |
| 7 | Infinity | $\omega$ 같은 무한 집합이 존재한다 |
| 8 | Replacement (도식) | 함수의 상도 집합이다 |
| 9 | Foundation | $\in$ 을 따라 내려가면 끝이 있다 |
| 10 | Choice | 임의의 집합족에서 동시에 하나씩 고를 수 있다 |

> ZF = 1–9, ZFC = ZF + Choice.
> 도식 (6, 8) 은 매 형식 $\varphi$ 마다 한 공리 — 합치면 *공리는 무한히 많다*.

## 다른 길도 있었다

ZFC 가 *유일한* 답은 아니다.

![ZFC 와 친척들](/images/blog/set-theory/ch01/zfc-comparison.svg)

| 이론 | 차이 |
|---|---|
| **Z** (Zermelo, 1908) | 치환 빠짐 |
| **ZF** | Z + 치환 |
| **ZFC** | ZF + 선택 |
| **NBG** (von Neumann–Bernays–Gödel) | 클래스 (proper class) 명시 도입 |
| **MK** (Morse–Kelley) | 클래스 양화까지 허용 — NBG 보다 강함 |

일상 수학에선 ZFC 로 충분.
NBG·MK 는 *클래스를 객체처럼 다루고 싶을 때* — 예를 들어 범주론 — 등장.

## 다음 장으로

ZFC 라는 *언어와 약속* 이 손에 들어왔다.

이제 본격적인 일이 시작된다 — **수학적 대상을 집합으로 짓는 일**.

다음 장 예고:
- *순서쌍* $\langle a, b\rangle := \{\{a\}, \{a, b\}\}$ — 짝 공리의 첫 본격 응용
- *관계* = 순서쌍들의 집합
- *함수* = 특별한 모양의 관계
- 그 위에 *데카르트 곱*, *동치류*, *정렬* ...

이후 Ch 3 에서 자연수, Ch 4 에서 순서수, Ch 5 에서 기수 — *모든 수* 가 차례로 *집합의 언어로* 다시 태어난다.
