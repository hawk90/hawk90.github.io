> **OSTEP** Chapter 23 요약

## 디스크 구조

```
| 수퍼블록 | 비트맵 | inode | 데이터 |
```

- 수퍼블록: 메타데이터
- 비트맵: 할당 정보
- inode: 파일 메타데이터
- 데이터: 실제 내용

## inode

```c
struct inode {
    size;
    mode;
    uid, gid;
    times;
    direct[12];    // 직접 포인터
    indirect;      // 간접 포인터
};
```

## 디렉토리

```
| inode# | reclen | strlen | name |
```

## 접근 경로

`/foo/bar` 읽기:
1. / 의 inode 읽기
2. / 의 데이터에서 foo 찾기
3. foo의 inode 읽기
4. foo의 데이터에서 bar 찾기
5. bar의 inode 읽기
6. bar의 데이터 읽기
