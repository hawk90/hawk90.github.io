> **OSTEP** Chapter 22 요약

## 파일

바이트 배열:
- 이름 (저수준: inode 번호)
- 타입 (일반, 디렉토리, 장치)

## 기본 API

```c
int fd = open("file", O_RDWR);
read(fd, buf, size);
write(fd, buf, size);
lseek(fd, offset, SEEK_SET);
close(fd);
```

## 디렉토리

파일 목록:
- (이름, inode) 쌍
- 계층 구조

```c
DIR *d = opendir(".");
struct dirent *e = readdir(d);
```

## 하드 링크 vs 심볼릭 링크

**하드 링크**: 같은 inode 가리킴
**심볼릭 링크**: 경로명 저장

## 파일 시스템 마운트

```bash
mount /dev/sda1 /mnt/usb
```
