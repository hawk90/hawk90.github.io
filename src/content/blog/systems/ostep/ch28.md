> **OSTEP** 시리즈 마무리

## 세 가지 핵심

### 가상화

물리적 자원 추상화:
- CPU → 프로세스
- 메모리 → 주소 공간

### 동시성

동시 실행 관리:
- 스레드
- 락과 동기화
- 버그 방지

### 영속성

데이터 보존:
- 파일 시스템
- 장애 복구
- 저장 장치

## 핵심 교훈

1. **추상화**: 복잡성을 숨겨라
2. **분리**: 정책과 메커니즘
3. **성능**: 필요할 때 최적화
4. **신뢰성**: 장애에 대비

## 앞으로

더 배울 것:
- 분산 시스템
- 보안
- 가상화 기술
- 새로운 하드웨어

> **운영체제는 컴퓨터 과학의 핵심이다.**
