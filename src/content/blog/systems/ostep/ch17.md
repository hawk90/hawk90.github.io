> **OSTEP** Chapter 17 요약

## 세마포어란?

정수 값 + wait/signal:
```c
sem_wait(&s);   // s-- (0이면 대기)
sem_post(&s);   // s++
```

## 용도

### 이진 세마포어 (락)

```c
sem_t lock;
sem_init(&lock, 0, 1);

sem_wait(&lock);
// 임계 영역
sem_post(&lock);
```

### 조건 동기화

```c
sem_t done;
sem_init(&done, 0, 0);

// 자식
sem_post(&done);

// 부모
sem_wait(&done);
```

### 생산자-소비자

```c
sem_t empty, full, mutex;
sem_init(&empty, 0, N);
sem_init(&full, 0, 0);
sem_init(&mutex, 0, 1);
```
