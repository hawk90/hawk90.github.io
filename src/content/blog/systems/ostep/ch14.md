## 스레드란?

같은 주소 공간의 여러 실행 흐름:
- 코드, 데이터, 힙 공유

## 왜 스레드?

- 병렬 처리
- I/O 중첩
- 자연스러운 모델링

## 기본 API

```c
pthread_create(&thread, NULL, func, arg);
pthread_join(thread, &retval);
```
