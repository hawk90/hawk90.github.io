> **OSTEP** Chapter 15 요약

## 락이란?

상호 배제 보장:
```c
lock(&mutex);
// 임계 영역
unlock(&mutex);
```

## 평가 기준

1. **정확성**: 상호 배제 보장
2. **공정성**: 기아 방지
3. **성능**: 오버헤드

## 구현

### 스핀락

```c
while (test_and_set(&flag));
// 임계 영역
flag = 0;
```

### 티켓 락

```c
myticket = fetch_and_add(&ticket);
while (myticket != turn);
// 임계 영역
turn++;
```

### 뮤텍스 (with queue)

대기 시 잠들기:
- 스핀 낭비 제거
- 컨텍스트 스위칭 비용
