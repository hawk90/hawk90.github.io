> **OSTEP** Chapter 2 요약

## 프로세스란?

**실행 중인 프로그램** = 프로세스

구성요소:
- 주소 공간 (코드, 데이터, 힙, 스택)
- 레지스터 (PC, SP 등)
- I/O 정보

## 프로세스 상태

```
생성 → Ready → Running → Terminated
         ↑       ↓
         └─ Blocked ─┘
```

- **Running**: CPU 사용 중
- **Ready**: 실행 대기
- **Blocked**: I/O 대기

## 자료 구조

**PCB (Process Control Block)**:
- PID
- 상태
- 레지스터
- 메모리 정보
- I/O 상태

## 시사점

프로세스 = CPU 가상화의 핵심 추상화
