> **OSTEP** Chapter 1 요약

## OS란?

운영체제의 역할:
- 하드웨어를 **쉽게** 사용하게
- 여러 프로그램이 **동시에** 실행
- 자원을 **효율적으로** 관리

## 세 가지 핵심 개념

### 가상화 (Virtualization)

물리적 자원을 가상화:
- CPU → 많은 프로세스
- 메모리 → 각자의 주소 공간

### 동시성 (Concurrency)

동시 실행의 문제:
- 공유 자원 접근
- 레이스 컨디션
- 동기화 필요

### 영속성 (Persistence)

데이터 보존:
- 파일 시스템
- 장애 복구
- 일관성

## OS 디자인 목표

1. **추상화**: 복잡성 숨기기
2. **성능**: 오버헤드 최소화
3. **보호**: 격리와 보안
4. **신뢰성**: 장애 대응
5. **에너지 효율**: 전력 관리
