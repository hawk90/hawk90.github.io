## 조건 변수란?

조건 만족까지 대기:
```c
pthread_cond_wait(&cond, &mutex);
pthread_cond_signal(&cond);
```

## 생산자-소비자

```c
// 생산자
lock(&mutex);
while (count == MAX)
    wait(&empty, &mutex);
produce();
signal(&fill);
unlock(&mutex);

// 소비자
lock(&mutex);
while (count == 0)
    wait(&fill, &mutex);
consume();
signal(&empty);
unlock(&mutex);
```

## 중요한 규칙

1. **while** 사용 (if 아님)
2. 락 보유 상태에서 wait
3. signal은 언제든 가능

## Mesa vs Hoare

- Mesa: signal 후 계속 실행
- Hoare: signal 후 즉시 대기
