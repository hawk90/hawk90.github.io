> **OSTEP** Chapter 12 요약

## TLB란?

Translation Lookaside Buffer:
- 페이지 테이블 캐시
- 하드웨어 관리

## 동작

```
VPN → TLB 검색
  ↓
히트: PFN 반환 (빠름)
미스: 페이지 테이블 접근 (느림)
```

## TLB 내용

```
| 유효 | VPN | PFN | 보호 | ASID |
```

ASID: 프로세스 구분

## 컨텍스트 스위치

선택지:
1. TLB 플러시
2. ASID 사용

## 교체 정책

- LRU
- Random
