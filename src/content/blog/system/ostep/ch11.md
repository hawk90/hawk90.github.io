> **OSTEP** Chapter 11 요약

## 페이징이란?

고정 크기 단위로 분할:
- 가상: 페이지
- 물리: 프레임

## 페이지 테이블

```
가상 주소: | VPN | 오프셋 |
              ↓
페이지 테이블: VPN → PFN
              ↓
물리 주소: | PFN | 오프셋 |
```

## 페이지 테이블 엔트리

```
| 유효 | 보호 | 참조 | 수정 | PFN |
```

## 장단점

장점:
- 외부 단편화 없음
- 구현 단순

단점:
- 페이지 테이블 크기
- 메모리 접근 2배
