> **OSTEP** Chapter 3 요약

## 핵심 시스템 콜

### fork()

새 프로세스 생성:
```c
pid_t pid = fork();
if (pid == 0) {
    // 자식
} else {
    // 부모, pid = 자식 PID
}
```

### wait()

자식 종료 대기:
```c
wait(NULL);  // 아무 자식
waitpid(pid, &status, 0);  // 특정 자식
```

### exec()

프로그램 교체:
```c
execvp("ls", args);
// 반환하면 오류
```

## 왜 분리?

fork + exec 분리 이유:
- 리디렉션, 파이프 설정
- 유연한 쉘 구현
- UNIX 철학

## 쉘 동작

```
while (1) {
    prompt();
    cmd = read();
    if (fork() == 0) {
        exec(cmd);
    }
    wait();
}
```
