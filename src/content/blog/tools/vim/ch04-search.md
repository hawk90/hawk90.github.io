> **Vim 마스터하기** Chapter 4: 검색과 치환

## 1. 기본 검색

### 검색 명령

| 명령 | 동작 |
|------|------|
| `/{pattern}` | 정방향 검색 |
| `?{pattern}` | 역방향 검색 |
| `n` | 다음 매치 (같은 방향) |
| `N` | 이전 매치 (반대 방향) |
| `*` | 커서 아래 단어 정방향 검색 |
| `#` | 커서 아래 단어 역방향 검색 |
| `g*` | 부분 일치 정방향 검색 |
| `g#` | 부분 일치 역방향 검색 |

### 검색 옵션

```vim
:set hlsearch     " 검색 결과 하이라이트
:set incsearch    " 입력하면서 검색 (incremental)
:set ignorecase   " 대소문자 무시
:set smartcase    " 대문자 포함시 대소문자 구분
:nohlsearch       " 하이라이트 끄기 (일시적)
```

### 검색 패턴 수정자

```
/pattern\c    " 대소문자 무시
/pattern\C    " 대소문자 구분
/\<word\>     " 단어 전체 매치 (word boundary)
/^pattern     " 줄 시작
/pattern$     " 줄 끝
```

## 2. 정규표현식

### 기본 메타문자

| 패턴 | 의미 |
|------|------|
| `.` | 임의의 문자 1개 |
| `*` | 앞 문자 0회 이상 반복 |
| `\+` | 앞 문자 1회 이상 반복 |
| `\?` | 앞 문자 0회 또는 1회 |
| `\{n}` | 정확히 n회 |
| `\{n,m}` | n회 이상 m회 이하 |
| `\{n,}` | n회 이상 |
| `\|` | OR (택일) |
| `\(pattern\)` | 그룹화 및 캡처 |

### 문자 클래스

| 패턴 | 의미 |
|------|------|
| `[abc]` | a, b, c 중 하나 |
| `[^abc]` | a, b, c 제외 |
| `[a-z]` | a부터 z까지 |
| `\d` | 숫자 [0-9] |
| `\D` | 숫자 아님 |
| `\w` | 단어 문자 [a-zA-Z0-9_] |
| `\W` | 단어 문자 아님 |
| `\s` | 공백 문자 |
| `\S` | 공백 아님 |

### Very Magic 모드

정규표현식을 더 직관적으로 사용하려면 `\v` 사용:

```vim
" 일반 모드 (more escaping)
/\(hello\|world\)\+

" Very magic 모드 (less escaping)
/\v(hello|world)+
```

## 3. 치환 (Substitute)

### 기본 구문

```vim
:[range]s/{pattern}/{replacement}/[flags]
```

### 범위 (Range)

| 범위 | 의미 |
|------|------|
| (없음) | 현재 줄만 |
| `%` | 전체 파일 |
| `1,10` | 1~10줄 |
| `.,$` | 현재 줄부터 끝까지 |
| `'<,'>` | 비주얼 선택 영역 |
| `g/pattern/` | 패턴 매치 줄들 |

### 플래그 (Flags)

| 플래그 | 의미 |
|------|------|
| `g` | 줄의 모든 매치 (global) |
| `i` | 대소문자 무시 |
| `I` | 대소문자 구분 |
| `c` | 확인 후 치환 (confirm) |
| `n` | 치환 안 함, 매치 개수만 표시 |
| `e` | 에러 무시 |

### 예제

```vim
" 현재 줄의 첫 번째 foo를 bar로
:s/foo/bar/

" 현재 줄의 모든 foo를 bar로
:s/foo/bar/g

" 전체 파일에서 모든 foo를 bar로
:%s/foo/bar/g

" 확인하면서 치환
:%s/foo/bar/gc

" 대소문자 무시
:%s/foo/bar/gi

" 정규표현식 사용
:%s/\<foo\>/bar/g        " 단어 전체 매치
:%s/foo\d\+/bar/g        " foo + 숫자
:%s/\v(\w+)@(\w+)/\2@\1/g  " 이메일 앞뒤 교환
```

### 특수 치환 패턴

| 패턴 | 의미 |
|------|------|
| `&` | 전체 매치 |
| `\0` | 전체 매치 (동일) |
| `\1` ~ `\9` | 캡처 그룹 |
| `\u` | 다음 문자 대문자로 |
| `\U` | 이후 전체 대문자로 |
| `\l` | 다음 문자 소문자로 |
| `\L` | 이후 전체 소문자로 |
| `\e` | \U, \L 종료 |

```vim
" 단어 첫 글자 대문자화
:%s/\<\w/\u&/g

" 전체 대문자로
:%s/.*/\U&/

" 그룹 교환
:%s/\(\w\+\), \(\w\+\)/\2 \1/g
" "Smith, John" → "John Smith"
```

## 4. 전역 명령 (Global Command)

### 구문

```vim
:[range]g/{pattern}/{command}
:[range]g!/{pattern}/{command}  " 매치하지 않는 줄에 실행
:[range]v/{pattern}/{command}   " g!와 동일
```

### 예제

```vim
" 패턴 포함 줄 삭제
:g/pattern/d

" 빈 줄 삭제
:g/^$/d

" 패턴 포함 줄 표시
:g/pattern/p

" 패턴 포함 줄을 파일 끝으로 이동
:g/TODO/m$

" 패턴 포함 줄에서 치환
:g/class/s/foo/bar/g

" 패턴 미포함 줄 삭제
:v/pattern/d
:g!/pattern/d

" 연속된 빈 줄을 하나로
:g/^$/,/\S/-1j
```

## 5. 검색 후 작업

### 검색 결과에서 연산

```vim
" 검색 후 삭제
/pattern
dgn           " 다음 매치 삭제
cgn           " 다음 매치 변경

" 이후 .으로 반복 가능!
```

### gn 모션

| 명령 | 동작 |
|------|------|
| `gn` | 다음 검색 매치 선택 (Visual) |
| `gN` | 이전 검색 매치 선택 |
| `cgn` | 다음 매치 변경 후 `.`로 반복 |
| `dgn` | 다음 매치 삭제 후 `.`로 반복 |

## 6. 실용 예제

```vim
" 후행 공백 제거
:%s/\s\+$//

" Windows 줄바꿈(CRLF)을 Unix(LF)로
:%s/\r//g

" HTML 태그 제거
:%s/<[^>]*>//g

" 중복 공백을 하나로
:%s/  \+/ /g

" 각 줄 끝에 세미콜론 추가
:%s/$/;/

" 각 줄 앞에 번호 추가
:%s/^/\=line('.').' '/

" CamelCase를 snake_case로
:%s/\v([a-z])([A-Z])/\1_\l\2/g
```

## 실습

1. 코드 파일에서 `*`로 현재 단어 검색, `n`으로 탐색
2. `:%s/old/new/gc`로 확인하며 치환
3. `:g/TODO/p`로 TODO 줄 모두 표시
4. `cgn`으로 변경 후 `.`로 반복

## 요약

- `/`, `?` = 검색, `n`, `N` = 다음/이전
- `*`, `#` = 커서 단어 검색
- `:%s/old/new/g` = 전체 치환
- `gc` 플래그 = 확인하며 치환
- `:g/pattern/command` = 매칭 줄에 명령 실행
- `cgn` + `.` = 검색 결과 반복 변경
