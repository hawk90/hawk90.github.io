> **Vim 마스터하기** Chapter 5: 레지스터와 매크로

## 1. 레지스터 (Registers)

레지스터는 텍스트를 저장하는 공간이다. Vim에는 여러 종류의 레지스터가 있다.

### 레지스터 종류

| 레지스터 | 이름 | 용도 |
|----------|------|------|
| `""` | 무명(unnamed) | 기본 레지스터, d/c/y/x/s가 저장 |
| `"0` | 복사(yank) | 마지막 y 명령 내용 |
| `"1` ~ `"9` | 삭제 | 마지막 9개 삭제 내용 (줄 단위) |
| `"a` ~ `"z` | 이름 있는 | 사용자 지정 |
| `"A` ~ `"Z` | 이름 있는 (추가) | 기존 내용에 추가 |
| `"-` | 작은 삭제 | 한 줄 미만 삭제 |
| `".` | 마지막 삽입 | 마지막으로 삽입한 텍스트 |
| `"%` | 현재 파일명 | 현재 파일 경로 |
| `"#` | 대체 파일명 | 이전 파일 경로 |
| `":` | 마지막 명령 | 마지막 : 명령 |
| `"/` | 마지막 검색 | 마지막 검색 패턴 |
| `"*` | 시스템 클립보드 | 시스템 선택 버퍼 |
| `"+` | 시스템 클립보드 | 시스템 클립보드 |
| `"_` | 블랙홀 | 저장 안 함 (삭제만) |
| `"=` | 표현식 | 계산 결과 삽입 |

### 레지스터 사용법

```vim
" 레지스터 지정 후 연산
"ayy       " a 레지스터에 줄 복사
"ap        " a 레지스터 내용 붙여넣기
"Ayy       " a 레지스터에 추가 복사

" 시스템 클립보드
"+yy       " 시스템 클립보드에 복사
"+p        " 시스템 클립보드에서 붙여넣기

" 블랙홀 레지스터 (삭제만, 저장 안 함)
"_dd       " 줄 삭제, 무명 레지스터 변경 안 함

" 레지스터 확인
:registers
:reg
:reg a     " 특정 레지스터만
```

### 실전 활용

```vim
" 여러 텍스트 동시에 저장
"ayiw      " 단어를 a에 복사
"byy       " 줄을 b에 복사
"cy$       " 줄 끝까지 c에 복사

" 나중에 사용
"ap        " a 붙여넣기
"bp        " b 붙여넣기

" 삭제해도 복사한 것 유지
yiw        " 단어 복사 (0번 레지스터)
dw         " 다른 단어 삭제 (무명 레지스터)
"0p        " 복사한 내용 붙여넣기 (삭제된 것 아님!)
```

### Insert/Command 모드에서 레지스터

```vim
" Insert 모드에서 레지스터 삽입
Ctrl-r {reg}   " 레지스터 내용 삽입
Ctrl-r 0       " 마지막 yank
Ctrl-r "       " 무명 레지스터
Ctrl-r +       " 시스템 클립보드
Ctrl-r =       " 표현식 (계산 결과)
Ctrl-r /       " 마지막 검색 패턴

" Command 모드에서도 동일
:echo @a       " a 레지스터 내용 출력
:%s/<Ctrl-r />/new/g  " 마지막 검색 패턴으로 치환
```

## 2. 매크로 (Macros)

매크로는 키 입력을 기록하여 반복 실행하는 기능이다.

### 매크로 기본

| 명령 | 동작 |
|------|------|
| `q{a-z}` | 매크로 기록 시작 (레지스터에 저장) |
| `q` | 기록 종료 |
| `@{a-z}` | 매크로 실행 |
| `@@` | 마지막 매크로 반복 |
| `{n}@{a-z}` | n번 반복 실행 |

### 매크로 기록 예제

```vim
" 각 줄을 따옴표로 감싸기
qa           " a 레지스터에 기록 시작
0            " 줄 처음으로
i"<Esc>      " 시작 따옴표 삽입
$            " 줄 끝으로
a"<Esc>      " 끝 따옴표 삽입
j            " 다음 줄로
q            " 기록 종료

" 실행
@a           " 한 번 실행
5@a          " 5번 반복
100@a        " 100번 반복 (줄 끝나면 자동 중지)
```

### 매크로 모범 사례

1. **재사용 가능하게 설계**
   ```vim
   " 좋은 예: 상대적 이동 사용
   0            " 줄 처음
   w            " 다음 단어

   " 나쁜 예: 절대 이동
   5l           " 5칸 오른쪽 (다른 줄에서 깨짐)
   ```

2. **줄 이동으로 끝내기**
   ```vim
   " j로 끝내면 연속 실행 용이
   ...작업...
   j            " 다음 줄로
   q
   ```

3. **검색으로 위치 찾기**
   ```vim
   " f, t, / 등으로 위치 찾기
   f:           " : 문자로 이동
   ```

### 매크로 편집

매크로는 레지스터에 저장되므로 편집 가능하다.

```vim
" 매크로 내용 확인
:reg a

" 매크로 수정
" 1. 빈 줄에 붙여넣기
"ap

" 2. 수정

" 3. 다시 레지스터에 저장
0"ay$

" 또는 직접 설정
:let @a = 'I// <Esc>j'
```

### 재귀 매크로

매크로가 자기 자신을 호출할 수 있다.

```vim
" 모든 줄에 주석 추가 (파일 끝까지)
qa           " 기록 시작
I// <Esc>    " 주석 추가
j            " 다음 줄
@a           " 자기 자신 호출 (재귀)
q            " 기록 종료

" 주의: 먼저 qa 레지스터 비우기
qaq          " a 레지스터 비우기 (빈 매크로)
```

### 병렬 매크로 실행

```vim
" Visual 모드로 선택 후 실행
V5j          " 6줄 선택
:normal @a   " 각 줄에 매크로 실행

" 또는
:'<,'>normal @a
```

## 3. 매크로 + 레지스터 응용

### 매크로를 명령으로 저장

```vim
" vimrc에 저장
let @a = 'I// ^[j'  " ^[ = Esc (Ctrl-v + Esc로 입력)

" 또는 function으로
function! CommentLine()
    normal I//
endfunction
nnoremap <leader>c :call CommentLine()<CR>
```

### 여러 파일에 매크로 적용

```vim
" argdo로 여러 파일에 적용
:args *.js
:argdo normal @a | w

" bufdo로 열린 버퍼에 적용
:bufdo normal @a | w
```

## 4. 실용 예제

### CSV를 SQL INSERT로 변환

```
" 원본:
John,25,Engineer
Jane,30,Designer

" 목표:
INSERT INTO users VALUES ('John', 25, 'Engineer');
INSERT INTO users VALUES ('Jane', 30, 'Designer');

" 매크로:
qa
0                           " 줄 처음
IINSERT INTO users VALUES ('<Esc>
f,                          " 첫 쉼표
s', <Esc>                   " 변환
f,                          " 둘째 쉼표
s, '<Esc>
$                           " 줄 끝
a');<Esc>
j
q
```

### 변수명 일괄 변경

```vim
" 특정 패턴의 변수명만 변경
qa
/oldName<CR>      " 검색
cgnNewName<Esc>   " 변경
q

" 이후 n으로 확인하며 . 또는 @a로 반복
```

## 실습

1. 여러 레지스터(`"a`, `"b`)에 다른 텍스트 저장 후 붙여넣기
2. `"+y`, `"+p`로 시스템 클립보드 사용
3. 간단한 매크로 기록 후 `@a`, `@@`로 실행
4. 파일 전체 줄에 매크로 적용 (`99@a`)

## 요약

- `"` + 레지스터명 + 명령 = 레지스터 지정
- `"0` = 마지막 복사, `""` = 기본, `"+` = 클립보드
- `"_` = 블랙홀 (삭제 내용 저장 안 함)
- `q{reg}` = 매크로 기록, `@{reg}` = 실행
- `@@` = 마지막 매크로 반복
- 매크로는 레지스터에 저장되어 편집 가능
