> **Vim 마스터하기** Chapter 3: 모션과 텍스트 객체

## 1. 문자 검색 모션

### 줄 내 검색

| 명령 | 동작 |
|------|------|
| `f{c}` | 오른쪽으로 문자 c 찾기 (커서 이동) |
| `F{c}` | 왼쪽으로 문자 c 찾기 |
| `t{c}` | 오른쪽으로 문자 c 앞까지 이동 |
| `T{c}` | 왼쪽으로 문자 c 뒤까지 이동 |
| `;` | 마지막 f/F/t/T 반복 (같은 방향) |
| `,` | 마지막 f/F/t/T 반복 (반대 방향) |

```
예: "Hello, World!"에서 W 위치로 가려면
f,;      # 쉼표 찾고 다음 반복? 아니면
fW       # 바로 W로
```

## 2. 텍스트 객체 (Text Objects)

텍스트 객체는 **의미 단위**로 텍스트를 선택한다. 연산자(d, c, y 등)와 함께 사용한다.

### 구문

```
{operator}a{object}  # "a" = around (객체 + 주변 공백/구분자)
{operator}i{object}  # "i" = inner (객체 내부만)
```

### 단어 객체

| 명령 | 범위 |
|------|------|
| `iw` | inner word (단어만) |
| `aw` | a word (단어 + 공백) |
| `iW` | inner WORD (공백 기준) |
| `aW` | a WORD |

```
"Hello world"에서 world에 커서가 있을 때:
diw → "Hello "     (world만 삭제)
daw → "Hello"      (world + 앞 공백 삭제)
```

### 문장/문단 객체

| 명령 | 범위 |
|------|------|
| `is` | inner sentence |
| `as` | a sentence |
| `ip` | inner paragraph |
| `ap` | a paragraph |

### 괄호/따옴표 객체

| 명령 | 범위 |
|------|------|
| `i(` 또는 `i)` 또는 `ib` | 괄호 안 |
| `a(` 또는 `a)` 또는 `ab` | 괄호 포함 |
| `i[` 또는 `i]` | 대괄호 안 |
| `a[` 또는 `a]` | 대괄호 포함 |
| `i{` 또는 `i}` 또는 `iB` | 중괄호 안 |
| `a{` 또는 `a}` 또는 `aB` | 중괄호 포함 |
| `i<` 또는 `i>` | 꺾쇠괄호 안 |
| `a<` 또는 `a>` | 꺾쇠괄호 포함 |
| `i"` | 큰따옴표 안 |
| `a"` | 큰따옴표 포함 |
| `i'` | 작은따옴표 안 |
| `a'` | 작은따옴표 포함 |
| `` i` `` | 백틱 안 |
| `` a` `` | 백틱 포함 |

### 태그 객체

| 명령 | 범위 |
|------|------|
| `it` | inner tag (태그 안 내용) |
| `at` | a tag (태그 포함) |

```html
<div>Hello</div>
cit → <div>|</div>   (Hello 삭제, Insert 모드)
dat → |              (전체 삭제)
```

## 3. 실전 예제

### 코드 편집

```python
# 함수 인자 변경
def greet(name, age):
    pass

# name 위치에서
ciw         # name만 변경
f(ci(       # 괄호 안 전체 변경

# 문자열 변경
message = "Hello, World!"
ci"         # 문자열 내용만 변경
ca"         # 따옴표 포함 변경
```

### 중첩 구조

```javascript
function example() {
    if (condition) {
        console.log("test");
    }
}

// "test" 위치에서
ci"         # 문자열 내용 변경
ca"         # 따옴표 포함 변경
ci(         # console.log 괄호 안 변경
ciB         # if 블록 안 변경
```

## 4. 점프 (Jumps)

### 마크 (Marks)

| 명령 | 동작 |
|------|------|
| `m{a-z}` | 현재 위치에 마크 설정 (파일 내) |
| `m{A-Z}` | 전역 마크 설정 (파일 간) |
| `` `{mark} `` | 마크 위치로 점프 (정확한 위치) |
| `'{mark}` | 마크 줄로 점프 (줄 처음) |
| ``` `` ``` | 이전 점프 위치로 |
| `''` | 이전 점프 줄로 |

### 특수 마크

| 마크 | 의미 |
|------|------|
| `` `. `` | 마지막 변경 위치 |
| `` `" `` | 마지막으로 파일을 떠난 위치 |
| `` `^ `` | 마지막 Insert 모드 위치 |
| `` `[ `` | 마지막 변경/yank 시작 |
| `` `] `` | 마지막 변경/yank 끝 |

### 점프 리스트

| 명령 | 동작 |
|------|------|
| `Ctrl-o` | 이전 점프 위치로 |
| `Ctrl-i` | 다음 점프 위치로 |
| `:jumps` | 점프 리스트 보기 |

### 변경 리스트

| 명령 | 동작 |
|------|------|
| `g;` | 이전 변경 위치로 |
| `g,` | 다음 변경 위치로 |
| `:changes` | 변경 리스트 보기 |

## 5. 매칭 (Matching)

| 명령 | 동작 |
|------|------|
| `%` | 매칭되는 괄호로 이동 |
| `[{` | 이전 매칭 안 된 { |
| `]}` | 다음 매칭 안 된 } |
| `[(` | 이전 매칭 안 된 ( |
| `])` | 다음 매칭 안 된 ) |

## 6. 그 외 유용한 모션

| 명령 | 동작 |
|------|------|
| `*` | 커서 아래 단어 앞으로 검색 |
| `#` | 커서 아래 단어 뒤로 검색 |
| `gd` | 지역 선언으로 이동 |
| `gD` | 전역 선언으로 이동 |
| `]]` | 다음 섹션/함수로 |
| `[[` | 이전 섹션/함수로 |

## 실습

1. 다양한 코드 파일에서 `ci(`, `ci"`, `ci{` 연습
2. `f`, `t`와 `;`, `,`로 빠른 줄 내 이동 연습
3. 마크 설정 후 `` ` ``로 점프 연습
4. `Ctrl-o`, `Ctrl-i`로 점프 리스트 탐색

## 요약

- `f`/`t`/`F`/`T` + `;`/`,` = 줄 내 빠른 이동
- `i{object}` = 내부만, `a{object}` = 주변 포함
- 텍스트 객체: `w`, `s`, `p`, `(`, `{`, `"`, `t` 등
- 마크: `m{a-z}` 설정, `` `{mark} `` 이동
- `Ctrl-o`/`Ctrl-i` = 점프 히스토리 탐색
