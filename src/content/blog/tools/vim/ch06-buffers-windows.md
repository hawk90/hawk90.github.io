> **Vim 마스터하기** Chapter 6: 버퍼, 윈도우, 탭

## 1. 개념 정리

```
+----------------------------------------------+
|  Tab Page 1                    Tab Page 2    |
+----------------------------------------------+
|  +------------------+------------------+     |
|  |     Window 1     |     Window 2     |     |
|  |                  |                  |     |
|  |  [Buffer A]      |  [Buffer B]      |     |
|  |                  |                  |     |
|  +------------------+------------------+     |
|  |          Window 3                   |     |
|  |          [Buffer C]                 |     |
|  +-------------------------------------+     |
+----------------------------------------------+
```

- **Buffer**: 파일 내용을 메모리에 로드한 것
- **Window**: 버퍼를 표시하는 뷰포트
- **Tab Page**: 윈도우 레이아웃의 집합

> 하나의 버퍼가 여러 윈도우에 표시될 수 있다.

## 2. 버퍼 (Buffers)

### 버퍼 관리 명령

| 명령 | 동작 |
|------|------|
| `:e {file}` | 파일 열기 (새 버퍼) |
| `:e!` | 현재 파일 다시 로드 |
| `:ls` 또는 `:buffers` | 버퍼 목록 |
| `:b {n}` | n번 버퍼로 전환 |
| `:b {name}` | 이름으로 버퍼 전환 |
| `:bn` | 다음 버퍼 |
| `:bp` | 이전 버퍼 |
| `:bf` | 첫 번째 버퍼 |
| `:bl` | 마지막 버퍼 |
| `:bd` | 버퍼 삭제 (닫기) |
| `:bd!` | 강제 삭제 |
| `Ctrl-^` | 이전 버퍼로 전환 |

### 버퍼 상태

`:ls` 출력 예시:
```
  1 %a   "file1.txt"      line 10
  2  h   "file2.txt"      line 1
  3 #    "file3.txt"      line 5
```

| 플래그 | 의미 |
|--------|------|
| `%` | 현재 윈도우의 버퍼 |
| `#` | 대체 버퍼 (Ctrl-^로 전환) |
| `a` | 활성 (로드되고 표시 중) |
| `h` | 숨김 (로드되었지만 표시 안 됨) |
| `+` | 수정됨 |
| `-` | 수정 불가 |
| `=` | 읽기 전용 |

### 숨김 버퍼 (Hidden Buffers)

```vim
" 저장 없이 다른 버퍼로 전환 허용
:set hidden

" 이 설정 없이 수정된 버퍼에서 이동하려면
" E37: No write since last change 에러 발생
```

### 여러 파일 열기

```bash
vim file1.txt file2.txt file3.txt
vim *.js
```

```vim
" 인자 목록 관리
:args           " 현재 인자 목록
:args *.js      " 인자 목록 설정
:argadd file    " 추가
:argdelete file " 제거
:next           " 다음 인자
:prev           " 이전 인자
:first          " 첫 번째
:last           " 마지막
```

## 3. 윈도우 (Windows)

### 윈도우 분할

| 명령 | 동작 |
|------|------|
| `:sp` 또는 `Ctrl-w s` | 수평 분할 |
| `:vs` 또는 `Ctrl-w v` | 수직 분할 |
| `:sp {file}` | 파일과 함께 수평 분할 |
| `:vs {file}` | 파일과 함께 수직 분할 |
| `:new` | 새 파일로 수평 분할 |
| `:vnew` | 새 파일로 수직 분할 |

### 윈도우 이동

| 명령 | 동작 |
|------|------|
| `Ctrl-w h` | 왼쪽 윈도우 |
| `Ctrl-w j` | 아래 윈도우 |
| `Ctrl-w k` | 위 윈도우 |
| `Ctrl-w l` | 오른쪽 윈도우 |
| `Ctrl-w w` | 다음 윈도우 (순환) |
| `Ctrl-w W` | 이전 윈도우 |
| `Ctrl-w p` | 이전 (최근) 윈도우 |

### 윈도우 배치

| 명령 | 동작 |
|------|------|
| `Ctrl-w H` | 윈도우를 왼쪽으로 이동 |
| `Ctrl-w J` | 윈도우를 아래로 이동 |
| `Ctrl-w K` | 윈도우를 위로 이동 |
| `Ctrl-w L` | 윈도우를 오른쪽으로 이동 |
| `Ctrl-w r` | 윈도우 회전 (아래/오른쪽) |
| `Ctrl-w R` | 윈도우 회전 (위/왼쪽) |
| `Ctrl-w x` | 현재와 다음 윈도우 교환 |

### 윈도우 크기 조절

| 명령 | 동작 |
|------|------|
| `Ctrl-w =` | 모든 윈도우 균등 분할 |
| `Ctrl-w _` | 현재 윈도우 높이 최대화 |
| `Ctrl-w |` | 현재 윈도우 너비 최대화 |
| `Ctrl-w +` | 높이 증가 |
| `Ctrl-w -` | 높이 감소 |
| `Ctrl-w >` | 너비 증가 |
| `Ctrl-w <` | 너비 감소 |
| `:resize {n}` | 높이를 n으로 |
| `:vertical resize {n}` | 너비를 n으로 |

### 윈도우 닫기

| 명령 | 동작 |
|------|------|
| `:q` 또는 `Ctrl-w q` | 현재 윈도우 닫기 |
| `:close` 또는 `Ctrl-w c` | 현재 윈도우 닫기 |
| `:only` 또는 `Ctrl-w o` | 다른 모든 윈도우 닫기 |
| `:qa` | 모든 윈도우 닫기 |

## 4. 탭 페이지 (Tab Pages)

### 탭 관리

| 명령 | 동작 |
|------|------|
| `:tabnew` | 새 탭 |
| `:tabnew {file}` | 파일과 함께 새 탭 |
| `:tabe {file}` | tabnew와 동일 |
| `gt` | 다음 탭 |
| `gT` | 이전 탭 |
| `{n}gt` | n번째 탭 |
| `:tabclose` | 현재 탭 닫기 |
| `:tabonly` | 다른 탭 모두 닫기 |
| `:tabs` | 탭 목록 |
| `:tabmove {n}` | 탭을 n 위치로 이동 |

### 탭 사용 팁

```vim
" 현재 버퍼를 새 탭에서 열기
:tab split

" 모든 버퍼를 각각 탭으로
:tab ball

" 인자 목록을 각각 탭으로
:tab all
```

## 5. 추천 워크플로우

### 버퍼 중심 워크플로우 (추천)

```vim
" 설정
set hidden

" 파일 열기
:e file1.js
:e file2.js
:e file3.js

" 버퍼 전환
:b file2        " 이름 일부로 전환
:b2             " 번호로 전환
Ctrl-^          " 이전 버퍼

" 목록 확인
:ls
```

### 윈도우 분할 워크플로우

```vim
" 참조할 파일과 수평 분할
:vs reference.js

" 윈도우 이동
Ctrl-w h/j/k/l

" 윈도우 닫기
Ctrl-w c
```

### 탭 사용 시나리오

- 프로젝트별로 탭 분리
- 작업 컨텍스트별 분리 (코드, 테스트, 설정)
- 비교 작업 시 탭으로 레이아웃 저장

## 6. 유용한 키맵

```vim
" 버퍼 탐색
nnoremap <leader>b :ls<CR>:b<space>

" 버퍼 전환
nnoremap <Tab> :bn<CR>
nnoremap <S-Tab> :bp<CR>

" 윈도우 이동 단순화
nnoremap <C-h> <C-w>h
nnoremap <C-j> <C-w>j
nnoremap <C-k> <C-w>k
nnoremap <C-l> <C-w>l

" 윈도우 분할
nnoremap <leader>v :vs<CR>
nnoremap <leader>s :sp<CR>
```

## 실습

1. 여러 파일 열고 `:ls`로 버퍼 확인
2. `Ctrl-w v`, `Ctrl-w s`로 분할
3. `Ctrl-w h/j/k/l`로 윈도우 이동
4. `gt`, `gT`로 탭 전환
5. `:set hidden` 후 수정된 버퍼에서 이동 테스트

## 요약

- **버퍼**: 메모리에 로드된 파일, `:ls`로 목록, `:b`로 전환
- **윈도우**: 버퍼를 보여주는 화면 영역, `Ctrl-w`로 조작
- **탭**: 윈도우 레이아웃 집합, `gt`/`gT`로 전환
- `set hidden`으로 저장 없이 버퍼 전환 허용
- `Ctrl-^`로 이전 버퍼 빠르게 전환
