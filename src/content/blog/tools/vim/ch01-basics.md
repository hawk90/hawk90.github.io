> **Vim 마스터하기** Chapter 1: 기초

## 1. Vim의 모드

Vim은 **모달 에디터(modal editor)**이다. 같은 키가 모드에 따라 다른 동작을 한다.

### 주요 모드

| 모드 | 진입 | 용도 |
|------|------|------|
| **Normal** | `Esc` | 이동, 명령 실행 (기본 모드) |
| **Insert** | `i`, `a`, `o` 등 | 텍스트 입력 |
| **Visual** | `v`, `V`, `Ctrl-v` | 텍스트 선택 |
| **Command-line** | `:`, `/`, `?` | Ex 명령, 검색 |

### 모드 전환

```
Normal → Insert:  i, a, o, I, A, O, s, S, c, C
Insert → Normal:  Esc, Ctrl-[, Ctrl-c
Normal → Visual:  v (문자), V (줄), Ctrl-v (블록)
Visual → Normal:  Esc
Normal → Command: :, /, ?
```

## 2. Vim 시작과 종료

### 시작

```bash
vim              # 빈 파일로 시작
vim file.txt     # 파일 열기
vim +10 file.txt # 10번 줄에서 시작
vim +/pattern file.txt  # 패턴 위치에서 시작
vim -O file1 file2      # 수직 분할로 열기
vim -o file1 file2      # 수평 분할로 열기
```

### 종료

| 명령 | 동작 |
|------|------|
| `:q` | 종료 (변경사항 없을 때) |
| `:q!` | 강제 종료 (변경사항 버림) |
| `:w` | 저장 |
| `:wq` 또는 `:x` | 저장 후 종료 |
| `ZZ` | `:wq`와 동일 (Normal 모드) |
| `ZQ` | `:q!`와 동일 (Normal 모드) |

## 3. 기본 이동

### 문자 단위

```
h  ←  왼쪽
j  ↓  아래
k  ↑  위
l  →  오른쪽
```

> **팁**: hjkl 대신 화살표 키도 사용 가능하지만, hjkl을 익히면 손이 홈 포지션을 벗어나지 않아 더 빠르다.

### 단어 단위

| 명령 | 동작 |
|------|------|
| `w` | 다음 단어 시작 |
| `W` | 다음 WORD 시작 (공백 기준) |
| `b` | 이전 단어 시작 |
| `B` | 이전 WORD 시작 |
| `e` | 현재/다음 단어 끝 |
| `E` | 현재/다음 WORD 끝 |
| `ge` | 이전 단어 끝 |

### 줄 단위

| 명령 | 동작 |
|------|------|
| `0` | 줄 맨 앞 |
| `^` | 줄 첫 번째 문자 (공백 제외) |
| `$` | 줄 맨 끝 |
| `g_` | 줄 마지막 문자 (공백 제외) |

### 화면 단위

| 명령 | 동작 |
|------|------|
| `gg` | 파일 맨 처음 |
| `G` | 파일 맨 끝 |
| `{n}G` 또는 `:{n}` | n번 줄로 이동 |
| `Ctrl-f` | 한 화면 아래 |
| `Ctrl-b` | 한 화면 위 |
| `Ctrl-d` | 반 화면 아래 |
| `Ctrl-u` | 반 화면 위 |
| `H` | 화면 맨 위 |
| `M` | 화면 중간 |
| `L` | 화면 맨 아래 |

### 스크롤 (커서 고정)

| 명령 | 동작 |
|------|------|
| `zz` | 현재 줄을 화면 중앙으로 |
| `zt` | 현재 줄을 화면 상단으로 |
| `zb` | 현재 줄을 화면 하단으로 |

## 4. 숫자 접두사

대부분의 명령 앞에 숫자를 붙여 반복할 수 있다.

```
5j      # 5줄 아래
3w      # 3단어 앞으로
10G     # 10번 줄로
2dd     # 2줄 삭제
```

## 5. 실습

1. `vimtutor` 실행하여 기본 튜토리얼 완료
2. 파일을 열고 hjkl만으로 이동 연습
3. 화살표 키 사용 금지 도전
4. `:set relativenumber`로 상대 줄 번호 표시 후 `{n}j`, `{n}k` 연습

## 요약

- Vim은 모달 에디터: Normal, Insert, Visual, Command-line 모드
- `Esc`로 항상 Normal 모드로 복귀
- hjkl로 이동, 숫자 접두사로 반복
- `:wq`로 저장 후 종료, `:q!`로 강제 종료
