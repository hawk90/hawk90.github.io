> **Vim 마스터하기** 시리즈 개요

## Vim이란?

Vim(Vi IMproved)은 1991년 Bram Moolenaar가 만든 텍스트 에디터로, Unix의 vi 에디터를 개선한 버전이다. 터미널 기반으로 동작하며, 모달 편집(modal editing) 방식을 사용한다.

## 왜 Vim인가?

1. **어디서나 사용 가능**: 대부분의 Unix/Linux 시스템에 기본 설치
2. **빠른 편집 속도**: 키보드만으로 모든 작업 가능
3. **가벼움**: 리소스를 거의 사용하지 않음
4. **확장성**: 플러그인으로 무한 확장 가능
5. **지속성**: 30년 이상 사용되어 온 검증된 도구
