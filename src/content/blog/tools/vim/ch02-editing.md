> **Vim 마스터하기** Chapter 2: 편집 명령어

## 1. Insert 모드 진입

| 명령 | 동작 |
|------|------|
| `i` | 커서 앞에 삽입 |
| `I` | 줄 맨 앞에 삽입 |
| `a` | 커서 뒤에 삽입 |
| `A` | 줄 맨 끝에 삽입 |
| `o` | 아래에 새 줄 생성 후 삽입 |
| `O` | 위에 새 줄 생성 후 삽입 |
| `s` | 문자 삭제 후 삽입 |
| `S` | 줄 전체 삭제 후 삽입 |

## 2. 삭제 (Delete)

Vim에서 삭제는 **잘라내기(cut)**와 동일하다. 삭제된 내용은 레지스터에 저장된다.

### 기본 삭제

| 명령 | 동작 |
|------|------|
| `x` | 커서 위치 문자 삭제 |
| `X` | 커서 앞 문자 삭제 |
| `dd` | 현재 줄 삭제 |
| `D` | 커서부터 줄 끝까지 삭제 |

### 동작(motion)과 결합

`d{motion}` 형식으로 삭제 범위를 지정한다.

```
dw      # 단어 끝까지 삭제
daw     # 단어 전체 삭제 (a word)
diw     # 단어 내용만 삭제 (inner word)
d$      # 줄 끝까지 삭제
d0      # 줄 처음까지 삭제
dG      # 파일 끝까지 삭제
dgg     # 파일 처음까지 삭제
d}      # 문단 끝까지 삭제
dt{c}   # 문자 c 앞까지 삭제
df{c}   # 문자 c까지 삭제 (c 포함)
```

## 3. 변경 (Change)

`c`는 삭제 후 Insert 모드로 전환한다. `d`와 같은 방식으로 사용한다.

| 명령 | 동작 |
|------|------|
| `cw` | 단어 끝까지 변경 |
| `ciw` | 단어 전체 변경 |
| `cc` 또는 `S` | 줄 전체 변경 |
| `C` | 커서부터 줄 끝까지 변경 |
| `ci"` | 따옴표 안 내용 변경 |
| `ci(` | 괄호 안 내용 변경 |
| `ct{c}` | 문자 c 앞까지 변경 |

## 4. 복사 (Yank)

Vim에서 복사는 **yank**라고 한다.

| 명령 | 동작 |
|------|------|
| `yy` 또는 `Y` | 현재 줄 복사 |
| `yw` | 단어 끝까지 복사 |
| `yiw` | 단어 전체 복사 |
| `y$` | 줄 끝까지 복사 |
| `yi"` | 따옴표 안 내용 복사 |
| `yi(` | 괄호 안 내용 복사 |

## 5. 붙여넣기 (Put)

| 명령 | 동작 |
|------|------|
| `p` | 커서 뒤/아래에 붙여넣기 |
| `P` | 커서 앞/위에 붙여넣기 |
| `gp` | 붙여넣기 후 커서를 끝으로 |
| `gP` | 앞에 붙여넣기 후 커서를 끝으로 |

> **팁**: 줄 단위로 복사/삭제한 것은 줄 위/아래에, 문자 단위는 커서 앞/뒤에 붙여넣기 된다.

## 6. 실행 취소 / 다시 실행

| 명령 | 동작 |
|------|------|
| `u` | 실행 취소 (undo) |
| `U` | 줄 전체 복원 |
| `Ctrl-r` | 다시 실행 (redo) |

## 7. 반복

| 명령 | 동작 |
|------|------|
| `.` | 마지막 변경 반복 |
| `@:` | 마지막 Ex 명령 반복 |

> **팁**: `.`(dot) 명령은 Vim의 가장 강력한 기능 중 하나이다. 반복 가능한 작업을 설계하여 `.`으로 효율적으로 반복하라.

## 8. 기타 편집 명령

### 대소문자 변환

| 명령 | 동작 |
|------|------|
| `~` | 대소문자 토글 |
| `gU{motion}` | 대문자로 변환 |
| `gu{motion}` | 소문자로 변환 |
| `gUU` | 줄 전체 대문자 |
| `guu` | 줄 전체 소문자 |

### 들여쓰기

| 명령 | 동작 |
|------|------|
| `>>` | 현재 줄 들여쓰기 |
| `<<` | 현재 줄 내어쓰기 |
| `={motion}` | 자동 들여쓰기 |
| `==` | 현재 줄 자동 들여쓰기 |

### 줄 합치기

| 명령 | 동작 |
|------|------|
| `J` | 현재 줄과 다음 줄 합치기 (공백 추가) |
| `gJ` | 줄 합치기 (공백 없이) |

### 숫자 증감

| 명령 | 동작 |
|------|------|
| `Ctrl-a` | 커서 위치/이후 숫자 증가 |
| `Ctrl-x` | 숫자 감소 |

## 9. 편집 문법 요약

```
[count] operator [count] motion
```

**Operator (연산자)**:
- `d` - 삭제
- `c` - 변경
- `y` - 복사
- `>` - 들여쓰기
- `<` - 내어쓰기
- `=` - 자동 들여쓰기
- `gU` - 대문자화
- `gu` - 소문자화

**Motion (동작)**:
- `w`, `W`, `b`, `B`, `e`, `E` - 단어 단위
- `0`, `^`, `$` - 줄 단위
- `gg`, `G` - 파일 단위
- `{`, `}` - 문단 단위
- `t{c}`, `f{c}` - 문자 검색

## 실습

1. 텍스트 파일을 열고 `daw`, `ciw` 연습
2. `yy` + `p`로 줄 복사 연습
3. 변경 후 `.`으로 반복 연습
4. `u`와 `Ctrl-r`로 undo/redo 연습

## 요약

- `d` = 삭제(잘라내기), `c` = 변경, `y` = 복사
- `p` = 붙여넣기, `u` = 실행 취소
- `.` = 마지막 변경 반복
- `operator + motion` = Vim 편집의 기본 문법
