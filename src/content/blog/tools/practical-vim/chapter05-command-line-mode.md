## 작성 중

### 예정 내용 (Tips 30-39)
- : 진입 / 히스토리 (q:)
- 범위 — 라인 번호 / 마크 / 패턴 / %
- :delete / :yank / :put / :copy / :move
- :normal — 노멀 명령 일괄 실행
- :@: — 마지막 ex 반복
- :! — 셸 실행
